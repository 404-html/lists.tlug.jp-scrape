You may also need to tell the parent app to pass that combination
through.

Eg, from KTerm:

! kterm use XMODIFIERS environment for XIM
!*VT100*translations: #override \
!  Shift<Key>space: open-im(XMODIFIERS) \n\

Note that if you start kterm in -xim mode, this translation isn't
needed.
