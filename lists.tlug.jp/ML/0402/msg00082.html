There are several (too many indeed) options.  One of the major options  
is to use EasyPackage, which is a package management system like Fink  
or OpenDarwin, but its packages support Japanese because their  
developers are mostly Japanese.  Canna should start when you turns on  
your iBook.
http://www.ie.u-ryukyu.ac.jp/darwin2/index.php

Apple's X11 differs from most others as it doesn't reflect the  
environmental variables you set in .xinitrc.  You need to modify (or  
create0 the file:
  ~/.MacOSX/environment.plist
and add the variables such as

LANG = ja_JP.EUC
LC_COLLATE = C
LC_TIME = C
XMODIFIERS = '@example.com=kinput2'
