I'd recommend Postfix with Amavisd-new or clamav. I've found the
following HOWTO to be very good:


"Fairly-Secure Anti-SPAM Gateway Using OpenBSD, Postfix, Amavisd-new,
SpamAssassin, Razor and DCC"

http://www.flakshack.com/anti-spam/


Though the above HOWTO discusses OpenBSD, It has some Linux related notes
at the end including clamav. Clamav works very well.

In addition you can use postfix's body_checks and header_checks to filter
most viruses.  I use the following...

http://jimsun.linxnet.com/misc/header_checks.txt
http://jimsun.linxnet.com/misc/body_checks.txt
