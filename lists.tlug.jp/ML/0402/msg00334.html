I'm not sure why your having trouble with blackdown-jdk or perl. A stage3
is a fully functional system. Try following the install instructions
without upgrading. I mean install grub/lilo, build your kernel, edit files
in /etc, install cron, logger, etc Once you've rebooted and are able to
log into a working Gentoo system, then upgrade. 

You have many potential ways to fix your broken install but without more
detail we can't do much. For example, what errors are you getting? Which
stage3 did you use? Did you modify /etc/make.conf? Which profile are you
using? 

One last suggestion is to check the Gentoo forums...

http://forums.gentoo.org/
