Try to set the following parameter and check. Becoz it may be that japanese 
man pages are not installed.

LC_CTYPE=ja_JP (set system locale to japanese)
LANG=en_US  (but this sets the intaractive messages like error, man pages to 
display in english)

If nothing works, set LC_ALL=C and check.

-Thomas
