GNU.  The "Free" Documentation License is incompatible with the GPL.

Strictly speaking, you are prohibited from copying example code in
Emacs docs into your (redistributed) Emacs extension unless you get
explicit permission from the FSF.  And you are prohibited from copying
comments and docstrings from Emacs code into your extension's
documentation if that is FDL.

Not to mention the (plausible but not court-tested) practical issues
that you're not allowed to transmit FDL material over an encrypted
channel or store it on an encrypted filesystem if anyone has access to
the system but lacks the password.
