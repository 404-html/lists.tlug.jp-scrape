Ahaa :-) Been there, done that and got the t-shirt!

You obviously installed the IME and Japanese locale files
for KDE? Are you using an English keyboard? If yes to
both questions open up an app and press the SHIFT
key and SPACE key then start typing.

If you haven't installed the relevant files you'll have to
fire up Mandrake's install manager and do so. You can
also do it at a command line with 'urpmi' 

I'm also running Mandrake 10 and if you need help 
with updates and installing packages the Mandrake
news groups are really helpful.
