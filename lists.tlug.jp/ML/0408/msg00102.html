of course not.
it is logical too. the same happens if you remove a file. any process
that has the file open, will not notice that it is gone.

only the entries in the directory are gone because they are removed
before the directory is removed itself.

try recreating the directory after you removed it.
the process won't notice that the directory is back.
it can't because the new directory is a different one, 
it just happens to have the same name.
