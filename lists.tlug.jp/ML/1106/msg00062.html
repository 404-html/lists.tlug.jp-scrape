One bug that RH based systems had (reported and if not fixed, will be
fixed) is that it needs dbus-x11 to be installed.  (Hopefully, the RH
based and Debian based names are the same.) 

Sometimes, if one didn't install Gnome, dbus-x11 wouldn't get installed.
Probably not the case with a full KDE desktop environment install, but
possibly worth checking.
