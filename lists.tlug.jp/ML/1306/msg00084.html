Darren,

I've worked converting those ordinary USB (printers) to network attached
devices using so called print servers and the like for so many years.

You bet it works if your "print server" has features good enough (don't
go too much for the cheapest one). You route print jobs through the
server via "queues" that are assigned to the USB ports where the printer
is connected. This device must understand the IPP and the LPD protocols.

I'd be glad to help you out.
