One thing you can try before giving up is to look for the PPD file for your Canon printer on the Canon web site. If there is one, then installing that might work. Look for Mac drivers or even the Windows driver (expanding the file using cabextract).
I don't know if it's Canon's fault, but I've had to dig around for drivers like this many times. Rarely has things just worked out.
And if you haven't already, you should install CUPS to make things easier on yourself.
