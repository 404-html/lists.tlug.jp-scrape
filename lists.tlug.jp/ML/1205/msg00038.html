In America, we have a machine-based solution to this problem called the Marine Corps Torque Wrench Method. We apply it to problems such as this. The rule is to tighten the bolt until it breaks then back off half a turn. I use it frequently when trying to upgrade an old machine for friends and I can't find any specs for the box they've handed me.
What you do is download and burn several key versions of the OS -- from the version you know works up through the latest -- on to CDs and then keep installing newer versions from CD until the machine doesn't run any more. Then, you go back to the newest version that worked. If you're feeling especially masochistic, you can then try to upgrade individual packages until it doesn't work again and then remove the last thing you installed.
It is time consuming, illogical, unscientific and, I am sure it will upset some of the OS Gods on the list. But, it works, for basic values of "works."
