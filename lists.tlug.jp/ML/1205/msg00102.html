Here is a free/good programming guide for Linux system. Although it is
a little out-dated, it is still a good elementary reference if you
want to get into Linux system. You can get the concept of every
aspects of Linux.

http://www.advancedlinuxprogramming.com/

For general command reference, you can get manpages.

http://manpages.ubuntu.com/

Also, don't forget these two not-too-short guide for bash programming

http://tldp.org/HOWTO/Bash-Prog-Intro-HOWTO.html
http://tldp.org/LDP/abs/html/

>
> "The UNIX Programming Environment" by Kernighan and Pike

This is a good book too. However, I think it is for experienced
programmers on the topic of general ideas of programming for Unix-like
system. It is might not be a good reference for beginners.
