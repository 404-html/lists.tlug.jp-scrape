A DVD player that has been set to be Region 0 will play DVD's from any region with very few a exceptions. There are many web sites that explain how to set a DVD player to Region 0 and many popular DVD players can be set for this region. Google for "region 0 dvd player" brings 13,800 hits, and this one was in the advertising section of the google search:
Region 0 Dvd Player $79

Region 0 may not be official, but it's certainly popular.
