On 09/08/2007, Stephen J. Turnbull <stephen@example.com> wrote:

> For the interested, RFCs can be accessed by number via the template
> URL ftp://ftp.rfc-editor.org/in-notes/rfc${RFC#}.txt.

Thus making it ideal for a Firefox Quick Search bookmark:

<DT>
  <A HREF="ftp://ftp.rfc-editor.org/in-notes/rfc%s.txt";
       ADD_DATE="1186616474" LAST_MODIFIED="1186616494"
       SHORTCUTURL="rfc"
       LAST_CHARSET="UTF-8"
       ID="rdf:#$8O67e3">rfc-editor.org - RFC Lookup</A>
<DD>rfc

Now type "rfc 822" in the location bar and hit Enter:

ftp://ftp.rfc-editor.org/in-notes/rfc822.txt
