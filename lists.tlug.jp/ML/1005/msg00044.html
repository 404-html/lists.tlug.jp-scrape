Many current MUA's (mail user agents) don't thread properly, just going
by subject line. Mutt and some others, however, do thread properly,
using header information.  Even some Windows clients will also do it
properly, I believe.  THe first time I did it, not knowing better, I was
told about it by a Windows user, so....)

For an illustration

http://home.roadrunner.com/~computertaijutsu/threadhijack.jpg

Note that the linguistics post is showing up as part of the thread Hello
to everybody.

It's also bad for the person doing it--if someone has decided to ignore
a thread and has a proper MUA, the hijacked post may be missed.
