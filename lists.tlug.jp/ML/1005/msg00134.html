It's true now we have X3D to adapt to XML type.
However, for historical reason some implementors stick to
traditional VRML formats and indeed it's simple for beginners
to write in VRML style.
Wikipedia in Japan says that it will need some more time
for X3D to flourish in the world.  As of now only a few editors
have implemented this format including Blender.
