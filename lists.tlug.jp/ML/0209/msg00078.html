On Wed, Sep 04, 2002 at 12:43:52AM +0900, Batara Kesuma wrote:

>My friend sent me some files with Japanese file names from Windows

Some friend :-p

Seriously, though, the files are probably encoded in SJIS, or
maybe Unicode (try SJIS first).

I'm not sure how to go about converting the names of files to
a different encoding (must be a way), but one thing you
might try is changing LANG to ja_JP.SJIS (usually this will
be ja_JP.eucJP if you're in a Japanese locale) and see if
that helps.

If it doesn't help, you could then try ja_JP.utf8.

If that doesn't help either, ask somebody else and 
ignore me :-)
