Fixed the input box problem. After I emerged openoffice-bin, the UI font
was all strange. It appears that ooffice is looking for a font called
Andale Sans UI that I don't seem to have on my system. I checked
www.openoffice.org and there was an entry in there FAQ about this. Using
the Font replacement screen in Options, I replaced Andale Sans UI with
Kochi Gothic (from the kochi-fonts ebuild) and now I get あ in the input
box.

Hope this helps
