Tony Laszlo wrote:

>(sending utf-8)
>
>Any favorite tools that will convert 
>html entities (special characters) to 
>real characters? 
>I need a toold that will take something like this: 
>
>
>&#273;&#432;&#7907;c
>
>and convert it (convert, not display) to this: 
>
>được
>
You could use tools like Mozilla as follows:
Open your source text in the Mozilla browser.
Grab the text you want to convert with the mouse
Open Mozilla Mail tool
Start composing a new message
Paste the text into the mail composer window.
In the mail composer window: select document encoding UTF-8.
In the mail composer window: select "Save file as..." and the text will 
be saved in a file with UTF-8 format.

Since Mozilla is not primarly a encoding conversion tool, it is not 
optimized for this usage, but it works and could be alternative.
