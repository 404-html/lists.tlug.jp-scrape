Quoth Mike Gauthier (Sun 2002-09-15 03:28:49PM +0900):
> 
> Be sure to set your XMODIFIERS variable before you run kinput2 (example
> XMODIFIERS="@example.com=kinput2"; export XMODIFIERS).

It does not matter when you set $XMODIFIERS, as it is during the execution of
the application that $XMODIFIERS is queried, causing kinput2 to be used for
input. kinput2 itself cares not.

In fact, I usually start apps for Japanese input like:

LC_CTYPE=ja_JP XMODIFIERS=@example.com=kinput2 foo &
