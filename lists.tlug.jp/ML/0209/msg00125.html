I forgot to mention the easy way, which is invoking X like this:

startx -nolisten tcp

Since you are running Mandrake, I assume that you use gdm (graphical
login, right?). In this case, you need to edit your
/etc/X11/gdm/gdm.conf file and change the 'command=/usr/bin/X11/X'
line in the [server-Standard] section to:

command=/usr/bin/X11/X -nolisten tcp
