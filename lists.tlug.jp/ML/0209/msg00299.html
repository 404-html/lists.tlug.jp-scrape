On Mon, Sep 16, 2002 at 10:23:53AM +0900, Tony Laszlo wrote:
> 
> will "chsh -s bash"  do it? 

It should be 'chsh -s /bin/bash'. One more way to do this is to run bash
when you login. Add this line to your '.profile':

. /bin/bash

This is useful if you dont have permission to change your default shell.

In both cases check that bash really is in /bin.
