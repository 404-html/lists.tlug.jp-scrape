On Mon, 25 Aug 1997 matt@example.com wrote:

matt>I guess I have to ask for help with this one. What I want is pretty
matt>simple (ha-ha): just to print plain (Japanese) text on a (Japanese)
matt>postscript printer.

Matt,
are you sure you have PostScript fonts in your printer?  Is 
it one of those expensive Epson printers?  Most printers,
like the Hewlett Packard printers come with Japanese fonts,
but these are _not_ PostScript fonts.  Also, if you have MS-Windows,
it will even print to an English printer by sending the fonts
to the printer.  

Try this, print Japanese to a file with something like MS Word.
The file should be PostScript. 
Mount your Windows partition under Linux.  Open the file with
vi and make sure there are no funny headers.  As root,
do 
  # cat filename > /dev/lp1

assuming your printer is attached to the parallel port.   It
should be able to print the Japanese.

If your printer does not have the Japanese _PostScript_ fonts,
you will have to filter through GhostScript to send the font
data to the printer.

What kind of printer do you have?  Is it connected to the
parallel port or to ethernet?  If to ethernet, is a machine like
Windows NT used as the spool, or is it a UNIX machine?  Or, is
the printer acting as the spool?
