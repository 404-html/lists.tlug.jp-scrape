Hi all,

I mirror the InterNIC Scout Report on my homepage. An HTML
version is sent weekly, so I wrote a SED script which deletes
the mail header, substitutes the BODY tag to one with a BGCOLOR,
and appends a line with a link back to my homepage.

It works fine, but I was curious how Perl would do it so I tried
using the s2p utility. Look what happens:

jim:~/bin$ cat scout
#!/bin/sh
sed '1,/^$/d;s/<BODY>/<BODY BGCOLOR=white>/;
 /<H1 ALIGN=CENTER>The Scout Report<\/H1>/a\
 <CENTER><A HREF="index.html">via the Schweizer Home
Page</A></CENTER>' scout.html > scout.htm
rm scout.html

jim:~/bin$ s2p scout scout.pl
Malformed substitution at line 2
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Any ideas on what this means? It looks like a pretty
straight-forward substitution to me...

Thanks,

Jim S.
