Hi,

To obtain a functional Japanese environment with English solaris:

install Canna in /opt/Canna (Canna builds easier than Wnn)

install MIT X11R6 in /opt/X11R6, and disable the Xsun server binaries
	by renaming them since they do not support all framebuffers anymore

add latest kinput2, kterm to X11R6

install Mule in /opt/Mule

fix /etc/.login and/or /etc/profile so /opt/X11R6 is before /usr/openwin
	for PATH and MANPATH (LD_LIBRARY_PATH should never be set by
	default since binaries should be correctly built to look for
	libraries in the place where they end up installed)

If you additionally copy across /usr/lib/local/{ja,japanese} from a
Japanese solaris, you get full Japanese processing with Solaris vi
within a kterm.

Recommended: also install Tcsh in /opt/Tcsh.

I can supply solaris packages for Canna, X11R6, Mule and Tcsh if anyone
is interested.

	-Huw
