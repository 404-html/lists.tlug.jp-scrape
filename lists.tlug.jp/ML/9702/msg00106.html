Hi all,

I'm trying to learn a little about bash programming to make
cleaning up a lighter job but I've run into a problem.

Purpose: find files by date and rm, cat, cp or whatever.

This works fine:

ls -l | grep -i 'feb 27' | cut -c56- 

for finding todays files, but I want to write a shell script for
it like:

function lsdate {
  date=$1
  ls -l | grep -i "^.\43\}$date" | cut -c56-
}

then call the script like lsdate 'feb 27' but the script isn't
accepting the filename arguments.

I realize this is pretty basic, but what am I doing wrong?
