On 2008-07-31 07:26 +0900 (Thu), Stephen J. Turnbull wrote:

> In any case, that route would imply reconstructing monadic behavior
> in Java (or C++) which is surely not straightforward, since you think
> it's impossible.

It's easy to reconstruct monadic behaviour in either language. It's just
very difficult or impossible to type check it. Once again, *the* major
win of STM in Haskell is not that it does anything at all that's not
been implemented before; it's that Haskell's type checker can tell you
at compile time when you're making a mistake while trying to use it.
