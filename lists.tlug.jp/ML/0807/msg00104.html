CL wrote:
Okay, I now have an AMD X2-6400 box with 64-bit Kubuntu installed on the HDD. I would like to preserve as much of the setup I already did on my 32-bit version as possible.

Generally reinstalling and reconfiguring is going to be your easiest option. What setup did you do? Things like home directories should mostly just work if you install the same packages. You can obviously run a 32-bit version of Ubuntu on an amd64 system but you won't get the full advantage of running a 64-bit system. It is also possible to run a 64-bit kernel and a 32-bit userland but this might not be supported by Ubuntu.
