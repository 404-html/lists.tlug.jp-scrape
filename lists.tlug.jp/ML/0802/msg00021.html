It suggest I can manually repair the disk, as the message suggests?

Can I recover this installation? I went through a lot to get it set up just right... I really don't want to have to redo all of it.

In the maintenance shell, remount your root partition as read only if it isn't so already.

Then run fsck.ext3 on the root partition.

fsck.ext3 /dev/hda1

You will probably be prompted for various errors, have it repair it whenever you can.

Depending on how this goes you might be able to remount in write mode and continue your upgrade.
