Subject: OpenOffice Hanzi Lookup Toolbar

Louis Dominique-Dubeau has written the code for an OpenOffice add-on toolbar that allows direct lookup into various online Kanji/Hanzi resources, wherein you can select text on-screen and find it in those lexicons. He is still working through a few bugs on the Windows version, but it works fine in Linux.

Description and downloads can be found at:

http://lddubeau.com/avaktavyam/2008/02/07/oohanzi-02-released/

and it is hosted at Launchpad:

https://launchpad.net/oohanzi

At the moment he only has a few lexical resources included on the toolbar, but since the basic coding groundwork has been done, it would probably be relatively easy to add in more at this point. He would be delighted to hear from anyone with any input.
