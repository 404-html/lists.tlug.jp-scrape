If you need just forwarding, then setting up a postfix should be simple enough.

If you're looking for a commercial-grade hosted 'free' forwarding service, then there's Zoho Mail which we have been using for almost 5 years at bunsenlabs.org. We had zero (known) issues over the years.

You can create an organizational/master account for free and add a number of slave users. All users can have mailboxes hosted on Zoho but forwarding is a first-class feature that is easy to configure. Think G-Suite lite.
