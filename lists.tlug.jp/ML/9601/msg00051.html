Hi guys,

      With all this discussions about hardware upgrades and per-
formances, I think it's good timing to ask this question which 
has been on my mind since I've run across this topic weeks ago.
So here goes. 
 
      Any thoughts on Linux and the new multimedia processors -
Microunity's Media processor, Chromatic Research's Mpact processor,
and Philips' TriMedia processor? It seems that computer systems 
using this chip will be available by the middle of this year and 
Gateway 2000 and Toshiba are rumored to be implementing them. What
I want to know is if our favorite Linux will support them. It looks 
like Windows 95 has an edge over Linux since Mpact processor, which 
incorporates SEVEN multimedia functions - video, 2D graphics, 3D
graphics, audio, fax/modem, telephony, and videoconference, is 
supporting the Windows 95 API, whatever that means.

      For more info about this multimedia processors, check out

* "Super chips" cover article on the October 1995 issue of Byte
  ( or was it November? )
* MicroDesign's Web page on 
  http://pomo.nbn.com/microdesign/report/toc/toc.html
  for more on Media and Mpact processor
* Microunity's Web page on http://www.microunity.com/
* Chromatic Research's Web page on http://www.mpact.com/
* Philip's Web page on 
  http://conquest.oakridge.com/philips_semiconductors/multimed/
         backgrnd.html

      Thanks in advance.   
 
      By the way, I'd like to attend the meeting on the 28th.
Yoroshiku.

TFN, 
George
