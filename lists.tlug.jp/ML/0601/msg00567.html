Here's what is in bittorrent-gui:

[nick@example.com ~]$ rpm -qpl bittorrent-gui-4.2.2-2.fc4.noarch.rpm
/usr/bin/bittorrent        <----- that's the one you want
/usr/bin/maketorrent
/usr/share/applications/fedora-bittorrent.desktop
/usr/share/icons/hicolor/48x48/apps/bittorrent.png
/usr/share/pixmaps/bittorrent-4.2.2
/usr/share/pixmaps/bittorrent-4.2.2/bittorrent.ico
<lost of other images/icons>

You can get it from fedora-extras. (for FC4)

Or, just get get the source tarball from www.bittorrent.com, untar in your home directory and associate torrent files with ~/Bitorrent-4.2.X/bittorrent in Firefox.

Or,  start launchmany-curses in some directory and have Firefox download all torrent files there. Downloads will start automatically.
