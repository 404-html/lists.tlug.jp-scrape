I'd use something like planner-el coupled with remember-el, which
makes tasks and notes that are automatically hyperlinked to the file,
e-mail, webpage, chat, or anything else you write a function for. It's
easy to keep insanely detailed time records (timeclock.el
integration), which is pretty handy too.

If you think hierarchically, you might want to check out org.el. It
does hyperlinking as well, I think, but is slightly less featureful.

Both are good for managing non-source tasks as well.
