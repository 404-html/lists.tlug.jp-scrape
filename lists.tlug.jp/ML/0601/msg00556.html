The utilities installed with bittorrent start with
"btdownload..." name. Open a console, type
"btdownload" and hit <TAB>, you will see the different
applications you can use. I prefer to use the
ncurses-based application.

Or, if you want a nice GUI, you can install Azureus.
It requires Java though.
