Azureus is great, but the reason that it works the same in both is
that it is written in Java. Which means a *huge* footprint, in terms
of memory and CPU usage.

If you are cool with that, Azureus is a feature-rich and stable BT
client. But the Python stuff in the official BitTorrent package is far
nicer to your machine.
