The usual installations of sox don't have built-in mp3 support. You can see 
which file extensions are supported by:

sox -h

You need to install external libraries to support mp3 (reading from their 
mailing list in sourceforge, libmad and libmp3lame. I don't know if there are 
packages (for Suse) that includes sox and these libraries. *searches*
