> (Red faced) - Yes... I was unaware of that.  I just wrote in red ink in

>
> "~" = $Home

"~" == $HOME

would have been better to write. In a terminal, run the following
commands, and look at the output:

ls -d $HOME
ls -d ~
ls -d $Home

See how the last one doesn't work? $HOME is an environment
variable[1], and case matters for those.
