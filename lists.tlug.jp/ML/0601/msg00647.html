The change in mailing list software brings some features you may wish  
to take advantage of:

1.  If you like to read your TLUG email at one address, but post from  
several different addresses, you can now subscribe each of your  
posting addresses to the list and set the "Mail delivery" option to  
disabled for all but one of them.

2.  If you are a digest subscriber, you now have the option of  
choosing MIME formated digests.

You can visit your list options page by using the "Unsubscribe or  
edit options" button at the bottom of <http://www.tlug.jp/mailman/ 
listinfo/tlug>.

Also note that if you were subscribed to both the tlug and tlug- 
digest lists with the same email address, you are now only subscribed  
to the tlug list.  Please use a different email address if you wish  
to resubscribe to the list in digest form.

If you encounter problems with the mailing list software, please mail  
either me or listmaster@example.com
