Actually, that's not entirely true.

I'm using my own MUA (sylpheed) to send this message via gmail's SMTP
servers using SSL/TLS. I'm also Bcc'ing the message to another address of
mine so that I get a local copy of it. Meanwhile, gmail is also keeping a
record of this mail. Any replies sent to TLUG are stored by gmail *and*
forwarded to my other address.

So, you get to have correct threading of messages by using your own MUA,
*and* you can delete mails regularly because gmail is backing them up
anyway.
