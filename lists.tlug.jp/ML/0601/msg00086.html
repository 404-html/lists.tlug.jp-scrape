On Jan.12 16h56, bofh@example.com wrote :
> My wireless connection doesn't seem to start when I start up my Ubuntu 
> box.

You should try to modify manually the file /etc/network/interfaces

This file should have an entry for your wireless connexion (I would 
guess eth1)

In order to allow automatic configuration, wou can try to add the 
following line, as the 'man interfaces' suggests:

allow-hotplug eth1

If it still doesn't work you can try the following lines in addition (I 
saw that somewhere I don't remember exactly where...):

mapping hotplug
    script grep
    map eth1

Hope this helps :)
