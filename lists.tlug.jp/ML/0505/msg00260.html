It seems that with SuSE there are two ways to upgrade/fix a borked 
system: one is to use the Upgrade function in YAST while running 
the old system, and the other is to boot from the CD or DVD and 
pretend that you're doing a new install.  When it gets to "partition", 
it will detect the existing system and ask if you want to overwrite it 
or fix it.  Have you tried this "fix"?
