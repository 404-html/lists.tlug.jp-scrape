> Is there a quick and easy way to find out what machines SuSE runs well
on? 

The SUSE LINUX Professional 9.3 Live (evaluation) version is a
downloadable DVD ISO. 
No hard disk space is required, you run this live demo version off the
DVD. 
www.novell.com/products/linuxprofessional/downloads/suse_linux/ 
If the Live (evaluation) version won't run on the hardware then that
doesn't 
necessarily mean that the full version won't install -- but if it does
run, then 
the full version will install.
