            llsi.sh v0.2 Released Dec 12, 1999.

'Tis the season.  Not only are the holidays upon us but Linux related
businesses are going public.  This is extremely important as these
businesses now affect many areas of Linux.  In order to keep up to
date on these companies, I have written a small script which uses
Linux Weekly News' Linux Stock Index.  

llsi.sh is a shell script which when run as a cron job, produces an
html file that can be used as a Server Side Include (SSI) suitable for
inclusion on a web page.

Features include configurable colors and other HTML parameters,
coloring of index change: negative=red, no_change=black,
positive=green.  These too can be configured.

You can see it in action on the Mecworks page:

   http://www.mecworks.com/

Simple, easy to use, released under the GPL...you can find it at:

Get it here:
   http://www.mecworks.com/software/llsi.tgz   (always the current ver.)
 or
   http://www.mecworks.com/software/llsi-0.2.tgz
