... better than a dancing bear.  Actually works, you get Japanese on
buttons and in menus without pain, redisplay does leave popup
droppings but so far has always cleaned itself up (it's not so good
about extending backgrounds when you resize the window).  Javascript
worked one the mitsumorisho at www.gw2k.co.jp.

I have not tried Japanese input yet.

Only objectionable feature so far is the use of "poor man's bold" and
scaled bitmaps for kanji.  No way to turn it off, unfortunately.
You're probably OK if you have a Postscript or TrueType kanji font,
though.

Uptime 185 seconds and counting.  That's a record for Mozilla on my box!

I'm using the Debian package from yesterday....
