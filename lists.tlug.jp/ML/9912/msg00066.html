>Ulrike Schmidt (lists.tlug):
>>I am thinking of buying a CD burner and was wondering whether there are
any
>>limitations for use wiht Linux or any recommendations.
>
>If you get an IDE one, things are, well, suboptimal.
>You'd have to compile in support for IDE-SCSI emulation and then pretend
>the thing's actually a SCSI drive. It works, but it's not fun. If you've
>actually got a SCSI bus, things get easier, but of course, more
>expensive.
>
>I'm using a relatively bog-standard Mitsumi IDE 4x burner which was
>about 170 pounds (28,000 Yen) when I bought it last year. Took me a
>while to get it going in Linux the first time, but I set it up again the
>other day and didn't have many problems with it.
>


Bought a Panasonic (Mitsumi) SCSI CD-R writer 8x20x (no re-write) at T-Zone
for about 21,000 yen (tax included) last weekend...  Used xcdroast (with all
the dependancies satisfied) and burned ISO images at 8x...;-)  (You also
have to buy a SCSI controller though...;-( )
