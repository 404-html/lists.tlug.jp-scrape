> 1) what is the correct mime.type line for mpeg-4 in the server side?

In either srm.conf or .htaccess put these lines:
  
   (AddType video/x-ms-asf .asf)
   (AddType video/x-ms-asf .asx)

> 2) what plug-in do I need in a PC side to view the mpeg-4 files?

Using Windows, you need the latest version of the "Windows Media Player"
(available as an update for Win98 or with IE 5).

More info (in Japanese) here:
   http://inet-viewcam.sharp.co.jp/q_a/q_and_a.html
