YMMV, but it depends on the net card you're using -- 2.2.9 has a nasty
habit of locking up machines with certain ethernet cards in them (seen
it with 3c905, eepro100, 3c509, and a couple others).  SMC cards seem to
be ok... 

> No problems with 2.2.11 on any of my boxes, either.

on the linux-kernel digest, many people are complaining about FS
corruption with 2.2.11.  Also, there is a severe (and currently known)
memory leak in the base networking code for 2.2.11 which can bring some
machines to their knees very quickly.  Again, YMMV, but these are the
reports I've seen from the kernel list.
