On 05/06/2014 10:31 AM, Raymond Wan wrote:

I wonder if there is any Linux distribution that allows you to keep
your system at one version but upgrade software like your web browser
in an isolated way such that all of its dependencies are put somewhere
so as not to disturb the rest of the system?  Though this goes against
software development, it would make upgrading far more optional!  :-)

I think that's exactly what they do.
If there's an app you depend on and you want to upgrade something critical to it, shouldn't you be able to do something similar: ie, package the executable with the libraries it depends on (which could be many) in a directory and use an sh script to set LD_LIBRARY_PATH to access your local .so's?
