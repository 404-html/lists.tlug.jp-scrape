On 05/16/2014 11:38 PM, Nikolay Elenkov wrote:



If you get an error again, you can compare responses from
different servers with something like:

dig +dnssec +multi jprs.jp @8.8.8.8

or even

dig +dnssec +multi jp. @8.8.8.8



There are also these, which might be easier to follow:

http://dnssec-debugger.verisignlabs.com/
http://dnsviz.ne

Note that not many .jp domains are signed, so if you were
getting errors, there was probably a problem with validating
the .jp domain.
