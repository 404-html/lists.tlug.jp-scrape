>>>>> "VR" == Venkatesh RAGHAVAN <raghavan@example.com> writes:

    VR> hello, I use a SCSI 230MB MO DISKunder win'95. I wish to use
    VR> it under Linux for back-up purposes.  Can any one please give
    VR> me some tips about configuring the MO so that I can use it
    VR> under Linux

I don't think you should have any problem.  Just plug it in, paying
attention to the usual issues of SCSI ID and termination.  Your SCSI
driver should automatically recognize it as a hard drive with
removable media.

I don't know whether it's my SCSI host, the SCSI subsystem, or the
ext2 file system, but I have a devil of a time unmounting and swapping 
MO disks.  The directory cache seems not to get reset.  You might want 
to watch out for that one.
