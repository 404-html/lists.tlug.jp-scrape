On Tue, 15 Apr 1997, Jim Tittsler wrote:

> Can anyone recommend a "software problem tracking" solution for the
> Linux platform?  We are currently outgrowing the HTML/CGI-based
> system I put together for our LAN a couple of years ago.  It is
> time to either update it or replace it.

Only one I've seen lately was GNATS, the GNU project's bug tracking
system. Debian has a package set for it. Haven't looked at it closely,
think it was based on e-mailed problem reports. Seems to be a web
interface available for it. Check out the GNATS homepage at
   http://www.alumni.caltech.edu/~dank/gnats.html
