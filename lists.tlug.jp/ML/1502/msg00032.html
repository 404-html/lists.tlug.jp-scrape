I am working on a few Linux-related projects, one of which should be ready
by March 14th.  If there is interest, I would be happy to present the
following:

Title: Flexible Dependency Management Using Ansible

Description:

Travis shows how to implement parametric dependency sets using
[Ansible][ansible], a configuration management system.  His
implementation is sort of like having [Gentoo][gentoo]
[USE flags][useflags] in [Debian][debian].  With [Packer][packer]
and [Preseed][preseed], highly-customized systems can be
provisioned very quickly!

Travis
