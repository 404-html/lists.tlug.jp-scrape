For me, and judging from various posts on forums and mailing lists, ibus
has gotten more problematic over the years, and I've come to prefer
fcitx-anthy. 

I don't know how many use CentOS as a desktop, but fcitx-anthy has made it
into the EPEL-7 testing repos and works without problem for me.  

Anyone who wants to try it, as long as you have epel-release installed, can
run yum --enablerepo=epel-testing install fcitx-anthy.

Feature request is here. 

https://bugzilla.redhat.com/show_bug.cgi?id=666427
