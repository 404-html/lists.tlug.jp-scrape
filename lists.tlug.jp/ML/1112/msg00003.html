[tlug] Kubuntu 11.10 日本語入力 How?

  • Date: Fri, 9 Dec 2011 20:12:08 +0900
  • From: Lyle H Saxon <llletters@example.com>
  • Subject: [tlug] Kubuntu 11.10 日本語入力 How?
Several versions of Ubuntu/Kubuntu back I just started installing the
Japanese version so Japanese input would "just work", and then put up
with all Japanese menus.  But... having installed Kubuntu 11.10 in
Japanese, I have all Japanese menus and no ability to input Japanese.
I found this on-line:

Kubuntu 11.10でIBusのUIが表示されない理由がわかった
2011年11月16日 23時42分13秒 | Ubuntu/Kubuntu
よくよくエラーメッセージを読んでみるとアイコンがないといってるので、インストールしてやればいいんじゃね？
ということで、"sudo apt-get install gnome-icon-theme"をやって再ログインしたら問題解決。めでたしめでたし（ホントか？

Kubuntu 11.10で日本語入力したかったら
sudo apt-get install ibus ibus-anthy gnome-icon-theme
でいいのかな。ibus-anthyのかわりにibus-mozcでもいいです。
あとはシステム言語も変更する必要があるのと、/etc/X11/xinit/xinput.d/ibusも書き換えなきゃダメか。めんどいなぁ。

追記:
ああごめんなさい、ibus-mozcはダメですね……。

From which it appears I need to do something to the
/etc/X11/xinit/xinput.d/ibus file - but what exactly?

Any suggestions welcome.

Lyle
