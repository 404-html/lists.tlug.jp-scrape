If kinput2 works for you only in kterm, you have probably not set
XMODIFIERS. Use

   export XMODIFIERS="@example.com=kinput2"

kterm supports XIM and kinput protocol. kinput protocol
is the default and kinput protocol will work without
a correct setting of XMODIFIERS.

kterm is the only program I know which supports kinput protocol,
most other programs only support XIM. 

XIM usually will not work when XMODIFIERS is not set correctly,
therefore you need to set it to make kinput2 work with other programs
than kterm.
