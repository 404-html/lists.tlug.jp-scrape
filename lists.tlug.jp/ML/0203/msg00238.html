"Eric O. Flores" <goripi@example.com> writes:

> When I receive a message written in Japanese only the subject line is
> properly displayed.  The message body will show small blocks.
>
>
> Which is the appropriate character set that will appropriately display
> Japanese Language.

'Auto'. KMail has a setting 'Auto' or 'Automatic' or something like
that in it's menu. If you choose charset 'Auto', it will use the
charset indicated in the MIME headers (supposed that the message has
correct MIME headers).
