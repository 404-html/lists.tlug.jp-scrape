> The RH7.1 netcfg has a panel about DHCP and you can force it to run from
> there, but that seems to have vanished with RH7.2


I just run

/etc/rc.d/init.d/network restart

from a command line to restart dhcp (if dhcp has been already been selected for networking for eth0) .

To specify to use dhcp, from the commandline in RH7.2 you can use 

/usr/sbin/netconfig

and select dhcp .  This sets up the appropriate files in 

/etc/sysconfig

for dhcp once OK is selected.  Then restart the networking with

/etc/rc.d/init.d/network restart
 

and maybe...just maybe your dhcp will be working!
