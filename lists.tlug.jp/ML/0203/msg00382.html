> Hector,

> I've found this which might help.
> 
> M-x hscroll-global-mode
> 
> (where M-x is Alt-x)
> 
> This stops emacs from line-feeding the text and
> stops the horrible jumping thing.  You can switch it
> off using the same command.
> 
> You can make it permanent by adding
> 
> (hscroll-global-mode t)
> 
> to your .emacs file.
