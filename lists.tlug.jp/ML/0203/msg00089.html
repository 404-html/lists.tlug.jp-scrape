TAO Group had announced new Free Intent Application Development Kit for Linux (RedHat 7.2) inside DIGITAL magazine (UK).

http://www.tao-group.com

http://www.digitalmagazine.co.uk

Intent is a portable software layer that enable the same application programs to run on various platforms (Linux, Windows, iTron) and various CPU (X86, PowerPC, ARM, HITACHI SuperH) at optimal speed without any modification / re-compilation. Intent will run on top of desktop, PDA (SHARP ZAURUS), Video Camera (JVC), and Cell Phone.

Intent had received support from various leading Japanese companies: SHARP, JVC, SONY, EPSON, KYOCERA etc.
