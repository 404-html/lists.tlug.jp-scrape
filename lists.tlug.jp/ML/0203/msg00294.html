The Hexa Media Drive is a USB digital media (card) reader/writer.  It has
three slots, which you can use at the same time (some restrictions apply).

Slot 1:	SmartMedia
Slot 2:	Memory Stick / SD Memory Card / MultiMedia Card
Slot 3:	CompactFlash / MicroDrive

Technically, the HMD is a USB 1.1 full speed (12 Mbps) mass storage device
with 4 logical units.  You need Linux kernel 2.4.x with the USB mass storage
driver stack.  The HMD appears as SCSI logical units.  BTW, the HMD also
works on Macintosh and MS-Windows machines. :-)
