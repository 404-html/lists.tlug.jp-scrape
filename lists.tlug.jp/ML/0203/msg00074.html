The if you MUST have an search engine at your own site, the most popular
would be namazu. See http://www.namazu.org/

It's an open source solution. The drawback is that you need to create an
index file once in a while (which takes time;It's kind of like Sherlock
in MacOS.) As an index creation engine, kakashi is the default, but for
better results, use ChaSen.

But I think the easiest way is to use google, if your site is connected
to the outside world.
