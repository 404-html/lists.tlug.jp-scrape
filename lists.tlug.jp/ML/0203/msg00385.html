>>>>> "CM" == Charles Muller <acmuller@example.com> writes:

    CM> One of the greatest obstructions I experience in trying to use Xemacs
    CM> for any serious, sustained writing is that the default fonts are so
    CM> small (at least on my system). I can set them larger for one session,
    CM> but as soon as I turn it off, I have to reset next time. Is there a way
    CM> to set a larger font size as default?

You can specify the font you want in ~/.Xdefaults, like this:

emacs*font: fixed
#emacs*font: -misc-fixed-medium-r-normal-*-14-*-*-*-*-*-*-*
#emacs*font: -sony-*-*-*-*-*-16-*-100-*-*-*-iso8859-*

and restart emacs.

Then you can switch fonts by pressing Shift-Button1 inside emacs, the
font given in .Xdefaults becomes 'default' in the font menu.

There's more to it, read the emacs manual :)

Hope this helps
Viktor
