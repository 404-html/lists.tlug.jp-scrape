G'day all,

I'm on about the last step of completing the RH7.2 installation on my
T2800 laptop. The final issue is the sound card (Yamaha YMF754), which
never worked satisfactorily with RH6.2.

The installation went fine, with it being identified OK. 

The CD drive is reported in "dmesg" as:

hdc: ATAPI 24X DVD-ROM drive, 128kB Cache, UDMA(33)
Uniform CD-ROM driver Revision: 3.12
cdrom: This disc doesn't have any tracks I recognize!

I can play
MP3s, .ram files etc using RealPlayer, however when I try an (audio) CD,
I get no sound. In "dmesg" I see:

hdc: packet command error: status=0x51 { DriveReady SeekComplete Error }
hdc: packet command error: error=0x50
ATAPI device hdc:
  Error: Illegal request -- (Sense key=0x05)
  Invalid field in parameter list -- (asc=0x26, ascq=0x00)
  The failed "Mode Select 10" packet command was: 
  "55 10 00 00 00 00 00 00 18 00 00 00 "
  Error in command data byte 2

CD-ROMs can be loaded and read OK - it's just the audio ones which fail.

Any thought/suggestions what to do about this?
