Not sure about gamma correction perse but if you load the photo, then go
to Layer -> Colors -> Curves, you can pull that curve up and down to
adjust the value to where you like it. Helps with brightness and
contrast. Also you can correct colors as well using that dialog box
(instead of value, select red/green/blue then play with the curve).

Other than curves the Layer -> Colors -> Levels dialog might be useful
for what you are trying to do.
