Hi Keith,

On Nov 16, 2007, at 10:56 AM, Keith Bawden wrote:

Hello Everyone,

I have a little project I would like to undertake at home that
requires I change IP addresses as often as possible. Therefore I'm
looking for an ADSL service in Japan that has the shortest possible
lease times on the IP addresses they hand out.


I'm curious to see what type of pet project you're cooking :-)




So far my experience has been that Japanese ISPs have very stable
service and quite long lease times (some have *near* static
assignments).


The Dion DSL service that some family here had a few years back had DHCP
lease times that I informally measured in hours.

One option you could do is to write a script to do the following and see
if that helps. I imagine you could do something similar if you need to use
PPPoE although add steps for the authentication process.

Every arbitrary time interval
1. Release DHCP lease
2.  Change your mac address to some different value
3. Try to re-acquire a IP address via DHCP
