> How about Sabayon (http://www.sabayonlinux.org/)?
> It's a distribution based on gentoo, binary only.

Sabayon Linux provides binary *releases* but is 100% compatible with Gentoo 
after the install.  So, if you would like to upgrade software between 
releases or install software that is not included in the Sabayon release, 
then doing so would require compilation just as with Gentoo.
