On Friday 28 December 2007 02:56:10 phillip tribble wrote:
> Is it possible to have Gentoo only install binary packages instead of
> compiling?

Gentoo is a source based (meta) distribution, and compiling from source is 
what allows it to achieve its goals of "extreme" configurability and 
performance.  Using binary packages (exclusively) would negate the whole 
purpose of using Gentoo...

While there are some binary packages available in Gentoo, I do not think that 
it would be possible to use them exclusively.  (I have never tried, 
though...)
