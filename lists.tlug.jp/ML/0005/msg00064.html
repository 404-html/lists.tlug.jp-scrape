> hmmm, I'll go with the easy way out for now, and go

> For now, I've been using aterm with something like

> obsessed with psuedo-transparent consoles)

You could try wterm, which is a hacked version of rxvt that allows
pseudo-transparent/shaded windows, and a bunch of other stuff that is very nice
for people who use Window Maker, such as NeXTStep-like scroll bars/buttons and
menu. Build it with --enable-kanji. It's much faster than eterm. You can get it
from:

http://wm.current.nu/files/wterm-6.2.6.tar.bz2

but the page is unavailable at the moment. It should be fixed soon (I hope).
