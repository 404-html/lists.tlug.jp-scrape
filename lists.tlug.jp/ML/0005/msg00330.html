That's not going to be easy if the rest of your system has already
been upgraded.

I also don't know gjiten's source; normally all you need to do is
change the linker flags, but in this case, with version skew as well
as doing the static link, what might be easiest is to somehow get all
of the .a's of the relevant libraries (glib, probably gdk, gtk, maybe
gnome, who knows what else, keep adding libraries until you get a
binary that works) in the gjiten source directory and add them to the
link step by hand (actually, adding `-L.'  early in the link command
line, before any other `-L' flags, should pick up any libraries you
have put in that directory.
