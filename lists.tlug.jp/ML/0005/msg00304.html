> Question #3:
> I have a 108 key japanese keyboard with the M$ windows and menu buttons.
> Since the jp106 keymap that comes with most distros is not perfect,
> neither is it under X, so I made my own Xmodmap file a while ago. But it
> doesn't use the extra keys (M$ and the japanese conversion keys), so I'm

The Window and Menu keys are keycodes 115 and 117, at least on my
English keyboard. 

But, if you're looking to put together your own custom keyboard map,
xev is your friend. Run it from an xterm with the keyboard in
question plugged in, and start taking notes.
