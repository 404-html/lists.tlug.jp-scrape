"Drew C. Poulin" <poulin@example.com> writes:

> I'm trying to use ndtpd, the server software for accessing
> Japanese dictionary CD-ROMs, with a client called bookview,
> which provides a GUI for performing the lookups and
> displaying the results.

[...]

> But in order to use bookview, I need to first install Tcl/Tk
> with their Japanese patches.

> So my questions:  Can anyone recommend versions of Tcl/Tk
> and their Japanese patches that play together nicely?

Do you already know lookup.el? lookup.el is an Emacs-package to
ndtp, dict, and various other dictionary formats:

    http://lookup.sourceforge.net/

I am using lookup a lot and like it very much. Maybe this is an
alternative for you to using a Tcl/Tk interface.

Unfortunately I have no idea about Tcl/Tk and Japanese patches.
