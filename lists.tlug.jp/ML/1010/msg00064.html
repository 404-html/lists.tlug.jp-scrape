> Here's the annoying part. I've found a workaround. I use a program on 
> the Mythbuntu machine called Wicd as a GUI interface for my network 

if you always need the network to be up why use wicd at all? Just set
the interface to automatically be brought up at boot (man interfaces and
search for the "auto" word).

...
> Not sure what else might be pertinent, so please let me know if there's 
> other info I can provide.

More relevant would be your /etc/network/interfaces file and maybe how
wicd is configured (i.e. /etc/default/wicd and maybe other details you
could find about auto connecting at boot).

Also have a look at the NEWS.Debian.gz and README.Debian files in
/usr/share/doc/wicd to see if there is any hint there that could solve
your issue.
