 > Where does one get the patch to remap the keyboard to Japanese (what's it
 > called?), and is there a file somewhere that tells you the name of each
 > program and what it does? I can't seem to find the TCP/IP dial up PPP
 > connection program.

1) Look in the JE package for a japanese keyboard (106 keys) map to be used 
   with the loadkeys package. Then you can create a new key table that goes 
   in the kernel 'loadkeys --mktable > linux/drivers/char/keyboard.map'. If 
   you don't want to recompile the kernel, add the 'loadkeys' command in
   your rc.local

2) descriptions of each package are logged in /var/adm/installed/packages
   or somewhere close (sorry, I'm running Win95 now and would have to reboot
   the machine to get Linux; BTW does anybody know if the installable drive
   that would enable ext2 filesystem support under Win95 was ever developed?)

3) the dial-up PPP connection program is 'chat'
