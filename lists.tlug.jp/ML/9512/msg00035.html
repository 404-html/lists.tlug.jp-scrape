EDICT is a great E-J/J-E dictionary. It has versions for DOS, Windows
(winjdic) and Unix (xjdic). The unix version is not an X-Window 
applications, it runs in a shell under kterm, but there is tkjdic, a good
front-end written in Tcl/Tk.

Edict can be found at: ftp.monash.edu.au/pub/nihongo
