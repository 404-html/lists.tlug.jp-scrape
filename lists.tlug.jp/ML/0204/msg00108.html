On Sat, Apr 06, 2002 at 02:53:56AM -0700, Matt Gushee wrote:

> (however I think XFree86 4.x is still 'unstable'--but more on that below).

4.1.0 is rock-solid.  4.2.0 is solid IFF freetype plays nicely with one's libm.

> The above notwithstanding, unless you're a hardcore X11 geek, you probably
> don't want to compile X from source.

X is one of those things that distro makers take great delight in munging.
X should be built from pristine XFree86 sources, modulo driver patches that
should only be obtained from XFree86.

tar xzf X420src-1.tgz
tar xzf X420src-2.tgz
tar xzf X420src-3.tgz
cd xc
make World
make install install.man
/usr/X11R6/bin/XFree86 -configure
vi /root/XF86Config.new

That's not so hard, is it?
