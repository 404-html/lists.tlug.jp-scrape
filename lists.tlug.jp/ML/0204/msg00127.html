On Sun, Apr 07, 2002 at 08:19:17AM -0700, Marlo Rocci wrote:

> The primary rule with windows users is that you do not want to have them
> type anything.

The primary rule with UNIX users is that of least privilege.

Users should not be editing the fstab, as that is a task assigned to the
system administrator.

If you find yourself in a situation where you have to grant fstab write
access to the Great Unwashed(tm), you should reconsider carefully whether
UNIX is the proper solution to the problem.

> Ideally, mounting a network share should be a matter of selecting from a
> list of network resources already known by the computer.

The resources do not have to be known by fstab.  One can work wonders with
judicious use of shell scripts.  Look up the requirements for a successful,
non-root mount.
