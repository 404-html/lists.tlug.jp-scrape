By default MacOSX comes (at least mine 10.1.2 upgraded to 10.1.3/10.1.4
came with) with sendmail. So any source of sendmail information will be 
helpfull (Bat book, http://sendmail.org, etc.).

You can replace it with postfix if you fill more comfortable with it (I 
prefer sendmail myself). 

To install postfix (or any other of ported to MacOSX debian tools) you 
can use fink (http://fink.sourceforge.net). Grab it, install, then in 
terminal start dselect and ... then probably you'll need some Debian 
documentation, rather then Mac-specific).

Hope that helps.
