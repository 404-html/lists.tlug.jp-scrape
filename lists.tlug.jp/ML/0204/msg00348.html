> 
> And make a rescue boot floppy.
> 
> And make a backup of the MBR, and copy the backup to a file on your
> C:
> drive.
> For example:
>   dd if=/dev/hda of=bootsect.lnx bs=512 count=1
>   cp bootsect.lnx /windows/c/bootsect.lnx
> 
> Ordinarily an install of Windows 2000 will overwrite the MBR so that
> Windows 2000's own boot manager will be invoked.  This isn't so 
> bad, but you'll still have to recover from it one way or another.  
> If you stick with the Windows 2000 boot manager then add a line 
> to C:\BOOT.INI to load C:\BOOTSECT.LNX as a possibility.  
> Otherwise boot the rescue floppy and restore the MBR from the backup.
> 

When I have to have both NT and Linux on one box, I usually use
the NT boot loader. A nice utility for setting things up is
BootPart. It can automatically extract the boot sector and set
up BOOT.INIT. 

http://www.winimage.com/bootpart.htm
