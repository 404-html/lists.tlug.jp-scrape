Ok, probably not that many use sylpheed, but I do, so I thought I would
throw it in there.  It has wonderful support for japanese.  Better than
the English version of OE.  You can at least read the subjects with 
sylpheed.  Only problem is it doesn't handle all OE/Outlook mail well.
This is because OE/Outlook will cavalierly lie about the charset that
the email is in.  eg it might send a Japanese email with the following
header line:
  Content-Type: text/plain;
	  charset="iso-8859-1"
