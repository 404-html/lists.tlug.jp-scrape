Sure would.

Majordomo has an optional feature called "Administravia," which (tries to)
trap and forward to the list owner things that look like attempts to send
Majordomo commands to the list.  However, it casts a very wide net.  Any
mail that contains a keyword on the Administravia watch list is likely to
get sent to the list owner instead of the list, so if your mail contains
the word "subscribe" you need to obfuscate it to get it past Administravia.
For example, by writing "s*bscribe" instead.

Administravia is a sufficient nuisance that few list owners use it.
