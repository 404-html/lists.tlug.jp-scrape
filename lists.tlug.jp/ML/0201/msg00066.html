Gentoo is currently slashdotted, but they have an article on
a hardware bug in Athlon/Duron that can hang the Linux kernel
through memory corruption in extended paging.  It's apparently
particularly common when using an Nvidia card or when using
the preempt kernel patch.

For those who have had Athlon/Linux problems lately, this may
sound very familiar.

There is a workaround (can't get to the details) and AC is said to
be working on a patch to do that automatically when an Athlon/Duron 
is detected.
