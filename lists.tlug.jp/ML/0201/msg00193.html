hi,

Mandrake Linux server handles samaba and netatalk in my office and works
fine. More info on netatalk,
 
http://freshmeat.net/projects/netatalk/

for using Japanese characters, set smb.conf to 

# language coding
	coding system = cap
	client code page = 932

You can share Japanese file names on both Windows and MacOS, however there
are some limitation, some kanji-characters don't work but I don't know
details. And don't use hankaku-katakana for file names.
