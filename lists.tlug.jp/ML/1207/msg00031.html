I'm using Gthumb and I'm really happy about it.

It has lots of extensions/add-ons: 33 Extensions installed by default (on Fedora) for Import/Export, Search, Edit meta-data, Renaming, etc...

It support export to: FB, Flickr, Photobucket, Picasa and more

Main reason I'm using it is the ability to assign "commands" to numpad, I use it tag my pictures while browsing them. By pressing one key, I run i.e. "exiftool -keywords+="Rating 3 %F"

Then I use the Filter or Search feature to browse only pictures matching my tags.

Cheers,
