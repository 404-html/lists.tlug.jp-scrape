Your hard disk might also be able to tell you if it is dying.
Check out smartmontools http://smartmontools.sourceforge.net/
Unless your hard disk is really old it probably has SMART.
If so with smartmontools you can run something like this:

# smartctl -d ata -a /dev/sda

to get a report about what your hard disk thinks.
