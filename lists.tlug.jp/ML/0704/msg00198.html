Your SD card seems to be detected and all the needed kernel modules
are loaded, so udev is not at fault here. The problem is that it isn't
auto-mounting. Ubuntu uses a combination of gnome-volume-manager, hald
and dbus to perform automounting. Make sure they are all running and
setup correctly. The following link might help a bit, just ignore the
gentoo parts and pay close attention to the last section which might
hold the solution to your problem.

http://gentoo-wiki.com/HOWTO_gnome-volume-manager
