On 2007/04/04, at 7:22, Joseph Essertier wrote:

Hello Tluggers,

Is there such a thing as a linux-compatible Mac laptop on the new- computer market today? I want to order one in the next few weeks, if there is. Any recommendations?


Well, there are only 2 Mac laptops, and they are just about equally Linux-compatible. If you use (free) BootCamp, you can install Linux dual-boot with MacOSX. If you use ($80) Parallels, you can run Linux in a VM on MacOSX. Installation is very easy with Parallels, provided you know that you must choose a non-SMP kernel at boot.
