I've had the same frustration (using *.txt extension for tab-separated;
but you should consider filing a bug report for it doing the same with
the .tsv extension). Personally I think doing it with *.txt is a bug:

paragraph).

Um... Actually that is a feature. It one of the things my mother
mentioned she prefers in 'office' to 'Word' is that when you open an
excel (or Calc) document in Writer, it opens correctly in Calc. MS
Word just opens it as gibberish (because it was an excel file, not a
word file).
