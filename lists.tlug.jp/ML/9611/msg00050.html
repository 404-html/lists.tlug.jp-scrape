  Hi all,

  I've mostly managed to get HylaFAX set up and working (it has been
a ridiculous pain in the ass!).  I can now send and receive faxes (and
it is nice to be able to send ascii or postscript files).

  The problem is that unless I run 'faxsetup' each time I have rebooted
the machine, I can't send outbound faxes (can receive, though).  The error
I get is odd:

	"Your facsimile job to 7719155 was not sent because
	the document conversion script was not found.
	...
	 Status: Can't find initialization file gs_init.ps.
	..."

  Now, this gs_init.ps is in the right place (there's even a link to it
from the one other place it might be).  Ghostview, ghostscript and so on
run fine (as does 'sendfax', but only if I've rerun 'faxsetup').  And in
fact I just had a look and the document conversion has at least gotten
as far as converting the file I'm trying to send from ascii to
postscript.  I shouldn't be having this problem.  Any ideas what is
going on?
