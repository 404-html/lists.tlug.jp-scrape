Hi,
I'm trying to get cnprint to output to an HP LaserJet IV.  The
printer has English PostScript fonts but not Japanese PostScript.
Can I print to it?  I get segmentation faults when I attempt to
run cnprint on an EUC text file.  I've also tried to save
a Japanese Web Page as PostScript and then print it.  Both
attempts failed.  

I can print ASCII text fine with cnprint.

I'm using jiskan.bin, jiskan24.hbf, cnj24.hbf, kanji48.hbf, kanji48.bin.

My cnprint.cmd is:

#   put PostScript print command (or port name in DOS, like LPT2:) bellow
PS_PRINT_COMMAND: lpr 
#
#   put default GB font (HBF) name below (such as cnj24.hbf)

#
#   put default BIG5 font (HBF) name below (such as cn5b24.hbf)

#
#   put default JIS (Japanese) font (HBF) name below (such as cnjb24.hbf)
