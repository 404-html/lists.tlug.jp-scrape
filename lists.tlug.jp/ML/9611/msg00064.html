      *** Input Works Only With Cut and Paste ***

http://www.bpel.tutics.tut.ac.jp/~take/Netscape/

Explains Japanese input under Netscape.  I followed 
the example and it worked.  I have included screen captures
and a link to the page on
http://www.twics.com/~craig/linux-nihongo/web/index.html

The page at tut.ac.jp is entirely in Japanese.  I'll give an
English explanation when I get some free time.

I can now do the following under Linux with English Linux Netscape 3.0:

   o  display Japanese in buttons and other form elements
   o  display Japanese titles in my bookmark file
   o  display Japanese in that little "go" menu item[craig@example.com netscape]
   o  input Japanese via cut and paste into Netscape

I've modified my netscape.sh script to load the libs as follows:
$ more netscape.sh 
export LANG=ja_JP.eucJP
export LD_PRELOAD="/usr/local/netscape/liblocale.so:/lib/libc.so.5.2.18"
exec /usr/local/bin/netscape

I call the netscape.sh to run netscape.  Source for liblocale is 
found on the web page at tut.ac.jp.
