> Unfortunately it doesn't work. What happens when started up in the Japanese

Contrary to KDE, GNOME applications *do* use LANGUAGE, LC_ALL,
LC_MESSAGES, LANG (in that order) to decide the language to use for
the user interface. If you want your user interface in
English, you can use for example

    LANG=en_US LC_CTYPE=ja_JP XMODIFIERS=@example.com=kinput2 <some-program>

See also

    http://www.suse.de/~mfabian/suse-cjk/locales.html

and

    http://www.suse.de/~mfabian/suse-cjk/xim.html
