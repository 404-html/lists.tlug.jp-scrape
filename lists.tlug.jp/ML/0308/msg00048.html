> Don't know what I have done but whenever I use vim now (v6.1) the first 
> column is always highlighted. And if I start typing a new line and 
> insert only spaces, all the spaces until the first character are 
> highlighted too.
> 
> It only recently started doing this but I don't remember making any 
> changes to my .exrc file ...


One way to check is to temporarily rename your .exrc file. Also check /etc for a
vimrc (probably not a dot file--it helps in cases like this, if you
mention your distribution). And, of course, your home directory for a
.vimrc. (You can also log in as another user, to see if it's system wide
or just for the one user.)

For example, Arch Linux, my current favorite Linux, has an /etc/vimrc
which has all sorts of (to me) annoying behaviors.
> 
> How can I get rid of this behaviour? Can anyone point in the right 
> direction?

Hope one of these ideas helps.
