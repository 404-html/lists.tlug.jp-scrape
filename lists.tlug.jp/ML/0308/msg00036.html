I created file ~/.Xresources and inserted this line:
*ConversionStartKeys: Ctrl<Key>space

Great, now I can safely type a space even if I was still holding down
the shift key after the previous character.  Ctrl-space is a perfectly
fine way of firing up canna.

But ctrl-space doesn't turn off canna.  I still have to press the old
shift-space to turn off canna.

Anyone know how to make it really work?

By the way, I know that questions like this have been asked before on
the TLUG mailing list.  But when searching the archives, "shift" hits
no matches and "space" hits no matches.
