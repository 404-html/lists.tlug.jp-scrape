On Tue, Aug 12, 2003 at 04:05:59PM +0900, Iizuka, Hirotaka, J-Phone wrote:

>I have problem with installing RedHat Linux7.1 

Stop!

Red Hat 7.1 is far too old and has too many old security problems in it.
The current version of Red Hat is 9.0, so if you want Red Hat, I 
strongly recommend using that.  It is also more likely to install
correctly on your Vaio, especially if the Vaio is newer than RH 7.1, but
maybe it's not newer if it has a Pentium II :-)


>For RedHat Linux7.1, is Sony Vaio PCG-C1XF capable to be a platform?

It should work.  Here is a description of installing Red Hat 6.1 (old
article):

http://www.bagley.org/~doug/c1xs/

And a description of installing Debian 2.1r4 (also an old article):

http://cowboyneal.org/vaio/

Here's one for installing Red Hat 7.1 (but don't do it, anyway :-)
This is also an old article):

http://www.animaproductions.com/linux3.html

Sony used to have links regarding Linux on the C1X, but they have
all been taken down, probably because it's old and they want to
sell and support new ones.

Could you tell us exactly what kind of problem you are having 
with your install?  With detailed information, someone can 
probably help you.  Also, using a newer version of Red Hat
might help a lot, too.  Hardware support is much better for
everything than it was in the days of Red Hat 7.1.
