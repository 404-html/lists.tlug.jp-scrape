Hi all,

I had to try out NcFTP on something really big so I downloaded
the lastest Netscape 4 beta. Here are some observations...

It's HUGE - by the time everything unpacks it's close to 12 MB,
but it loads in about the same amount of time as 3.0 (which
means on a 486, crack a beer and look for the lastest Linux
Journal.)

The java bug from 3.0 is fixed, but there still is no sound
support!? Nothing in the README about it. I'm not a big fan of
icons in the tool bar but can't reduce them to text<grrrr>
Since I'm stuck with 256 colors I've always been dissatisfied
with Netscapes handling of the color map. 4.0 makes a big
improvement here. I haven't figured out how to clear the cache.
Could it be Netscape only caches the current session? Seems that
way. I was really looking forward to playing with style sheets;
the Embedded Style Sheet and Linking Style Sheets work but
Inline Styles don't.

Hmmm, unexplained crash on the Real Audio site, but on the
second try, no problems playing a .ram file. Is there a Shockwave
plugin for Linux? I'll have to check that out...

Bottom line: if you're happy with the 2.02 or 3.0 you're using
now, don't rush off to download Communicator...

Jim S.
