My company has been looking for a cross-platform suite so I picked 
up a copy of Applixware 4.2 for Linux from RedHat to test it out.  
RedHat has a $199 deal where if you buy it now you get the 4.3 upgrade
free.  There are a few pre-configured utilities for the financial 
industry, and the word processor/spreadsheet imports and exports 
english M$ formats.   

The install was a breeze even on top of Slackware.  The mail utility 
has a couple of quirks that I'm still working on, but seems OK.

I could set it up at the next meeting if anyone's interested in having 
a look at it.  Are there any other ApplixWare for Linux users in TLUG?
