root>Does anyone have any ideas on how to time-synchronize a Linux
root>box automatically from a known standard, like Universal Time?
root>I've heard that the GPS can be used, but it's expensive.  

If you have an internet connection then most probably the most cost
effective way would be to use the NTP (Network Time Protocol) Daemon.

You can get the source code from the home of NTP software at: 

	http://www.eecis.udel.edu/~ntp/

Also available at this site is the list of publicly accessible clock
servers.  I've recently gone through a NTP network setup so feel free to
contact me.
