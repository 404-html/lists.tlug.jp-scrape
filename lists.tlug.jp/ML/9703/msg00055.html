tlug note from joem <root@example.com>
--------------------------------------------------------

I'd like to be able to remote boot a linux/NT box so that I can choose
which OS will come up.  Ideally, this would be ftp-ing into NT, setting
a switch which would give me a Linux boot next, and vice-versa.

The goal here is to be able to occasionally remotely boot linux on 
several NT machines in order to use Linux to tar new filesystems
onto the NT partitions, and then re-boot into NT.  This works manually
really well, but I want to automate the process.

Any good solutions to this one?  

-Joe.
