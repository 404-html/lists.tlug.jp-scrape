I run my old modified TL6.0 + 2.4.4 kernel hybrid thingy on my new T-bird
just fine.  I have:

1267mhz AMD Athlon-C socket-A processor (266 FSB)
Gigabyte 7DX motherboard (AMD761 northbridge/VIA 686B southbridge) DDR board
256mb PC2100 266mhz DDR ram

runs great.  compiles 2.4.4 (make bzImage && make modules) in 3 min 40 sec.
runs X.  does everything the way it should.  kernel complains that it can't
understand the AGP bus, but it seems to get by with the generic equivalent,
and it runs X really well, so I'm not worried.
