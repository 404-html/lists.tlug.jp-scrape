the relevant command for testing would be:

ipchains -F
ipchains -t nat -F

(that would flush out all the rules and make it "wide open").  Useful for
testing.  You can leave it like that, too, unless the box is connected to
the 'net directly.

ipchains -L

you maybe able to change the settings for samba by running "setup" as
root and choosing "Firewall Configuration", otherwise you will have to
change them from the command line with

ipchains
