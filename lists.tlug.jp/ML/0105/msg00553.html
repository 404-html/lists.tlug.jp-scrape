> Do you *need* to use perl? awk can handle this without the fuss..

"Without the fuss" translates to

basename `head -1 /var/spool/mail/cronthing | awk 'BEGIN{ FS=" ";} { print $4; }'`

perl -M File::Basename -nae 'print basename($F[3]); last' <
