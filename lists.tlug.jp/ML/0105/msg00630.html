Since you have a bunch of PCs going through a hub to a TA or router
connected to Farts ISDN, and since AFAIK Farts ISDN only gives a single
IP address, the device on the other side of the hub is probably a
router doing NAT.  If so, it's may well also be doing DHCP.  

Set the WinME box up to get its IP address by DHCP (might be something
like a check box that says "Get address from server" but not sure
about this), reboot it, and see if it works.
