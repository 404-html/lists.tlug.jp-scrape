is there a DHCP server?  if so, just do ipconfig /renewall or /renew (one of
those...).  If not:

1.  does the machine have a name?  If so, do an nslookup on that name on one
of the other machines, and get the IP address that way, and manually
reconfig.

2.  if it doesn't have a DNS record, pick an unused IP address in the subnet
and apply it.

Get the DNS and gateway settings from one of the other machines, they should
be identical.
