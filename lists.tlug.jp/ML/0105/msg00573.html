Sorry, I didn't try it out before mailing :( I usually run exim
but thought I'd play with sendmail since it seemed like an interesting
problem.

In /etc/sendmail.cf there is a line...

O DaemonPortOptions=Port=smtp,Addr=127.0.0.1, Name=MTA

If you comment this, it will work as I believe this restricts it to
only accepting connections from localhost. There is probably an m4
file somewhere, perhaps /etc/mail that also has the option from which
you can generate your sendmail.cf file.
