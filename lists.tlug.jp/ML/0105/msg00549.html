On 23 May 2001, at 8:04, Jean-Christian Imbeault wrote:

> >The only editor I use is vim and by default it creates a backup file
> >with a "~" at the end. Have recovered data many times because of
> >that.
> 
> Is this a setting you had to put into you .rc file?
> 

Yeah it is. The command you are looking for is 'set backup'. put 
that in your .vimrc file and the backup file will be left when you exit. 
Won't help you out now, but in the future... However after you put 
this in there, you might want to setup a cron job to go around and 
delete all the backups that haven't been modified for a few weeks 
as they wind up all over the place. *grins*
