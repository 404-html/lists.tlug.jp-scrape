I've been busy reinstalling linux. Because everytime I run "sndconfig" my
machine freezes and I can't boot linux so I reinstall.(I don't have a floppy
drive).
I spent the hole night doing the same thing again and again.
But the cdrom mounts fine. I add this in /etc/fstab

"/dev/hdc /mnt/cdrom auto user,noauto,nosuid,exec,nodev,ro 0 0"

Thanks a lot.
