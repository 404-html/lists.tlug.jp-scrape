http://www.macssh.com/FAQ.html

claims that you can do CVS over SSH (SSH2 only, unfortunately) on a Mac. I 
haven't tried it myself, though. What has never been available on the Mac,
strangely, is scp. How do security-conscious Mac users send files around
these days, anyway?

FWIW, Mac OS X includes the command-line version of ssh. This I've tried,
and it seems to work normally. Japanese display doesn't work because OS X 
doesn't seem to come with any terminal program that supports double-byte 
characters, but that isn't ssh's fault.
