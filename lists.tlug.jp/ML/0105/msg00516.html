I think Netmeeting uses either the old or new Directplay format for message
passing.  If you forward the right ports to the intended machine, it should
work:

OLD DIRECTPLAY:  tcp and udp 47624, tcp and udp 2300-2400 inclusive.
NEW DIRECTPLAY:  9110/udp only
