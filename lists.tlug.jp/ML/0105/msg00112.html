was created by some Japanese software. It works fine (the document has
just scanned images), but each time it starts it whimpers:

	"Warning: locale not supported by C library, locale unchanged"

wants is there, and is being told it isn't.  Or possibly it is trying to
change the locale and can't. Should I or can I have a different C library
