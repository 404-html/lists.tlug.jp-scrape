The easiest option I have seen is to install it on a Windows
box, and copy the entire directory over.  I carry my copy
around on a thumb drive.  Use wine to execute it.  In my 
experience, it works for a little while, then needs to be
closed and restarted again. Looks like resource issues to me.

For those who have asked IrfanView is a light weight image
Viewer with a few nice image manipulation features.  You can 
rotate, adjust lighting, adjust color depth, resize and crop 
images.

There are a few other things as well, but that covers what I
generally use it for.  :)

It's negative behavior is most likely caused by differences
between how Wine and actual Win32 handle memory management.
