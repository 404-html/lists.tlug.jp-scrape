Godwin Stewart wrote:
> How does ImageMagick react to a multi-page PDF?

It works like the following:

$ convert input.pdf output.png
$ ls
output-0.png
output-1.png
(snip)
output-11.png

If you want to convert only one page, you need to add the page number
with the input filename, like input.pdf[0].
