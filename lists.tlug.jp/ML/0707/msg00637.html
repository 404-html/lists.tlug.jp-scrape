[tlug] Tip of the Day: Viewing (almost) any online PDF as HTML

  • Date: Sun, 15 Jul 2007 10:58:44 +0900
  • From: "Josh Glover" <jmglov@example.com>
  • Subject: [tlug] Tip of the Day: Viewing (almost) any online PDF as HTML
Use Google:

<filename> site:<sitename>

For example, I want to be able to search and copy from
d20060206slides.pdf, which is the FY 2007 Department of Defense
Budget. The original file is here:

http://www.defenselink.mil/news/Feb2006/d20060206slides.pdf

So I perform the following Google search:

http://www.google.com/search?q=d20060206slides.pdf+site%3Adefenselink.mil

i.e. type the following into the Google search box:

d20060206slides.pdf site:defenselink.mil
