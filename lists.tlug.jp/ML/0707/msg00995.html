    Godwin> "Ogg" is the container format, in much the same way as
    Godwin> AVI, MPG, MOV or MP4. The audio stream contained in the
    Godwin> Ogg container is encoded using the Vorbis codec. Thus, a
    Godwin> ".ogg" file can in fact be a movie with a vorbis audio
    Godwin> stream and a theora video stream.

Not to be a wise guy, but be careful with calling MPG a container
format there are MPEG program streams, MPEG system streams and MPEG
transport streams, which all contain MPEG video and audio elementary
streams and sometimes private streams like AC3.
I am just mentioning it, because you can have quite some trouble
converting a DVB or ATSC transport stream into a program stream that
can be used for a Video DVD, and some people just ask you: "Why it is
both MPEG2, isn't it".
