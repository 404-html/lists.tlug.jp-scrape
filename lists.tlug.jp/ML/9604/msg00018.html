Jim:

I finally got my Compaq Deskpro running Linux ( Red Hat 3.0.3 ) smoothly. There
are a number of hurdles I had to
jump. Not sure if you will run into the same problems, but here is some help.

Is your "D:" drive a physically different drive or a partition on the same
physical drive?

Check to insure what device your root partition is on. i.e /dev/hda2  or
/dev/hdb1. You might have to force feed lilo or loadlin
the root directory "root=/dev/hdXX".

Did you format the "D:" as msdos or a Linux file system?

If you can provide a little more information, I might be be able to help you
more.
