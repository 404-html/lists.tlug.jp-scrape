If you got this far AND have backed up any important data from the old 
system, I suggest
(1) boot your installation floppies in installation mode.
    If they show drives at /dev/hda and /dev/sda, and a CD-ROM at
    /dev/sbpcd (or wherever the "thingee" lives)
    THEN
(2) simply install Linux to /dev/hda1 and figure that adding a line
    like 
    /dev/sda1        /scsi-hdd    ext2        defaults   1   1
    to /etc/fstab should win.
