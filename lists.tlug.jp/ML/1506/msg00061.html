On Thu, Jun 25, 2015 at 3:12 PM, Thomas Blasejewicz <nyuwa@example.com> wrote:
The machine has been sitting here for a few days, during which I did not
use it.
Today I tried to start it up and get:
"Cinnamon has crashed. It is currently running in fallback mode. Do you
want to restart Cinnamon?"
Which does not do anything.

I would be you, I would try to create another user. This looks like a profile issue, better to confirm it.

You may try:
CTRL-ALT-F1 (which should give you access to a text console)

Then login, and type "sudo adduser testuser"
testuser being a chosen username.

Then try to login with your new user (CTRL-ALT-F7 to come back to graphical interface).

If CTRL-ALT-F1 does not give you a terminal at first, you have a real trouble.
