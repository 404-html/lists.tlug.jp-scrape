# talking with github booth: 
Always wondered about the business model.
They sell private-github-repo hosting, and a VM to host 
a github in companies own perimeter.  US based datacenter 
so far, they research on getting more distributed like 
google.  Hard to do as git protocol is not designed for it.

# NTT docomo talk on network virtualization.
At 9/11 they had 60fold phonecalls, you want tech to 
survive that time, without buying 60x hardware.

# cumulus booth
Many network switches have actually the same hardware 
chip.  Cumulus is a linux distro installable on these 
boxen.  The benefit: instead of the vendors OS you get 
all of the linux tools and software, can even run own 
code.  Also many different switches look to the outside 
all the same.
debian based, speed same as original manufacturers OS.
I guess that is good news to companies, they can ex-
change switch hardware and get a similiar interface to
the outside.  In times of openstack when switches need
to be tightly integrated into some bigger infrastructure,
thats good.  Reminds of nowadays x86 hardware with Linux,
also kind of replacable (but the hardware monitoring is
different everywhere).
