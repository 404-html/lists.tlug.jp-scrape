The trick to getting fcitx working in Fedora seems to be editing one's
$HOME/.config/fcitx/profile file and making sure that it reads anthy:True
as opposed to the default of false. In the next day or so, I'll put this on
my page.  And just in case there's someone searching, and we all know how
we hate to find  something saying nevermind, I fixed it, the way to get it
working in FreeBSD is to install from ports, make sure you have a few
fonts, then follow the instructions given when you run pkg -D -x.  Also, at
least when using it with dwm, one has to again edit their
$HOME/.config/fcitx/profile and this time change mozc to true.  This seems
to only stick if you first close X and edit the file.
