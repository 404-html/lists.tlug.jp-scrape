> Is anyone else using MythTV at all? Or does anyone know of any cards
> that can be inserted into a computer that handle ISDB broadcasts?

I bought myself a Buffalo DT-H50/PCI a few months ago.  I didn't
expect it to work in Linux so I didn't try until I upgraded my Windows
partition to Windows 7, only to find out that there weren't even vista
drivers for this card.

I then started investigating the possibility of getting it working in Linux.

lspci identifies it as "03:06.0 Non-VGA unclassified device: ViXS
Systems, Inc. XCode 2100 Series"

A search for ViXS Systems quickly turns up this page which describes
the architecture of the tuner card:
http://www.vixs.com/sections/markets/applications/applications_jap_dig_ter.htm

It pretty much looks like it's impossible to get any meaningful data
out of this card since the data is encrypted before it reaches the PCI
bus.  And it looks like that's the way things are for every chideji
tuner card.

1seg isn't encrypted though, but it is really low bandwidth and the
quality sucks for anything bigger than a cell phone screen.

it to science if anyone wants to try hard hacking it with a soldering
iron.
