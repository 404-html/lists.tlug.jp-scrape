Now that you mention it - I just tried opening up both 1.1.1 and 1.1.2, 
and 1.1.2 does seem to open quite a bit more quickly (on SuSE 9.1 
Professional)!  Another version I'm testing is a an early test version 
of 2.0 (680), which has one very nice feature lacking in the current 
main release - a very handy and effective Word Count that gives not only 
overall statistics, but also word count for highlighted text.  People 
have been clamoring for this for a long time!  There's a macro available 
for the older versions, but I haven't tried it yet.
