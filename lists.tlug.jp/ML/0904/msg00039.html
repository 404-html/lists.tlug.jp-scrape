Positions are open at an Web venture company looking to expand . Japanese is not so necessary for the roles, however high skills in Java script, Ruby and other object orientated language exp is. For the Scrum master position the certification is desirable but not necessarily a prerequisite. Those interested and looking to find out more please contact grant.habgood@example.com or find Grant Habgood on Linkedin.


1
User Interface Software Engineer (JavaScript/RIA)


Responsibilities

As a UI engineering expert, you will build responsive and elegant web UIs of our next generation ad server product with AJAX and similar technologies.

- You will both implement new UIs and redesign existing UIs.

- You will have the focus on the client side in many cases, but sometimes work on the front-end of the server side as well.

- You will work together with our ad server engineering team and product designer


Technical requirements

Extensive knowledge and experience of DHTML, CSS, JavaScript/AJAX, JSON and web services APIs. Expertise in jQuery is a plus.

- Experience with W3C DOM methods and properties, cross-browser development techniques, and browser degradation strategies.

- User Interface design knowledge

- Deep experience in designing modular, object-oriented JavaScript

- Well versed in code optimization and software design principles.

- Experience with modern server-side web frameworks such as Ruby on Rails

- Experience in RDBMS such as MySQL

- Experience in at least one of the object-oriented programming languages (Java/ Ruby/Python/ C++/C#/Objective-C/OO Perl, etc). Ruby experience is a plus.

- Experience working in a Unix environment.

- BS or MS in Computer Science or relevant subject.


Qualifications

- Excellent verbal and written communication skills

- Team player with interpersonal skills

- A self starter and a creative-problem-solver.

- Can learn new technologies and frameworks on-the-fly.

- Fluent English is required.

- Japanese skill is a plus.

- Passion to make people's lives better with science and technology.


Notes

- This position is based in Tokyo.

- Growth potential is more important than the current capability --- so junior developers and senior developers are all welcomed for application if following this philosophy.









2
Scrum Master




Responsibilities

- Scrum Master is a technical project manager type roll but a mastery of Agile development practices.

- As a Scrum Master, you will facilitate the execution of front-end engineering projects for our mobile ad server product.

- You will be responsible for the team to provide on-time and quality deliverables of the projects.

- You will provide guidance for team activities. You will coordinate and communicate across team members and teams.

- You will perform variety of project support tasks such as facilitating meetings (planning, review, retrospective), maintaining backlog and monitor project metrics.

- You will work together with our ad server engineering team and product designer/planners.


Technical requirements

- Experience in software project management using Agile practices in a technical environment.

- Scrum experience is a plus.

- Ability to understand User Interface design issues

- Knowledge and experience in DHTML/CSS/JavaScript/AJAX and web service APIs.

- Knowledge and experience with modern server-side web frameworks such as Ruby on Rails

- Experience with RDBMS such as MySQL

- Experience in at least one of the object-oriented programming languages (Java/ Ruby/Python/ C++/C#/Objective-C/OO Perl, etc). Ruby experience is a plus.

- Experience working in a Unix environment.

- BA or BS degree.


Qualifications

- Excellent verbal and written communication and presentation skills

- Strong leadership with interpersonal, facilitation and cross-team coordination skills

- A self starter and a creative-problem-solver.

- Can learn new technologies and frameworks on-the-fly.

- Fluent English and Japanese are required.

- Passion to make people's lives better with science and technology.


Notes

- This position is based in Tokyo.

- Growth potential is more important than the current capability --- so junior candidates and senior candidates are all welcomed for application if following this philosophy.
