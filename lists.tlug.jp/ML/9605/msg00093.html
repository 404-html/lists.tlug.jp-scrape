>Jim, how do I call up the text installation manager that asks me

It should have been started automagically when you ran the 2.1
boot/ramdisk1/ramdisk2 disks.  There are a bunch of Perl scripts
in /ramdisk/usr/lib/perl {install,dispatch,textinst} (or xinst
if you are doing an X installation) that control the process.
The text installation uses the 'dialog' program and so looks
vaguely Slackware-like.
