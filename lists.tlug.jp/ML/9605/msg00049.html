Jim,
thanks for getting back on this.  I have been battling with Debian for the
past few days and I am going to wipe my disk again and try Red Hat this 
time.  What are the differences between the commercial and non-commercial 
versions?  

I believe Debian may be the way to go at some time in the future, but for 
the information pool of the group, I will contribute my experience:

	1) can't get it to work with NFS install from a laptop
	2) default kernel did not work from a desktop machine with
		NE2100 clone ethernet card.  I was forced to reboot
		every time I used the ethernet
	3) default disk setup could not repartion my hard disk
	4) program manager is still a mystery after about 3 hours.  This 
		probably pretty normal when using a new way of doing things,
		but it was a bit tedious.  I didn't like the program manager
		too much
	5) it installed a lot of texinfo files which I don't really use.
		Who does?  HTML seems the way to go for online docs.

I got it to work, but am going to try RedHat.  Debian is still in Beta.
Perhaps it needs more time.
