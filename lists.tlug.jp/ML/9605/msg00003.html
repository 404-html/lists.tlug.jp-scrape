Yes. With Xedit you can only display Japanese, but you can not write Japanese:-)

I use Canna (kana-kanji conversion server) and kinput2 (Japanese input method) for writing Japanese.

I prefer Canna to Wnn, because Wnn require too much memory.

Installing Canna and kinput2 is not difficult. You only need editing configuration file before compiling.

here is my configuration:

Toshiba J3100SX001VW
i386SX(20MHz) + i387
10M memory
200M Hard disk
VGA compatible LCD monitor
PS/2 mouse

Linux 1.2.13     (from slackware 3.0)
Canna32p2        (from ftp.astec.co.jp)
kinput2-v2fix1   (from ftp.sra.co.jp)
