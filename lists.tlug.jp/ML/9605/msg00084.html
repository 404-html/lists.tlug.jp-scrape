On Thu, 23 May 1996, Jim Tittsler wrote:

> >After 5 failed attempts at installing RedHat 3.0.3 via FTP I gave up and
> I'm curious how it failed.  I was envious of your net connectivity
> and the ability to just start the process and walk away after every-
> thing had be slurped over and magically installed.

The process would hang when trying to load a large file.  I would
get about 50 packages installed then it would hang.  The net connection
would still be up and other machines on the ethernet would be fine.
However, all TCP/IP traffic from the machine I was trying to install
to would stop.  This would continue even if I left it for an hour
or so.  

I used 

	Root disk: RedHat 3.0.3 single floppy 
	Boot disk: kernel 1.3.57, image number 13
	Ethernet: unknown cheapo LANCE 10BaseT clone 
		(this could be the problem)
	Disk: IDE 800 meg disk with 100 meg partition for root and
		/usr mounted on about 300 megs
	Other: 32 meg RAM, Intel Atlantis motherboard, 166mhz Pentium

Installation off of a 2x Mitsumi CD-ROM went fine without any problems.

> ways.  I prefer using the text installation to put in the basic
