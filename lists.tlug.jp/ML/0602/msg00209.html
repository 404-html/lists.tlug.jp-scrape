TLUG,

    I got a solution on another linux list that I cross posted to.

    The SD card, which was formatted on my Palm Pilot, is in FAT16 
format. There is a limit of around 512 MB, give or take depending on 
some system overhead, for the *root* directory.

    For whatever reason, the storage limit only applies to the drive's 
root directory. By simply creating one subdirectory and putting all 
files in there, I can use the full 1 GB capacity.

    Thanks to everyone for suggesting possible solutions.
