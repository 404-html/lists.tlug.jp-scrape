Synaptic must have come a long way since I last tried, but I just see no
reason to use it.
Unless your keyboard is broken and your hand has a mouse implant,
probably clicking is the way to go, but a lot of people prefer apt-get.

If you can memorize those few keywords you need to use, apt is a better
choice I think.

apt-cache search <keyword>
apt-get install <pkgname>

If you really need a gui there are other apt front-ends as well
(gnome-apt for example), but synaptic might be still the best amongst.
