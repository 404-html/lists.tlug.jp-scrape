On Feb.10 16h07, Jim Maricondo wrote :
> Did you know you can use GMail as a free secure SMTP server?  It's 
> a bit tricky, you have to use these settings for SSL/SMTP/Gmail. 

I use Gmail with mutt, and after some google search I can now send 
emails with msmtp with the following config in ~/.msmtprc [1]

Evan

[1] ~/.msmtprc
defaults
logfile ~/.msmtp.log

# gmail account
account gmail
host smtp.gmail.com
port 587
auth on
tls on
user my.address@example.com
password abcdef

# set default account
account default : gmail

Description: Digital signature
