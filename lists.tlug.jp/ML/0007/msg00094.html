You might also consider using a swap file.  'man mkswap'
will explain the details.  Basically:
  1. make a file the size of the swap space you want to add ('dd' is easy)
  2. mkswap filename
  3. swapon filename

Particularly handy if you just occasionally need to use some precious
hard disk space for swap.
