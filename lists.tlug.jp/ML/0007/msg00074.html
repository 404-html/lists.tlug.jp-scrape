b) Regarding a post long ago about extracting PDF text:
A convenient way to extract Japanese text from a PDF file is to use xpdf.
As usual, you have to get the crypto stuff from outside of the US.
 1) get crypto patches from http://www.foolabs.com/xpdf/decryption.html
 2) patch xpdf
 3) run configure with --enable-japanese
 4) make install


c) SMI (Silicon Motion Inc) Lynx works with XFree86 3.3.6, although this
doesn't seem to be mentioned in the docs at the XFree86 web site.
Some people apparently use older XFree86 with the framebuffer, too.

http://www.siliconmotion.com/lynxemplus.htm
http://www.xss.co.at/sysinfo/laptops/asusP6300/stepbystep.html

You can find information like this at the "LCD-NOTE-PC list":
