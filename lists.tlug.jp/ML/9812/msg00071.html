Michael Casinghino writes:

 > I have a file that is using ^M for line delimitification.  How can I
 > replace these with LF's?

Chris Sekiya writes:

 > No sed.
 > 
 > cat file | tr \\015 \\012 > newfile

Peter Linsley writes:

 > Here is a sed solution:
 > 
 >      sed 's/.$//'

Good, simple answers which I can't improve on. Gets a little more
complicated if you have to do CRLF (DOS) to LF (Unix). If you have a
continuing need for this sort of thing, you might have a look at my
webpage <http://unnamable.home.ml.org/crunch/textconv.html>, where
I've collected several text conversion methods.

Chris, Peter: if it's alright with you, I'd like to add your methods
to my page ... properly attributed, of course -- or not, if you
prefer.
