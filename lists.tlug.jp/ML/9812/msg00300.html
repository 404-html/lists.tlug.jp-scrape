On Fri, 11 Dec 1998, Uva Coder wrote:

> - Add the following to the top of your /etc/lilo.conf file:
>   
>     append="ether=0,0,eth0 ether=0,0,eth1"
>   
>     Don`t forget to type "lilo" after you edit the file. ;)
>     This may get one card to show up. Once the card(s) are detected
>     edit this file and make changes as necessary 
>     (i.e. append="ether=10,0x300,eth0 ether=9,0x6100,eth1"

This will only work if the ne driver is compiled into the kernel.  If
you're using a modular kernel, these parameters have no effect.
