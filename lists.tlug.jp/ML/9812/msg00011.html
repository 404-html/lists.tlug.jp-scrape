> Hi,
> 
> I have just installed TL2.0J onto my machine and I'm on my way of
> upgrading X to XFree86 3.3.3 with binaries that I've got from the
> XFree86.org website.
> 
> When I did "preinst.sh" it gives me a message like 
> 
> "you appear to have an ELF system
> Make sure you are installing the 'Linux-ix86' binary dist 
> and *not* the 'Linux-ix86-glibc' binary dist
> You are strongly advised to backup <blah> <snip>......"
> 
> 
> I thought TL2.0J has glibc support? Should I really be installing the 
> glibc version or should I listen to what the message says and install
> the libc one? If I'm installing the libc one, then what is the glibc version
> for?


well, 2.0 is a libc5/glibc hybrid.  the X libraries installed by 2.0J are
actually libc5 X libraries.  I'd go with the libc5 ones like preinst.sh
says to do.
