Do you need to search for *software*? Your original question

 > > > I need to find out how to get this Snappy connected to Linux,

sounds more like you need *information* ... in which case Deja News is
probably your best bet.

To find software, I often use FTPSearch ... the old URL is 

http://ftpsearch.ntnu.no/

which will redirect you to the new URL, which is part of Lycos. If you 
use FTPSearch, I'd suggest you start with a case-insensitive substring 
search (the default), filter out all the BSD stuff, and limit the
domain to "jp" (no quotes). Then if you get no hits, broaden the
conditions a bit.

Hope this helps.
