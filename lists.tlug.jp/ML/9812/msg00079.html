On Fri, 4 Dec 1998, Michael Casinghino wrote:

> I have a file that is using ^M for line delimitification.  How can I
> replace these with LF's?  An answer containing the string "sed -e" would
> be most interesting for me.  BTW, 'man ascii' says CR = \r = 0x0D and LF =
> \n = 0x0A.

Really all you need to do is get rid of the ^Ms.  there should already be
linefeeds in there.  there's a program called 'dos2unix' which was
designed for this special case, but if you want to use sed, I think:

(sed 's/\r//g' file > file.new;mv -f file.new file)
