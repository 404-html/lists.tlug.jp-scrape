Mule/Mew

When I start Mule from KTerm the window opens up rather low on the 
screen placing the mini-buffer behind the lower row of icons (default 
setup for TurboLinux 2.0J; I don't mind them being where they are). 
I am inevitably lifting Mule up with the mouse each time I use it.

screen and make life just a bit easier to live? 

Netscape

Netscape on a Linux system would do well to turn 
off Java. FWIW, I have had more problems with 
Java (and Java script) disabled than under the 
default settings.
