On 05-Dec-98 Raimund Buellesbach wrote:
> When booting linux, everyone can enter single user mode as
> root

AFAIK if you allow physical access to a computer, there is the
risk of compromise. The only way to really secure a box is to
lock it in a room 20 feet underground and sever all links to it
- including the power.

You're going to have to find some compromise between usability
and safety. Your first line of defense is a good back-up policy
which is actually done and checked from time to time. 

Your second line is deciding how much unsupervised access to the
computers you'll allow and what security measures you will
insist on.

HTH
