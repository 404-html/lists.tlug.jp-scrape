    Hi all,

    I got a problem. I want to autostart a song from a homepage.
My HTML is as follows. 

------------------------------------------------------------
<P ALIGN="CENTER"><embed src="song1.mid" height=55 width=135
 autostart=true loop=true></embed></P>
------------------------------------------------------------

But this doesn't work. When I put some *.wav file into "xxxx"
things get all right to my sheer surprise. *.wav starts without
any problem.

Also when I put *.wav file, the place in the homepage where the
song is embeded shows a nice audio icon, whereas *.mid doesn't.
It shows only a rectangular frame and doesn't start until I
double-click on it. 

I added "audio/x-midi    mid" into mime.type file in NCSA server
directory's conf/. But still it doesn't work. Netscape Navigator
Document Information doesn't recognize it and says it is a
text/plain. When I download some mide files from other sites,
they start automatically and also information says it is a
audio/x-midi.

I also tried to put AddType things in srm.conf but in vain.

I need a nice icon as well as autostart. Please give me a clue.
Thanks in advance.
