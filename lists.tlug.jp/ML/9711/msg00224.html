Someone the other night asked if there was a Redhat Linux 4.3.  To my
knowledge 4.2 was the last version put out before the release of 5.0.  I
have ordered a copy of 5.0, so if someone would like to use it AFTER I
load it up you're welcome.

Also, if you check out the RedHat Errata for 4.2 there are some kernel
upgrades to correct the Intel chip f00f problem, an update for the
svgalib security hole, and an iBCS update if you use it.  If you follow
the directions on upgrading the kernel, all goes well.  If you don't
read the directions you'll be reloading like I did the last couple of
nights.

Anyway, thought you might like to know.
