As others already suggested, you first have to get your
nameserver right.  To check that everything apart from the
nameserver works, do a ping to a numeric IP address with one
hopp.  

You said, you get

> ppp0      Link encap:Point-Point Protocol
>           inet addr:165.76.21.201  P-t-P:165.76.23.253  Mask:255.255.0.0
>           UP POINTOPOINT RUNNING  MTU:1524  Metric:1
>           RX packets:13 errors:0 dropped:0 overruns:0
>           TX packets:13 errors:0 dropped:0 overruns:0

So do

  ping 165.76.23.253

(That is, use the numeric IP address given after `P-t-P'.)

This doesn't need a name server.  If this works, try

  ping 133.31.180.200

(This is the numeric address of `sunsite.sut.ac.jp'.)

Now, you can try to get your nameserver right, as explained
in the other e-mails.
