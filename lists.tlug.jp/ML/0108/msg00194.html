Use a script or program that set rules for you. There is a number of
free scripts that setup a firewall more or less automatically according
to your preferences.

I personally used seawall (for 2.2) (http://seawall.sourceforge.net/),
and now shorewall (http://shorewall.sourceforge.net/) for kernel 2.4

Both of them are pretty simple to use. You have to set various config
variables, then rules are automatically generated.
