put solaris cdrom in drive, power on sparc.  at banner, STOP-A.  at the "ok"
prompt, type:

boot cdrom -s

this will drop you into a single user shell, so you can mount the hdd and
fix /etc/passwd and /etc/shadow.

However, if root is *gone*, your system has either suffered severe
filesystem corruption or you've been 0wn3d (ie, you've been hacked badly),
and you probably are going to be in "recover any remaining valuable data and
reinstall the box" mode now.
