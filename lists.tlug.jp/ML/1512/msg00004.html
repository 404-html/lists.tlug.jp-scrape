On Thu, Dec 3, 2015 at 8:34 AM, Jim Breen <jimbreen@example.com> wrote:
> >From what I'm reading about Ubuntu Gnome 3 it's a fresh install, and
> you can't easily upgrade/convert an existing 15.q0 installation. Or can
> you? Any advice/suggestions?


After seeing Charles' e-mail last night, I went ahead to Google about
this :-P and ended up installing on my Ubuntu 15.10 Unity systems
(two).  So no, you don't need a fresh install.

Just install the "gnome" package and all its dependencies.  Oddly
enough, on one machine, I had an annoying circular dependency.  Forgot
the package in question, but the error was something like:

dpkg: error processing X.deb
 trying to overwrite '<some file>', which is also in package Y

As they weren't different versions of the same package and the file
seemed unimportant (IMHO :-) ), I did a:

sudo dpkg -i --force-overwrite ...

Hopefully, you don't encounter such a problem!  I think that's the
advantage of a fresh install.  Less chance of a conflict.  But, once
you install the gnome package, in the login, you can select "Gnome"
and it should be good.  I did that and even did a "sudo aptitude
search unity | egrep ^i" and started uninstalling all of it.  :-)  I
didn't loathe it, but happy to try an alternative...one person calling
it "perfect" in a blog is better than what I've heard about Unity!

Oh!  Oddly enough, gdm display manager gave me a blank login screen.
I don't know why...so I ended up going with lightdm + gnome.  Seems ok
so far...

Hope this helps and thanks Charles for opening our (my) eyes!  :-)  I
was grudgingly accepting Unity and didn't have the motivation to try
something new.

Ray
