Sounds like I got it in one... if the "some time" is the 30
seconds you mentioned earlier.

If you don't want to do the ident lookups, set:
rfc1413_query_timeout = 0s

Or, if you don't want to disable it globally, add some
"negative" nets to the rfc1413_hosts at the beginning of
the host list so that for those entries in the host list, no
ident query will be issued:
rfc1413_hosts = !127.0.0.0/8:!172.16.1.0/24:0.0.0.0  

> Yes, 113 is blocked. I didnt know it should be open. Will unblock it and try
> again later. For now, got to pick up my new bicycle. :)

With the decline in multi-user systems (and the slim chance of
actually finding a co-operating postmaster at the other end
that is going to require ident info when you make a complaint)
I think a case can be made for disabling the lookups.
