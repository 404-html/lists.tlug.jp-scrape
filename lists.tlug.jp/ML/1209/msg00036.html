On 2012-09-09 at 16:30:51, David J Iannucci wrote:
>Wait.... is it maybe just because I didn't freaking unmount sda6 from
>the hibernated OS before going into the live boot!?!?

That was going to be my guess. You had a filesystem mounted by 2 Linux 
installations at the same time. That's a big problem unless the 
filesystem is one of the clustered filesystems, which i'd assume on a 
normal Linux installation's hard disk is not the case.

If you need to modify a filesystem on a hibernating Linux installation, 
just unmount the filesystem before hibernating and then remount after 
resuming from hibernation. Then it should all be OK.
