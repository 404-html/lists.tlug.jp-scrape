that /usr/local/info/dir is an info file---"mule /usr/local/info/dir"

RMail files, and goes into RMail mode automagically when an RMail file

You can (if you have write permission) edit info nodes from inside of
Emacs Info by typing "e".  You may have to do ESC ESC "(setq
Info-enable-edit t)" RET first to enable editing.
