  Hi,

  Thinking it might be fun to play around with the JIS X 2012 character
set, I picked up some stuff at Jim Breen's site at monash.edu.au.
In addition to an edict format dictionary file for these characters,
he provides a font file (jis21216.zip), and kterm sources and a vi
clone (jstevie.gz) patched to support the character set.
Unfortunately, I can't get the patched kterm sources he provides in
kterm212.tar.gz to compile.  When I do the 'xmkmf ; make', things churn
away for a bit, and then I get this fatal error:
