% Has anybody on the list ever used the Tektronics mode of xterm?

   Yes, here's a very rough and ready example, knocked together on my
   (sharp intake of breath) Solaris system at work, using graph(1) and
   plot(1). 
   
   The short uuencoded data file below contains the "graph"'ed output
   from the "Raw ASCII Input" section. It will uudecode to
   /tmp/plot_data.  Start up an xterm and use <CTRL>-mouse-button to
   switch into "tek" mode. In the tek window do "cat /tmp/plot_data" to
   display the graph.
