On Sun, 13 Oct 1996, C. Oda wrote:

> 
> Hi,
> Andrew Howell was nice enough to post a patch for xjdic so that it
> would compile under Linux.
> 
> I was unable to patch it properly.  Can someone make the
> patched source, binary distribtion, or instructions on
> how to apply the patch available?  I've tried applying the
> patch to xjdic2.2 with no success.  If I get it working,
> I can archive it on our FTP server so others can get it.

I got the patches and it didn't work either...BUT, it was just the clue
that I needed to get it working.  Finally, I've been trying to get this
program to work off and on for months!  Its a nice program to look up
stuff from web pages.

What should work is if you define DEFAULT_DICDIR in both xjdic.h and
xjdxgen.c.

Mine is:

  #define DEFAULT_DICDIR "/usr/local/lib/xjdic"   

Hope that works.

 	Shige
