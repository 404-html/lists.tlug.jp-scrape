I have a Perl script that updates my web pages via cron.  However, the
directory is locally mounted.  Anyway, it might be a helpful example for
recursively going through a directory structure. 

Basically it goes through my local html directory, compares the time to
that of the corresponding file in the target directory, and if it is newer
the target is updated. Then it goes through the target directory looking
for files that exist there which are not in the local directory.  It is a
70 line Perl script.
