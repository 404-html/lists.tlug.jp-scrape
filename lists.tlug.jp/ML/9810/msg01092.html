On Tue, 27 Oct 1998 masatomo@example.com wrote:

> Hello, everybody.
> 
> I happen to have a machine with kernel-not-recompiled Redhat5.1.
> 
> > 2.When I try "mount /mnt/cdrom" I get this error:
> >   mount: fs type iso9660 not supported by kernel
> Do as follows:
> # insmod /lib/modules/2.0.34-0.6/fs/isofs.o
> # mount -t iso9660 /dev/hdb /mnt/cdrom
> 
>  Until you recompile kernel this will be a solution.
> 
> Cheers.

You don'ot have to do the 'insmod', you just say 'modprobe isofs'.

And it's supposed to load that module automatically when you request to
use the filesystem.  if not, something's wrong with the module
dependencies file (try 'depmod -a'), or kerneld is not running.
