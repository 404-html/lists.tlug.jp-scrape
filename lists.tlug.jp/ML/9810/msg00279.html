Hi,

It may be neccecary to edit the /etc/pcmcia/pcmcia.conf file to get your 
NE2000 card working correctly, since the default file has a fairly 
limited selection of cards available.

For example I had to add the following to get my NE2000 compatible card 
to work.

card "My NE2000 Compatible card"
  version "PCMCIA", "LNT-10TN", "1.0"
  bind "pcnet_cs"

Good luck,
