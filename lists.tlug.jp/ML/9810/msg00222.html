I've been trying to pull together some "interesting" things to demonstrate
on the NetWinder at Saturday's TLUG meeting.  Since it is a small server, it
is running my favorite server daemons, but that doesn't lend to a compelling
demo. :-) Many of the intriguing interfaces (video, telephony) for client
use are barely supported by software at present.  I've built some tables of
features and a simple comparison to what I know about the Cobalt Qube.
