> Is there a software package that will change kanji in to hiragana

I use kakasi for converting incoming Japanese mail into furigana but
still leave the kanji next to it. Every incoming mail is being checked
by procmail and then sent through a perl script to kakasi. Works fine
only M$ mail doesn't declare it's content correctly so I end up without
translation there <rant>.

Quick&Dirty[TM] perl script that glues the thing together.
