Sorry for posting about a purely technical matter ... but I
received a message this morning from Peter Braam, the project
coordinator for Coda filesystem development, indicating that they
will be releasing Coda for Windows 95 soon.

Good news for me, because I can then offer shared Coda-space and
a dual-boot workterminal configuration to EVERYBODY here.

For anyone who was away for the one day it took this thread to
peter out ... Coda is a distributed filesystem like AFS, with a
couple of neat extra features like server replication and
disconnected operation out of client file-cache.
