I've never experienced this particular problem, but my guess is that 
you're trying to do this while offline.  (A sensible thing to do.)  
Probably, Netscape likes to check for you that it's real.  

Another 'feature' of at least some versions of netscape that I 
experienced is that it will  "check it's  network connection" by 
trying to contact some netscape.com hosts.  Similar idea.  It doesn't 
check if you use a proxy though.

I still like Karl-Max's suggestion of using a local mail 
program like sendmail though, even if you want to use Netscape.  (Does
netscape even allow sending mail in an 'offline mode'?)
