Many thanks to Jim, Rex, Scott, Andrew and Tom for pointing out
that "L1-A" or "Stop-A" is the magic key combination to bring a
Sparc around to the PROM prompt.  It worked "ok", and I was able
to carry out the install yesterday evening.  Much enjoyed the
"Redneck" install messages from RedHat.  Yup.

All I need to do now is string a bunch of spare SCSI disks off
the Sparc, install Coda, and write some scripts to take care of

few more scripts and troubleshoot the installation.
