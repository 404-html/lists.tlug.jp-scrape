Well, I've finally motivation to make at least a minor contribution
to a program for Linux.  First, I want to make sure no one else has
done it.

I couldn't find a xkeycaps configuration for my 106 key Japanese
keyboard (bought at one of those famous TLUG auctions).  Has anyone
else done one?  

Even if so, I plan on making one for the Dvorak layout as well and

the program maintainer will want it, but I'll offer.)

If anybody has any suggestions, please let me know.
