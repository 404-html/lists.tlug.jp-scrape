> > through another smtp server (ie, it's behind a firewall of some sort),

> The technical term for this is "smarthost" or "relayhost".
> 
> in sendmail.cf
> 
> DS<smarthost>

Better practice is to edit the master "*.mc" file, and generate
the new sendmail.cf using m4; the content of the master file is
easier to interpret.
