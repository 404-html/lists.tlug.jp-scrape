Hi, folks--

I'm experimenting w/ a TeX virtual font based on the Wadalab fonts,
and I have a couple of quick (?) questions:

1) Am I correct in thinking that Wadalab Mincho contains the full
JIS-X0208 character set, sequenced in the standard Kuten order? So
that if I want to know which character is in which position, I can
just look it up in a kuten table such as the ones in Lunde's
_Understanding Japanese Information Processing_? By the way, is it the 
JIS-X0208-1983 or 1990 character set (I know the difference is very
small)?

2) I generated a set of VPL files, and I notice that the CODINGSCHEME
is defined as DMENCODING + TEX-TEXT. Can anybody tell me what
DMENCODING means and, more importantly, whether dvips and ghostscript
actually need to know about it?
