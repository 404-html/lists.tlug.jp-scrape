Generally, bzip2 is make smaller compressed file than gzip, 
especialy patch files. This is why the kernel patch is compressed
by bz2 format.
but bzip2 cost much time than gzip, this is not adaptable for 
communication ussages (eg. on time compresssion in wu-ftp tar.gz file)

I select gzip and bzip2 with my file type.
