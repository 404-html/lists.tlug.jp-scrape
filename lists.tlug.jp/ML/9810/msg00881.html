Maybe somewhere along the line you must have set some X resource. Try:

	xrdb -q |grep background

My guess is that you set something like

*background: black

or some such thing. Since you may have changed several resources, the
easiest ( whimpiest ?? ) way to fix it is just restart X.

You can also do something like:

echo "*background: white"|xrdb -m

which will change any background resource to white. It is much better
to set these on an application by application basis in your .Xdefaults
( .Xresources ?? ) file. Example:

Emacs*foreground:		lightblue
*XTerm.VT100.saveLines:         2048
xclock.Foreground:              white
