Yong-Ming Hua wrote:
> According to some primer book, mem=size is necessary to be 
> typed after boot: prompt when we make Linux system with memory over 
> 64MB in the beginning. 

Put an entry in /etc/lilo.conf like:
    append="mem=??M"  where ?? is your real memory amount.  
Then run lilo again.


BTW - For anyone who's got this (remotely related) problem ...
I just heard from Werner Almesberger, the author of lilo, that
he has a fix for the 4Gb disk/NT/Lilo boot incompatibility and it will 
be out in lilo 0.21.   

-Joe.
