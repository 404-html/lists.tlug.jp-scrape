I set Thunderbird not to show that dialog box by using the config
editor to set mailnews.nav_crosses_folders to 2... and I change
folders from the keyboard using the must have Nostalgy extension.

# mailnews.nav_crosses_folders
# 0 - Always go to the next folder without prompting
# 1 - Ask before going (default behavior)
# 2 - Don't prompt and don't go to the next folder with unread
