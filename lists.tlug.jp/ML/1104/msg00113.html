Using thunderbird 3, gnome, ubuntu 10.04. When thunderbird brings up a
dialog box (*) it makes a beep, and I want to stop this beep, but I
cannot find out where it is configured. I've been through thunderbird's
"about:config" looking for sound, beep, dialog, etc. but no luck.

I've already configured gnome's Preferences|Sound choosing a custom
alert sound. The sound thunderbird makes is a different sound (it sounds
like "duh-dh-dh"). The thunderbird sound is also louder than the alert
sound used in the rest of gnome.

...semi-breakthrough: in that gnome dialog if I set the alert to be
mute, then thunderbird's annoying beep also goes away.

So... I still don't know if I should be hunting for a gnome or a
thunderbird setting! Any ideas?

Darren

*: E.g. pressing N to go to next message and it asks me about going to
the next folder. In fact I've realized this might be the only place it
does this particular beep; e.g. when thunderbird prompts for a password
it does not make a sound.
BTW, I have thunderbird configured to not make a sound when new email
arrives, and that works fine.
