If you haven't been to the
<A HREF="http://www.ccil.org/retro/retromuseum.html">Retrocomputing Museum</A>,
it is an absolute must.

Implementations of several OISC (One Instruction Set Computer)
architectures are there.  If that doesn't pique your interest ... sell 
your Hacker's Dictionary and go learn Cobol.

Many thanks to Andy Howell for the pointer to U-Dub's IMAP page, which 
led to ESR's Freeware page, which led to the Museum.
