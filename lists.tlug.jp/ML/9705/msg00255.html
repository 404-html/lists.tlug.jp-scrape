some more.  And guess what?  It works, but not quite the way it is supposed
to:  what finally worked was C(trl)+Shift+^ .  This will work from plain
                                   ^^^^^^  
Roman letter entry mode or Hiragana input mode (as the book says it should).
I find it necessary to repeat that key combination twice to get it to come
up the first time.  After that once is enough.  Subsequent switching to
that mode takes you directly to whichever sub-mode you last used (e.g.
symbol, Greek, Russian, etc.).  To get the main menu in the minibuffer again
you need to press C-g.
