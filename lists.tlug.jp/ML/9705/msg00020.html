One piece of software that I've been really pleased with over the last
few days is a proxy server called junkbuster. You specify a list of url
patterns that you don't want to see and it blocks access to them.

If your block file contains, for example,

/banners/
/Banners/
/graphics/advert
/home/ads/
/image/ads/
/images/ads/
/img/ads/
/pictures/sponsor/
ad.doubleclick.net

you won't see many gifs that are likely to be banner ads (unless they
are already in your cache).

It's GNU GPL software; http://www.junkbuster.com

I use a blockfile I got from http://www.teclata.es/junkbuster/english/blocklist.html
