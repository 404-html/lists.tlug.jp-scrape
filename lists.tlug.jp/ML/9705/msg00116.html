Andy:

You might actually be able to save a bit more than you think when you look at 
what is included in the motherboards these days (especially those designed for 
high-end, dual processor systems).  For instance, my motherboard (an Intel 
PR440FX "Providence" Dual PPro has:
        Adaptec 7880 family Wide/Fast scsi
        Ide (of course)
        Soundblaster compatible audio
        Intel pro100b Ethernet
        USB
built in.  So you probably can kill 30,000 - 40,000 with the built-in SCSI and 
networking on most dual ppro motherboards.  (SuperMicro also has a good 
reputation for dual MBs, from what I've heard)...

Let me know if you want any more info...

I've got a Gateway 2000 486/33. I've been toying with the idea of
putting together a dual PPro machine for some time. Having gone to
Akihabara many times, only to return overwhelmed by the choices and
prices, I haven't gotten very far. My rough estimates are that it
would cost:

     Mother Board           50,000
     64M memory           40,000
     PPro 200mz x 2          140,000
     SCSI 3/wide Controler      30,000
     2.0Gig wide drive     100,000
     Video Card           20,000
     17inch Monitor           50,000
     Ethernet Card           10,000
     Case                15,000
               ---------------
                    455,000

These are very rough estimates, from memory. I'm sure I could do a lot
better doing main order through the US.

Well, I can dream anyway....
