On Thu, 8 May 1997, Craig Oda wrote:

> I can read the Japanese text okay.  However, my display is evidently
> set incorrectly.  Does anyone know how I can change this display.

Go to the main menu.  Hit "S"etup.  Hit "C"onfig.  Look for a line that
looks like this:

character-set            = (none)

... and replace it with:

character-set            = ISO-2022-JP

Caveat: I've found that pine is pretty inconsistant about displaying
Japanese.  My inbox will have nonmapped EUC, but once I reply to the
message things map okay.  I haven't had time to squash this one.

> The Pine-mule system I am using now would work okay if I could
> change the Content-Type: mime header to indicate proper Japanese
> encoding.

This one I can't answer.
