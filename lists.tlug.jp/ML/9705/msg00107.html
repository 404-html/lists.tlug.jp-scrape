I've used SKK for my henkan/FEP thing for a year or two now.  I got fed
up with these huge systems like wnn which were a real pain to
compile/understand (plus there was lots of documentation which I didn't
want to read).

It doesn't get much easier than SKK.  No fancy-pants lexical stuff, no
servers, no configure scripts.  Just an elisp file and a big plain-text
dictionary.  To install, you just copy the elisp & dictionary files in
the right place.


Maybe I'm missing out by using SKK, but I kind of doubt it. :-)


Jason

PS- I should note in fairness that I hate emacs.  I use vi except when
I have to write Japanese.  I guess people who actually like emacs might
prefer some of these 2MB-of-source-code-henkans. :)
