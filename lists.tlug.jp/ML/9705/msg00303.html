Hello Alan,

>tlug note from "Alan B. Stone" <stoneab@example.com>
>>Could someone give me some advise on how I might go about setting up the
>>sound card on my Linux Box?  I've read the HOW-TOS, and they mention
>>xconfig, menuconfig, and config, but when I try to run any of them I get
>>command not found.  Can someone help?

cd to the linux kernel source (usually /usr/src/linux) and enter "make 
menuconfig" - this'll build the menu config utility and run it for you.
You
can now configure and rebuild your kernel.

If that seems too daunting, and if you have a 2.x kernel with module 
support compiled in, you could try and grab the modularized audio driver
from http://www.4front-tech.com/usslite/.

HTH,
Thomas
