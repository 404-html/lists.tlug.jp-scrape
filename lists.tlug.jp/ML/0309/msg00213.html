It also occurs to me that POL is dead easy to abuse. All a spammer needs to
do is set up a petition of which the content is basically something hawking
$BODY_PART enlargers. Then, you just need to script an HTTP request which
you fire off (through an open proxy, of course) to POL once for each address
you want to spam. POL will then mail a confirmation complete with spam off
to each of the target addresses.
