Darren Cook wrote:
> I'm planning a linux/samba file server (*) for a mixed network of
> windows and Macs. Are there any issues I need to be aware of? Both the
> Mac and Windows users will be using Japanese filenames some of the time.
> ...

Caveat:  I'm no expert, and I don't read Japanese... =)

From some work I've done recently, there are some issues with having M$
Windows and MacOS playing on the same network with Japanese character sets.
If I have my info correct, Macs generally use UTF for the character sets,
while M$ uses SJIS -- resulting in some bad mojo between the two.

As an addendum, I've called RedHat JP about this sort of issue, and the
person with whom I spoke said that they (at RH JP) don't use Japanese-named
files.

If someone's got a silver bullet for EN/JP Win/Mac to play nicely, I'm all
ears. =)
