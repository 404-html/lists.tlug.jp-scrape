Jim Breen <jwb@example.com> writes:

> [Antony Stace (Re: [tlug] Does rxvt support kinput?) writes:]
>>> 
>>> I am also wanting to know what utilites exist for changing the character encoding, ie 
>>> 
>>> euc <-> shift_jis.
>>> 
>>> And also finding out what encoding a file is.  I have found the program kcc,  what other
>>> program/s exist which can do these type of operations?
>
> Ken Lunde's "jconv" has been around for a decade or so. It tests
> code-type and converts to whatever target you want. Doesn't do UTF8, but
> I use it for everything else.
>
> Then there's "nkf" which comes on most Linux systems.

glibc >= 2.2 also comes with 'iconv' which can convert a lot of
different encodings.

    iconv --list

shows you a list of the supported encodings.
