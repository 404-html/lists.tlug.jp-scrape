Jonathan Byrne writes:
 > Does anyone know of an mpeg editor that will allow me to supply
 > a packet number in the mpeg stream upon which an operation
 > should be performed (just deletion is good enough)?  Ideally, a command-line
 > utility that would take the mpeg file and the packet number as 
 > arguments.
 > 
 
What do you mean by packet? Do you mean a frame?
There are some tools in the DVB driver package on linuxtv.or. There is
also an application called gopchop
(http://outflux.net/unix/software/GOPchop/) which can edit an mpeg
stream on GOP level.
