I picked up an "Easy Disk" keyfob to see what I could do with it.  It looks
like an interesting gadget.  When I threw the usb-storage and sd_mod drivers
into a 2.4.16 kernel, the device turned up, but was unwritable with lotsa
SCSI errors.  I found a success report and a patch to work around the readonly
limitation at:

  http://www.stray.ch/articles/dok.html

I don't know about the one I purchased, but these things are now apparently
being made with boot capability.  So you will eventually be able to carry a
Linux-based backup and recovery system for Intel machines on your (physical)
keyring.  Check it out at:

  http://www.missl.cs.umd.edu/linuxbios/msg06107.html
  http://www.nvnews.nv.net/envy/news/DOK16/DOK16.shtml

I will run another trial of the backup CD at home tonight.  If all goes
well, I should have a working version of the RepliCoaster up tomorrow morning.

Cheers,
Frank
