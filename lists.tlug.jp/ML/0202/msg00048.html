If you run Debian, and have a non-xterm terminal emulator installed
(e.g., kterm, rxvt, uxterm), can you try the following for me?

1) run
	update-alternatives --config x-terminal-emulator
and select something other than plain xterm.

2) run
	man xterm
What man page do you get?

I keep getting the man page for x-terminal-emulator, which means I get
whatever I chose in step 1, rather than xterm(1). This is WRONG. For
starters, kterm's man page assumes you've read the xterm man page.
For another, just because I chose kterm/rxvt as the default doesn't mean
I never use xterm. Finally, none of the other packages that provide
alternatives do stunts like this--"man more" gets me the man page for
more(1), not less(1) or lv(1), for example.

Any "me too" or "not here" reports appreciated. Once I get some
confirmation, I'm going to file a bug report.
