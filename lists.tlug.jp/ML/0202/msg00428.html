Or you could install the compat-egcs-c++-6.2-1.1.2.16.i386.rpm package
(and the other supporting packages). This contains the missing library,
and is included with RH7.2.

You will probably have to run
/usr/i386-glibc21-linux/bin/i386-glibc21-linux-env.sh or set the
required variables by hand before running javac.
