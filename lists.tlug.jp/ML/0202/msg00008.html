I was wondering if anyone has sucessfully integrated the 5.1 version of
the Eudora client with CMU's Cyrus Server.

Has anyone had any recent problems with Eudora 5.1 and Cyrus ?

From the 5.x release notes, it appears that Eudora has had a
long history of bugs related to the IMAP protocol. More specifically, it
appears that IMAP was not working for the Macintosh client until recently.
Many people on the Cyrus list complained about Eudora.

Despite the negative information about Eudora, we
may be implementing the Eudora client with the Cyrus IMAP server.

We have both Mac OS 9, OS 10 and Windows 98, 2000 clients.

The current Eudora client release notes state that all these bugs are

resolved.

I worry that the release notes may not tell the full story.
