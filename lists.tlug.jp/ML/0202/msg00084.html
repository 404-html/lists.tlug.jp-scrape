> > rpm -qf `which netscape`

> This is answered by "No such file or directory"

rpm is the package manager under mandrake 8.1. It is installed on every
system, sou you shouldn't get a "No such file" error unless you mistyped the
command (space before -qf, etc), or ran it as a normal user who doesn't have
rpm in his path. Be sure to run it as root.

By the way, mandrake has a GUI package manager called "rpmdrake". This should
be easier for you to use i think.
