item would be to create a set of web pages that could explain how to
set up a Japanese IME.  It would be nice to "click a button" and have

not quite there yet.  Until that "click a button" day is here, I think
such a web page would be helpful -- even more so if, someone outside

installed it on Ubuntu many times...yet I seem to be unable to repeat
