The IP address is, again, my external web hosting ISP (my web site is on isris.pair.com) and is definately outside of my home network. I let the thing run for about 14 packets and after the 3rd hop, nothing comes back. Given the trace on 137/138, it seems that's still within the Usen network. So, as someone mentioned before, they've probably blocked it.
