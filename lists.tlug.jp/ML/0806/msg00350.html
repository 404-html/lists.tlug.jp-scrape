There are a lot of Linux distributions, but that's a great example of
The Long Tail.  Most Linux users will use a distribution that is based
on Debian or RedHat, and that means they will be using dpkg or rpm
packages and have similar tools.  Even file system structure is not
too different between distributions; the reason it seems worse is that
often it is the location of configuration files that differ, and those
are the files we want to tweak.
