[tlug] AR5007EG--Madwifi now supports 64 bit

  • Date: Thu, 5 Jun 2008 00:01:18 -0400
  • From: Scott Robbins <scottro@example.com>
  • Subject: [tlug] AR5007EG--Madwifi now supports 64 bit
  • User-agent: mutt-ng/devel-r804 (FreeBSD)
For those with one of the many lowend laptops that use that AR5007EG
wireless card, there is now a patched MadWifi snapshot that supports 64
bit systems.  (It also supports kernel 2.6.26--the earlier patched
snapshots that supported the card, even on 32 bit systems, don't seem to
want to build on 2.6.26 kernels.)

Information can be found towards the bottom of of a ticket with around
220 comments, http://www.madwifi.org/ticket/1679

Of course, I have a page on it
http://home.nyc.rr.com/computertaijutsu/rhwireless.html#5007
