On Sat, Jun 07, 2008 at 09:25:52AM +0900, Nick Bikkal wrote:
> Can anyone tell me please where I can get some intro books on Linux? Any
> suggestions as to which book is good, etc?

Well, one problem is that a book on say, RedHat, isn't going to help
that much with Ubuntu. 
However, there is a free book online, do a google for the Rute book.
It's downloadable as a PDF, I believe.

Also, on the Linux Documentation Project site, there's an old book,
something like Installing and Running Linux, by Matt Walsh.  The
installation section is way out of date, but then, there's a section on
basic shell commands which is still quite useful.
