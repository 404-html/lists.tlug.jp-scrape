Okay, I`ll assume you are using Debian or a Debian-derivative.

If so, then review interfaces(5). Also, review your
/etc/network/interfaces file and evaluate whether the pre-up entry
therein is correct. If you are using Debian, then you could `zless
/usr/share/doc/ifupdown/examples/network-interfaces.gz` for an example
to how it might be done.

 In the interface file you might need define the interface the script
is to use, for example:
 pre-up iptables-restore $IFACE

Regards, Jed
