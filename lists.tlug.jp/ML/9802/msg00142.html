On Mon, 16 Feb 1998, Jim Schweizer wrote:

> On  Mon, 16 Feb 1998, Craig Oda wrote:
> >I would also like to propose use of *.html files as part of the
> >FTP archive, similar to sunsite.
> 
> Sounds good! Can I help?

It should be okay to do almost everything by FTP if TLUG had its
own server.  If this was the way we made the CD-ROM, there would
have to be some type of TLUG FTP administrator in the same way
that there is a TLUG webmaster.  I was thinking of two sets
of html files.  One that was created by a cron job and was text 
only and the other that was created by merging the text-only html
file with multimedia templates.  You could do this by having
screen captures and long descriptions of programs in certain
directories with standardized names, then having a cron job
walk through the directories every so often. 

Sounds like fun.  :-)
