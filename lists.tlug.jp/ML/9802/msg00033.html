What does "dselect" do? I have xjdic v2.2 installed and don't
remember having used it. Is there a new version?
 
1) edit jdic.h, setting DEFAULT_DICDIR to the directory
   where you will install the dictionaries

2) "make client" and "make server"
   I prefer the client/server rather than the standalone
   version, as the dictionaries are kept in memory by the
   server; it takes a few seconds to load them with the
   standalone program

3) copy the binaries to somewhere in your PATH

4) copy edict and kanjidic (both unzipped) plus
   kanjstroke, radicals.euc, radicals.tm, radkfile, 
   romkana.cnv, vconj, wdict and kanjstroke to the
   directory set in 1)

5) "make xjdxgen" and execute "xjdxgen edict", "xjdxgen kanjidic"

6) copy .xjdicrc.skel to your home directory as .xjdicrc and edit it
