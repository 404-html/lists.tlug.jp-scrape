I'm a bit upset - tlug's majordomo will dump a list of all the email
addresses of list members if you tell it 'who tlug' - I don't know how
things work in Japan, but in the USA, this is how people build up their
mail spam lists.  Can we maybe turn this off?  It's easy to do, just edit
the appropriate line in the .config file for the list.

Speaking of lists, when we move the TLUG server here to PHT, will it
handle this list or do you want me to have my current majordomo server
handling the list?
