Craig & TLUG crew :-


The Netscape source release is still scheduled for the end of March -
which is about a week before I'm in Tokyo, so it will be a pretty good
time to talk.  The source distribution will be large, I'm not sure of an
exact size, but to give you a rough estimate, it currently takes about
45 minutes to pull the entire source tree on a Solaris Ultra via CVS. 
The codebase is large and complex - so it will really take people a
while to get their heads around it.

Additionally, there are a few upcomming events prior to the source
release that might be interesting. The licensing agreement for the
source will be beta-ed on the net soon.  In case you may not have heard,
we are developing our own source license - it won't be the GPL but it
will be something similar.  If you check on
http://developer.netscape.com/

Currently, Navigator is building with Motif 1.2 under Linux - which we
have a source license for.  Some of the Unix development teams have been
investigating moving to Redhat Motif 2.x which they say works well.  I'd
also imagine that a port to the athena widget set will happen once the
sources are released.

I don't know how much of a presentation it will be, what I'm more
interested in is trying to set up a working relationship between my
group (Netscape i18n) and yours.  I'll probably have a few slides with
information about the source release, some pointers from our group, and
a few suggestions as far as the direction we think some of the i18n
features should be moving.  I think mostly it will just be an informal
talk so that we can all get to know each other and I can answer
questions.  Hopefully though, you could arrange to have some machine
with navigator installed to view the slides.
