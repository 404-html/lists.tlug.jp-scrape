Yeah, you need to make that patch to ppp.c.  TurboLinux 1.4J will
incorporate it in its standard ppp driver module.  It also doesn't hurt to
do:

hdparm -u 1 /dev/hda (and /dev/hdb if you have it)
irqtune 3 (I'll need to RPM-ize irqtune for TL)
setserial /dev/modem spd_vhi


and I still think that it's cool to be able to talk on the phone and use
the computer at the same time on the same line.
