> You mentioned a version of Pine that can support Japanese characters.  
> Where can I go to learn more?
> 

Regular pine will support them.  Here's what you do:

1.  install Canna or Wnn6 (preferably Wnn6)
2.  Install kinput2, learn how to use it, and start it up.
3.  Run pine in a kterm window or from kon, if not using X.  Configure its
'alternate editor' command to run 'vi'.

4.  Compose a message.  To insert Japanese, invoke your alternate editor
with CTRL-_ ... now, push i for insert, and use kinput2 to enter your
japanese (I think it's shift-space to start, ctrl-c to end..).  When
happy, press esc, then :wq and you'll be back in pine.  finish sending
your message.


Reading Jmail is easier - just simply read it while running pine under a
kterm window or under kon.

But, isn't there a 'pinek' or 'kpine' or something like that that handles
it better?
