> Hi, 
> 
> It's the first time for me that I write an e-mail 
> to this mailing list. 
> I deal with Linux system, but I'm not an expert.
> I bought a laptop Canon 5500GX with neomagic 
> chipset (NMG4) and I've been told that Linux 
> doesn't not support such chipset. 

You are right the Neomagic is not supported by Xfree86, but
is supported by Accelerated-X.  You can order it for a little
less than 3万円 I believe.  In Japan you can order it from a
company called ぷらっとホーム.  You can find their web page
at:  http://www.plathome.co.jp.

I have this server for my notebook and it works well compared to
using the Xfree86 VGA server with 16 colors.  Now I have 256 colors.
Not great, but much better than 16.
