Here I am, steadily ascending the learning curve again :-)

Today's question is about Sendmail, about which I know virtually nothing but
am curious.  Sendmail seems to be an SMTP server that runs right there on a
person's local machine, judging by it's behavior.  If that is indeed the
case, how do I:

1) Set it up so that it will accept mail from an e-mail client running on my
Windows machine and send it (this is on a direct Internet connection);

2) Set it up so that it will accept mail from only my Windows machine or
from within my Linux machine (filtered by IP address);

3) What (hopefully starting at the basics and assuming little prior
knowledge) books would you recommend for learning more about Sendmail?

hopefully in a nice, friendly .rpm package, that will?
