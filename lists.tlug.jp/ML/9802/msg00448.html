I could be on the wrong track here.   But AFAIK ASCII is a 7bit char code std
and only cover codes 0 - 127.   Once you go above that it becomes
implementation dependent.   In the DOS world there are many different code
pages you can use in the 128 - 255 range to get a variety of different chars.

I would try using a char in the 0 - 127 range first.
