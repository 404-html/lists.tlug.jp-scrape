On Thu, 12 Mar 1998, Jim Schweizer wrote:

> Hi all,
> 
> I've got another puzzle with my Sparc Linux box and I think it's
> related to the PAM thing.
> 
> Whenever I try to add a new user it says the /etc/passwd file is
> locked and to try again later. John suggested I look for a ptmp
> or ptmptmp file but find didn't find anything.
> 
> Maybe with the passwd file locked ftpd can't find the user name,
> but then why is telnet working?

This was a problem with Redhat 4.0 or 4.1, if I remember right.  It's a
problem with either adduser or passwd (forget which) which doesn't clean
up the lockfile when it's done.  To get around the problem, just remove
/etc/.pwd.lock
