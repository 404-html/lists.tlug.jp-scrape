Hi,
after a bit of a mistake with sendmail this morning, I think 
I got the tpj-jp@example.com list working.... maybe.

I've set up an account so that

  http://tlug.pht.co.jp/~tpj-jp/

is now protected by a seperate password.  So the TPJ-JP
team can take control of it as they get organized. 

I've also done a trivial hack to the archive program of majordomo
to allow web access to the archive files.  
     http://tlug.pht.co.jp/~tpj-jp/list/

The modification is only for the tpj-jp files.  I'm using the
original archive2.pl as the default. 

I'd like to run glimpse or some search engine on the list
archives in the future. As we get a organized, we'll be
able to distribute the fun among interested people.  

I feel a little guilty hogging the fun for myself.  I hope
everyone else can jump in soon.  This is quite a blast. 


Regards,
Craig

for the system admin type tasks for the server.  Personally,
I think I'm going to volunteer more time on the Server Setup Team.
Joe Marchak is leading the Perl group.  Hopefully, Chris
Sekiya or someone will get into coordinating the server setup
team.
