On Thu, 5 Mar 1998, Stephen J. Turnbull wrote:

> I've got three plausible places to install my new CD-R.  They're all
> networked and have scads of disk to make the file system to be burned
> on.
> 
> The issue is do I need speed, or can I stuff it in the not yet
> overloaded box that handles all my background stuff like mail and Web
> server and such, leaving my workstation free to work?

I wouldn't put a CDR on a server.  CDR is realtime stuff, and if too many
interrupts get generated or what have you (ie, by a server accessing the
HD), you can ruin the one-off.  I always burn my CDs on a workstation that
isn't running much of anything.
