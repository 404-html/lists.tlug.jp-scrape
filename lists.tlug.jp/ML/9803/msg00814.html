Use Juggernaut. It was published in PHRACK issue 50. Juggernaut
is a drastic cracker's tool (remember, it comes from PHRACK - I
read it regularly because it's one of the best publications
about network security - they were the first to find that "Ping
of Death" - Alan Cox read it there and removed that loophole
within hours from the Linux networking code). It is available
at:

            http://www.fc.net/phrack

Maybe tcpdump does a good job, too. It is available at

    ftp://ftp.ee.lbl.gov/tcpdump.tar.Z

Hope that helps.
