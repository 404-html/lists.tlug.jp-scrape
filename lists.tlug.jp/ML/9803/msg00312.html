Hi,
as you can see from my new signature, the TLUG server has
functioning e-mail accounts and user web pages.  There
are a heck of a lot of programs on the server and we want to
let everyone have access to it.  However, we have a limited
amount of bandwidth, processor and other resources.  Also, we
need some type of control system to prevent abuse of the
accounts.  With the web page access, this could easily get
out of control. 

I was talking to Chris Sekiya about the security issue and
we propose that we give shell access to a team of people
working on setting up the server first.  After the server
config team sets things up and sets a usage policy, we open
it up more for the general membership.

Unfortunately, since this is a public mailing list that is
archived, I'm going to disable the guest account until
we take another look at the security issues.

--------
