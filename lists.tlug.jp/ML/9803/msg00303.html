Hi Mikio,

Mikio wrote:
> I was wondering if anyone knows how to use MIDI devices with Linux.  Does
> Linux support MIDI cables, sequencers, keyboards, etc?  Software-wise, are
> there any sequencing, editing, or wave-table synthesis software available?

I'm not sure on your questions, but I have a book:

    "Linux Multimedia Guide", Jeff Tranter, O'Reilly

There is one chapter "The Linux Sound Driver" in that book, which
introduces how to configure the kernel for a SoundBlaster Pro card as
an example.  I can see MPU401 I/O settings in a configuration output
message.
