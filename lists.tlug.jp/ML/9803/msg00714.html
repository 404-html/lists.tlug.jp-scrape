In order to facilitate voting on TLUG topics, I'm attempting to
install eVote.  I'm having some problems with the permissions,
especially the clerk permissions.  Has anyone actually installed this?
I've created a test mailing list where interested people can
practice voting.  I don't have it working yet.  If you do, help me. 

There are now three active mailing lists on the server:
  o  tpj-jp  The Perl Journal Japan group, (English and Japanese)
  o  tlug-server   The TLUG Server Configuration Group (primarily English)
  o  jtlug	Japan Linux Users Group, Tokyo Linux Users Group
		Cooperation  (Japanese)

There is a closed list called tlug-exec for executive committee
discussion. 

Archives of the tlug@example.com list are available 
online.

We have tlug.gr.jp registered.  We are waiting for a resolution
of Internet access with Pacific HiTech, developers of 
TurboLinux, before moving on the registration in the DNS
entry.   

Jim Schweizer is moving the home pages from TWICS to the
new server.  

We need some type of CGI script written to keep track of
outstanding tasks.  Anyone want to write one?

We're always looking for volunteers.  Join a project and
get involved.
