Ok, I set up a computer for my boss and his family, at their apartment.
AMD K6-233, nice machine... anyway, they wanted Windows and TurboLinux on
it.  SO, I install Windows, everything goes fine but then gives me a CPU
exception error right at the end.  Strange, I think.  Then I go to install
Linux.  While installing, at random points in the install, it will
spontaneously hard-reset itself!  No warning, nothing, just ZAP, and we're
back to checking memory.

Now, I suspect bad ram here.  Windows doesn't preallocate most of the
system ram like Linux does, so Windows might never be touching the bad
second simm, I think..  Any other ideas?  (yes, the cpu fan is working
this time :) )
