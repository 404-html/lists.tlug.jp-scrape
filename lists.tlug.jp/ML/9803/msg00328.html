Ok, I've set up an account 'tlug' on ftp.pht.com (gandalf.pht.com), with
the password being the same as the root pw on the tlug server (Craig, you
might want to change this..?)

Anyway, if you log in to gandalf (you will need ssh to log in to it), go
to ~ftp/tlug and there's the tlug ftp tree, owned by the tlug user and
accessible via anon-ftp.  Note that this dir is *different* from the tlug
user's home directory - that dir is for private tlug files that you only
want tlug members to have access to, as it's under /home/tlug and not
accessible via anon-ftp.

the tlug directory is currently on a drive with 1.5GB free.
