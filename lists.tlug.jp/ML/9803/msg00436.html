> i'm a little confused by the message included below...is 1.4j a beta

> support on a separate versioning scheme?

1.2 Hokkaido is the latest USA/non-Japanese release.

1.4J is an upgrade CD which upgrades TL 1.0J to TL 1.4J.  Yes, the
Japanese and non-Japanese versions have different numbering now.  The
Japanese versions are much more Japanized... but if you don't read
Japanese/speak Japanese/need to do any of that, TL 1.2 is the latest
version that you need to worry about :)


April will see v1.9beta for both USA and Japanese, and due to our improved
localization techniques, I think the 2.0USA and 2.0J will be released
around the same time, this time, and will be concurrent as far as version
numbers go.  Used to be that I'd make the USA ver and then we'd spend a
couple months japanizing tools and stuff.  Now that I"m over here in
Japan, we can do it concurrently.
