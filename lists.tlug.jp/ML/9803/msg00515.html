On Fri, 20 Mar 1998, Tim Meggs wrote:

> Is there any software for Newton <-> Linux connectivity avaiable?  
> (As there is for the PPilot)

There's a bit.  I use the NIE to connect the Newt (it's a MP130) to my
home LAN via PPP.  I use SimpleMail to grab my email via POP3.  InkSpot is
my offline newsreader of choice (except it isn't exactly stable -- I think
it has to do with my 10Mb flash card, though).  Package transfer is via
NewtFTP.  Entertainment via YAZI.

I haven't used Slurpee to move notes back and forth simply because I
haven't needed to.

Out of the programs I've listed above, only InkSpot and NewtFTP are
non-free (they're both USD$25).  Well worth the price.

( ... now if I can just find a third-party supplier of replacement
screens, I'll be set with my Newt until after the end of the millenium ...
)
