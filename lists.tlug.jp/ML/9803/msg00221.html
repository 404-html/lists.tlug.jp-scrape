   Yuk! Not one, but two revisions behind. I don't know whether it'll
   cure your PAM problems Jim, but you really should upgrade to Red Hat
   4.2. There's a particularly nasty bug with the 2.0.18 kernel which
   wouldn't have affected you in standalone mode, but will cause the
   system to panic every so often now that you're connected to a 
   network with other systems.

   There's also an "endian" problem with the dump program which shipped
   with the 4.0 release, so if you try backing up to tape you'll suddenly
   find yourself in the land of negative inode numbers. An amusing little
   party trick, but most people would prefer the dump.
