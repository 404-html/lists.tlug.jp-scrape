Hi all,

CGIwrap is now installed and working!

under your public_html directory make a cgi-bin directory and
put your scripts there. To run the script call it as

http://tlug.pht.co.jp/cgi-bin/cgiwrap/user/script.pl

and CGIwrap will run the script with your permissions - not the
server's. This is a *safety* precaution - not a *trust* issue.
There is also a dubugging mode so if you're having a problem
with a script, call it as

http://tlug.pht.co.jp/cgi-bin/cgiwrapd/user/script.pl

I've found this to be one of the best features of CGIwrap.

Thanks all,

Jim S.
