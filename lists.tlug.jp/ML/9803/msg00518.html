Well, we got perl/Tk working with Japanese (even Japanese input), so
TurboLinux 1.9beta/2.0 (J version) will have fully Japanized configuration
tools under X.  One more step towards improving J support in Linux, I
suppose.  If you have jperl and a japanized Tk, you can go to
ftp://mirror.vector.co.jp/mirror/turbolinuxj/updates/1.4 and get the new
perl-Tk Japanese RPM and install it.  You'll be able to get a new version
of pkgupdate there soon as well, which will work with Japanese.
