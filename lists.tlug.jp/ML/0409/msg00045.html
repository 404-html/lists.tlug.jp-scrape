hi.

i would not change the locale on your system.

rather, tell smbmount the charset stuff.

read "man smbmount",

search for "codepage" and "iocharset".

as i understand, iocharset should be set to YOUR encoding,
and codepage to the windows SERVER encoding.

so something like "smbmount //server/share /mnt/sharepoint -o
codepage=shift_jis,iocharset=euc-jp".
or something like that.
i am not sure if those encodings are named exactly like i wrote it (i
used "iconv" charset names), but the idea should work.
