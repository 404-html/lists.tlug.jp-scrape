I have a G550 hooked up to two monitors and am very pleased with the
performance. But for some reason, the HAL drivers are not in the stock
xfree tree (I don't know about the x.org or freedesktop servers), so
whenever a new X version comes out (or a security issue comes up) you
have to patch the source code (replace the mga directory with code from
the Matrox web site, and edit a config file) and rebuild. It's rather a
pain. (Binary distributions may do this for you -- I don't know.)
