Hello,

Trying to access the mini-sd card in my A5403CA keitai via usb, I get a 
mount: /dev/sdc1 is not a valid block device error message.

...and dmesg shows...

usb.c: USB disconnect on device 02:08.1-1 address 3
hub.c: new USB device 02:08.1-1, assigned address 4
usb.c: USB device 4 (vend/prod 0x7cf/0x7001) is not claimed by any active 
driver.

Which I assume means either:

1)the hotplug system doesn't know what driver (module) the
device needs/uses.

2) it's not supported as a usb-mass-storage device under linux.

So my questions are:

A)  Is the driver module for sd cards different than that for flash 
cards.  My flash storage mounts fine so if they use the same module I 
would have to think the problem is B (which I think it is).

B)  If the sd card in the phone is not a supported mass storage device, 
can anyone recommend a card reader for either minisd cards or sd cards 
(which I can use the included adaptor with)?
