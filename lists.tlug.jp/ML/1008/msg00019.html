What I wrote was certainly not cheery, but should not be construed 
as requiring you to give up on using Windows within a virtual 
machine within Linux. 

What my post suggests is that it can be _expensive_ to acquire 
the version of Windows that is legal to run in a virtual machine. 

Maybe the version of Windows included with the machine you buy 
is sufficient to run in a virtual machine; maybe not. 
Maybe you'll have to buy a separate copy of a more expensive 
version. 

The devil is in the details. Read the license(s).
