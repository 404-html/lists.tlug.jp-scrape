which says

"The Computing/Telecomms (COMPDIC) file, which contains terms used in the computing and (tele)communications industries. (Full documentation.) In June 2008 the entries in this file were combined with the full JMdict/EDICT file, and the terms or senses specific to information technology and telecommunications were tagged with "{comp}". The file is now simply a subset of the main EDICT file."

So the computing/telecomms entries have been merged into the EDICT :
