One thing with Ubuntu's latest and greatest--the latest Edgy release
doesn't support many Matrox cards (and possibly others).  

It's poor advocacy to say "Watch this," put it in the machine and get
nothing but a dark screen, and perhaps a message, sync out of range. 

It's a known bug, with a fix available, but at present, one would have
to install with the special install CD or whatever they call it, in text
mode, then get the package which, so far, I've only see available from a
link on their bug reports.  

Because of this, many Ubuntu based distros will also not work.  It had
to a do with a bug in the Matrox drivers in X-7.1.something, which has
since been fixed.   Anything still running X 6.9 will work too.

I think ATI cards were another problem, but I don't really remember, I
just remember noticing, while looking for solutions to the Matrox
issues, that there were a few other cards with issues.
