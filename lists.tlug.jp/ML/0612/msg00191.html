On 08/12/06, scott <scott@example.com> wrote:

Anybody know if it is possible to add this "open with" option to
Firefox? Would be handy...


Yeah, it's possible. I cannot find anything handy in about:config at a
quick glance, but it is probably doable there.

Take a look at Edit -> Preferences -> Downloads -> View & Edit
Actions. See the default actions? If you remove one, you will be
prompted for files of that type. Grep your prefs.js file in your
profile directory and see if you can figure out how that stuff is
controlled. Try commenting one out, and seeing what Firefox does.

Since Firefox already has the functionality, it is just a matter of
telling it what you want.

It is also possible that someone has written an extension to do just
this. Have you Googled?

-Josh
