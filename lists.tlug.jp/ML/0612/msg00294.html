Good. This likely means you nic has been detected and the correct
modules have been inserted.

> Do you know how to configure networking under Linux?

no. 

On the command line run the following command. It should let you know
if eth0 is available:

ifconfig eth0

If all looks well then you should be able to use the gui to configure
your network settings.
