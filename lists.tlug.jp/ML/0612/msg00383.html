6.10 is Edgy, the one where they upgraded Xorg and it caused problems
with various video cards.  This will be fixed (and may already be fixed
if one upgrades the system, but the install CD won't work with a few
video cards.)

Also a quick note--on lists like this, it's considered good practice to
trim unnecessary stuff.  For example, your post basically answers my
first point, then you left in all the other things I wrote.  While ~I~
might enjoy reading my words again, most of the list doesn't want to do
so.  :)

It's better to try to trim out unnecessary things when answering a post.
(For example, note that I snipped out all the rest of the email.)

The only reason I mention this publicly is because it's a VERY common
thing with people, especially those new to *nix like operating system
tech lists, so hopefully, other newcomers will see it and learn.  

(If you're not a newcomer, and simply neglected to snip because you were
in a hurry, well, what the heck, they'll learn anyway.)  :)
