The dwm window manager is pretty simple, but it's actually a C
configuration file--you then recompile it (which takes about 3 seconds, it
is a tiny program--that 3 seconds is literal, by the way, it's really only
a few seconds for compilation.  I have a page on it at http://srobb.net/dwm
which has a link to a better page on the Debian forums.

Openbox is another nice, pretty lightweight manager. It uses xml files. I
have a page that covers it slightly, at http://srobb.net/fluxopen.html, a
page that is more aimed at fluxbox users.
