Of course it does:

Corollary #1.1 of the Internet: if it works on site A, it very likely
works on site B too!

The main Proposition for deriving the Corollary is axiomatic:

Rule #1 of the Internet: rough consensus and running code.

The proof of the corollary relies on

Lemma #1 of Software: if you know what it does, it's usually easy to
get code running that does the same.

Of course then you run into

Theorem #1 of Software: Not all programs run at the same speed, and
the average is always slower than the maximum.

and

Stylized Fact #1 of Software: Your program will run slower, usually
much slower, than average until you install a better algorithm.
