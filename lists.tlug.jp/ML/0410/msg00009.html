Hi,

Le 04/10/04 17:32, Ahmed Sarwar ecrivait :
> I removed zope by "apt-get remove zope"
> and I deleted "/etc/init.d/zope". After that I
> reinstalled zope but the problem is, I can't 
> find "/etc/init.d/zope". Anyone knows how
> to start zope now please. Thanks in advance.

Maybe you should try:
  apt-get remove --purge zope
then reinstall it.

If not, you can simply extract from the .deb the script file zope with a
command like:
  dpkg-deb -x package.deb directory/

The .deb should be there:
  /var/cache/apt/archives
If not, there:
  http://packages.debian.org/stable/web/zope

Good luck,
