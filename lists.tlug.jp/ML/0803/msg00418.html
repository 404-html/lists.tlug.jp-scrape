emiddleton@example.com writes:

 > So it requires remote root login?

I'm not sure what you are asking.  Of course you need to have a root
login to log in as root.  But AFAIK all Unix systems have a root
login; it's just that no password is set, so logins are effectively
disabled.

If you *have* a root key-pair for the target system (which need not be
different from your local system), and you have the private key
available, it will use it (either via agent or ssh -i).

TRAMP can also use sudo and maybe super, but those were sort of bolted
on later.
