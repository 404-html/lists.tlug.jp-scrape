A good tool for gathering system performance numbers (among other things) is Munin (blatant plug, the company I work for did the base development).

Munin is a GPL'ed, plugin based system for measuring almost anything, and somewhat resembling mrtg.

We use it to gather data for several hundred computers we manage for various customers, and I find it really useful.

Standard plugins include vmstat, iostat, cpu, memory, network and things like counting processes, email queues. In addition there are plugins for things like JVM heapspace usage, temperature readings, spam/ham distribution, top-ten-virii etc.

The project web site is http://munin.projects.linpro.no/ and a live demo is available at http://munin.ping.uio.no/
