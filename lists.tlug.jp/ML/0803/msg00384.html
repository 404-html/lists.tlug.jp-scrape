ASUS has announced that they will begin selling an Eee PC that comes preinstalled with Windows XP instead of the default Xandros based Linux distribution. A Windows XP based Eee PC currently sells in Japan, but not in the United States.According to this Reuters article, the Windows based Eee PC will cost (or start at?) approximately $400. ASUS expects that the Windows based devices will make up about 60% of sales as well.
