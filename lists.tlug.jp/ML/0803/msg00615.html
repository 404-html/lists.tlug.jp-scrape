info elisp

I have found the info pages very useful. It takes some time to learn where to
find stuff but in my mind this is time well spent.
My basic strategy is always to go to the reference documentation, the learning
curve may be steep but not vertical. If this does not work, google for howtos,
introduction, examples etc.
For style read code. Style may vary a lot and there are many good ones. Just
make sure to pick a style that is supported by the emacs lisp-mode.
