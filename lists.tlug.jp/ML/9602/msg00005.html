There is a dirent bug in make-3.74 which forces the use of the -f option
in order to find the makefile.

Exp.:

  # make

will not fine the makefile

  # make -f Makefile

will find the makefile. 

The fix involves modifying dir.c and glob/glob.c

Elf binary of the patched file can be found in file:

	make-3.74-direntfix-elf.tgz

I think in  /pub/linux/devel/make  ????   something like that.
