I just installed a 350 meg hard disk on my Linux box so I could
get room to install Windows '95 programs.. (itai!).  I mainly
wanted to install Netscape 2.0 and test out the JDK for Windows '95.
I know JWT is using this.  As far as I can tell the Windows '95 
JDK 1.0 is not an integrated development environment.  I don't
have the docs, so this is why I'm bringing this up for discussion.

I really don't see much advantage of developing applets under Windows '95..
Or, am I missing something here?  The only thing is that I can run
applications under Windows '95 and have not been able to under Linux
yet.  Though, this is only a matter of time.  

Since the next meeting is about Java, I'm trying to get a handle on
what all the hype is about.

The bottom line is that I want to use Linux for everything because
I can get work done faster under Linux, but I
don't want to waste time if the Java development tools are going
to be much easier to use under Windows '95 and thus, I would
be willing to use Windows '95 if it would save me time.  

To think about it, this whole Java thing is purely for recreation for
me and it sure is a lot more fun to use Linux than to use Win '95.
Might as well stick to Linux.  :-)

Regards,
Craig
