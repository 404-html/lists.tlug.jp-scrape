Get the Linux Java HowTo from the usual sites.

1.2.13 should work.  You need to be using an ELF system.  If you don't
know what ELF is, you might want to check the ELF HowTo.  If you just
bought the cd it should be ELF.

Confirm that you have the following:

  o  libc.so.5 => /lib/libc.so.5.2.18

  o  libX11.so.6 => /usr/X11/lib/libX11.so.6.0

  o  libXt.so.6 => /usr/X11/lib/libXt.so.6.0

  o  libXext.so.6 => /usr/X11/lib/libXext.so.6.0

  o  libXpm.so.4 => /usr/X11/lib/libXpm.so.4.3

  o  libdl.so.1 => /lib/libdl.so.1.7.9

After installation you want to set the links and stuff with

  # ldconfig     (as root)

or a 
 
  # ldconfig -n /lib


Make sure you get Linux JDK 1.0.  It has several improvements over
the Beta distributions which were not apparent at first.  Basically
the Betas will waste your time because there are some bugs in it.

The 1.0 works a lot more smoothly than the Beta versions.
