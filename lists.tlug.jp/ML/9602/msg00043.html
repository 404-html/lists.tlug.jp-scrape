You don't need to run kon to run X.  

For the fonts, the first thing is to confirm that you have the fonts
installed.  Use xlsfonts

 # xlsfonts |grep jis

Here is what I have:

Also, you might want to run mkfontdir again to make sure that the
fonts.dir file in each font directory is updated.  The X and
font servers use the fonts.dir file to find the fonts. 

 # mfontdir /usr/X11R6/lib/fonts/misc
