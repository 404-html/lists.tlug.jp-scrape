Now as a webmail and IMAP user I miss MH. It has separate shell commands for different functions like inc to incorporate new mail, comp to compose, rmm to delete and so forth. xmh was one of several gui wrappers. NMH does MIME. You can use a nifty built-in line-oriented editor similar to ed to send tweet-like messages or set your favorite one as default. Emacs client works well. You can reedit mail from inside comp or repl in a different editor with a simple command.

See http://rand-mh.sourceforge.net

Includes a link to nmh, a fork. The RAND corporation was where it was made.
