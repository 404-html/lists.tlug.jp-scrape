Lyle, what happens when you boot Kubuntu live-cd and try to mount the
Linux partition? It might also help to try to run fsck being booted to
Kubuntu and see if it can fix something.

I guess that the hard drive itself is OK since you can boot Windows, so
it's probably Linux partition problem, like unclean shutdown. And you
can also try to access the Linux partition from Windows, if it is ext2
or ext3:

    http://www.fs-driver.org/ - I know this one works, but I'm not sure
how it can handle ext3 that was not cleanly dismounted
    http://www.diskinternals.com/linux-reader/ - this one looks very
promising, but I did not use it
