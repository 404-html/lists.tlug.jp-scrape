Edward Davies wrote:
> Hi All,
> 
> Is anyone out there using KDE?  I've got KDE up and runnning
> under RedHat 7.2.  When I set the language option to Japanese,
> all menus and everything are in Japanese, but I can't figure
> out to enter Japanese from the keyboard.  

Urk. Have you checked out the archives (are they working today ;)? This 
is an oft asked and answered question on this list.

The answer boils down to this: input in X is application-specific, not 
global to a Window Manager.

E.g. if I want to be able to input Japanese in Mozilla Mail, I do 
something like:

kinput2 -canna &
LANG=ja_JP XMODIFIERS=@example.com=kinput2 mozilla -mail &

Ya dig?
