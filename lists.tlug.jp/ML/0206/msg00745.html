wrote to tlug@example.com:

> Now, say you wanted to install FreeBSD, which requires a primary. You
> still have room for two primaries

'fraid not.

The main partition table only has room for 4 partitions in all. These
partitions can either be primary, or they can be extended, in which case the
bootsector of the partition contains its own partition table which in turn
can only contain 4 partitions etc. etc.

If you take a primary for 98, a primary free for XP and an extended with a
FAT32 inside it, that's 3 partitions gone. You only have room for one more
in the main partition table.
