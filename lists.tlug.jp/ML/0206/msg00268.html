For KDE applications, especially KDE3, there is a common problem 
with  QT's inputstyle.  By default, it is set to "onthespot", 
which doesn't work with kinput2.  To fix this, run qtconfig and 
set to "overthespot" in Interface.

If KDE2.*, this shouldn't be a problem.
