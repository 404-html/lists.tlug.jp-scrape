LC_ALL will set the menus to Japanese *if* there are Japanese menus
installed. For example if you try setting LC_ALL with abiword you get
Japanese menus, however with Mozilla, unless you install the extra menus,
you get English menus.

You can also note with Mozilla, that the file chooser shows dates with
kanji. It just depends on what you like however, I prefer to set LC_ALL
because then there is not intermix of locales and no possible confusion,
for instance I usually live in:

LANG=en_GB.UTF-8
LC_ALL=en_GB.UTF-8

I am not terribly bothered about menus and dates in Japanese (after all
it's good practice ;-)  and I usually remember where specific menu options
are anyway.
