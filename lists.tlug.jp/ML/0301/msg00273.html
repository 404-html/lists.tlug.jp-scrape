Actually the sylpheed ebuild uses "nls" rather than "cjk", there isn't
any magic needed other than nls support (and libjconv if you want to
convert between encodings - but sylpheed finds this automagically id it
is installed)

For those Gentoo users interested in what USE flags are available for an
ebuild, I highly recommend etcat (avaialble from 
http://www.liquidx.net/projects/etcat/). It's syill in early deveoplment
but you can do stuff like this

stu $ etcat uses sylpheed
[ Found these USE variables in : sylpheed-0.8.9 ]
[ Colour Code : set unset]
+ crypt
- gnome
- ipv6
- ldap
+ nls
- pda
+ ssl
+ xface
stu $

It even tells you what flags you have currently set in make.conf
