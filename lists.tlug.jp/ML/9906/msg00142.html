Linux trivia: older versions of "file" didn't used to do this for linux
(err ... I should say "older versions of /usr/share/magic").  Someone
once posted a 2 cent tip to linuxgazette that the linux "size" command
also prints out the command name when run on a core file:

    [rex@example.com rex]$ file core
    core: ELF 32-bit LSB core file of 'wmmail' (signal 11), Intel 80386, version 1
    [rex@example.com rex]$ size core
       text    data     bss     dec     hex filename
      81920  184444       0  266364   4107c core (core file invoked as wmmail)
