On Fri, 11 Jun 1999, Jim Schweizer wrote:

> Does anyone know how to configure the remaining function keys
> (F8, F9 etc.) for virtual consoles?

Configure via /etc/inittab.  Add lines like:

x:12345:respawn:/sbin/agetty 38400 ttyx

... where 'x' is the appropriate VC.  Once you've got it rigged, kill -HUP 1.


> Also, does F7 have to remain dedicated to X

Doesn't work like that -- X will grab the first available VC.  If you've
got getty running on VCs 1 through 4, then F5 will take you to X.
