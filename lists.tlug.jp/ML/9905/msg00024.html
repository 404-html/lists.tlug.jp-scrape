I am able to unzip files with no problems, but that is if they are .zip
extensions.  What unzip is telling you is that the file you want to unzip is
not .zip extension.  My first attempt would be to copy the file to hp990509.zip
then try to unzip.  If that doesn't work (I would think it should) then you
need your friend to resend as a standard .zip file not an auto extractable exe.
Hope this solves your problem.
