It sounds like your BIOS is not set to use LBA mode with that disk.  Try
turning that on and see if the problem clears up.  What's basically
happening here is that LILO needs to be in the first 1024 cyclinders (I
*do* hope that will some day be fixed), and if you do install LILO under
that setup it will probably hang on boot.
