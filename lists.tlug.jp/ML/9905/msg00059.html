The latest issue of Computing Japan features interviews with TurboLinux and
RedHat regarding who will get Japan. What I want to know is, must there be
just one?

As far as I have studied, if it's Linux, you can use the same software and
data, so it seems that it would be just a matter of preference. Are there
concrete incompatibilities between distributions? I know that different
distr's put certain system files in different directories, but isn't the
bottom line that, if I am a trained and competent Systems Admin or
programmer for Red Hat, I can get work in a company that uses Turbo, no
problems? Please fill me in on this, since I am trying to learn Linux for
future job prospects. If I have to learn three or four distributions, I
will. If I can learn one really really well and consider myself informed,
then I will do that.
