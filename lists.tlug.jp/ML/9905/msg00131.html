On 25-May-99 Tetsuji Rai wrote:
> Hi,
>    Somehow my Linux goes into runlevel 5 after I upgraded Linux to RH6.0.
> And somehow it runs into gnome mode which I don't like.   How can I set
> the initial runlevel to 3?   When I used RH5.2 default runlevel was 3.
> Thanks in advance....

The initial runlevel is set in the /etc/inittab file. You want a line like

id:3:initdefault:

instead of 

id:5:initdefault:

man inittab for more info.
