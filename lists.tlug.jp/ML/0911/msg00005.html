You might be interested in Clojure, it's a functional lisp that runs on the JVM and would cover functional programming as well as macros, DSLs, etc. Clojure and Scala are the big two new languages in the Java community now. The value of running on the JVM really can't be understated, since it offers access to production quality libraries for just about everything.

A good book would be,
http://www.amazon.co.jp/Programming-Clojure-Pragmatic-Programmers-Halloway/dp/1934356336/ref=sr_1_1?ie=UTF8&s=english-books&qid=1257078893&sr=8-1
