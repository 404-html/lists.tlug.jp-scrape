been available, it's just whether you had it turned on or not.

> I think if you right-click on the icon on the Panel, select SCIM Setup

And just to be geekily thorough, if there are any KDE/skim users out
there (yes, "skim" is some kinda KDE-specific wrapper for scim that I've
never entirely understood the need for, even though I use it :-), you
can effect the same toolbar removal by going into the skim config-
uration, to Panel > Skim Plugins, and uncheck "Main Toolbar".

Personally I've always liked seeing the toolbar :-)  Although it is a
bit problematic to find a place to put it on the screen so that it
doesn't obscure something else...
