Hi,
I'm playing around with mailserve.  This is a perl CGI script
that puts a web interface around the basic majordomo commands.
I'm not sure how useful it would be, so I have it linked to
the author's server in the U.S. now.  The people on the TLUG
mailing list have a pretty high level of technical knowledge,
so the majordomo commands probably wouldn't scare someone.

However, it is something to play with for a few seconds:

	http://www.twics.com/~tlug/list.html

Click on the TLUG mailing list web button at the bottom of the
page.  Actually, it would be pretty simple to write a Perl
script that outputted stuff to a web page instead of sending
it by e-mail....
