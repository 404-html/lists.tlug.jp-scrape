I finally got around to converting to mh-e, (an interface to MH for
Gnu Emacs), and I'm pretty happy with it so far.  It was pretty easy.
It's rather RMail-ish in its interface, although not enough to avoid a 
few typing mistakes.  The customization interface is different, too.

The big advantage is much better MIME support than RMail has.  I also
like the message-per-file format of mail folders, which is useful with
glimpse.  Also, it's occasionally useful to be able to get at the mail 
system from the command line without going into emacs.

If you're looking for an Emacs-based mail user agent with decent MIME
support (it's not hugely excellent), this may be for you.
