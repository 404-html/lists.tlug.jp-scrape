>>>>> "leo" == pal  <pal@example.com> writes:

    leo> Hi All Received Some Data for Family Tree

    leo> Wondering whether there is Linux Software for this ?

Don't know, but there is a huge collection of genealogy software for
DOS at Simtel, some of it written in C.  It's a natural for a database 
application, but you should be sure that the database permits "BLOBs"
(binary large objects) like images, movies, and audio, as you will
surely want to add these.  Although, actually, if you're going to do
it as a Web-style structure, you could get away with a text-only
database with file pointers for the BLOBs.

    leo> Was Thinking of HTML otherwise

??? HTML is a terrible format for a database (hard to keep conssitent
and maintain), although it's great for presentations (including by
Web).  It shouldn't be too hard to write some CGI scripts (in Perl,
say) that look up the database and output a page.

    leo> Any ideas??

I strongly recommend doing a Web search and a check at Simtel
(ftp://oak.oakland.edu/ and many mirrors).
