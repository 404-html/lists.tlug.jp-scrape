There are some cabocha repositories on Docker Hub:

https://hub.docker.com/search/?isAutomated=0&q=cabocha

If you do not want to use Docker, the Dockerfiles for those repositories
might be useful in determining how to get it to compile for you.  Note
that this is a good use case for Docker, though: running "sudo make
install" outside of a container would pollute your system.
