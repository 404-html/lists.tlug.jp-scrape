The error messages are there because the "buggy" ifconfig (with support for
ipx and appletalk) triggers auto-loading of the appletalk and ipx modules,
which don't seem to be there. Put

alias net-pf-4 off
alias net-pf-5 off

into /etc/conf.modules.

I don't think it's the general consensus that this is an ifconfig bug,
because the net-tools maintainer, phil blundell, is quite attentive and
reads linux-kernel (I recently found a bug in arp).

-Klaus
