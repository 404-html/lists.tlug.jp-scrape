I'm not sure if it is or not, however, the other day I found a utility
program to verify mail aliases called 'vrfy'. You give it an alias and
optionally a mail server, and it tries to verify the alias:

	vrfy casillas yoshiko
	<casillas@example.com>

	vrfy garbage yoshiko
	garbage... User unknown
	User unknown

I found this at:

http://www.harker.com/sendmail/sendmail-ref-body.html#Sendmail_Tools

If you run it in verbose mode, it will show you the complete
interaction with the mail server.
