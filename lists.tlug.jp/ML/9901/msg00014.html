The things that computer makers are doing these days are really just
driving me crazy.  I am talking about the "recovery disk" that are now
shipping with new computers.  I bought a Sharp Mebius PC-FJ10 and I
am trying to install TurboLinux 3.0J on it.  The problem is this.
It seems I can not use FIPS with Windows 98.  I searched various
places and that seems to be a common problem.  So I just went ahead
and installed TurboLinux and destroyed the old Windows 98.  I
created a partition during this install to use for Windows 98, but
with the the "Recovery Disk" system, I can only put the system back
to it's original state (with one big partition for Windows).  Anyone
have any ideas about how to get around this other than:

1.  Buying commerial disk partioning software
2.  Buying a real copy of Windows 98
3.  Install only Linux and forget about Windows (although I would like to
    this world still requires it)
