Scott Stone wrote:
> OK, I may have found a bug in the soundblaster driver...?  Anyway:
> 
> 1.  I'm using realplayer to listen to a live radio feed.  Working great.
> 2.  I open a new page in netscape.  Page opens.
> 
> BUT
> 
> realplayer now gives a 'general error #1' message, and on the console:
> ****audio: write error: 588 bytes errno: 11
> 
> according to <asm/errno.h>, 11 == EAGAIN "Try again"
> 
> What's going on here?  this same procedure does NOT produce this error
> under kernel 2.0.36.

A kernel interface changed in the 2.1.x kernels and rvplayer has not been
updated. Try going to http://www.i2k.com/~jeffd/rpopen/ and following the
direction there.

(Of if you happen to use debian, get the rvplayer_5.0-7.deb package from the
frozen distribution, which will also fix it.)
