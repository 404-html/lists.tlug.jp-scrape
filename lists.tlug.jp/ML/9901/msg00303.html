On Wed, Jan 20, 1999 at 11:30:27AM -0700, Scott Stone wrote:
> cdrecord claims to support CDRW now.  I've never seen anything for Linux
> that can use IDE CD writers, though - is there such a thing?

You can use the "SCSI emulation support" for IDE that is in recent kernels.
This mimics a SCSI host adapter, allowing you to use SCSI device drivers...
handy when there is no native driver.
