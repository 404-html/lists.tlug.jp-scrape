> When I try to access some of my CGI-scirpts I get an internal
> server error message: "Premature end of script headers"

That error usually means that the permissions are wrong on one
of the script files. Try running the script through cgiwrap with
the debigging flag on. That will tell you exactly where the
script is exiting.

cgiwrap http://www.umr.edu/~cgiwrap/
