> I've never managed to get Thunderbird to obey a geometry command, and
> never found any alternative.  :-(

If you use Compiz,

* open "CompizConfig Settings Manager" (or ccsm)
* category "Window Management"
* activate the "Place Windows" plugin
* open the "Place Windows" plugin
* tab "Fixed Window Placement"
* add an entry in "Windows with fixed positions"
  class=Icedove (or whatever) plus the X and Y position

That's it. Thunderbird will always open at the same place. Note that
you can also choose the placement mode (Cascade, Centered, Smart, ...)
and the viewport (the virtual desktop).
