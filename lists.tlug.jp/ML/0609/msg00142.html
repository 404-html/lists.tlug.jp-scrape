Dave M G wrote:
I'm a little confused right now about how caching works. Is it possible for a web site, by JavaScript or some other method to stop itself from being written to a cache?
The following HTML tags instruct a browser not to cache a page or to expire it immediately. No guarantees the browser will honor it but if they are present on the page you are viewing it might explain it. 

<meta http-equiv="Pragma" content="no-cache"/>

<meta http-equiv="expires" content="0"/>
