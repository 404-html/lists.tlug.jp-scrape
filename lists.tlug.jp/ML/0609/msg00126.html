I thought it worth mentioning that after some more hunting, I've discovered that if I type "about:cache" in Firefox's address field, I can see some information about the cache.
However, it's not exactly what I was hoping for. This method of looking at the cache is more like looking at the browser's history, not the data it's cached on disk.
In other words, it provides links and information about each page and image and whatnot visited, and stored in cache. But you can't access any of the data directly. Instead, if you try to see the data itself, it will take out to the source on the web.
What I'm looking for is a way to scan the contents of the local copy of the data.
