On Wed, Apr 19, 2000 at 10:55:02AM -0600
Marc E. Christensen wrote...

> I wm using x-chat and wondered if it supports Japanese input and char
> sets.  Anyone know if something like that is a restriction of the
> server, client or irc protocol?

Just like with e-mail, japanese IRC messages are sent/received in ISO-2022-JP (JIS)
code. Since what we use normally is EUC, the client must support some kind of
code conversion. There is a patched version of x-chat in Kondara but you'd need
thier glibc locales for it to work.
