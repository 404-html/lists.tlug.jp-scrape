Yasuaki Kudo さん曰く:

> Hi there,
> Can I run an X server on Linux without a video card?
> 
> I want to setup a Linux server without the video card ( no $$$ ) but still
> want to run an X server on it.  It would be great if I could use VNC to
> access it from my NT machine!  I checked out X servers that run on NT but
> they were not free.

Actually, VNC itself is a X server that runs on the Linux machine and
redirects all it's output to a VNC client. 
It's all you need.

# There's also something called xvfb, which is an X server that runs
# totally in memory without ANY graphical output, but that's certainly
# not what you are looking for.
