Hector Akamine (lists.tlug):
>I'm trying to use the times postscript fonts including the following
>line in latex (under RH 6.1J)
>\usepackage{times}

Should do the trick.

>The problem is that when I run latex, I get errors like the following
>one:
>
>LaTeX Font Warning: Font shape `OT1/ptm/m/n' undefined
>(Font)              using `OT1/cmr/m/n' instead on input line 33.
>
>LaTeX Font Warning: Font shape `OT1/ptm/m/it' undefined
>(Font)              using `OT1/ptm/m/n' instead on input line 6.

If most of your output is correctly coming out in Times, then it's
merely complaining that it can't find some characters in the Times font
because they don't exist.
There's nothing you can do about that.

If this error happens huge numbers of times and the actual text is in
Computer Modern, you have another problem. 

I suspect from this that it's the former, because LaTeX is very
obviously finding the Times font.

>I downloaded the psnfss directory from the latex archive (ctan.org), copied
>it under
>latex/packages and run "make" to install it,  but the problem continues.
>I think that latex does not "know" that I installed the new package.
>How do I "tell" latex that I did that?

The usual answer to this question is "texhash or mktexlsr" but not in
this case. You haven't needed to install a new file; if you had needed
to, LaTeX would have previously complained about not being able to find
a file. LaTeX 2e comes with psnfss as standard, IIRC.
