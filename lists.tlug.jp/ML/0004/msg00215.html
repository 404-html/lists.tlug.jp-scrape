So it looks like the AT1700 card may be good for something besides a
doorstop after all. I got mail from Kinichi Kitano saying that there is a
patch available for the AT1700 driver that is only circulating in Japan--it
somehow never made it into the main kernel tree. Here is the header he sent
me:

/* at1700.c: A network device driver for the AT1700.

	Original:
		at1700.c (1993-94 by Donald Becker) and
		fmv18x.c (1994 by Yutaka TAMIYA).

	Modified by Hiroaki Nagoya (nagoya@example.com)

	This software may be used and distributed according to the terms
	of the GNU Public License, incorporated herein by reference.

	This is a device driver for the AT1700/RE2000, which
	is a straight-forward Fujitsu MB86965 implementation.

	Sources:
		at1700.c
		fmv18x.c
*/

static const char *version = "$Id: at1700.c,v 0.1 1997/03/25 03:56:16 nagoya Exp nagoya $\n";

I don't know where you can get the rest, but you can email Mr. Kitano
at Kin-ichi_Kitano @ yokokawa.co.jp and ask.
