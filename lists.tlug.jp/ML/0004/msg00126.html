The scsi id is probably set
by some jumper on the disk, so need to open the case to know the id.
Wait a minute, that reminds me, you should get all that info 
at the boot time. After the bios messages and before the lilo
prompt you will get scsi messages listing the adapter and connected
devices. You can also probe the bus using the adaptec scsiselect
utility by pressing ctrl-A at boot time. There is a menu item named
disk utility or some thing like that to scan the bus and show all
connected devices (make/vendor) and their ids. 
</selva>
