On Thu, Apr 06, 2000 at 01:31:35PM +0900, Selva Nair wrote:

> You need to use  "ide0=dc4030" as kernel boot parameter. For example,
> boot as 
> 
> linux ide0=dc4030

Not for a Promise Ultra66 you don't.  That parameter is for the old ISA
Promise card.  The Ultra66 uses a different driver ... one that isn't available
in the main-line 2.2 kernels (though I did hack a patch together that allowed
it to work with the generic PCI IDE driver).  2.3.x has the proper driver.
