I used the rpm from Adobe's site.  


You have to install both nspluginwrappers, for i386 and x86_64.
After that, run mozilla-plugin-config -i -v

<minor rant>  In FreeBSD, one installs nspluginwrapper and gets a
pkg-message telling you what to do afterwards.  In RH based systems, one
installs it, and finds no documentation about how to use it. The README
says run nspluginwrapper, but of course, in CentOS/Fedora, there's no
such command.  

Actually, in Fedora 10, you don't have to do anything but install the
two (x86, x86_64) nspluginwrappers, the postinstall script takes care of
the rest. </minor rant>

I don't remember doing anything else.  I'm running the 64 bit version of
firefox. 

It also works with opera.
