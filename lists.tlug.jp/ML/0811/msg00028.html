Ty, Curt,

Thank you for responding.

Ty said:
Hm.. Make sure that there are php5.conf and php5.load in /etc/apache2/mods-available/ and
those files are symbolic linked in /etc/apache2/mods-enabled/ just like other modules in that directory.
If they are not linked, you just can make those links yourself. After that, restart apache.

This seems to have done the trick! Thanks for the clear and concise instructions.

Kind of strange that this manual step was needed this time round, but hey... in the grand scheme of things it wasn't as bad as other computer issues I've faced.

On the upside, that nagging MySQL sock file problem that I had before, where I had to keep creating a symlink every time I booted, seems to have been resolved in the new Ubuntu Intrepid. So there's some silver lining.
