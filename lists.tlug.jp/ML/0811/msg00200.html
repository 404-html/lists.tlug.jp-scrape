Well, there are mozilla/plugins all over the place, and it seems easy
enough to guess wrong.  :)

Did I mention that one doesn't have to remove nspluginwrapper,
especially if using adobe reader as well?  (removing nspluginwrapper can
also cause problems with SELinux if you use it.)

Instead, one should just add it to the IGNORE_WRAP line in
/etc/sysconfig/nspluginwrapper.  After the = sign, add this to the end
of that line (but within the quotes.)

:libflash*

If you have nothing in that line at present, then don't use the colon
before libflash, but by default, it seems to at least put libtotem in
there.  (I'm guessing that's the default since I never specifically
chose to install totem and I see it in there.)   

Just in case that's not clear, what I mean is the line should look
something like this. 

export IGNORE_WRAP="libtotem*:libjavaplugin*:libflash*"
