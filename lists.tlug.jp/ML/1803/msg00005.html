I'm very interested in this; shame I cannot make it. In our comparison
with Dropbox (this is for integration into an app), Syncthing wins in
all areas except for that "no 3rd party advantage": with Dropbox you can
be sure it will be working 24/7, but Syncthing requires the user to run
a server and guarantee that themselves.
