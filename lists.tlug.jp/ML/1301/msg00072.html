If you are going to look at memcached I would look at varnishd[1], or
redis[2].


Any particular reason you prefer these?
I have found varnishd to perform better then nginx, which is saying something. It is also designed to perform well in real world situations were the dataset is larger then available memory. Redis offers some server side features you might be able to be leveraged to improve performance.
