> So is installation as simple as dropping the install disk in the CD

If the installation itself runs under Linux then yes, thet's all there
is to it.

You might want to boot from a live-CD that gives you a command shell
first of all. This will allow you to do this:

# fdisk -l /dev/hda

This will tell you what Linux thinks of the hard disk.
