I'm pretty sure the router that they use allows you to setup RNAT, which is what you are after by the way. I wouldn't suggest using the DHCP from the DSU 
because it is flaky.  Anyway, I know its all in Japanese, but you should be able
to use a static address space behind the router, and specify all inbound 
network traffic to the IP of your server box behind the DSU. I don't remember
if these settings exist but they might. I think it should be noted that the eaccess service is not intended to host services so it isn't really reasonable to expect eaccess and GOL to cater to that end.
