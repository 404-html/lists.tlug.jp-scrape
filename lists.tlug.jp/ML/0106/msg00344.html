The allow directive doesn't do that, it's for host based access restrictions.
What you should probably do is grant ownership of all your web files to a unique
web user who's a member of a unique group, then chmod them so that only the
owner can read/write them, but group and other can read. Any uploads you then do
as that user. Then you run Apache as the standard 'nobody' user, which should
allow it to read the files (as a member of the other group).
