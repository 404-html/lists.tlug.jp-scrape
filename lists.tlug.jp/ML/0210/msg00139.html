>
> IIRC, the rpms install java in /usr/java/. So, if this program is
> expecting to find the java executable on your path, you need to
> check your path:
>
> $ echo $PATH
>
> and verify that /usr/java/jre-[version]/bin is included in there.
>
> Many programs also respect the JAVA_HOME environment variable,
> so you can try setting JAVA_HOME to /usr/java/jre-[version].
>
