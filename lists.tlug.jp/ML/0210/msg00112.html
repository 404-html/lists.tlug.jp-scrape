> So, can anybody suggest a solution, or has anyone set lm_sensors up

I could set up lm_sensors as a data source for the Palantir project:

  <http://www.fastpath.it/products/palantir>

The demo server at palantir.santinoli.com reads the CPU fan speed and
core temperature (just two examples) via lm_sensors, by accessing
directly the /proc files. Some other system status info are gathered via
sysinfo().

In my case, lm_sensors were compiled from source (no RPMs).
If you need some help with the installation, don't hesitate to ask!
