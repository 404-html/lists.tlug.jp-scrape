Mephisto Backup v1.1Beta has been released. This program can do simple backups and syncs over a network or locally.; it uses tar and rsync in the background. The key feature of this program is the ability to backup your system to an image and copy it to a Linux Live DVD for future restoration. The user can then take that cd and move his or her system to any PC.
