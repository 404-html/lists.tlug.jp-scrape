Hi,

Recently I started using afterstep for my window manager and like it a lot,
except, it seems to take all my colors.  I am using a Depth of 8 8bpp and
don't seem to have this problem with fvwm or fvwm2.  But in afterstep,
I can not use any color but black and white after it is started.  For example,
I have a background color in mule.  If I start it before afterstep starts,
it starts okay.  If I start it after afterstep is running, it will not run
because it cannot allocate the color for the background.  Anytime I run any
program, I get the following type of error:

Warning: Cannot allocate colormap entry for "green3"
Warning: Cannot allocate colormap entry for "yellow3"
Warning: Cannot allocate colormap entry for "blue3"
Warning: Cannot allocate colormap entry for "magenta3"

etc, etc....

Does afterstep really take so many colors by itself?  Is there a way
I can limit how many it takes?
