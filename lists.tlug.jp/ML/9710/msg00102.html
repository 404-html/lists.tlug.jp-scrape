I think this might be interesting: a set of scripts that convert the
JE (Japanese Extensions) packages to the RPM format (for use with a
RedHat Linux installation).

The scripts are at ftp://muse.brain.riken.go.jp/pub/rpm/tools

and the converted JE-0.9.8a packages are at

 ftp://muse.brain.riken.go.jp/pub/rpm/je098a/je-rpm-i386

According to the documentation, they can be used to convert not only
JE, but any package that can be expanded using 'tar xzvf <file.tgz> -C /'.
