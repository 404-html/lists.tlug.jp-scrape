2 afterstep problems:

1. if you are using graded titlebars, menu items and/or dock buttons
then you will eat up lots of colors - there is a way to turn this
off in the .steprc file, but Afterstep is uglier that way (of course)

2. The pixmaps it comes with for your icons are very color rich.  There
is a set of icons somewhere on the net that all use the same 21 colors.
You could also check out the AIcons site for replacement icons (they
also all use a small, shared set of colors).
