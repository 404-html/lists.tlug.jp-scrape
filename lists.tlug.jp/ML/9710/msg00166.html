I've been using XFMail for about six months and find it pretty good. It has an
annoying habit of crashing when I open other programs, but other than that it's
okay. Make sure you read the KNOWNBUGS file as it has some pointers for hacking
.xfmailrc which is *important* if you're using POP. The most recent version
(1.1) is _less_ stable than earlier versions - I don't know why.
