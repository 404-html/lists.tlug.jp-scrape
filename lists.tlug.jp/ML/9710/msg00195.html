I've been using mutt for a while now.  There is also a Japanized
version of it.  I have two dot-rc files, and aliases so I run "mutt"
for us-ascii and "muttja" for iso-2022-jp.  I use different editors
depending on the language, too, so this works best for me.  Of course,
it means I have to exit my MUA and restart it to switch languages.

This setup means that if I want to forward an iso-2022-jp note, I can't
do it from the us-ascii setup (unless the person on the receiving end
is a mime-by-hand yatsu like me :-) because the header will claim it is
us-ascii.
