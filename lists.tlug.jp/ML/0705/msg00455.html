G'day,

During the last server hackathon those that were in attendance agreed
that the TLUG site could do with a redesign. In order to get the ball
rolling on this we would like to collect ideas and comments on what the
new site should be, and what it should not be. This is a wish list and
as such there are no promises that all/any of these ideas will see the
light of day ;-)

Some ideas thrown around so far include:

# Existing content should remain accessible in one form or another.
The current favourite method is to create a new sub directory and move
the existing site into that and preserve it as is. This will kill
existing URLs, however any popular content will soon have new/updated
search engine entries.
# Admin interface to easily add and update announcements to the front
page with links to the relevant wiki entry.
# Media gallery. Each event to have a wiki entry with "highlight"
photos on that page. We can then throw all the extra photos in a sub
dir and proved a link to that. Also, users can add links to their own
relevant photo galleries if they wish.
# All pages to display creation and modification dates.
# Links to sponsors.
# Links to  other groups - links on the front page to "like minded
groups" in Japan.
# A more detailed and extensive links page in the wiki.
# Links to the wiki homepage, and maybe a random wiki page, or "page
of the week" or "featured" pages...
# Wiki and site to be searchable from the front page.
#
#
#

Regards, Al & Keith
