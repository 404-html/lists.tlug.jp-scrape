According to the Japanese Instructions, it shows that the recovery
disc has it's own partition, that I suspect is the first
/dev/sda
in the menu.


/dev/sda is not a partition it is the whole device. It is likely that
the "recovery partition" is something like /dev/sda1 and will contain
a fat (fat-16 or fat-32) file system.

During the install of any distro on a new laptop you can usually
identify your recovery and Windows partitions by the file system type.
That is if you see a partition with a filesystem type similar to *fat*
then there is a good chance that this is you recovery partition. A
filesystem type of ntfs would likely contain be your Windows install.

Note down the partition numbers of these partitions and do not touch
them if you want to keep them.
