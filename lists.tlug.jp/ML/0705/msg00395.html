 > Or would this be better handled with Webmin?

Unless Webmin emulates the worksheet interface, the worksheet idea is
far better, IMHO.

The problem with older versions of Webmin was that any decision-
making it did (yes, it did some) was hidden in hard-coded
"intelligence", and much of the user input was lost, with only the
final configuration remaining.  That documents what you *did*.  A
worksheet would help document what you *were thinking*.
