Do yourself a favor and download the trial version of Parallels first. Then install it in a virtual machine. It is more fun than a barrel full of monk sleeze. The Parallels docs have detailed instructions on installing Linux, and the installation helper pretty much walks you through it. The nice part is that you can use Linux alongside MacOSX and then install Linux with BootCamp later when you have settled on a distro.
