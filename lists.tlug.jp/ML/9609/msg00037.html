> Also, this is off-topic, but how much does Win95J sell for in Japan? 

29,800 yen retail price.  You may be able to find it close to 20,000
yen on the street.  However, my humble opinion is that Linux is
a better choice.  Windows '95-J is known to be more unstable than
Windows '95.  Windows NT 4.0-J is coming out this week at 45,000 yen.
This is probably going to be better than Windows '95-J.  If you do
real work on your computer, stability is probably a high priority.
Linux has been very stable for me.  Linux is infinitely more fun, 
perhaps the real reason for its following.  I did have some fun
with Windows '95-J by upgrading it to 4.00.950a, patching the kernel,
and updating all my MS Office applications.  However, without the
source code or really knowing what I was upgrading, the satisfaction
is greatly diminished.
