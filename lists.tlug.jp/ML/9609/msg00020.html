Dennis,

   As you've probably already been told a hundred times, you 
   should probably change that to be just one line, ie:-


		127.0.0.1     donegal localhost

   Having two lines with the same IP address is probably going to 
   cause some things to fail. Having "donegal" and "localhost" on
   the same line correctly aliases both names to the same IP address.
