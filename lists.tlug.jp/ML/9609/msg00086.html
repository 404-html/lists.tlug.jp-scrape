Now that the licensing deal with Sun seems to be resolved, I am happy to
announce the release of pl1 (release 1) of the Linux/x86 port of the Sun JDK
1.0.2.

The web pages at http://java.blackdown.org/java-linux/ will be updated shortly
with the new licensing information, a current known-bug list (not many) and a
new HOWTO.  And of course information on where to get it.

This release fixes a number of AWT bugs and some very serious bugs with
multi-threadeding.  This should especially help anyone who is trying to use the
Linux JDK for server operations.  I would suggest that everyone upgrade.
