As of August 27, 1996 the Linux User Group list project has started.

	http://www.nllgg.nl/lugww/LUGWW.html

This project wille be copied in the UK within a couple of days.

Current project members are:
Gareth Blades for the UK part.
Hugo van der Kooij (me) for the other lists.

maintainers. Others are also welcom to participate in any way.

Please read the charter carefully before you participate as it might
