I also use Gnome. My approach is to install the gnome-light Gentoo
ebuild, which installs just the desktop environment, none of the apps.
As I need apps (Gconf Editor, Gnome Mixer, Character Map) I install
them.

This gives me a good balance between bloat and a desktop environment
on which almost all applications/applets/tray icons are supported. My
Gnome Menu contains only apps I actually use!

To spice things up a bit and to keep up to date with the 3D desktop
stuff, I also installed Compiz Fusion [1] from the xeffects [2]
overlay.
