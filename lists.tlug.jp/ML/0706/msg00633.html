I use green-on-black terminals, and by default, the Vim syntax
highlighting colours comments dark blue, which is basically unreadable
on a black background.


set background=dark

Either type that each time or throw it in you .vimrc file

Also, you can change colour schemes as follows (where darkblue is the
name of scheme you want to use)[1]

colorscheme darkblue

Regards, Keith

[1] http://www.vim.org/scripts/script_search_results.php?keywords=&script_type=color+scheme&order_by=creation_date&direction=descending&search=search
