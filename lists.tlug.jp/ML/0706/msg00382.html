> do I just use the -f option on rpm

I believe multiple kernels can be installed (*), So try without -f. Then
in grub when you boot up you should see the alternative kernels. Edit
/etc/grub.conf to control which is the default.

Darren

*: Actually since Fedora core 5 (and I'll assume "el5" is very similar
to "fc5") I think this is controlled by a yum option. I *think* it is
/etc/yum/pluginconf.d/installonlyn.conf which on my machine looks like:

[main]
enabled=1
# this sets the number of package versions which are kept
tokeep=4
