Hi,

I don't have CF-W5 but have experience to write a driver for Panasonic notebook.
For temperature, you can use cpufreq kernel framework to reduce power
consumption.

In root privilege do
# aptitude install cpufreq-selector

# modprobe speedstep-centrino
if it fails by 'no device' , try
# modprobe acpi-cpufreq

#modporbe cpufreq-ondemand

# cpufreq-selector -g ondemand

With this modules and settings, kernel controls power according to
cpu loads.

If you can try my driver,
can you please tell me a report?
Ubuntu-ja includes my driver or see my page!
