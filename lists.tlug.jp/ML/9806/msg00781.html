[snip]

> Does anyone know of any good resources on the web for tieing SQL
> and Apache together? I have 'The Practical SQL Handbook -
> Addison Wesley' as a primer on SQL but need more info on the web
> server side.

Try the new PHP3, a Hypertext Preprocessor comparable to the much hyped
ASP (active server pages from M$). PHP3 supports a variety of DBs like
MySQL, mSQL, PostgreSQL, Oracle, Sybase..... you name it. PHP3 is Open
Source and can be compiled as a module for Apache or standalone for use
in the CGI bin.

The syntax is rather Perlish and easy to learn, I think. Actually I just
ported some DB search from a CGI script to PHP without problems.

have a look at:
<http://www.php.net/>
