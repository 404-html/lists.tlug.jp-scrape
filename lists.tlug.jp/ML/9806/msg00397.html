In a minute I thought you were too modest to show what to do.

The situation is that it is easy to send or receive a fax on Linux.
A command like "fax send 555-1212 dummy.ps" does the job.
Probably it is easy to send command thru lpr. I don't know how to send 
mail to fax thru sendmail especially when a fax modem is a modem that
is connected to ISP thru diald and pppd. How sendmail suspends diald
and pppd to send a fax?

So once Japanese text as in E-mail is converted into a PostScript
file, it is easy to send a fax.
People suggest using a2ps. But it seems a2ps does not support
Japanese though I have not got to reading the tutorial yet.

Anyway one advantage of a analog modem over isdn is that it can send a 
fax.
