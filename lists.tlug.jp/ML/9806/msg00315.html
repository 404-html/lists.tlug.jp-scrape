KDE is the K Desktop Environment, a graphical desktop for Unix that is based
on the Qt graphics library.  http://www.kde.org/

> you know where to get the gui for the pilot-link stuff?  I would like to
> put that in and then see how it is written.

KPilot is a graphical front end for pilot-link for the K Desktop Environment.
http://www.slac.com/pilone/kpilot_home/

PalmPython is a set of Python packages that build on top of pilot-link, and
includes (a starting point) for Tk interaction.

PilotManager is another graphical tool for connecting your Pilot to Unix
(separate from pilot-link).   PilotManager is written in Perl with Tk.
http://www.moshpit.org/pilotmgr/
