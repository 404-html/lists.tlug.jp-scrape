Okay,

I was using the "xdmish" startup under CDE, however, when I went to exit
it would always hang, and I would end up rebooting.  I've never had one
sucessful exit from CDE xdm.  This leads me to believe I'll probably
have the same problem in xdm.  Also, sometimes when I start the system I
want to be at the prompt, other times I want to go right into a X window
session.  I have to say I like the "Command Prompt Option" under Sun's
CDE.  If I'm not using fvwm, fvwm2 or afterstep, I wish RedHat would
leave those options out and just had a command prompt option.

Also, along the same line, contrary to RedHat representatives, CDE-1.2
does not work with RH-5.1.  If anyone has it running I sure would like
to know how.  They say on their site that you must already have it
installed when you apply the patches, but you can't upgrade so what's
up?  I tried to install CDE under RH-5.1 and as expected it failed.

Someone on the mailing list once told me how to get it running, and I
followed your instructions but it still would not start up.  Any
suggestions would be appreciated.

Blaine
