It`s easier if you go directly to:
http://kichijoji.mech.tuat.ac.jp/~rob/NetscapeV.html

I just got it and installed the hack. It works great. Took me about 10 
min. One little warning : If you are using communicator 4.05 you have
to edit the file  ja_JP.eucJP/Netscape-404j-DeskTop_kinput2.ad and set 
all the 4.04 to 4.05. You also need to set your LANG variable to
ja_JP.eucJP, i.e. export LANG=ja_JP.eucJP. After that it works fine.
