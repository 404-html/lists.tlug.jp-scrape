This may sound too obvious. But I started using atx machine
that implements apm power down recently.
I use this so I can go away while compiling which supposedly
takes a lot of time:
	# while pidof make ; do sleep 60; done ; shutdown -h +5
This automatically shutdown computer when compiling is done
successful or not.
