>Hmmm, this is interesting... say, has anybody tried it out yet? How are

I understand that MK Linux now works on the G3.  I'm not sure about
PPC Linux, and don't know anyone who is running one.  But a fast RISC
chip like that with Linux would probably cook.  However, for the price
of a well-equipped G3 with lots of a memory and a big hard drive, you're
better off buying a 500 MHz Alpha.  If you're going to go to a system
that is not Intel-compatible, go for something that really rocks.  Alpha
machines that are currently available will completely smoke a G3 Mac and
don't carry a price tag that is prohibitively higher.  They also offer
the additional advantages of:

1) Being just plain cool :-)

2) Can  run Alpha NT, for those who might also need to use Windows;

3) Can run DEC UNIX, for those who might need to do that

>Oh, BTW, are macs pricey in Japan? I heard the user base is still
>comparatively wide there?

They cost more than they do in the U.S., and prices on used PPC Macs can
be pretty high, comparable to a brand new Intel-based machine.  The user
base is fairly wide, but then, the user base in North America is also
quite a bit wider than Apple's current market share.  I'm not sure where
their market share stands here, although it may be better than in North
America.  It has taken (and is still taking) a beating, though.
