Hi all,

On 06-Jun-98 Matt Gushee wrote:
> Jonathan Byrne - 3Web writes:
>  > What's
>  > the file that tells X what window mangler to give you?
> 
> Probably ~/.Xclients, but possibly ~/.xinitrc -- 

on Slackware, it's .xinitrc. The last line of the file is 

exec whatever-you-want-as-a-window-manager

You should also be able to change window managers on the fly.
>From my .fvwm2rc95

#
# This menu is invoked as a sub-menu - it allows you to quit,
# restart, or switch to another WM.
#
AddToMenu "Quit-Verify" "Really Quit Fvwm?" Title
+ "Restart%mini-turn.xpm%"              Restart fvwm95-2
+ ""                                    Nop
+ "Start fvwm%mini-olwm.xpm%"           Restart
/usr/X11/bin/fvwm 
+ "Start olvwm%mini-olwm.xpm%"          Restart
/usr/openwin/bin/olvwm
+ "Start olwm%mini-olwm.xpm%"           Restart
/usr/openwin/bin/olwm
+ ""                                    Nop
+ "Yes, Really Quit%mini-exclam.xpm%"   Quit
+ "No, Don't Quit%mini-cross.xpm%"      Nop


HTH

Jim S.
