I used to use a program called xwin from Starnet communications. 

        http://www.starnet.com

Their X-server is a commercial product but they have a demo version available.
The demo that I used was fully functional and not time limited but it was
limited to running one copy on any individual subnet. The University bought 50
licences for this for something like $500 - there was a large educational
discount. The web page says that an indivual license costs $200 though.

The program seemed to work reasonably well. I don't remember any major
problems.
