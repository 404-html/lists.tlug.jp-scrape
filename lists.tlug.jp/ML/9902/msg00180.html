TLUG CD Febuary 1999
====================

This CD contains the following:
/2.2.xUpdates	All of the utilities that you need to get Linux 2.2.x running. 
                Among them are glibc-2.1, binutils-2.9.1.0.19a, and modutils-2.
                1.121.
/Kde1.1		The K Desktop Enviorment 1.1 (Sources).
/Kernels	Linux 2.0.36/2.2.1 and patches.
/Communicator	Netscape Communicator 4.0.8/Muriari Japanese Patch
/Monash		Monash Nihongo Archives
/Mozilla	Mozilla source excerpts.
/Emulators	Various emulators for Linux.
	/WINE		Latest version of WINE and misc HOWTOs.
	BasilliskII	The GPL'd color MacII emulator.
	Bochs		A portable i386 emulator (It boots Linux/95!).
	Dosemu		Dosemu with 2.2.x support.
	vMac		Mac Plus emulator binary.
/Games		Games for Linux.
	/Quake		Quake/Quake II for Linux.
	/Heretic	Heretic for Linux (With shareware data files).
	/Descent	Linux Descent (With shareware data files).
/Ggi		The GGI project.
/Japanese	Misc Japanese packages.
/Misc		Misc Files.
	Mesa3.0		The Mesa OpenGL library.
	Blender1.56e 	The 3D program Blender 1.56e.
	Maximum-RPM	The Maximum RPM book (Post Script).
	vnc3.3.2r3	Virtual Network Computing 3.3.2r3 (Binary).
/XFree86	XFree86 3.3.3.1 (Binaries).
/TLUGWeb	The TLUG Website and Mailing List archives.
