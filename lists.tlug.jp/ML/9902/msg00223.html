The best idea depends on what kind of application you will write.

Well, OpenGL do realtime rendering based on polygon. If you want
to write interactive graphics application like game or visual
simulation, use Mesa.

If you don't need realtime capability, you can use PovRay.
PovRay renders an image using ray tracing. So you can get high
quality rendered image.

Visualization Toolkit(vtk) does not have C binding, since it is
an object oriented visualization library.
