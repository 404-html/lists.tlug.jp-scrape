On Mon, 1 Feb 1999, Dave Gutteridge wrote:

> I've got this Planet ISA Ethernet Adapter Card which I intend to use in my
> server machine.
> This same net card was in a previous machine where I had installed RH 5.1,
> and it auto-detected it and installed it with neither fuss nor muss.
> However, this time i'm installing 5.2, and i not only did it not autodetect
> it, I couldn't find the brand listed among the available drivers.
> Theoretically, this thing is NE2000 compatible, so i tried selecting that
> the driver (or do we call them kernels?) for that, but it still wouldn't
> detect it.
> What might I be doing wrong here? I went ahead with the rest of the
> installation and Linux is running happily on my machine, but I'm wondering
> how to get the net card properly installed.

older RHs will try io=0x300 if the PCI probe for ne2k cards fail.  Maybe
they took that out of 5.2?  try loading the module 'manually' in the
installer.  Or, install first, fight with net card later..? :)  I can
confirm that those cards are Ne2k compatible, and I've used them before.
They come from the factory on io=0x300, so unless you've changed that...
