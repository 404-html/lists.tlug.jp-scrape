I got a trial addition sometime ago.  I wanted to be able to run Matlab
(which I have for windows, but is terrible expensive for Unix) from 
linux.  It worked fine except for one problem.  The drop in speed seemed
to depend on what application.  Matlab in particular was very, very
slow.  So slow, I decided it was not worth buying.  You may want to
get the trial edition yourself and evaluate for the applications that
you plan to use.  

One other problem is that, you can not just install this on top of
an existing Win NT, but you must replace the Win NT with this version.
It is in itself a version of Win NT, not an extension.  When I got the
trial edition, they only supported English Win NT and even though Win NT
4.0 was released, they only had Win NT 3.51.  This was all at least
a year ago, so things certainly have changed by now, but probably
there is still a delay between the current Win NT and this version of
Win NT.
