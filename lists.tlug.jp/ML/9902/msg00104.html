> I got a Sun Sparc Ultra-5 machine with the system on it.

*laugh* If you want to get rid of that Ultra, I have a nice home for it
...

> I am comfortable with RedHat Intel platform, but not on Sparc
> platform. Is the same "easy" way to install and configure?

Configuration is near identical.

> What about the screen/netcard problems?

Full framebuffer support, full netcard support.

... though, with an Ultra, you'd want to use the Ultrapenguin distribution
rather than RedHat.  Kernel runs in 64-bit mode (but due to processor bugs
userspace is still 32 bit).  Tuned for the ultra (RH is actually tuned
_against_ performance).
