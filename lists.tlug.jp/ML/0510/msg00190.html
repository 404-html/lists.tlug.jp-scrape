I used to (when still using the evil-W for e-mail) transfer the data folders of Netscape 4.x from one box to another and regularly obliterated everything on the hard dive of one machine after moving data to a new box. Since moving all e-mail and Internet functions to Linux though, I've yet to do this - instead doing upgrades from SuSE 9.0, to 9.1, to 9.3. The upgrade to 9.3 produced sporadic mouse support issues, and on a test machine, upgrading to 10.0 obliterated all mouse support - both PS/2 and USB, connected either through a KVM or directly. Obliterating everything on the hard drive and doing a new install of 10.0 on the test machine provides mouse support with no problem, so I think it's time to get the data out of my main box and into its near twin, which is already set up with a new install of 10.0. The question:
I think I can do this without any difficulty if I install Mozilla in the user folder, but I'd rather drop the data into the system-installed version of Mozilla. Where does the hidden folder with all the mail go in that case? I'm ashamed to admit it, but up until now, all of my installs, etc., have been into the user folder. I'd like to do it right this time....
