> What's a kludge?  A bug?

  http://en.wikipedia.org/wiki/Kludge

  "A 'solution' for accomplishing a task, originally a mechanical
  one and usually an engineering one, which consists of various
  otherwise unrelated parts and mechanisms, cobbled together in an
  untidy or downright messy manner."

Actually, as far as this particular bit of font bizareness on my
system, I'm think it's being generous to call it a kludge. I think
"somebody's really stupid idea" is a better description.  The
cause is some font package that has a font file (or files) in
which some of the correct glyphs for certain code points have been
replaced with other glyphs.
