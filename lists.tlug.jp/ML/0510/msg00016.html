> Is there a way to have the source code for "emerged" packages extract
> themselves into a special directory automatically? 

ebuild /usr/portage/<class>/<package>/<package-1.2.3>.ebuild unpack

The resulting files will be located in /var/tmp/<package-1.2.3>/work
