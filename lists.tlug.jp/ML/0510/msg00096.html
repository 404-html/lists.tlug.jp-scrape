After my upgrade troubles, I better report on two successful installs of SuSE 10.0.
1) I successfully installed in on an old HP (P-III 800MHz) that I had put in SuSE-9.1 into, but had been unable to upgrade or newly install 9.3 into.
2) I successfully installed it on an old Dell (466MHz) as a new install. On the same machine I had previously put in SuSE 9.3, which worked fine, except for it not holding the same screen size with the other machines on my KVM switch. I'm happy to report that 10.0 doesn't need to have the screen size adjusted when switching between it and the other boxes *and* that the mouse is working fine (through a KVM switch).
One thing that seems to be different from 9.3 is that not all the packages are on the set of CD's. Naturally, installing it from the DVD is the best way to go, but I still have some old boxes with no DVD read capability.... The 466MHz machine has one though, and the install went smoothly, although taking about three hours (466....).
As for the test machine with no mouse support... I would still like to fix that without having to do a new install. I realize that a clean new install is the best way, but there are cases where I'd prefer to upgrade.....
