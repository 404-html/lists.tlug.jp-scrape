Does anyone have a filter that can decode header lines like this

	Subject: =?iso-2022-jp?B?GyRCSlYbKEo6IBskQkpWGyhKOiBQb3N0Y2FyZCAbKEo=?=

My local installation of MH doesn't decode it (ie display as JIS) when
it does a show or scan command. I thought a simple hack would be to
pump the output of show thru some simple perl script, if someone has
such a script. Yes, I know I could fire up some version of emacs or
view it with Netscape's mail to see the decoding but I like MH.
And I could probably install some localized version of MH, but I'm hoping
for a quick fix. Not to mention it could be useful for other email processing
tasks.
