X11 Doom for UNIX

  • Subject: X11 Doom for UNIX
  • From: udo@example.com (Udo Munk)
  • Date: 29 Dec 1997 21:25:37 GMT
Dec 23th ID Software made a nice Christmas gift to the UNIX community
and released the sources for Linux X11 Doom. Here is a quote from
John Carmack (ID Software) about it:

    I don't have a real good guess at how many people are going to be
    playing with this, but if significant projects are undertaken, it would
    be cool to see a level of community cooperation.  I know that most early
    projects are going to be rough hacks done in isolation, but I would be
    very pleased to see a coordinated 'net release of an improved, backwards
    compatable version of DOOM on multiple platforms next year.

    Have fun.

    John Carmack
    12-23-97

OK, so I ported it to SCO OpenServer 5 and Unixware 2.1.2 and it still
should compile under Linux too. I modified the whole layout some and
named it X Doom, this isn't just Linux Doom anymore. I fixed some bugs
and minor glitches I never liked in the Linux Doom versions, details
are in the ChangeLog. Yes, both versions come with sound support, if
you have a 16bit sound card and the voxware or OSS drivers installed.
I am using OSS, but the Linux version works with the voxware driver,
I guess it will do too.

It plays Doom 1 shareware and retail WAD, Ultimate Doom WAD, Doom 2 WAD
and both WAD files from Final Doom. It also playes custom made 3rd party
WADS for Doom 1 and Doom 2 and the Eternal Doom 3 TC WAD plays too.

Current sources and binary distributions for OS5 and UW2 and shareware
WAD can be downloaded from:

ftp://www.umnet.de/pub/unix/doom

At this URL you'll find a subdirectory 12-19-1997, it includes everything
done so far. Newer versions will become available in new subdirectories
just with the release date as name, depending on progress and contributions
done by others.

Same as John Carmack I just hope we can have a Doom engine for all
UNIX platforms sometime and that others might help to get this
ported to more platforms, get bugs fixed and the TODO topics done.
In this hope I crosspost to several UNIX newsgroups, even if I haven't
ported it to all this OS's yet and there is no binary distribution for
all the OS's yet. Well, maybe there will, here you have something to
start from and it might be easier already then everyone starting from
the sources ID released.

Have fun and a Happy New Year.
