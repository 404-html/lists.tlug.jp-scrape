Fernando,

   Let's take these two issues separately...

% 
% unable to  boot up ipc sun with red hat linux
% booting stops at  loading appletalk protocol
%

   You IPC *is* booting up, but as I told you at the meeting, it is
   booting in "serial console" mode. That is, the login prompt is coming
   up on serial port "A". Now, you can either attach a serial cable
   to the port and use the "setserial" command to change the console
   device back to the video/keyboard ports, or...

%
% tried to boot up with red hat linux boot up disk..returns to red hat
% setup screen.
%

   ...use the virtual console available from the second-stage install
   step of Red Hat install ( <ALT> <F2> ) to get you to a shell prompt
   where you can mount the disk partitions and then use "setserial".

   I would recommend that you use the first method, as it is by far the
   simplest. Please see my previous posting to the list on the pin-out
   connections for the IPC serial port.

   Good luck!    - If all else fails, bring it in to the next meeting
   and we'll dissect it on the table.  :-)
