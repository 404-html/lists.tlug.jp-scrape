On Mon, 11 May 1998, Martos-Riano wrote:

> Scott Stone wrote:
> 
> > amd question - how the heck do I get it to automatically mount a local
> > filesystem?  I want it to automount my CD drive.. I added a line to
> > amd.conf:
> >
> > cdrom   host==armadillo;type:=ufs;dev:=/dev/hdc;sublink:=cdrom
> >
> > now, if I do a tail -f /var/log/messages while I try to go into the proper
> > dir, it says that it is *trying* to mount it, but it's trying to mount it
> > as MINIX, not iso9660.  The man pages/texinfo docs for amd are 7 years old
> > and are horribly bad as far as revealing anything on the subject, ... does
> > anyone know the solution to this?
> 
> I thought automounting was to add a line at /etc/fstab with the same options
> used with the command mount. are there other orthodox methods?
> 

amd is different... I can theoretically set it up so that I pop in a Cd
and then do 'cd /mnt/cdrom' and it automatically mounts it, then unmounts
it when I leave the dir (after a specified keepalive interval).  It works
great for NFS... if i'm in an environment with servers that export NFS
volumes, for example... if 'myserver' exports '/home', I can just do:

cd /net/myserver/home

and there it all is....
