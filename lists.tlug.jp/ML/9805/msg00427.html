On 23-May-98 Scott Stone wrote:
> On Sat, 23 May 1998 craigoda@example.com wrote:
>> From: Scott Stone <sstone@example.com>
>> sstone> > Say, does anyone know why incoming on the FTP site
>> is not working?
 
> hm, well, the incoming dir creates files as write only... 

Actually, when I set up the incoming directory I gave it 1733
permissions. This means you can upload files to the directory
but can only view them if you are logged in via SSH as user tlug.
I did this to prevent the TLUG site from being used as a pirate
repository.

The plan was to let people upload files at will, but have them
send me a note about where in the FTP structure they wanted the
files to be placed.

I thought that it might also be a good idea to have a cron job
run periodically to clear out this directory... but I never got
around to it.
