Has anyone gotten the Unix Amiga Emulator for linux to compile and work
under X?  I can get it to work in SVGALib (somewhat)... but in X, I just
get core after core.. even in SVGALib, not many games and stuff work...
usually it'll say, "Your amiga program just did something terribly
stupid".

Any hints?  If anyone wants the source, i've got it.  You'll need a lot of
ram to compile it, since the 68000 cpu emulation is all in one .c file
that's almost 2MB (yeah, 2mb .c file).  My machine w/ 64MB ram and 64MB
swap has to have almost nothing else running on it in order to compile it,
or it'll run out of memory (and swap!).
