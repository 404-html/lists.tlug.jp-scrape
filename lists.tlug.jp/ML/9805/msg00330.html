    David> When I'm not in X, it's in my path, but apparently not so
    David> (?)  under X.  If I try to start up an xterm through

Once in X, try to start the xterm and do Ctl-Alt-F1 and look at the
console output.  Then do Alt-F7 and see if you have an xterm.  Then do 
Ctl-Alt-F2, log in as the same user on the 2d virtual console and do
`xterm -display :0 &'.  See if you now have an xterm.

    David> Does this sound familiar to anyone?  Know what's wrong

    David> Has anyone out there _ever_ used the mono server? :^)

No.  No.  Probably not.  Yes, once---too painful because the color
scheme left me with lots of black on black and white on white.  (This
is a big problem with RHL 4.2 on Sparc.)  So I don't know much about
it.
