>Is there a Netscape out there that takes a Jse FEP for
input?

Unfortunately, there's nothing out there now.  The Netscape
Communicator version for Linux will display Japanese
correctly in main text, but it won't take Japanese input and
it displays mojibake in buttons.  There is a hack for
Netscape 3 to make it display Japanese in buttons and allow
cut-and-paste input of Japanese, but this is a sledgehammer
approach and something the students would A) need
instruction on doing, and B) get fed up with pretty quickly.
I can't find my bookmark on that hack right now, but I think
it's also written up in English somewhere in Craig Oda's
Japanese in Linux instructions on his home page.

Basically, the bottom line is that for now, there is nothing
you'll be able to find that is going to satisfy people who
are used to doing their web surfing under MacOS or Windows.
Indeed, that's why I don't do much of my web surfing under
Linux, either :-(

I think the Linux workstation idea is a great one, but at
least for now, you're going to run up against obstacles in
the area of applications.
