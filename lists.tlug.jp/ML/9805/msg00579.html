Hi guys,

I currently use (on NT I'm sorry to admit) a CD player that can talk to
a CD database on the Internet. This lets a CD player provide track
listings. This database is a voluntary project (under GNU), and
basically relies on its' users to mail in track listings for CDs in a
special format for inclusion in the database. Anyway, they're looking
for server space in Asia to mirror the service. They currently have
around 30 other servers.

Anyway, if anyone feels like helping them out:

http://www.cddb.com/info/news.html
