Jim Schweizer wrote:

> I looked through the Linux Applications page trying to find a
> web page hits counter but none were listed. Does any one have
> any suggestions for a hits counter to put on the TLUG server?

I used the WWW Access Counter 2.4 from Muhammad A Muquit once for a page
on a BSDI system though. I just checked the page again and it says that
it will compile in Linux (version 2.3).

The program is written in C and uses strips instead of single gifs which
speeds up the process a little bit. Beside counting the hits it can
display the current time. The biggest advantage over other counters is
that you don't need SSI at all. 

here the URL:
http://www.fccc.edu/users/muquit/Count.html
