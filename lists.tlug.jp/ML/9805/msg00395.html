It has its disadvantages.

To use: C-X C-F /<user>@<host>:<file-spec>

The <file-spec> can be absolute (if the ftp server will let you do it) 
or relative (in which case it's relative to wherever ftp would start you).

XEmacs uses efs, which is claimed to be better than ange-ftp, although
I don't know why.  Uses the identical interface.  First time users are
in for a shocking contrast first time they try it in XEmacs.
