Netscape on Linux works moderately well.  Look at 

    http://www.bekkoame.or.jp/~egota/NetscapeV.html

If you need help, ask questions to the list and I can probably help you.

If you ask a question, include this info:
  what Xlib are you using?   There is a Locale support problem in
	standard Debian Hamm and Red Hat 5.0
  script you are using to pre-load the lib 
      (export LD_PRELOAD=/usr/lib/netscape/INSTALL/liblocale.so)

Your relevant .Xresources for Netscape.
Netscape*international: True
Netscape*inputMethod: kinput2

If you don't use kinput2 normally, we may have to deal with this 
problem seperately.

First thing, check out the home page.  

I only use input on forms regularly.  I've tested it with that editor that 
comes with Communicator and it worked, so you could use it to write a
short memo in 日本語。
