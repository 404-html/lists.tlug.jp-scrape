On Wed, May 27, 1998 at 05:52:00PM +0900, Jim Schweizer wrote:
> 
> In a sense, JavaScript provides a Mozilla presentation mode.
> It's possible to 'turn off' most of Mozilla's features like this:
> 
> aPopUp=window.open('','Note','toobar=no,location=no,
> directories=no,status=no,scrollbars=yes,resizable=yes,
> copyhistory=no,width=300,height=200')
> 
> and then set the width and height to match your screen. When the
> new window opens, most of the clutter is gone and you have a
> full screen for the presentation.

Hey, cool!

I've used javascript for mouseOver stuff, but that's it.  The stuff I
don't know ....

What I'd really want is:

    o Run completely full screen without scroll bars (probably no way to
      avoid requiring authors to design for a given screen size, alas)

    o map keystrokes & mouseclicks to "next page/prev page" type actions.

    o Embed "live" X clients like mgp (that Xeyes demo is just too cool).

>From your javascript above, it seems like "scrollbars=no, resizable=no"
would effectively accomplish the first bullet.  Is there any way in
javascript to accomplish the second?  (We can always hope that Joe
Hacker will come up with a solution for the third bullet, now that
Mozilla source is available.  :-)
