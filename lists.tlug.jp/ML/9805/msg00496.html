Eric wrote:

> 2) I followed the ISP-HOWTO to the letter and I still cannot get Linux
> online.  HELP?

> Ps.  I am working with a 166Mhz 32MB 2.5GBHd Sony Vaio with ATI 3D Rage 2Mb
> video and 28.8 DSVD ?brand? modem.

1) There are two type of telephones, pulse and tone. Both type are popular in
Japan. I use pulse-type telephone at home, while I use tone-type at office.
Use 'atdp' for pulse-type. Use 'atdt' for tone-type.

2) Can you use a terminal type connection? Try 'cu -l /dev/cua0'
('cu -l /dev/cua1') and type 'atdp03-xxxx-xxxx' ('atdt03-xxxx-xxxx').

3) Which provider are you going to use? One provide use 'login'-type
connection, while others use 'pap'-type connection. If you don't know,
try 'pap' first.

4) Which pppd do you use? I'm using 'ppp-2.2.0f-5'. The latest version
is '2.3'. Arguments were changed a bit.

Cheers.
