> As it stands, NT does nothing cheaper than Linux.  It does some things
> easier than Linux, owing to a better GUI (I know some people are going to

XFree86 supports drag and drop.  The applications in question must be
built with the DnD libraries to do it, though.  For example, in TurboDesk,
you can drag a file from the file manager to the (iconified or
un-iconified) astrash that's swallowed in the bottom-left button bar, and
it does what you'd expect it to.

I think the problem is that most people who write X apps don't really care
about drag and drop.  Heck, i don't even really care about drag-and-drop
INSIDE my apps that I write (not that I write many, but for this game I'm
writing...)..

Oh well, the world is too mouse oriented anyway, it's time for people to
learn to actually *use* their computers again, IMNSHO :)
