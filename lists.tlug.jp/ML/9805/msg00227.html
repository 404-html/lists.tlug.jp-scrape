Is there any way to use symlinks with cvs?  Ie, if I have a directory with
10 items, 5 of which are regular files and 5 of which are
differently-named symlinks to the regular files, and I check that stuff in
via cvs, I'll end up with 10 regular files in the cvs repository, since
it'll follow the links and check in the linked files instead of the links.

This is bad when I'm checking in trees that are used to build floppy
disks, for obvious reasons.  Is there an elegant solution to this?
