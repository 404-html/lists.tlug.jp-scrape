> Hi,
> does anybody know if there is a vm version that works with fsf emacs 20.2.
> I just build the 20.2 emacs and it works fine except for the vm.

Kyle Jones has said he won't port vm to FSF emacs 20 until emacs 20
has settled down (my paraphrasing and bad memory :-), I use it, and it
works fine, even using mailcrypt.  But he says in particular it can
munch things that involve multibyte chars.  I would say find the one
line which checks for FSF 20, disable it, and test it thoroughly, as
if your mail depended on it :-), before trusting it with multibyte
email.
