Are you sure that you don't have a shell with the device in the
current path? That is, if you do a 

	cd /mnt/cdrom/some/path/down/here

Then to automounter will never be able to unmount. It gets tricky when
you then rsh, telnet, start another subshell etc. Until you exit back
out and cd, umount will not work. 

Try to manually unmount. If you get a "device busy" message, then
thats the problem. You might be able to use lsof to find the culprit.
