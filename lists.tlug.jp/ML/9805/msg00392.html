> On Thu, 21 May 1998, Jim Tittsler wrote:
> 
> > Or even better, Emacs with ange-ftp, which will allow you to treat
> > remote files as if they were local.
> 
> Now *that* sounds cool!  Is this a program that installs itself in emacs,
> or how does this work?  Does it work with Xemacs?
> Doesitdoesitdoesithuhuhuhuhuh?!  :-)

Its part of any descent Emacs distribution.  Just try to
load a file (C-x C-f) and then use as a file name something
like

  /<username>@<machine-name>:<local path>

and it should ask you for a password on this machine for the 
given user and go off and get the file.  Note that if the
file is a directory you will get into dired-mode as usual -- 
very convenient.
