the results from the KeyNavi Linux survey are available.  Since the
URL is way too long, I've made a link to it from
  http://tlug.linux.or.jp/whatsnew

It is one of the top links now.

 2,005 respondents, March 23 to April 3 1998.

For those that can't read Japanese.  Top uses in order of popularity:

  www server   332
  mail server  323
  dns server   279
  ftp server   237
  file / printer sharing  152
  news server  119
  rdbms		27


Reasons for using Linux
  lightweight, does not require high-end hardware
	does not consume resources	176

  does not crash, stable		140
  price					139
  good availability of free software	43
