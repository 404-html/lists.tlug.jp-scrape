At 23:20 98/05/25 +0900, you wrote:
> 1) is it possible to HotSync a Palm Pilot in Linux?

I got my Linux Journal (June 1998, issue 50) today and found an
article entitle "Linux and the PalmPilot".

It lists several interesting packages:

	- pilot-link: includes the pilot-xfer program
                     to transfer programs and data between 
                     Linux and the Pilot
	- PilotManager: GUI
	- MakeDoc: formats text files into Pilot Doc format
	- prc-tools: SDK
