Indeed - to get your linux box working with their equip just set your
gateway to be 192.168.1.1 (default gateway address), connect your box via
10base-T to the gateway/router or to a hub which is connected.

There is one issue though, they don't provide a static DNS server address -
you'll have to setup DHCP to locate this or use another known server (I do
the later).
