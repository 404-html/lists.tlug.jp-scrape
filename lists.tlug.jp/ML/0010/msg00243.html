With Chris' help, I've done it! I have 1280x1024 now!

I installed X version 4.0.1 (3.3.6 was there before), and an important
thing to know is that you don't use xf86config any more but XFree86
-configure instead -- which doesn't ask you questions.

Switching the monitors with LCD/CRT button doesn't work, but when the
external monitor is disconnected, X starts in 600x800 mode. I have 2
Monitor and 2 Screen sections in XF86Config.
