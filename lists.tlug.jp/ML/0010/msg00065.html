On Wed, Oct 04, 2000 at 12:32:51PM +0900, Hector Akamine wrote:

> 1) Is there a way to make .xinitrc able to work with both [Gnome and
> wmaker]? (for example
> that it recognizes the computer in which I am running startx, or if it's
> possible to pass any parameter to tell it which window manager I want to
> start). How should the format of .xinitrc would be like?

The answer to your first question is "Yes!" The answer to your second
question is ...

  man hostname

  $ locate xinitrc
  /.commons/X11R6/lib/X11/xinit/xinitrc
  /etc/X11/xinit/xinitrc

  $ cat /etc/X11/xinit/xinitrc | sed -e q | grep ^#!
  #!/bin/sh

  $ ls -l /bin/sh
  lrwxrwxrwx    1 root     root            4 Aug  2 12:07 /bin/sh -> bash

  man bash
