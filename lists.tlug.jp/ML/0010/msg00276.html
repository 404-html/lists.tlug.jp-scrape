I'm using it.  It took about 5 months to get it
installed.  I haven't tested it out very well yet.

I've got the NAT option and a "private IP address",
not a "dynamic IP address".  That means that it is
impossible for a client to open a connection to your
server--not merely difficult, as with a global IP
address that changes periodically.
