It is fairly easy to start using CUDA on Gentoo:
$ emerge nvidia-{drivers,cuda-sdk,cuda-toolkit}

But check if your nvidia card has CUDA capability first:
http://en.wikipedia.org/wiki/CUDA#Supported_GPUs
