On Tue, 6 Jul 1999, Jordi S.B. wrote:

> I'm really sorry. It's just that I just installed Red Hat 6.0, and still can't set the ppp up

Fetchmail should do it for you.  Just make your .fetchmailrc look like this:

-------------
set daemon 600

# runs as a daemon, fetches every 600 seconds.  Adjust the fetch 
# interval to suit yourself, or remove completely if you don't
# want it to run as a daemon.

poll your.popserver.here proto pop3 user your-id-here pass your-password-here

# Place this file in your home directory, and make sure you set the
# permissions to 600.

------------------
