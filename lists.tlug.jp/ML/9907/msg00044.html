On Thu, 8 Jul 1999, Eric Standlee wrote:

> I cannot believe that pine has to be on-line to send messages properly..

You need to edit sendmail.cf. Find the line that says "DeliveryMode=" and set
it to "deferred."  Right now, it's probably set to "background."  In
background mode, you need to have a network connection up to send mail.  In
deferred mode, sendmail will spool the mail until it has a network connection.
