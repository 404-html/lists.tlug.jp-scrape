On Tue, 6 Jul 1999, Scott M. Stone wrote:

> FWIW, I'm using make 3.77 and haven't had any problems - is this
> problem specific to make 3.77 + some specific version of libc or is
> there some other caveat here?

3.77 has broken globbing.  Manifests when one tries to build glibc-2.1.1
with a system running 2.1 or higher with make-3.77.  The right fix is to
rebuild make with a spruced up glob/glob.c -- but 3.76.1 doesn't have the
bug, 3.77 isn't a radical upgrade, so I don't bother.
