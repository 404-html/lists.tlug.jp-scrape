Hi,

I released 0.3a today, with too many enhacnements to count. (The size shrank 
since I only included the kernel headers, <- You can get the source from any 
ftp.kernel.org mirror anyway... and got rid of the profiled glibc libs).  A 
Nihongo-ka pacakge with a Japanese IM, editor and mailer in additin to KON 
is due out RSN (Nothing much else to Localise since the web browser w3m is 
already Japanese ready).

Main differences include, various package updates, made devfs optional( I 
use it though), size shrank to a ~80MBs full install.

Also a Boot/Mini-root disk installation subset is also in the works. (All 
that remains is to sync some of the minor details with the 0.3a release). 
(Installation will still be by hand however, bootdisks will be available).

It is available from http://members.linuxstart.com/~austin (I have all the 
big FTP'd files on another server this time ;-) )

Yours,
Austin K. Kurahone
