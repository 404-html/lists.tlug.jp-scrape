Yep. I'm using kerneld to load the device driver for a Meteor video
capture card. I took these three steps:

	1) compiled and installed the module

	2) created the file /etc/conf.modules, with the line

		alias char-major-40 meteor

	3) edited rc.local, adding:
	
		if [ -x /sbin/kerneld ]; then
		   /sbin/kerneld
		fi

Now, everytime I start an application that accesses /dev/mmetfgrab0, its
module is loaded automatically.
