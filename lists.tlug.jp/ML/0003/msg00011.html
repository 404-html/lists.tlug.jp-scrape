Only if it's for OCN.

The best deal in ISDN routers is the Yamama RTA-50i, AKA the Net
Volante.  Reliable, straightforward to set up.  It has a browser-based
configuration utility that works under That Other Operating System (in
Japanese only; if you telnet to it, it will display in either Japanese
or English).  It has 3 or 4 (I forget) ethernet ports, so if you have a
few boxes that you want to NAT, you don't have to buy an extra hub to do
that.
