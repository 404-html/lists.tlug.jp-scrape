At 09:44 00/3/15 +0000, you wrote:
> The Win32::OLE is relying on the fact that it can talk to a local
> Lotus Notes client. Since Lotus aren't releasing a client for Linux,
> you can't go that way.

There is a Linux version for Domino (the Notes server that sometimes
is referred as "Lotus Notes Domino", giving the misleading impression
that the Notes client is available for Linux).

I'd suggest you install Domino in the Linux box, replicate the
database from the Windows server, and try to find some mechanism
to interface your Perl program with the locally served database
(if there is one :)
