On Tue, 7 Mar 2000, Darren Cook wrote:

> Does anybody know of a DNS how-to in Japanese I can give to these people?

Anyone doing DNS should have a copy of the O'Reilly book, "DNS and
Bind."  It is available in Japanese.  If they're doing DNS but don't
even know what a TLL is, they should read it ASAP.

If the change they are planning is to move a web site to a machine with
another IP address, I suggest running the sites in parallel until
all of the previous cached DNS records have expired.  That way,
whichever IP address someone's machine goes to, they will see the right
thing.  Then, once all of the old cached DNS entries would have expired,
they can terminate the old website.

Jonathan
