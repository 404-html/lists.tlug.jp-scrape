Yeah, more than four partitions are needed, but they don't have to be on
the same disk:

$ mount
/dev/hda2 on / type ext2 (rw)
none on /proc type proc (rw)
/dev/hdb1 on /tmp type ext2 (rw)
/dev/hdb3 on /usr type ext2 (rw)
/dev/hdc3 on /var type ext2 (rw)
/dev/hdb2 on /home type ext2 (rw)
/dev/hdc4 on /usr/local type ext2 (rw)
/dev/hdc1 on /usr/local/vmware/machines type ext2 (rw)
none on /dev/pts type devpts (rw,gid=5,mode=620)
/dev/hdd on /mnt/cdrom type iso9660 (ro,noexec,nosuid,nodev,user=tappel)

;-)

(Oh, I don't seem to have root on hdb in the current setup. It used to
be... And I seem to have partitions that aren't in use... Well, hda4 is
swap, but still.)
