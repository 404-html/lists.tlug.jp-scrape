doughtym@example.com wrote:

doughtym> Anyway, I'm looking for a japanese patch for recent versions of 
doughtym> Tk Perl. Can anybody help me with this?

I've had similar question before, and I believe there is no such
thing. Last patch released for Perl/Tk is for version 4.X, and
seem to be abandoned since then.

On the other hand, since Tk (and Tcl) started to include Unicode
support for version after 8.X, there's a hope that they'll someday
support Japanese by default (I've heard their Unicode support
isn't complete yet, and moreover, Unicode != Japanese).

My suggestion is to go with Gtk/Perl.
