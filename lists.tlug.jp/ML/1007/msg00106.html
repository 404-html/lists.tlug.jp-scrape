I think the group the device file belonged to was "disk" or something.
Anyway you can change the permission with write access so that
general users other than root can access it like this :

$ sudo chmod 666 /dev/sg1
$ ls -l /dev/sg1
crw-rw-rw-  1 root disk 21, 1 Mar  2 09:06 /dev/sg1
