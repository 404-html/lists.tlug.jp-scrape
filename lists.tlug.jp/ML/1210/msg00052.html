Hi CL(?),

Please check your hardware first.
You should be able (as root) to check if you receive the clicks:

cat /dev/input/mice |xxd

No need for X, works from console. Move the mouse and you'll see
garbage, click (left, right, middle) and you'll see (different kind of
garbage).
Clicking produces mosly zeros, moving produces all kind of garbage.
(xxd is a hexdumper, replace with something else if you don't have it
installed).

Make sure you don't move the mouse while clicking (hold with one hand,
click gently with the other).

If that works, then install gpm
(http://www.nico.schottelius.org/software/gpm/), run it and check if
mouse works in console.

If it doesn't, we'll look into the layers above (X11).

And write your name unless you prefer to be referred to as CL ;-)
