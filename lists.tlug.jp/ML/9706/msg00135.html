tlug note from craig@example.com
--------------------------------------------------------
Dennis's comment about grepping through his mail made me contemplate
that many TLUGers are probably archiving their own mail and 
performing grep searches on it.  The list is automatically
archived by month since the beginning of the group.  The
digest is also archived.  Unfortunately, there is no easy
way to search through it.  It occurred to me that if
John Little's proposal to set up a TLUG server came to fruition,
another use of the server could be to set up a good
search facility for the list.  The entire contents of the
list since the dawn of time could be on the TLUG server.   
There are also numerous search tools that could be implemented
on a server like that.  The data could also be interfaced
in a number of ways.  

-- Craig
