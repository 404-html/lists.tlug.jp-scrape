And just to be really strange, I use a 6th.

I use "jemtex", which is a combination of a front-end utility that
converts jis x 0208 characters into TeX font calls, and a heap of font
files which are made up of the 24x24 bitmapped ex-.bdf font turned into
pseudo metafonts. 

Good points: simple to install & run, uses vanilla non-Japanese latex,
dvips, gs, etc.

Bad points: not super-high quality, only one font available. (I find the
quality quite acceptable.)

One of my must-do's is to modify the utilities to handle alternative font
files (I have asiya24 and heisei24) and to handle jis x 0212.
