> It might be a good idea to install it if you think someone has
> (or may) cracked into your system.

> Tripwire reads a configuration file of files or directories to
> monitor, and then tracks changes to inode information and
> contents. The database is highly configurable, and allows you to
> specify particluar attributes to monitor and particular message
> digest alogorithms to use for each file.

> 
> -----------------------------------------------------------
> #!/bin/sh
> 
> find `cat /usr/adm/filelist` -ls -type f -exec md5 {}\; >
> /tmp/now
> diff -b /usr/adm/savelist /tmp/now
