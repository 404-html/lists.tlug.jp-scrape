Hi all,

Many moons ago Craig wrote:
>Dennis's comment about grepping through his mail made me
>contemplate that many TLUGers are probably archiving their own
>mail and performing grep searches on it.  The list is
>automatically archived by month since the beginning of the
>group.  The digest is also archived.  Unfortunately, there is
>no easy way to search through it.  It occurred to me that if
>John Little's proposal to set up a TLUG server came to fruition,
>another use of the server could be to set up a good
>search facility for the list.  The entire contents of the
>list since the dawn of time could be on the TLUG server.   
>There are also numerous search tools that could be implemented
>on a server like that.  The data could also be interfaced
>in a number of ways.  

It occurs to me that if we can find a internet feed for John's
baby that it might make a good August project for me to set up
the TLUG server and get some of the search functions working. We
could also move the majordomo list to it as well.

I've messed around with some Perl-based search engines that
'grep' through html files for keywords. That might be an
interesting place to start. It would also be interesting to see
if we could set up some sort of web type glimpse interface,
although I'm not sure exactly how to go about that.

Anyway, I'm willing to give it some time. The only thing is I
can't offer any help with the connection:-(

Jim S.
