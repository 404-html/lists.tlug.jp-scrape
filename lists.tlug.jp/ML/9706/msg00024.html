> > /wutil.o -static ../../Image/jpeg/linux/libjpeg.a ../../Image/libpng/linux/libpng.a ../../Image/zlib/linux/libz.a -L/usr/X11R6/lib -lX11 -L../../Library/linux -lwww -lm

> 
> Yep, you don't have a static X11 library on your system (for whatever
> reason), so this link command fails.  A static X11 library would be called
> something like libX11.a.  You only have shared ones, libX11.so*.
> 
> Just pull out the -static--I would guess they put this in there so that
> you could redistribute the resulting binary without having to include all
> the various libraries that are linked in.

    "Arena browser built successfully and can be found in Arena/linux"

  Thanks Jason.  This did the trick alright.  For anyone else trying this,
you need to comment out the LFLAGS= line in Arena/linux/Makefile.include
to get the last stage of the build to work properly.  Once again, for
those who are interested, the source is at:
 ftp://ftp.yggdrasil.com/pub/dist/web/arena/  and you want:
 arena-CJK-beta3a.tar.gz

  I'll let you all know how this browser performs, as soon as I have
had a chance to try it out.  Jim Schweizer's comment "Ah, arena 0.97g
doesn't work well at all. It keeps crashing when encounterning a JPEG"
was not too encouraging.
