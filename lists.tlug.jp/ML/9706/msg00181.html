------------------------------------------------------------------
Hi all!

It has been quite awhile now since you heard any news from me about
html2ps. Finally, over a year later than I originally had planned, a
new version is (more or less) ready. I have discontinued the work on
the 0.2 version, and release yet another beta: 1.0. You can download
it from:

  http://www.tdb.uu.se/~jan/x.tar.gz

You must have Perl5 (the command 'perl -version' shows this) installed
to be able to use this version.

This version is only announced to the mailing list. But if no major
problems arise, I will very soon announce it more publicly. The
documentation is not yet finished, but hopefully it should be
enough for you to get started.

There are several new features, see the enclosed documentation. The
most important change is the use of configuration files for layout
control, resource information, etc (it was simply not possible to
control everything with command line options only).

Execute the installation script 'install' (for Unix only) to generate
a global configuration file and install the files.

Enclosed is also a sample configuration file 'sample', that is used
by the installation script to convert the HTML document html2ps.html.
>From this config file, and the not yet complete documentation, you
may get an idea of how the config files are used.

If you manage to get the script running, but encounter some other
problem: use the option -d, this creates debugging output to a file
html2ps.dbg. Send the content of this file to me.

Please let me know about your experiences, good as well as bad.
Especially, if you try to install it on some non-Unix platform.

/Jan
