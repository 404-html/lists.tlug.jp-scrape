If you ever wanted to run Linux under windows and actually be able to do things, please visit:

http://www.pendrivelinux.com/2007/09/19/portable-qemu-persistent-pendrivelinux/

This runs under qemu + kqemu acceleration and has Debian . You simply double click a batch file and Debian boots. You can even browse the internet, and get package updates.
