I don't know much about Linux Mint, so I'm not sure if this is
relevant.  But at least for Ubuntu, there are web sites such as this:

https://repogen.simplylinux.ch/

(NB:  This isn't an official Canonical web site, AFAIK)

which can help you regenerate the sources.list file that you would put
in /etc/apt/ (do back up your old one).  You can try regenerating a
new one and then doing an "sudo apt-get update" to see if you can
access the repository.  Maybe even try various repositories.

As Curt pointed out, this doesn't seem to be a Firefox problem at all
but a problem with the packaging system.  But since I don't know Linux
Mint, I don't know how sources.list differs from Ubuntu's (which is
what the above link is for).
