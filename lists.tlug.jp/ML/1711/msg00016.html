Sorry to hijack the thread, but how do you switch between them? I start to have
trouble with more than about 5 terminals with similar names. I've tried
nimbler/superswitcher [1], but if you have a large number of desktops and
terminals with very similar names it can get confusing. Rofi [2] is better, but
its ordering of windows is unintuitive and again it doesn't help with windows
with very similar titles. Skippy-xd [3] is probably closest to what I'm looking
for, but doesn't have previews for windows on different desktops.

I'm using a heavily-customized Fluxbox and switching to something that supports
this natively isn't really an option :)
