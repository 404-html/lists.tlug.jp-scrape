anthy.  As far as I understand it (but I could be wrong), anthy is no
longer developed.  There is also, in Fedora, at least, libkkc, and a Fedora
fcitx-kkc.   (I just built an rpm for CentOS-7x too, if anyone cares, but

file and it built. )

effort.  Should one try to avoid anthy in favor of kkc?  And is there any
advantage or disadvantage between using say, fcitx-kkc as opposed to
