>>>>> "Atlas21" == Atlas21 Shawn <gray_s@example.com> writes:

    Atlas21> Any hints on how to get the environment to use EUC and/or
    Atlas21> how to convert the file to EUC?

In a kterm, Ctrl-Button-2 (middle) gives a menu that you can use to
set the kterm to EUC.  Or start it with kterm -km euc.  For file
conversion, get one or more of the utilities nkf, kcc, or jconv.

I don't know how to do those operations with rxvt other multilingual
xterm derivatives.  It kinda has to be a kterm, nobody else but the
Japanese care that they have three different standards.
