Stephen, have you heard of mew?  Check out http://www.mew.org

It support MH style mail folders, but without using mh.  In mh,
folders are directories, and messages are files.  Great for think like 
storing a message in multiple folders.  It's represented as a link, so 
really only a single copy is stored.

Mew also has good support for MIME, PGP, and I think it has good
support for internationalization (at least Japanese).  Works under GNU
Emacs and XEmacs.

Do check it out.  BTW, you'll need to install the mh replacement
called im (Internet Message).  It is written in Perl.
