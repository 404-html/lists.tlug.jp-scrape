Considering the exorbitant prices of some of the mentioned devices and
that your child also plans to use it, I'd suggest going for a Nintendo
DS Lite plus a software called 漢字そのまま楽引辞典 (Kanji sono mama rakubiki
jiten). It's cheaper, has more functionality besides being only a
dictionary, and the software itself provides a nice set of features
such as a virtual keyboard, writing recognition using the stylus pen,
a list of kanjis similar to the one you intended to write, J-E, E-J,
J-J dictionaries, a small flashcard application, etc.

Best regards,

Marcelo
