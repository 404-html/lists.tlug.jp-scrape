Well, I installed mlterm. It is slightly better than Gnome
terminal in that it doesn't grab (ASCII) spaces abutting Japanese
text, but it does grab parentheses. Clicking on (ひらがな) gets the
( and ) as well as the ひらがな, whereas clicking on (hiragana) just
gets hiragana. Not very helpful, or consistent.

Also, since I have IIIMF running with Gnome Terminal, Firefox, etc. I 
sort-of expected it to be there with mlterm. It wasn't. I tried to
repeat it with the iiimf option turned on in configure, but the 
make blew up, whingeing about a missing "iiimf.h" file. Poking around
the mlterm lists, all I can find is mentions that iiimf seems broken.

Anyone got good advice about building an iiimf-capable mlterm?
