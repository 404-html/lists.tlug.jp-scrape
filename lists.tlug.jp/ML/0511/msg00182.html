Good point

I need 3 1/2 inch, 2DD discs.
As someone mentioned, I think that "dime-a-dozen" HD disks work fine. I had forgotten, but when I read that, I just remembered that I discovered with my old Ricoh word processor (a *very* stupid purchase in about 1988 when I could have bought a computer!) that it worked fine with HD disks. So I think there is absolutely no need for the old DD type. With MO disks, there is a *mechanical* difference on the disks that prevents one type from being reformatted as another type, but with floppies, they're purely magnetic, so that's not the case. Just reformat HD disks for whatever old machine you need the disks for. Obviously this only works in one direction - you can't use DD disks as HD but you can use HD disks as DD.
