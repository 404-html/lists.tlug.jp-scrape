I think you've conflated two things. Lookup is a regex etc. package in C.
There was an old Perl system for glossing text, but it hasn't been sighted 
for years.

Straight glossing with EDICT is not that good. In WWWJDIC  I use a big
(650,000 entry) composite file with priority ranking, plus auxiliary
stuff to undo verb inflections, etc. etc.
