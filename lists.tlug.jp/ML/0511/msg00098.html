production experience required. Prefer bilingual (English and Japanese),

System Administration skills (main duties)
  - Strong Linux, xBSD, Solaris or other unix os skills. Has a lot of
experience setting up production servers (mail, file, web-Apache).
  - Strong experience with Communigate required. If not, then strong
experience with Sendmail, PostFix, Qmail and/or EXIM will be considered.
  - Strong network skills: knows Samba, NFS, Atalk, MySQL, Postgres, dhcp,
bind, iptables, bash/perl/similar shell or command line, and other unix
services. Cisco knowledge a plus.
  - Mac OS X Server (Tiger) knowledge a plus
  - Experience setting up Open LDAP service
  - Can troubleshoot and look up solutions on his/her own.
  - Acquires knowledge quickly.
  - Works and communicates well with others.

Secondary duties (escalation for Help Desk Staff)
  - Mac OS 9, 10 knowledgeable.
  - Windows 2000 Pro, and Windows XP knowledgeable
  - Microsoft Office knowledgeable. Able to diagnose and resolve problems
- on his/her own - with Word, Excel, Powerpoint, Outlook.
  - Microsoft Outlook 2000, XP, and/or 2003 mail client knowledgeable.
  - Mac Mail.app knowledgeable.
  - Able to diagnose and resolve encoding problems  "mojibake" with mail,
browser and/or MS Office applications.
