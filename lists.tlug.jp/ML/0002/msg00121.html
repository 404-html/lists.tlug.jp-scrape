-------- Original Message --------

It sounds like a networking configuration problem if the card is
recognized
at boot time.  (You can double check the boot messages with the 'dmesg'
command.)

As root, the 'ifconfig' command will show you all of the interfaces that
are
configured in your machine.  If eth0 doesn't appear there, you should be
able to use RedHat's "control-panel" application to set up your network
configuration.  (Or, if you prefer doing it by hand, editing the
/etc/sysconfig/network and /etc/sysconfig/network-scripts/ifcfg-eth0
files.)
