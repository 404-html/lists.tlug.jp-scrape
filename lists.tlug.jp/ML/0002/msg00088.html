I use :

(i)   ghostscript  : gs -sDEVICE=pdfwrite -sOutputFile=foo.pdf foo.ps

(or the shell script 'ps2pdf' that essentially does the same thing)  

(ii)   pdftex /pdflatex

both of which are free.       

Of course LZW compression is not supported because of the (in)famous
Unisys patent, but both do a decent job. The resulting pdf look good on
acrobat reader.
