[tlug] Disintegrating Firefox Functionality

  • Date: Sun, 14 Nov 2010 01:40:29 +0900
  • From: Lyle H Saxon <llletters@example.com>
  • Subject: [tlug] Disintegrating Firefox Functionality
I've been experiencing increasing levels of Firefox dis-functionality
(running on Kubuntu 10.04-LTS) - mainly in the form of dead functions:

 - When sending an e-mail, selection of an attached file functionality
is sporadic.  Sometimes it works on the first try, and sometimes I
have to repeat the process as many as eight times before it takes.
Usually, I click on the file to attach - and then nothing happens.

 - When attempting to open an HTML file saved to disk, clicking on the
"Open" [URLを開く], produces no response.  For this feature at least,
using [Ctrl]+[O] works, but why is the menu option for Open broken?
This doesn't seem like rocket science - so how can it *be* broken
anyway?

 - Right clicking on a photo to save to disk is broken.  While
[Ctrl]+[S] works, that saves the entire page, and not just the photo.

Fortunately, all of the above dis-functionality is functional with Chromium.

Is anyone else experiencing problems with Firefox?  There are some
aspects to Firefox that I prefer over Chromium, so I'd prefer not to
have to abandon Firefox, but it seems to be increasingly broken.
What's gone wrong with Firefox - they were so good before....

Lyle
