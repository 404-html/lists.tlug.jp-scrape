I haven't used this myself but was recommended by a colleague.

http://backuppc.sourceforge.net/

Here is what he said about it:

We are using a program called BackupPC. Don't let the name fool you, it's actually very good at backing up servers as well. The best "feature" is that it's free. It also does very good datade-duplication and compression. If it sees the same file in multiplelocations, even on different machines, it will only store it once. 

It also has an easy to use web interface that allows you to recover individual files or entire directory trees.

It can backup via smb, rsync, tar, or rsync/tar over ssh. So you don't *need* to install anything on the client, but I highly recommend rsyncrunning on the client as it greatly reduces the amount of data that istransferred every night.
