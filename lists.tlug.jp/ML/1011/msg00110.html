I'm a big believer in stuffing as much RAM into a computer as possible
- especially with older machines.  Things should run a lot smoother
with 1GB compared to 512MB.  That's been my experience in any case.
I've got an old desktop Hitachi I use that has 1GB (that machine's
maximum) and it works fine with Ubuntu 10.04 LTS.

Incidentally, that じゃんばら place used to have different stuff at
different stores - so I don't know how carefully they post things
on-line.  I used to just walk into one of the stores in Akihabara and
see if they had what I needed in the glass case with the memory.  In
any case, I'm glad you found a source for more memory!
