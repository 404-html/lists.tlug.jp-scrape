On 1/16/19 12:33 PM, Stephen Lasseter wrote:

This is what currently works for me:

Distro: Fedora 29, KDE spin

Install software:
  sudo dnf install fcitx fcitx-kkc fcitx-skk kcm-fcitx
Fix start link (maybe don't have to do this anymore):
  mkdir -p ~/.config/autostart ; cp /usr/share/applications/fcitx.desktop ~/.config/autostart/fcitx.desktop
Add the following to ~/.bashrc or similar:
  export QT_IM_MODULE=fcitx
  export XMODIFIERS=@im=fcitx
  export GTK_IM_MODULE=fcitx
Run the fcitx setup from within KDE.  Usually I have to log out and back in once or twice until it works.  I've not got around to looking into why.
