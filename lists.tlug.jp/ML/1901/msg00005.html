I know at one point, first found it on FreeBSD, but also had the same
problem on some Linux installs, I had to run a little script to get ibus to
switch to and from Japanese.  I mention it on my page on Japanese input
http://srobb.net/jpninpt.html in the FreeBSD section.  Go to the FreeBSD
section and do a search for engine, the script is there.

But, I gave up on ibus, finding it to get worse in most versions of Linux
and FreeBSD, at least with the window manager choices I was making and
switched over to fcitx. I find it more reliable.
