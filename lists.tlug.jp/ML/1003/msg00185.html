Hi folks,

Some PostgreSQL developers are planning to have a hackathon
event for the upcomming release PostgreSQL 9.

* PostgreSQL 9 TestFest Japan on Apr.4 
  http://pgsnaga.blogspot.com/2010/03/postgresql-9-testfest-japan-on-apr4.html

PostgreSQL 9 is now under "alpha" phase, and introducing many advanced
features for users. So testing and fixing bugs are very important
for the production release.

We will have several kinds of tests on several kinds of platforms,
and online conversation with SFPUG (SanFrancisco PostgreSQL User Group)
at that event.

If you're interested in PostgreSQL and this event, please join us.

Thanks.
