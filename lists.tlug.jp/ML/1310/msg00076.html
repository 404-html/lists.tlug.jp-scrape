If you have a fast connection in Japan and use the JP mirror, you will 
probably be much happier with the results. I have a Linode running in 
the Tokyo data center and i'm always pleased with how fast it pulls down 
updates.

You can switch your Debian mirror after the install so that you can get 
better speeds downloading updates. Take a look at /etc/apt/sources.list 
and change the hostname part of the URLs to "ftp.jp.debian.org". Do not 
change "security.debian.org" though; that one should be fine as-is.

If you have to add any additional repositories, i'd recommend not 
putting them in the main sources.list; there is a directory where you 
can put them: /etc/apt/sources.list.d/
Just create files in that with a .list extension and the 
apt/aptitude/etc. will pick them up.
