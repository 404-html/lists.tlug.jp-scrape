Unfortunately, I didn't keep that install.  I'm going to put it down as a
one time oddity.

Today I'm finding that as long as I choose ibus as input
method, on two quick test installs, it works in Lubuntu on everything but
Libreoffice.

However, if I then install xubuntu-desktop, Libreoffice will then allow me
to input Japanese. 

So, while it does still seem there are some issues with Lubuntu and
Japanese input--I think I also had to add export XMODIFIERS=@im=ibus to
.bashrc on another quick test install, it's nowhere near as bad as I
thought. 

As Ray said in the other response to my post, it's not hard to change Super
key to something else--actually, if using Lubuntu as opposed to a custom
install with DWM, I can leave it with the super key.
