On Thu, Jun 14, 2012 at 07:06:21PM +0900, Jawaad Mahmood wrote:
> Hi guys,
> 
> I'm curious, though, if anyone here has purchased a System 76 laptop.  Their
> top of the line, fully upgraded, doesn't cost that much - and is shipped with
> Ubuntu.

I've used rjtech, which seems to have something very similar to sytem
76.  It looks, judging from the pictures on System 76's site that
they're using Clevo laptops, which is what rjtech uses.  The only issue
I've had is that sometimes the wired ethernet card takes a little while
to come up.  


http://www.clevo.com.tw/en/products/index.asp


System 76 didn't say (at least in the quick look I
gave) whether the hard drive is 5400 rpm or 7200 rpm, which could make a
difference, but on those machines, it's easy to change the hard drive if
you wish.

I was not crazy about the keyboard, but I didn't hate it either--it's a
little mushy, but I was still able to type for it for long periods of
time.  The model I got (from rjtech) was the 14", with one of the lower
starting prices, so that might be a factor. 

http://rjtech.com/shop/index.php?dispatch=products.view&product_id=30124
