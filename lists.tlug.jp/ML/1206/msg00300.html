And I'm glad you asked.  I should have specified that that is one more
little thing making it easier.  From what I understand, one no longer
needs to use localinstall --nogpgcheck.  (Not sure about the gpgcheck,
maybe skype's gpg was OK). 

In other words, I downloaded the skype-4.0.0.7-fedora.i586.rpm

I then did a cd to the directory where I downloaded it.

I then ran

yum install skype-4 (at that point I hit the tab key, which completed
the name of the rpm)


It then downloaded all the necessary i686 libraries.  As I understand
it, localinstall is no longer necessary. I'm not 100 percent certain of
that, but it seems to be the case, judging from my experience with this
locally downloaded rpm.
