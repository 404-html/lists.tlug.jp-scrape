Brett,

Any update on this Dave?

Actually, yes. They've just released, within the last few weeks or so, a version 0.7 which is based on the KDE 4 system.

The interface, while slightly prettier, isn't really too different. Which is okay, because I always felt they were on the right track and it's easy to use.

The main thing is that it's a lot more stable, which you'll probably appreciate if you were using the earlier versions. I left it for a while because it crashed far too frequently.

I haven't really pushed it yet, but it's definitely an improvement over the previous builds. I'm still of the opinion that this could become the de facto video editing GUI for Linux because of it's accessibility.

I used the "builder wizard" to install it, which is pretty cool in how it works.

Download options are here:

http://kdenlive.org/content/2008-09-26-download-kdenlive


Hope that helps.
