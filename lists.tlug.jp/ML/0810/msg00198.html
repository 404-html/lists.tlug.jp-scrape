jep200404 wrote:

Are you sure that sdc is the internal DVD drive? 

Yes. Disk and file systems gives the information.

On my 8.04 Ubuntu installation, the DVD drive shows up as /dev/scd0, not /dev/sdc. The /dev/scd0 line in your fstab looks similar mine, with the exactly the same mount points. /media/cdrom0 makes sense as mount point to a DVD drive. 

Information from disk and file systems, external USB hard disk not attached

sda1 292.4GB partition mount point /
sda2 1KB partition
sda5 5.7GB swap file mount point none

sdb1 burner DVD-RAM mount point /etc

The /dev/sdc1 line in your fstab has a mount point of /home. Both the device name and the mount point make sense for a hard drive like medium, but not for a DVD drive.

Please tell us more about /dev/sdc. Do you have another hard drive or flash drive involved? 

No other hard drive than the devices already mentioned.
No flash drives.
