Running Debian Lenny, when I do the most recent upgrade including Xorg, I lose support for my Radeon X600 (RV 380) driver for a 1280x1024 screen resolution (it reverts to a lower Vesa resolution). Today I spent most of the day trying to resolve it, without success.

This is apparently a known issue, as one soon finds out Googling the relevant keywords. I have found a number of threads on various forums that discuss it, but nothing that worked for me. According to the Xorg site, they presently support this driver, and I do have all the Radeon Xorg driver packages installed on my system.

Running dpkg-reconfigure xserver-xorg simply regenerates the same xorg.conf file that contains no specific settings. I have tried hand-editing this file following examples that I found in a few of the threads on the topic, but I usually end up losing X altogether after I do this.

Has anyone come across this problem and resolved it?
