> times I unplug it and replug it, it doesn't show.

It's time for some diagnostics. 

1. Boot computer without the compact flash. 
2. Execute the following command as root: 

      dmesg >dmesg.before

3. Plug the compact flash into the computer. 
4. Execute the following command as root: 

      dmesg >dmesg.after
      diff dmesg.before dmesg.after

5. Post the results of the diff command 

      Avoid typing the results into email by hand. 

         Copy and paste is good. 

         Saving results of diff to a file, 
         and then attaching that file to the 
         email is also good.
