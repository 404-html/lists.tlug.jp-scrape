I saw this on comp.os.linux.misc    I thought some people using
Emacs as their mailreader might find it useful:

:    The mh-e.el package will compose a MIME message for you.  Unfortunately,
:    you will not be able to view mime message you receive.

: Use Mew instead of mh-e. 

:       ftp://ftp.aist-nara.ac.jp/pub/elisp/Mew/mew-current.tar.gz

: --Kazu


Also, I have uploaded:

	mpack-1_5-linux_tar.gz

to ftp.twics.com

in the "unix" sub-directory.

The other thing is that I used Ctrl-T the last time to attach an
ASCII file in Pine.  This converted it into BASE-64.  I should have
used Ctrl-R to leave it in ASCII.
