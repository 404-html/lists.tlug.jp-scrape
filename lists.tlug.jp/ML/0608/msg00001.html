As is always the case when the kernel gets upgraded, my VMWare needs to be reconfigured.
In it, it says it needs the location of C header files. Usually it finds it automatically. 

But this time when it asked:
What is the location of the directory of C header files that match your running 
kernel?

And suggested that I use:

[/usr/src/linux/include]

So then I tried to specify the right directory:

/usr/src/kernel-headers-2.6.17.7/include

Huh?
