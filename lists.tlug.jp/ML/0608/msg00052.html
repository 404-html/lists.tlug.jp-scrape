Seriously... I also used to dread kernel updates until I started using
Gentoo. Since then, I've learned to look forward to them.

The trick is that you have to find out where the '.config' file from your
previous kernel has been stashed away. If you copy that file to .config in
your newly d/loaded kernel source tree, the new kernel uou build will look
pretty much like the original one and that allows you to move forward from
there instead of wasting time re-creating previously existing functionality.
