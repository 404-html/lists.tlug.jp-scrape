Just tip that I thought could be helpful for some people  
I often like to read those Japanese e-mails in ROMAJI. 
Here is a mutt macro that does the trick for me:

Get KAKASI from http://kakasi.namazu.org/

Then add a macro:

# translates message into romaji
# set pipe_decode=yes
# macro pager \er "<pipe-message>iconv -f UTF-8 -t SJIS | kakasi -Ha -Ka -Ja -Ea -ka -s|less  <enter>"

then use <ESC>R to magically convert everything into ROMAJI.
