[tlug] Mirror for MPlayer in Japan needed

  • Date: Tue, 29 Aug 2006 08:33:16 +0200
  • From: Attila Kinali <attila@example.com>
  • Subject: [tlug] Mirror for MPlayer in Japan needed
Moin People,

I assume that most of you know what MPlayer[1] is, so i'll get straight
to the point. Currently most of MPlayers mirrors are located in
Europe. This results in a very high bandwidth-delay-product for
Japan, resulting in very slow downloads (even after some serious
tweaking i was not able to provide more than 200kByte/s/connection
from the main server located in Switzerland to Japan).

Thus i would like to have another mirror to serve our users
in Japan.

What we need is a well connected machine (at least 10Mbit/s)
which runs a webserver and if possible an ftp server.
Required diskspace is currently 500MB.
The machine does not need a lot of CPU as all pages are static,
but it should have a good portion of ram to have enough disk cache.

The bandwith requirement is around 1TB/month, so anyone
thinking about providing a mirror should better have a flat rate
connection.

An howto on how to create a mirror is available from
http://svn.mplayerhq.hu/mplayer/trunk/DOCS/tech/mirrors/mirror_howto.txt?view=markup

Any help is greatyl appreciated.

			Attila Kinali
