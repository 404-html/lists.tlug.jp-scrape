On 18/08/06, scott <scott@example.com> wrote:


[Matt Gross wrote:]


*     RAID 5 (Software or Hardware - what do you recommend?)

software raid 5 on 4 SATA drives


I would actually recommend hardware RAID, if you have the yen. I have
heard good things about 3ware[1][2] cards and Linux.

Not that there is anything wrong with software RAID, but why not
offload the RAID from the CPUs if you can?

Cheers,
Josh

[1] http://www.3ware.com/
[2] http://japan.3ware.com/
Hardware raid is of course better on some levels, but I've also heard valid arguments for software. You might check out the comparisons. Also be very careful with hardware RAID, do you're homework. It's generally only the more expensive cards/systems that do it right and support all OSes at a base level. Make sure it will work with Linux. 

PS.
400,000 yen is a lot for a system. I'd expect that a system like this could be built for around 150,000-200,000 unless you're really going top of the line.
