Ryan Shaw <ryan.shaw@example.com> wrote:

> I'm on OCN ADSL. Does NTP need ports open on the client side?

Make sure udp port 123 is open.  If you've a firewall or some sort of router device, make sure you are allowing udp port 123 out AND IN (it's probably already getting out) from ntp.cyber-fleet.net.
