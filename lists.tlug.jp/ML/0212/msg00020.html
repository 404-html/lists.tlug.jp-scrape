Or possibly LPD mode--FWIW, I have a little page of stupid cups errors
at
http://home.nyc.rr.com/computertaijutsu/cups.html  If you do it in
JetDirect mode be sure to change ServerName in /cups/client.conf.  I've
found success, with both JetDirect and LPD by just putting in the IP
addie for the Server name.

Also, I think you said you'd thrown RH 8.0 on something--if that's the
box in question, you can do redhat-switch-print (or possibly
printers--hit the tab key). If you don't, then you'll wind up with other
problems as lp and lpr will try to work with Lprng.  (Not sure if that
command is in earlier versions of RH--when did they start throwing Cups
on their distro?)
