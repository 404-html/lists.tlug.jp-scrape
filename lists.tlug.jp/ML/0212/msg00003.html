"Eric O. Flores" wrote:

> Diedrich,
>
> I am not sure how to reach the internet gateway, from the server system I can
> reach anything from within the network I can only ping each computer.
>
> I assigned the following values to eth1 192.168.0.1/255.255.255.0 the eth0 I
> am leaving it as DHCP for ISP connection via PPP0 (DSL), the other PC I setup
> ETH0 as 192.168.0.2/255.255.255.0  with the gateway set as 192.168.0.1.  I
> can ping 192.168.01 & 02.

Eric,
192.168.*.* is private addressspace (see RFC 1597), you will never be able to
"reach the outside" unless using some technique like NAT, or proxying. The
former is usually called "internet sharing" on the shiny distros. Try enabling
that. If it doesn't work you need to check your routing also. And read the NET &
IP Masquerading howtos.
