Matthew Tippett wrote:

> I was wondering if anyone could give any guidance on some useful
> suggestions on japanese input and use under Debian.

I'm using kinput2-canna-wnn with freewnn-jserver.

I have this in /etc/X11/Xresources/kinput :
 *inputMethod: kinput2
 Kinput2*useOverrideShellForMode: true
 *preeditType: OverTheSpot

And the following in /etc/X11/Xsession.d/60_kinput2 :
 /usr/bin/X11/kinput2&
 export XMODIFIERS="@example.com=kinput2"
