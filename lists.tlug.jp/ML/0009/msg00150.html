There is a typical problem where X and GPM argue about who owns the
mouse, and GPM usually wins.

My preferred solution is to run GPM by hand when I'm using a VC, which
is rare for me.

What is supposed to work and mostly does is to point GPM at the mouse
(typically /dev/mouse or /dev/psaux), and to tell X that the mouse is
at /dev/gpmdata.  /dev/gpmdata is simply GPM echoing all mouse data as
is.  But this means that X will not work unless GPM is running AFAIK.
