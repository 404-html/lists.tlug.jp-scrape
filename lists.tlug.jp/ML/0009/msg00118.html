As a preliminary to Steve's recommendations, have you recompiled your kernel
since installing the OS on this box?  PCMCIA is fussy about the kernel
version and configuration; it is prudent grab the sources for the PCMCIA
package, recompile and reinstall it anytime the kernel itself is altered.

I don't know enough to distinguish the changes that make this necessary, so
I've made it a mechanical habit; anything more signficant than compiling a
new module invites a PCMCIA recompile.
