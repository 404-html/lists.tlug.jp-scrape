Philip Greenspun, author of "Philip and Alex's Guide to Web Publishing
" ( http://www.arsdigita.com/books/panda/ ) and founder of ArsDigita 
Corporation
  (http://arsdigita.com) is going to be giving a talk on the subject of
web site design for online communities  at Digital Hollywood's building in 
Ochanomizu, 1st floor, Thursday evening,  October 5, from 19:00-21:00. 
Japanese translation will be provided.

While the talk will be about general issues of web service design for 
online community sites,
this talk may of specific interest to Linux users because the ArsDigita web 
server
toolkit (ACS) is distributed as free software under GPL, and is supported 
under Linux. The
OpenACS project (www.openacs.org) has a port of this code to use Postgres 
as the database backend rather than Oracle.

Here's a map to Digital Hollywood's building near Ochanomizu JR station:
http://www.dhw.co.jp/git/map/index.html

Digital Hollywood 2001 Building
2-3 Kanda Surugadai
