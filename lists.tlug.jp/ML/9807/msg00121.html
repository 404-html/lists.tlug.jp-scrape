To my knowlege there are 2 good methods to emulate a mac
other than executor.  They both are free, and they both need a fast
cpu, but they both emulate a macintosh far better that executor.

The first method is to use a program calles vMac.  This is available
at http://www.vMac.org and will emualte a macintosh plus.  There
are several drawbacks to this method the first being that it 
faithfuly impliments the 4mb ram limit present it that model of 
the computer.  The pluses are that it is free and it runs at a food
speed on anything above a 486DX2

The seccond method is a lot harder to setup, but is very good. 
You will need a VERY powerful cpu in order to do this because
it involvs emulation on two levels. You will need to get the UAE
Amiga emulator for Linux, configure it and then install 
ShapeShifter in your Amiga enviorment.  This done sucsesfuly 
will run almost any program that a m68020 will run.

Both methods require that you make a hardisk image.  These
are available from the VAS there is a link to the VAS from 
http://www.vMac.org

In addition to a harddisk image you will need a copy of the mac
ROM.  For vMAc you NEED TE MAC PLUS rom, NOTHING else will do.
For UAE+ShapeShifter, you will need any 512k mac rom.

ALso if you are going to use UAE you will need a copy of the Amiga
Kickstart rom v3.0 and Workbench 3.x or above.

All the emulators mentioed above come with utilities to obtain 
the mac rom from a existing mac, and it is not imposible to find
copies of them floating around on the net.

Good luck
