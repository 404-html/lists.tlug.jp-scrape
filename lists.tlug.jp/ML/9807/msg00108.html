On Wed, 8 Jul 1998, Eric Standlee wrote:

> I was just wondering if anyone knows if there is a Mac emulator and if that
> emulator can run a somewhat current version of FileMaker (a Mac database
> program).

There is a commercial one called Executor at http://www.ardi.com

I have never used it, and don't know anyone who has, so can't vouch for
the product at all. There is a demo available. At $150 for a single
license (they know it's expensive; they make you wade through several
pages to find the price list) it would seem more cost-effective to work
with one of the database packages already developed for Unicsen. But I
suppose that would probably not make your friend immediately happy.

> I have a friend who might switch to Linux if I can show him that his
> precious FileMaker database can run under Linux.

Well, if he has a powermac, he should be able to set up a flavor of
Linux on that, which can still leave his FM database intact when he
boots up the MacOS.
