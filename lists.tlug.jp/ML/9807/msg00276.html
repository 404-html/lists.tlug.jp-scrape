Just FYI, everyone, TL 2.0 USA and 2.0 J are available by anon-FTP at the
following locations:

2.0 USA:

ftp.turbolinux.com:/pub/turbolinux-2.0

2.0 J:

ftp.pht.co.jp:/pub/turbolinuxj/2.0


To do an FTP install, you need to go to the images/ subdir and get the
boot.img, supp.img, and (possibly) extrahw.img, and write them to
floppies, then boot the boot disk and select FTP install when prompted.
This procedure is the same for both USA and J.

Alternatively you can download the entire tree (not necessary to get all
packages, but preserve the dir structure) onto an ext2 filesystem (vfat
mangles things) and install from an HD partition.

*Note - you can NOT install by FTP over a dial-up connection.
