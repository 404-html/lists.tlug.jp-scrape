Mike wrote: 
> What sort of problems did you encounter with Redhat 5.0?

 I tried a couple of months ago. When I tried to use ezinst, segmentation
error occured. So I couldn't install PJE on Redhat 5.0.
 From Redhat 5.0 they use new libraries, while PJE & JE uses older libraries.
PJE library checking is strict. So it causes segmentation error. To avoid
this you need to use some tricks or to use JE, whose error checking isn't
strict.

fj.os.linux, those who uses PJE still uses Redhat 4.2.
