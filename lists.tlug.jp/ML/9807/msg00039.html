> I'd like 5.1, but I may just stick 4.2 on it for the time being.  Some
> of the TL Japanese packages need glibc, though.  No matter which I do

What you _could_ do is this:

* Install 4.2
* Compile the 5.1 RPM.  Perversely, the 5.1 RPM is itself a type 9 RPM.
* Install the 5.1 kernel
* Install the glibc runtime package
* Install everything else

That's sort of what I did.
