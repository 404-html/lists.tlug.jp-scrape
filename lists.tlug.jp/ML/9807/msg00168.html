On Thu, 9 Jul 1998, Totoro wrote:

> On Thu, 9 Jul 1998, Hiroshi Hasebe wrote:
> 
> > A good window manager has to be shell-user-friendly too. :-)
> > With fvwm, for example, a lot of key bindings can be defined, but I
> > usually end up reaching for the pointing device. Anyone using a good
> > keyboard-oriented, good-looking (like wmaker) window manager?
> 
> AfterStep has quite a few keymaps set up with it (1.4.5.3, anyhow).
> I had been using WindowMaker for quite awhile, but I like AS better
> now, and there a some nice add-ons by users available.

Yeah, when I first shopped for a Windowmanager for TurboDesk, I originally
wanted to use fvwm.  Then, I saw Afterstep and thought it looked nice, and
best of all it had just turned 1.0 stable, so we went with that.
Windowmaker can be more visually appealing at times, but it's still beta
and not as stable as AS, and AS can be pretty darned attractive too if you
put your mind to it :)

Anyone here use Enlightenment?  It's high on graphics and special effects,
low on stability, but it could be fun once it matures a bit.
