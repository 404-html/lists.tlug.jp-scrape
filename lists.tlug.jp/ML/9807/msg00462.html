Actually, there is a perl module which already does some HTML parsing.
It's part of the libwww bundle. I was just looking at the Parser.pm that
is part of this. It's still qualified as "alpha" by the author. But it
can clean up some areas, and it wouldn't be hard to add in some more.

HTML::Parser.pm

Maybe you already have it; I haven't looked at what all Scott included
in the TL distribution.

Look at one of the CPAN sites for more info on perl modules. There is
also a module called CPAN, which allows you to easily add new/updated
modules from (where else?) your favorite CPAN site. One that I find
works well here in Japan is:

ftp://ftp.eos.hokudai.ac.jp/pub/lang/CPAN/CPAN/
