Eric,

I stand corrected on the upgrade option for Red Hat.  As Matt pointed
out, this can indeed be done.  However, he does also mention some good
points for doing a clean slate upgrade.  Since this is a fairly new
install, it might be easier to do just that, saving anything important
that you have in a backup and then putting it back later.  As the Red
Hat manual points out (P.46 of the 5.1 manual), configuration files do
have a tendency to change format over time, so be very sure about what
you copy back from your 4.1 version to 5.1, and especially be sure to
back up the default 5.1 configuration files before replacing them with
files from 4.1.

Cheers,

Jonathan
