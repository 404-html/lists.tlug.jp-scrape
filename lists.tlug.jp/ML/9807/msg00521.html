> 1. Every time i try to mount my CD-ROM, it says:
>     mount: can't find /dev/hdc in /etc/mtab or /etc/fstab
>     How do i resolve this?
Answer A:
 #mount -t iso9660 /dev/hdc /mnt/cdrom

Answer B:
 Add such line in /etc/fstab
/dev/hdc              /mnt/cdrom              iso9660 noauto,ro       0 0

I'm using Redhat5.1. If you are using Slackware '/mnt/cdrom' should be
'/cdrom'.

> 3. Can i read MSDOS floppies and/or CD-ROMS? If so, how?
Use following command to read MSDOS floppies
mdir
mcopy -t a:*.txt .
mcopy a:* .
