The 4c machines may not work with RH 5.1 yet, but the 4m's sure do.  I 
installed 5.1 on an LX last night, and it was the easiest linux install
I've had the pleasure to do yet.

The install menuing is almost identical to RH 5.1 for the i386, but
there is no need to do any X setup at all - it's all automatic.  It
seems a lot faster than SunOS 4.1.4 as well.
