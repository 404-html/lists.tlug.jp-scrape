> One can use gpm as a sort of mouse server for X.  Set the XF86Config to
> /dev/gpmdata as type MouseSystems, and feed gpm the -R flag.  Really
> useful when I switch between quake2, console, and X -- everything is being
> handled by gpm, so my mouse won't mysteriously stop working when the X
> mouse handler stomps on quake (or vice versa).
