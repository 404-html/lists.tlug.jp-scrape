Hi all,

On 26-Jul-98 Jim Schweizer wrote:
 
> I haven't had much luck figuring out how to substitute a DOS
> carriage return with a new linefeed, ASCII code 10, in vi.

Turns out I made a mistake... It's not a DOS file after all but
a MAC file and there aren't any linefeeds.

Jim T. suggested:

"tr '\r' '\n' <infile >outfile"

which works great outside of vi so I guess I'm all set.

Special thanks to the computer science club members over in U of
Waterloo!
