> Is it that easy?

Absolutely not. 

Almost all of my computers will run Windows and Linux,  
but none of them will be able to run Windows and Linux 
at the same time under Xen, because I don't have the 
right CPU. 

You need the right CPU to support virtualization under 
Xen without having to modify the OS.

   http://tlug.jp/ML/0604/threads.html#00224
