> 
> First I tried to print a test page. It shows the job as complete. The 
> printer does nothing. I then click to reprint the job, and CUPS displays

> 
> Error

I wonder if it could be the Location Deny,Allow thing.

For fun, you could try changing that in cupsd.conf (as I recall,
eventually you found it, right?) :) 

By default it denies everything (though I think it's supposed to allow
from localhost, I seem to remember that not working either at some
point.) 

Try changing Deny to None temporarily and restart cups.  If that works,
then you you can fiddle with it to allow what you want to allow. 

This is in the Location section and will look something like


<Location />
Order Deny, Allow
Deny From All
</Location>

(It's line 767 in an edited cupsd.conf file on a FreeBSD machine if that
helps.)
