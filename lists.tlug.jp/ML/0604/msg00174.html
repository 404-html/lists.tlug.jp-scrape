> dave@example.com:~$ ls -l /dev/ttyUSB{0,1}
> ls: /dev/ttyUSB0: No such file or directory

> This is obviously a rather critical problem.
> But what's the deal there, because according to the following output,

Well I suspect it is one of two problems, firstly the driver is no
working with the device,  or udev is misconfigured and is not creating
the nodes.  Try creating static device nodes. i.e.

# mknod /dev/ttyUSB0 c 188 0
# mknod /dev/ttyUSB1 c 188 1

If it is a udev problem then creating static device nodes should work.
