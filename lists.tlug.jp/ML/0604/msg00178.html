> pi_bind error: /dev/ttyUSB1 No such file or directory

Once again, there's your answer right in front of you.

Check that you're not using udev. If you are then fish around in /dev/usb
to find something that looks like it could be "tty number 1". If not, then
create the device nodes.

# mknod /dev/ttyUSB0 c 188 0
# mknod /dev/ttyUSB1 c 188 1
