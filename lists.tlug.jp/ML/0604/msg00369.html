Hello TLUG,

I just released im-ja 1.5. 
This release features a new input mode that allows one to look up kanji
through its radicals. Someone wrote about the need for such a feature, so
here it is :)

See the ChangeLog for more details:
  http://im-ja.sourceforge.net/ChangeLog

I built a bunch of binary packages for most popular distros,
they're available from this page: 
  http://im-ja.sourceforge.net/binary.html

The source can be found at the project webpage as usual:
 http://im-ja.sourceforge.net

Enjoy.
