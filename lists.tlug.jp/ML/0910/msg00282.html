I'd be curious which collation you are using. As far as I know there is no utf8_japanese collation, so I have been using utf8_general_ci and utf8_unicode_ci. The documentation says that both are case-insensitive. But I found that for searches utf8_unicode_ci does not distinguish between hiragana and katana and also not between は and ば while utf8_general_ci does.
