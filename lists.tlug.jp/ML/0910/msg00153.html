> I am a neophyte with Linux and am not real sure here, but IIUC, running
> the shell script on bare console means booting at runlevel=3,

Firstly, please do trim your lines so that they're no longer than 72-76
characters.

http://linux.sgms-centre.com/misc/netiquette.php#wrap

Secondly, there's no need to reboot as such. You can change the runlevel at
will. Fire up a console, use "su -" to become root and then run "telinit 3".

Alternatively, if you need to be in single user mode, "telinit s" should do
that for you.
