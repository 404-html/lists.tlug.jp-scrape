Moinmoin could be an alternative. No database, just python
(cgi or apache mod).
http://moinmoin.wikiwikiweb.de/

The user rights are easy/trivial to setup (rights are defined
per page (to override default ones), user groups are simple wiki pages).
Also, by default, your login name is a wiki page, that you can edit as
you want (if you have the right to create it).

With moinmoin, the only admin job would be to maintain the "tlug" wiki
page, which could contain the list of tlug users logins.
These ones could create/edit pages for instance.
An "admin" group could also be allowed to delete/revert, etc...

Everybody can subscribe to any page (with regexp, such as .* for
everything), to receive notice when there is a change.

Very simple admin in fact, which could be a factor for the lucky guy
who will take the job ;-)

In latest versions, there is also a wisiwig editor, for the lazy ones ;-)
