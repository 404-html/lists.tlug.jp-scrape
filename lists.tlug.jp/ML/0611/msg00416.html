> Next I need to figure out why that isn't autodetected by Xft and/or
> the X server.

Do you have a "DisplaySize" entry in the "Monitor" section of your X
configuration file? Without it, X assumes your screen resolution is
72dpi IIRC, which results in ridiculously microscopic fonts on high
resolution screens.

If not, add "DisplaySize width_in_mm height_in_mm"

Note that this affects both the UI font size (what you want) *and* font
size of web content if FF is set to use the system screen resolution.
