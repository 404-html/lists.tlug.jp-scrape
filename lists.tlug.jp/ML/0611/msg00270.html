wrote:

> Today in the TLUG meeting someone (Josh I believe) mentioned that it
> is possible to subscribe all your email addresses to submit emails,
> and restrict the listserve to only send to one of them.
>
> How is this done?

Subscribe all the addresses from which you plan to send messages to the
list, then set to "vacation" mode all but the one at which you wish to
receive messages.


It is probably easier to use the web interface, rather than the mail
one, for this:

http://www.tlug.jp/mailman/listinfo/tlug
