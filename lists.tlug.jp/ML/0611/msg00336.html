You can get a fixed IP from ASAHI Net at an affordable price, 
either through BFLET'S / with FLET'S fiber-optic or FLET'S ADSL.

You can only have one Fixed IP address per User ID, (as an individual member)
but you can get second or more User ID with it's own Fixed IP address 
connected through the same line giving extra Fixed IP address for your location.

There is also an introducer program, so if your interested in signing up 
via ASAHI Net and you want to contact me offline before you sign-up, 
we may be able to figure out how to help the (introducer) proceeds from your 
sign-up go to TLUG.
     https://asahi-net.jp/en/join/campaign/syokai.html
