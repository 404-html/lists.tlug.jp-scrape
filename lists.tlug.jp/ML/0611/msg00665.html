I don't know how many times you're going to post this question, but
once again, any DVD playback software built around libdvdread and
libdvdcss (mplayer comes to mind, or, if you want the navigation
capabilities, xine) doesn't care what the DVD drive's region setting is.
My own DVD drive is set to region 2 but I can still watch region 1 DVDs
no problem.
