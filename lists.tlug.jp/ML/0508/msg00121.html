I have to admit I don't really know what information resides or should
reside in the directory .uim.d (I don't have that directory), but I do
know that my uim produced weird output (kanjis but not japanese) until
I put the line

(define default-im-name 'anthy)

into the file .uim, so I would suggest creating the file with that
content. It tells uim to use anthy as its conversion engine instead of the
default (Chinese?) option.
