My experience has been different though not a lot better. The interface
is very nice and polished, specially anti-aliasing, but the installer
destroyed my monitor (slightly old, though not *that* old) as soon as
it went into gui mode. Probably probed a little too much.

After completing the install and rebooting, both the mouse and keyboard
were not recognized and the only way I can use the machine now is
through VNC though it isn't really a problem.

One more thing I noticed is that folding@example.com gets killed within a
minute after starting. I haven't investigated this much, but I suspect
some new process management 'feature' is responsible. Since its an smp
machine, TLUG is losing a few units because of the problem.
