I am using the canna with kinput2 so it definetly works.  I am using

app-i18n/canna-3.6-r1
app-i18n/kinput2-3.1

> So it can't mount something. I've looked in /usr/share/canna, and there 
> are a lot of dictionaries there, just not the ones specified by the 
> error messages.

To start with the main config file is
/etc/canna/default.canna

The dictionary files are in /var/lib/canna/dic/canna/

if you look in
#less /usr/portage/app-i18n/canna/files/canna-3.6-r1-gentoo.diff
-DicDir   = $(cannaLibDir)/dic
+DicDir   = /var/lib/canna/dic

the following file lists all dictionaries.
/var/lib/canna/dic/canna/dics.dir
