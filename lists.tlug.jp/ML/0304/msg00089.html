On Tuesday 08 April 2003 15:18, Valter Fukuoka wrote:

> I like to install RH8 japanese edition.

Red Hat has been running a unified code base since 7.something (0,
maybe), so 
there is really no such thing as a "Japanese version" of Red Hat, per
se.  
There is a Japanese edition of the boxed set, which you can buy in
stores.  
The differences, if it follows previous patterns, would be:

1) Japanese-language manual;
2) Installer defaults to Japanese instead of English;
3) Some Japan-specific proprietary software is bundled (past bundles
included 
Atok for Linux and some other stuff).

However, there is no difference in the base distro.

If you're using the FTP version, all you need to do is install all of
the 
Japanese packages (or at least the ones you want) and set your locale to

Japanese instead of whatever locale you have now, and you have the
"Japanese 
version."
