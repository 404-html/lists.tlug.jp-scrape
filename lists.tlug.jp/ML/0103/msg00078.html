>> I have to create some pictures of kanji in TIFF format. Any hints about
>> which tool I might use for that? According to the gimp-FAQ Japanese
>> fonts seem not to be supported.

On ftp://ftp.cc.monash.edu.au/pub/nihongo/00INDEX.html is a Perl
program written by Jeffrey Friedl & enhanced by William Maton called
txt2gif/xtxt2gif It creates nice .gif files in various colours,
orientation, trtansparencies, etc. You feed it Japanese in an EUC-coded
file. It's set tu use 26x26 snf files as fonts, but you can tweak it to
do any size you like. To get tiffs from it, pass the result through
giftopnm and pnmtotiff.

I have also used it to make the JIS212 gifs (now pngs) that WWWJDIC
produces, and also to make the Korean and Big5 images that Silas Brown's
Access server uses.
