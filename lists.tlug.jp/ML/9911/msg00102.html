>Oh, yeah, can I have a pointer to the Partition Magic sources?  I've
>always been curious about why they're better than fdisk.
>
Well, I think that Partition Magic is better because is a no-destructive 
partition machine. Working under Windows everything goes smoothly and 
quickly without necessity to format the partitions and install the systems 
again. Also, one doesn't need to defragment their hard disks before the 
partition, it's done automaticaly during the creation of partition. As a 
bonus, the Partition Magic cames with the Boot Magic that is GUI boot 
manager very cool.

I don't have any problems with reconfiguring the lilo, but for the normal 
newbie user (even if one is not a newbie for Windows) that is not only a 
pain in the neck, it's something almost impossible to achieve...
