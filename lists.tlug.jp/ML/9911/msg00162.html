Hiya,


Try the classic three finger salute to X-windows,

CTRL+ALT+F2
CTRL+ATL+F1
CTRL+ATL+BACKSPACE

To get out of windows and try to configure your settings using Xconfigurator
(I think).

Choose the value according to your system's hardware, afterwards, it should
be okay (well not all the time).  If there seems to be more problems, the
I'd suggest checking out the documentation on X-windows (actually XF86
project).  And work on from there...

Hope that helps,
