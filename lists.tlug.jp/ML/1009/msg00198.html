On 09/26/2010 06:42 PM, Jean-Christian Imbeault wrote:

Thanks to all for the very instructive posts, as always.

I've decided to tempt fate and remove the unbuntu-desktop package so I
can do away with CUPS and a few other packages (e.g. Bluetooth) that I
will never use. My laptop is getting on in years (it just had it's
inverter go :) and I'd like to improve performance just a tad.
Its probably easier to just stop services rather then trying to remove packages. Many of the ubuntu dependencies are compile time so removing the packages will just cause crashes. If you want to install an optimized minimalist install, use Gentoo and compile out all unnecessary dependencies.
