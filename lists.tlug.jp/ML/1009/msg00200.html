Just wanted to give an update on my travails. In the end I went with
Puppy Linux and so far am quite happy. It was a bit of work trying to
get it to install and boot from a non-boot partition but I got there
in the end. (It's been so many years since I had to deal with
installation and boot loader; I'm very rusty).

I really wanted to go with XUbuntu but the live installation CD kept
crashing and giving me a blanks screen. Puppy also wasn't great and
kept crashing when I was running it off the Live CD and trying to
install a web browser. After I got puppy on the HDD though everything
went well. The only other hurdle was getting an IME working.

This ended up taking up most of my weekend. I hope I get recompensed
appropriately :)
