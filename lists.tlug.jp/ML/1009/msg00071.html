For the same quality it's generally cheaper to host in the US, so it's
only really worth using a Japanese service if you're running something
that will benefit from the lower network latency. Even then, you might
have some decent options like serving your applications off a US
server and using something like Amazon S3/Cloudfront to serve your
images etc.

But if you are going to use a dedicated host in Japan, I like Sakura Internet:
http://www.sakura.ne.jp/

Their phone support isn't great - sometimes you have to try a few
times to get an answer - but I've never had any problems with their
servers or their network so I don't need much support in any case.
