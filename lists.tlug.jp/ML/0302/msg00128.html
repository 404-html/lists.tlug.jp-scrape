Re:
"I had no better luck with RTF for the file that go me started on this quest
than I did with the native .doc from Office XP.  It was just as big, and just
as buggered up (and people who know RTF say that Microsoft's take on RTF is
quite buggered indeed), so that doesn't surprise me."

The only thing I've found that will consistently, reliably get rid of the
garbage in an MS-Word file is to save it as a .txt file.  That obliterates
formatting of course, but then you have clean text to work with and can usually
just reformat a new (clean) sheet more easily than try to work with the monster
mutated MS file.  Incidentally, I've found that when a Word file is
particularly mucked up, making an RTF file actually makes it bigger!  I don't
know the reasons for this, but have experienced it several times.
