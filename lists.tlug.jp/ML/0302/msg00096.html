  • Subject: [tlug] copy/paste got easier and harder
1) mozilla 1.3a does a better job at copy/pasting 
than mozilla 1.0. quite a pleasure to use 
by comparison. 
It seems to crash just a bit more, however. 
I guess it has to live up to the name unstable 
somehow. 

2) On RH8.0, I noticed that the data one 
is copying from point a to point b 
gets lost if one terminates the program 
of point a (e.g., kills the editor 
from whence the data is being copied). 
This has happened with gedit, yudit, 
mozilla, on gnome. It doesn't happen 
when point a is something ungui like 
vi, I noticed. 

Of course, one can simply remember 
not to terminate a program or close 
a window before pasting, but this is 
just a little bit inconvenient...

bug or a feature. :)
