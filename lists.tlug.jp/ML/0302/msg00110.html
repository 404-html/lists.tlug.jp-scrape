On Thu, Feb 06, 2003 at 08:53:12PM -0500, Josh Glover wrote:
> 
> ReiserFS is actually a great choice for a root partition, as it is great for
> filesystems with many small files. Root partitions are known to contain things
> like /etc, which on my relatively pristine system, contains about 7000 files
> averaging about 350 bytes.

I agree. All the partitions on my main system are XFS except for / and
/usr (/usr/local is XFS) since both of their average file sizes are way
below 100KB. It seems that XFS has the best performance for files over 
100KB while ReiserFS is best for many smaller files. I wonder how ext3
performs.
