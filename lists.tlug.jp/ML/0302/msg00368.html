> I have mousedev 3776 0 (unused)
> I also have usbmouse 1760 0 (unused)
You are trying to use two mouse drivers at the same time. Not sure if it's a
problem, but the kernel help (for usbmouse) says:

"Say Y here only if you are absolutely sure that you don't want
to use the generic HID driver for your USB keyboard and prefer
to use the keyboard in its limited Boot Protocol mode instead.

This is almost certainly not what you want.

The module will be called usbmouse.o."

Also for the keyboard, try the generic one (listed under input core).
