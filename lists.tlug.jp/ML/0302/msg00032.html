 TLUG - Tokyo Linux Users Group "nomikai"
===============================================


Date: 7th February 2003 (Friday)
Time: 20.00 - 22.00
Place: Tengu SHINJUKU PALETTE BUILDING 7F (Above Sakuraya)
http://www.teng.co.jp/tenpo/map/map137.html
http://www.shinjuku.or.jp/palette/

〒160−0023
東京都新宿区西新宿１−１−１
新宿パレットビル７Ｆ


Event: drink and eat (primarily a social activity)

Costs: will be divided up among participants
see http://www.tlug.gr.jp/meetings/index.html
for information on costs of previous events


This event is open to all Linux users and friends, and is
to promote casual Linux (and non-Linux) related discussions
over a few cold ones. Even if you're not a Linux user or a
LUG member, but want to hang out with Linux people, please come.


Tengu is

http://www.teng.co.jp/tenpo/map/map137.html


If you get lost, or simply want to get in touch with us, please
call:

BABA: 070-5561-0255
Mauro: 090-2492-0840


For more information, access the TLUG web page at
http://www.tlug.gr.jp


See you there!
