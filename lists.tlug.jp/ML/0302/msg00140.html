On Fri, 07 Feb 2003 22:23:57 +0900, Jean-Christian Imbeault
<jc@example.com> wrote to tlug@example.com:

> And worst yet, I have never manually created partitions ... I guess I 
> have to use fdsisk? If I use fdisk and guess wrong at a partition size, 
> location, will it mess anything up?

I don't think so. The partition won't mount - period.

The superblock will only be written to in the event of a successful mount
(in order to increment the mount count and run fsck if need be), which
implies that the starting point of the partition was found successfully. If
you mount the partition r/o you're not risking overwriting the beginning of
the next partition in the event that you guessed "too big".
