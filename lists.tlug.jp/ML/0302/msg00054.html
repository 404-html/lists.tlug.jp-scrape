  • Subject: [tlug] TTFs and RH7.3
When I changed to RH7.3 a few moons back, something funny happened to the
spacing of TTFs. It shows up particularly in Mozilla and Tk.

Normally it dosn't bother me, because I have other fonts selected, but if
I happen upon a page which specifies "Arial Unicode MS", it displays in a
hugely-spaced fixed format. For example, the page:

	http://www.arts.monash.edu.au/

looks like this sample:

	http://www.csse.monash.edu.au/~jwb/fontfunny.gif

This is being caused by the following in that page's .css:

	FONT-FAMILY: 'Arial Unicode MS', Arial, Helvetica, Sans-Serif

It's not just Mozilla - the same happens in Netscape 7, and also in Tk
apps when I select that font.

Any suggestions? Help?
