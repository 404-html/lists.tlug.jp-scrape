Hi all,

since some time sd-cards are around which besides offering 
i.e. 16GB of storage also span up a wlan and offer the files.

The cards contain an ARM cpu, the linux can be modified to
also offer other services.  

Nice small devices with storage and wlan:
http://haxit.blogspot.de/2013/08/hacking-transcend-wifi-sd-cards.html
http://www.wirelessgoodness.com/2012/08/29/transcends-first-wifi-sd-card-arrives-on-the-fcc-with-ios-and-android-apps/

Smallest linux system I have seen so far that can be fully
customized, great idea to use these for new purposes.

Christian
