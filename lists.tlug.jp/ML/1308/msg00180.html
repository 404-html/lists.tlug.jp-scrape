My own, by now very dated page, at
http://home.roadrunner.com/~computertaijutsu/jpninpt.html goes through the
following--done on Lubuntu, but might be close enough.

>From the start button, go to preferences, then language support.  There
should be an option to add and remove languages.  Select Japanese and click
apply, and it will then hopefully download the necessary pacakges.

After that, you should be able to run ibus-setup, and Japanese should be
available.
