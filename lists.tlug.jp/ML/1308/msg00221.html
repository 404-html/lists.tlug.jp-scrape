On Sat, Aug 24, 2013 at 10:50 PM, Darren Cook <darren@example.com> wrote:
>> Is there a way to install: Jim Breen's "WWWJDIC: Online Japanese
>> Dictionary Service" so it can be used off-line?
>
> I use EBView, wired up to search both JMDict and GG5.
>
> It is usable, but the UI is a bit rough (e.g. ctrl-c to copy to
> clipboard doesn't work, I have to right-click and use the context menu).

Just got it going - it's exactly as you describe - a bit rough.  And
the tenses in the left pane are jumbled together and barely usable
(and only with the mouse, not the keyboard).  Still - it works!
Banzai!

> I've not touched the configuration in almost 3 years, so I don't
> remember if it was trivial or hard work to setup; I may also be using an
> old version of JMDict.

I had to do some reading and searching, etc., but it's working fine
now.  I used to use EBView all the time, but fell out of using it
after one upgrade (from bare metal) install when I didn't install it,
and then a lot of time went by.  Referencing:

"How to use EBView 0.2.0"
http://ebview.sourceforge.net/help/en/index.html

 - it says:

"The first time you use the program, add one or more dictionaries
using the 'Add/Remove Dictionary' entry from the 'Settings' menu.
Follow the steps bellow:"

 - which is a little confusing, because there is no "Settings" menu!
The closest thing is:
Tools > Options > Dictionary Search > Dictionary Group > Add

Once past that, it works fine though (EBView 0.3.6.2).

Thanks very much for this tip!  It works great (although rough)!
