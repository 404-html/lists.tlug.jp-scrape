If you already have a WINS Server (Which WINSERV seems to be) you should
configure samba not to act as a wins server itself and use that wins
server instead:

local master = no
wins support = no
wins server = <ip or dns name of WINSERV>
interfaces = <local subnet (i.e. 192.168.0.0/255.255.255.0)>
