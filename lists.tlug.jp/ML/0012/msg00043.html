Thanks all, I did it!

1. on Windows IE5.0, choose Import/Export from file menu then ( chat with Wizard :-} ) export IE bookmarks as HTML file.

2. copy that HTML file to your Linux box, for example,

    /home/your-name/.netscape/ie-bookmarks.html 

3. If your bookmark HTML contains Japanese names, use nkf to convert EUC kanji-code.

4. Open this bookmark file from Netscape bookmark eidtor menu.

Now you can save bookmarks from windows aliens world!

Thank you.
