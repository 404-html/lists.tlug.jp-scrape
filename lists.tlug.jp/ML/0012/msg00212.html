hi!

Did you check password section? Windows 98 or later and NT clinets send password in Encypt mode. So you have to turn on this option in Samba as follows.

----  /etc/smb.conf -------------
# You may wish to use password encryption. Please read
# ENCRYPTION.txt, Win95.txt and WinNT.txt in the Samba documentation.
# Do not enable this option unless you have read those documents
 encrypt passwords = yes
 smb passwd file = /etc/smbpasswd
------------------------------------

then adduser to samba,

$smbpasswd -a user-name

Restart Samba and logon again from your windows client PC.
