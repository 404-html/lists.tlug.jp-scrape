Quoth Botond Botyanszki (Tue 2004-06-08 07:30:51AM +0200):

> On Tue, 08 Jun 2004 10:45:06 +0900
> "Stephen J. Turnbull" <stephen@example.com> wrote:
> 
> > It always used to piss me
> > off that traceroute, which does not require root privilege, is in
> > /usr/sbin which is not on the default path for ordinary users in
> > Debian.
>
> This did piss me off too, that's why I started adding /sbin and /usr/sbin
> to the global PATH from the early days. If the binary requires root
> privilege, you won't be able to run it as a normal user anyway.

Agreed, but from a sysadmin's point of view, you would love to:

chown -R root:wheel /sbin /usr/sbin /usr/local/sbin
chmod -R 770 /sbin /usr/sbin /usr/local/sbin

and be done with it. :)
