On Tuesday, June 01, 2004 7:43 AM +0900, Shawn wrote:
[Partial output of  cat /proc/bus/usb/devices  with phone plugged into the USB slot of the Linux machine.]

> T:  Bus=02 Lev=01 Prnt=01 Port=00 Cnt=01 Dev#=  2 Spd=12  MxCh= 0
> D:  Ver= 1.10 Cls=02(comm.) Sub=00 Prot=00 MxPS=16 #Cfgs=  1

Cls=02(comm.) --> A Communication Class device

> P:  Vendor=07cf ProdID=7001 Rev= 0.00
> S:  Manufacturer=CASIO COMPUTER CO.,LTD
> S:  Product=au A5403CA
> C:* #Ifs= 1 Cfg#= 1 Atr=c0 MxPwr=100mA
> I:  If#= 0 Alt= 0 #EPs= 2 Cls=ff(vend.) Sub=02 Prot=00 Driver=(none)

Cls=ff(vend.) --> A vendor specific interface.

This interface is not mass storage.  However, the keitai might have other configurations built in, where one of them might include a mass storage interface.

The developers on the Linux USB users mailing list <linux-usb-users@example.com> know more about this than me.  How about asking there?

Hope this helps,
jimb.
