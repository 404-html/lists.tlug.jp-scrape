wrote:

> Is there any way of running windows applications environment on a dual
> boot (XP/Linux) laptop besides wine?  Samba wouldn't do it, would it?

That isn't SAMBA's purpose. SAMBA is basically a file/printer sharing system
using Windows' own SMB protocol.

If you want something to run Windows applications on Linux then you're going
to need something which emulates the Windows API. WINE is one such solution.
Another is Crossover (non-free-as-in-beer-or-speech).
