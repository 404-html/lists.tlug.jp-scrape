http://ezine.daemonnews.org/200309/mutt-enhanced.html

Maybe this, too:

# Mailboxes - set $folder before $mailboxes
set mbox_type=Maildir
set folder=~/mail
set mbox=~/mail/inbox.`date +%m-%Y`
set record=~/mail/outbox.`date +%m-%Y`
set postponed=+postponed
set spoolfile ="/home/joe/mail/0in/frogfoot"

I got that one here:

http://blue.frogfoot.net/unix/mutt/publish/muttrc
