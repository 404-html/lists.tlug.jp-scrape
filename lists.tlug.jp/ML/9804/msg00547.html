> 1) Can Linux be booted from an MO disk?

> 2) Assuming the answer to the above is "yes," would an extremely small (only

Well, my name is Jesus an I'm a new member of this list.


About the MO disks with Linux (I have a Sony SMO-521-1.3Gb MO unit) 
and I install one time ago Linux in there, but I don't recomend do that, 
because the diks after some weeks began to have some problems (too much 
access "R/W operation") this disks have a live of about 100.000 cycles 
too insuficient for Linux partition.

Linux see MO as a /dev/sd* where (*) is the unit assigned to the MO, if 
you don't have any other SCSI disk MO will be /dev/sda

Yes,Linux with 230Mb can be installed.

And for the question about if linux can boot from MO disks: It depend of 
the SCSI adapter, in my case AHA1528 no, but can boot with a boot disk 
(A:) mounting as root the MO disk, or boot from DOS partition with LOADLIN
