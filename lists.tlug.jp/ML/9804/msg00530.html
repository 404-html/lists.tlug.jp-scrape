> The mail problem is still there, "Can't open mailbox lock.  Access is read

is the /var/lock directory writable by you?  If not, make it so.  just go
to /var and do 'chown root:jonathan lock' and then 'chmod g+w lock'

shouldn't be, but if it's writable by the user that's using pine, that
should be sufficient.
