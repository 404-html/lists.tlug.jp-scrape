It is, BUT:

Wabi and WINE are not like dosemu - they don't bootstrap Windows into
Linux (dosemu bootstraps a real version of MS-DOS into memory when it
runs).  Wabi or WINE will simply RUN the Windows application and try to
treat it as if it were an X app.  I have no idea if it'll work with
Japanese apps, but I'd be willing to bet considerable money that you'd
have considerable difficulty in doing this.

IMHO, your best bet is to get Applixware-J for Linux, preferably along
with a copy of TurboLinux-J PRO, since that includes the Wnn6 Japanese
Input System (fully licensed professional version).  OR, you can buy that
separately from us w/o purchasing TurboLinux.
