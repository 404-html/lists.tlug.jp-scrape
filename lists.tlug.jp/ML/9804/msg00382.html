Yes, it is still working (but I would need a webcam to see it from here :-).
You can find very inexpensive surples LCD modules in Akihabara.  Mine came
from Rohm and can be controlled with either a byte-wide or nibble-wide
parallel data path... like a spare printer port.

I've since gotten a vacuum flourescent display (the very bright, typically
blue displays similar to the ones seen on VCRs or microwave ovens)... so it
is easy to spot the server status from across the room.  :-) My Samsung unit
can be controlled by either a parallel or a (TTL-level) 9600 baud serial
channel.
