Thanks for volunteering!  The library has grown quickly, and I haven't had
time to catalogue it all.  There are stickers on almost all the items
saying "TLUG LIBRARY", and a list of what people signed out, but no
over-all organization to it.  

There are two types of submissions.  The first is a straight "give-away" 
that the owner doesn't want back.  The second is a "loaner" with the 
owner's name on them for eventual return.  We need to keep track of 
what is who's, and where it went ... but I think the most important thing
is to let everyone know at the meetings what is available so that it 
can be used instead of being on the shelf.  We have some good software,
from Applix to Yggdrasil, and ideally it should all be available to 
everyone.


Thanks again,
