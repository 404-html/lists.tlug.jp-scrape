You know, I think it's generally good that Linux is becoming more
desktop-oriented and easier to use, but if it was completely 'easy to
use', it wouldn't be as much fun for a lot of people, don't you think?  I
really have a lot of fun hacking around on my system(s), making things
work, compiling apps, etc.  Maybe I'm strange, but I actually like
watching things compile, esp. if I've written the code.
