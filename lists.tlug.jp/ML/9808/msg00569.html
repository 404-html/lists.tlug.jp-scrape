On Thu, Aug 27, 1998 at 03:04:52PM +0900, Jim Tittsler wrote:
> On Thu, Aug 27, 1998 at 02:43:06PM +0900, Scott Stone wrote:
> > 
> > Can a bootp client talk to a dhcp server?  I've heard that it can, but I'd
> > like to confirm that.
> 
> Yes.  See RFC1534.

Actually, the correct answer is "probably." In section 1.5, it says "DHCP
must provide service to existing BOOTP clients." But then if you read on, in
section 2, it says "In summary, a DHCP server... MAY support BOOTP clients."
So it is up to the server.  It appears to be typically implemented. (It
definitely is in the ISC's dhcpd which is popular in the Linux world.)

http://web.syr.edu/~jmwobus/comfaqs/dhcp.faq.html
