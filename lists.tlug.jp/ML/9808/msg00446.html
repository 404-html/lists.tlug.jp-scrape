===============================================================================
                         Trinux: A Linux Security Toolkit
===============================================================================

A base Trinux install fits on 2 floppies, runs entirely in RAM and 
contains the latest versions of popular network security tools.  Trinux
currently runs on 486s with at least 16MB of RAM with support for 
NE2000, 3C509/3C590, and SMC EtherPower II cards.  Additional hardware
support is available through (2.0.34) kernel modules.  Trinux is 
compatible with Microsoft and ISC DHCP.

Current List of Tools:   
======================
NETBASE: arp dhcpd ftp hostname ifconfig *ipfwadm* netconfig netstat 
nslookup ping route whois libpcap libreadline

NETMAP: drpscscan *fwping* *mscan* *nat* nc (netcat) neped nfs_check 
nmap *rdns* rpcinfo snmpget snmpset snmpwalk strobe tcp_scan traceroute 
*udp_scan* *z0ne* libsnmp

NETMON: icmp_redir ipgrab iptraf makearp makeeth makeip makeudp maketcp
netwatch readsmb sendarp sendeth sniffit statnet tcpdump tcpshow trafshow

UTIL: clear cut date find grep head sed sleep sort tail uniq watch wc

* added or updated in version 0.3a

Additional .tgz packages are available that provide email, world wide web 
(both client & server), and perl.
