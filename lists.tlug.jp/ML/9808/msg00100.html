Notes for non Macophiles:

Rhapsody OS is based on NeXTstep and is currently aimed more toward servers
it is more like NeXTstep/OpenStep with a MacOS emulator [blue box] included.
Old apps need to be rewritten to run under the 'true' Rhapsody [yellow box]
environment.

Mac OS X uses the Mach kernel and a new set of MacOS API functions so that old 
Mac OS apps can take advantage of the new architecture without needing to be
written [ie not emulation in a common address space]
