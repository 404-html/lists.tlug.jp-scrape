I tried to do it that way too, but somehow I couldn`t write to the
extrahw disk. I could mount it and read it, though. Anyway, I`ll send
you the patches on monday. I haven`t got them here right now. If that
is not soon enough, just add the Name of the new Ricoh controller
(rl5c475) to the controller list in i82365.c (There are two lists as
far as I remember and the entry has to be at the same place in both
lists) and instead of IS_RL5C475 use IS_RL5C466. In this way all the
settings for the older controller version will be used. You should
also add: 
#define PCI_DEVICE_ID_RICOH_RL5C475     0x0475
in rl5c466.h. After that the driver should work. There is a directory
on the TL1.9b2 where are all the necessary scripts to create a new
extrahw image. I haven`t checked on the TL2.0, but it should also be
there. I just don`t remember the name. I will also see that I get the
image done until monday. Right now I have to reconstruct all this
stuff from memory because I left it at home.
I hope that will help you.
