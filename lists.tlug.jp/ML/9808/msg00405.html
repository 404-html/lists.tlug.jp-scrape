> I'd like to telnet into a server and from there ftp some files to another
> server. In ws-ftp on Windows I click a button and it will transfer the
> directory structure, happily chug away and copy a complete web site.

Use the Midnight Commander. This is a Norton Commander lookalike
tool that has an ftp mode - that is you can establish an ftp
link to a machine, have its content listing in one of the
windows and copy stuff to and fro as if it were local. Entire
directory trees are no problem.

Hope that helps.
