On Sun, Aug 30, 1998 at 03:39:10PM +0900, Eric S. Standlee wrote:
> I want to start programming for the Palm Pilot, but where should I start?

http://www.cs.purdue.edu/homes/zamboni/Pilot/ is a good starting place.

http://www.acm.rpi.edu/~albert/pilot/ has a nearly complete archive of the
Pilot-UNIX mailing list (which covers programming in addition to syncing
and applications).

> If I write in C, how do I port to the Pilot?  Is there a Linux version of
> the Code Warrior for the Whinedoze?

Donnie Barnes of RedHat fame has made the GCC cross-tools available in RPM
format. ftp://ftp.redhat.com/pub/home/djb/
