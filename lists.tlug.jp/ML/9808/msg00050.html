On Tue, 4 Aug 1998, Marcus Metzler wrote:

> Hi,
> I just installed the acrobat reader 3.01 for Linux and all I get when
> I try to start it is a window with "Segmentation Violation Caught."
> and an "Unable to initialize user interface.". Does anybody know what
> the problem could be. The libraries the executable needs are all
> there:
>         libreadcore.so => not found
>         libpfs.so => not found
>         libagm.so => not found

these are libs that come with acrobat reader, you've got to install them
and run ldconfig.

you might want to symlink them into /usr/lib or /usr/local/lib and make
sure that the path is in /etc/ld.so.conf, as well.
