On Sat, Sep 26, 1998 at 10:48:46AM +0900, Scott Stone wrote:
> Where can I download this faq wizard?  I want to use it for the TL USA

The FAQ wizard is part of the standard Python distribution.  It can be found
in the Tools/faqwiz/ directory of the Python 1.5.1 distribution.

You can get Python from ftp.python.org:/pub/python/src/pyth151.tgz (and be
sure to get the patches from http://www.python.org/1.5/patches-1.5.1/)
