On 07-Sep-98 Fisher wrote:
> I am looking for a BBS system on linux. If someone in this mailing list
> has any experience setting up niftyserve like BBS system on linux,
> please tell me which program I can use and from where I can get it.

A couple of years ago I looked into some Linux-based BBSes, so my current info
is somewhat dated (although I suspect that the web has replaced most BBS
software.) Here's what an old sunsite archive looked like:

1bbs-0.10.tar.gz         script to giveLinux a BBS interface
BBLinux-0.1.tar.gz       utilities for adding BBS functionality to Linux
BBLinux-0.3.tar.gz       simple utilities to convert Linux system into a BBS
PH.BBS.linux.ver3.0.tar.gz shareware chat BBS program
adbbs-2.1.tgz            bbs/menuing system w/ ANSI color support
ambbs-.9.5b.tar.gz       BBS software
bbs-5.123-src.tar.gz     the No-Name BBS system
cafe.tar.gz              modified version of Citadel/UX
citux403.tar.gz          source for the Citadel BBS package
dbox-1.1.tgz             demo of BBS mail system
drealmBBS-2.1.tar.gz     Bulletin Board System for Linux
ebbs-3.01-Linux.tar.gz   ebbs binaries
ebbs-3.01.tar.gz         Internet BBS based on Pirates BBS
ft_bbs-1.0.tgz           allows secure file transfer via tty's (modems)
ix-mbox30.tgz            ix/MBox BBS
lime-0.1b.tar.Z          complete color menuing system for Linux
mbox26.tar.gz            ix/MBox BBS
party.taz                Public "party-line", w/ local, multi-user chat
pbbs-1.9.tar.Z           Source for Pirate BBS, a dial-in or net BBS system
rocat-1.05.tar.gz        full-featured BBS ported to Unix
tbbs.1.3pl8.1.tgz        BBS system w/ rip, nntp, qwk, etc
unib112a.tar.Z           part 1 of UniBoard BBS bins
unib112c.tar.Z           part 3 of UniBoard BBS bins
unib112l.tar.Z           part 2 of UniBoard BBS bins
xbbsv1-0.tar.Z           Source for xbbs ported to linux
yabbsnet0994.tar.gz      yabbs server (gopher, mail, chat)

Maybe your search could start here....

HTH

Jim S.
