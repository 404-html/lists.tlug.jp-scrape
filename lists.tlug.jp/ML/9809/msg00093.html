On Thu, Sep 03, 1998 at 04:03:22PM +0900, Venkatesh Raghavan wrote:
> The Kernel that I am using is 2.0.30. I have never patched the Kernel. So
> if it is not a bother can you please tell me how do I apply the
> patch after downloading . Do I have to remake the Kernel after I do the
> patch ?

A good place to start reading is the Linux Kernel HOWTO.  It will explain
the basics of how to apply a patch and how to compile the kernel.
http://w3.clat.hi-tech.ac.jp/LDP/HOWTO/Kernel-HOWTO.html

I'd suggest building an unpatched kernel first (and you may wish to take
this opportunity to move to 2.0.35) and make sure you can build, install,
and run it.

Then apply the patch I mentioned earlier:
  cd /usr/src/linux; patch -s <patchfilename.txt
and rebuild, reinstall the kernel.
