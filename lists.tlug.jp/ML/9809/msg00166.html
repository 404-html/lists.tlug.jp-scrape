I've read about people using a network of linux computers to crack
passwords, and to do ray-tracing. Does anyone know more about how this is
done? How easy is this to set up, what protocols are used, how are the
computers connected, etc?

As I mentioned in the go-thread, my interest is mainly for a go program (in
C++). This problem can be split up quite nicely - each machine is given a
candidate move to consider. It returns the result of a minimax search. This
would take at least one second, so communication overhead should be negligible.

One requirement is that it can be controlled from a Windows machine, because
that is where the user interface is, and also a DLL I don't have source code
for. The ideal would be a portable solution, so it can use a mix of both
windows and linux machines.

Initially I was thinking of setting up a simple test system where the task
is just to increment a number. Ie. it sends the other machine the number
"12" and gets "13" back. Anyone interested in experimenting with this with me?

Darren
