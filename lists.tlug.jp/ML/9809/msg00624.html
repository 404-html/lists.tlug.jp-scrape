On Tue, Sep 22, 1998 at 10:48:09AM +0900, Howard Abbey wrote:
>
>                 So, why wouldn't it call another program (Lynx /
> Netscape) to read HTML? 

It can.  Assuming mime types are set correctly (text/html) all you need
is an appropriate line in your .mailcap file.  You can even make set
things up to use netscape if running X or lynx if not.

Mutt is modular almost to a fault.  Mutt suffers from feeping creaturism
in a few places, but for the most part the developers are pretty adamant
about not adding unnecessary features.  Case in point: mutt doesn't even
speak SMTP directly (you need sendmail, qmail, or, easiest of all, ssmtp
to do the actual smtp conversation).
