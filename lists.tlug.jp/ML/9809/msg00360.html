If you are using NT4.0 SP3, then the problem is that as of SP3, NT
encrypts passwords, whereas samba expects them as clear text by
default. You can change samba, I believe, to use encrypted passwords,
or you can do the following:

1) Use regedt32

2) From the HKEY_LOCAL_MACHINE subtree, go to the following key:

	\SYSTEM\CurrentControlSet\Services\Rdr\Parameters

3) Click Add Value on the Edit Menu.

4) Add the following:

	Value Name: EnablePlainTextPassword
	Data  Type: REG_DWORD
	Date: 1

Click OK, Run the MS UPST ( Universal Problem Solving Tool)

AKA reboot


After the reboot, you should be fine. I forgot the URL for this, but
you can search MS's pages for article id Q166730

Hope this helps.

Another thing you could try in a Cmd window is:

	net use \\hostname\printer

And see if it asks you for a password, it should if its the first time
you've connected to that host, I think.
