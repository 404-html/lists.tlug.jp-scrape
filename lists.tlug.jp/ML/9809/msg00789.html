Hi,

On 27-Sep-98 Eric S. Standlee wrote:
> I have a problem shelling root and not being able to ppp-on.  If I get out
> of that user and exit altogether and then login root, it works just fine!
> 
> What's up?

Permissions on pppd. 

*If* you're on a standalone Linux box with no other users and you're just
dialing up your ISP, it's probably OK to set the sticky bit on pppd. For example

schweiz:~$ ls -l /usr/sbin/pppd
-rws--x--x   1 root     bin         70040 Aug 29  1995 /usr/sbin/pppd*

allows me to run pppd as USER schweiz on the aging Linux box in my home office.

HTH
