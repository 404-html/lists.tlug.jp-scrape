> I just ran across a horrible setup problem on my RH5.1 sys.  I ran lynx to
> www.redhat.com to get the errata for the kernel d/l'd.  I hit return on
> the d/l links and after d/l'ing the kernel source lynx said that there was
> no "player" or something like that for "rpm" files.

That means Lynx doesn't know about a helper application to
properly handle the file. It instead offers you to save the file
onto disk. Select that and you're set.

If you want to save a page on display under Lynx hit "p". You
then enter a menu whether you want to print, save to a file etc.
Select save to a file. You are asked then about a filename. Type
the name you wish in and you're set.

Lynx is great......
