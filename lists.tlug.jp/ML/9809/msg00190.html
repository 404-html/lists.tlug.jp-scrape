    ash> Question: Does anyone know if TL will run on an IBM ThinkPAD

TL I don't know about; I checked on the ThinkPad series a while back
and they all seemed to run Linux OK.  Don't remember the page I got
the data from though, sorry.

    ash> Anyway, I like to get away from this excuse of an OS as much
    ash> as possible, but unfortunately still need it for Word and
    ash> some of our own apps. If I can get it to dual boot I'll be
    ash> very happy to upgrade to TL.

I've been booting Linux on my StinkPad 330cs from the same
coffee-stained floppy for 3 years now; the kernel has been rebuilt
about 6 times, new modules added to the /lib/modules, XEmacs built
(and GCC removed to make space for Lisp libraries, it's a 230MB
HD)...  And it keeps on ticking, locked-out bad sectors
(pre-coffee-stain) and all.

I recommend using a boot floppy; it doesn't take much more time, and
you have the confidence that NT can't get at your boot data, since the 
floppy is NOT in the drive when NT is in RAM....
