Michael Casinghino wrote:
> If I use procmail to filter my messages as they arrive, how can I be
> alerted that I have new mail?  

I'll send you my actual recipe privately when I get home, but in your
procmailrc, and a recipe that fits all the conditions of mail you want 
to be notified of, have it not mess with the mail, but run a program
to notify you that you have new mail.  (My choice: play a sound file
from
Monty Python's Holy Grail: Arrow shot, 'Message for you sir'.)  If you
want
something like xbiff, 'touch newmailfile' & having xbiff look at that
file
instead would probably do it.
