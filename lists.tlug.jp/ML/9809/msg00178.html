On 08-Sep-98 Rob Bickel wrote:
> This is a simple question, but I can't remember how to do it.
> What can I use to count the words in a tex file?  I don't really
> care if it is absolutely accurate or not.  I just need a round
> number.  

One thing you could do is get a program called detex which removes all the tex
stuff from the file and then pipe the result into wc
ie.

detex file.tex | wc -w
