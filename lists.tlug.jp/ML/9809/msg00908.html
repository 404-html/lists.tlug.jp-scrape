I've not heard good things about twinbridge or similar options. If you just
need to read web pages, Netscape and Explorer have Japanese extensions. If
you need anything more, install Japanese Windows. 

You could try making a dual-boot system with Windows 95/98J, and English NT
if you only need Japanese some of the time. TPC (Tokyo PC Users Club)
newsgroups would be a good place to ask for more help on this:
  http://www.tpc.ml.org
