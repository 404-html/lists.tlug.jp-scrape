I don't know, and I'm too afraid to ask the author ;-)

I still use Mew on mule because it's much reliable.
The ability of handling MIME is the core part of Mew
and is implemented quite well.

The advantage of Wanderlust is the ability of handling
nested folders, combining e-mail and netnews seamlessly.
You can have a meta-folder consisting of something like this;

	Emacs {
	  %#mh/spool/mew-dist	<-- IMAP server mail folder
	  +ml/mew-faq		<-- local mail folder
	  -fj.editor.emacs	<-- New news group
	}

It also tracks unread mail, i.e., the functionality of
Mew + prom-mail.

I really hope if both mailer get merged in some way.
