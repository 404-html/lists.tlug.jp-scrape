Well, dpNote 2.0 looks *much much much* nicer than 1.0.  I haven't gotten
into using it much, but the UI is a vast improvement...

Anyone who's bought TLJ 2.0 Pro will get a free upgrade to dpNote 2.0 -
they should be mailed out soon to you.

We'll also donate a dpNote 2.0 (the TurboLinux version) CD to the TLUG
library.  The only difference is that the TL version is packaged with the
correct permissions and it doesn't live in /usr/local.  It's statically
linked, though, so it'll work with any Linux distribution, theoretically
:)
