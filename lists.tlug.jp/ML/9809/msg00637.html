Define "next release".  There hasn't been any discussion about the
Neomagic driver on the xfree86 dev list (but then core doesn't always tell
us everything).  "Next release" could mean the next 3.3.x release (which I
don't think is going to happen), or it could mean the 4.0 release (which
may happen someday, but they say that the messiah will return someday
too).

Bottom line -- non-XFree86-dev-folks aren't going to see that source code
until XFree86 core decides what to do with it.
