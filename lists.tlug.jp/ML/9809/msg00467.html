On Fri, 18 Sep 1998, Rob Bickel wrote:

> 
> I screwed up someone on one of my machines and I need to edit a couple
> files before I can get it to boot up.  My only problem is that I can't
> figure out how to get any kind of editor to work.  I am doing the
> standard boot/root disk to boot up from the floppy and then mounting
> my linux partition.  From there I should be able to run for example:
> 
> /mnt/usr/bin/vi
> 
> But I get
> 
> /mnt/usr/bin/vi: not found
> 
> I even put /mnt/usr/bin in my PATH.  No luck.  I go to /mnt/usr/bin
> and run ./vi.  Still no luck.  vi is there.  I looked.  What do I 
> need to do to get an editor?

shared libraries not found, probably.  do this instead:

chroot /mnt /usr/bin/vi

then it can find the shlibs on your installed system.
