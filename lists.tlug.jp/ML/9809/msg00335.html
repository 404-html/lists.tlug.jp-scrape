> This isn't for a TA, but for a 128K leased ISDN line with an NTT DSU on it.
> I believe those only have an RJ-45 connector, don't they?  I don't think the

cisco 2501 has the interfaces that is serial and ehternet.
it does not have 128k leased line (BRI) interface.  
CISCO 2503 has BRI interface but it can use only 64kbps line.
CISCO 2600/1600 series has 128k capable BRI interface. plese see

http://www.cisco.com/japanese/warp/public/3/jp/service/info/faq/LLfaq.html
  
If you want to use 2501 router, you need TA that can work with 128k leased 
line. 

you can see spec of 2500 series. please see figure.

http://www.cisco.com/japanese/warp/public/3/jp/product/catalog/2500/catalog.html

this is in English.
http://www.cisco.com/univercd/cc/td/doc/product/access/acs_fix/cis2500/2501/2500ug/ovr.htm

> Cisco to DB-25 cable is what they need, but more like something to connect the
> Cisco DB-60 to RJ-45, maybe?  Is it possible to connect a Cisco to that type
> of DSU?

I think answer is NO. If you have 2503, answer is probably yes. 

you can use TA like bellows:
http://www.nec.co.jp/japanese/product/kiban/densow/datacom/prod/ez128/index.html
this is NEC TA for leased line(I-interface 64k, 
                   128k, DA64, DA128, FR 64k, FR 128k).
It has V.35/X.21 interface.

this is figure for connection

NTT--128k leased line---NTT DSU----(RJ45)---TA--(X.21)-------(DB60)--CISCO 2501
