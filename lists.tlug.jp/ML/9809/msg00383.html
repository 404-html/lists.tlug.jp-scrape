Hi, I just joined this ML, and am happy to see so many helpful posts.

After trying out TurboLinux 1.0 Pro some time ago, and more recently
RedHat 5.1 and Caldera OpenLinux (with S.u.S.E. on order!), I thought
I'd install TurboLinux 2.0 and give it a whirl.  (After all, I've been
giving it free advertising on my Web site for a long time!)

The installation program is the best of all those I've tried so far, and
the list of bundled programs is not bad, either. Would be nice to have
Xfmail, though.

One problem I've had is with Japanese fonts. If I use Communicator for
email, the Japanese fonts in received mail look terrible. What's the
best way to fix this? Also, why is it that Navigator requires you to
quit the program and reopen it each time you change fonts? Is this a
limitation of Linux?
