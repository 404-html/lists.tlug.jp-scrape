On Mon, 28 Sep 1998, Eric S. Standlee wrote:

> But, that doesn't explain how su root doesn't have all the same priv.s as
> login root... Help.

make sure you do it like this:

su -

instead of like:

su

when you use the -, it implies 'login shell', so everything gets set
correctly, just like if you log in as root.

> 
> Also, if I plan to network my machines, what is the better solution for
> pppd then?

what's wrong with ethernet?  Or is this a WAN application?
