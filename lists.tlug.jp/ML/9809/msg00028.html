On Tue, 1 Sep 1998, A.Tomita wrote:

>Also, after been "digestified", I don't know if attachments would
>be correctly decoded by a mail reader. I would hate to have to scroll

That's a good point.  Attachments usually become junk when run through a
"composter."

>BTW, someone mentioned IMAP4. I'm not sure, but I think
>my provider (AT&T Worldnet) doesn't support this protocol.

They didn't when I used them, nor do we support it.  I don't know
any ISPs that do, for a couple of reasons.  First, it would take a lot more
disk space if a lot of people started using it, and 4 GB Sun disks are
expensive, and second, it would probably create a support nightmare,
especially in the early days. 

We used to have IMAP on the employees' mail server, but a serious security
flaw was discovered and IMAP was subsequently disabled and we went back to
POP3.
