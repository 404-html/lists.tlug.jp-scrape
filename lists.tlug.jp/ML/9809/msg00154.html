On Mon, 7 Sep 1998, Dave wrote:

> I'd like to start customizing my environment to suit my needs, and i have the
> following questions with regards to that:
> 
> 1. What are the configuration files used when you (a) log in and (b) start an
> nxterm session. I need to know where to store all my aliases and whatnot so
> that i don't have to keep typing them in and various other settings.
> 
> 2. I'm using Afterstep, and I would like to set it up not only so that when i
> log into my account the OS automatically starts some programs, but also that
> it places them in specific places on my deskop so that everything isn't piled
> all on top of each other at the outset.

well, .profile, .login, .bashrc when you login, and for stuff that starts
when you start X, there's a section in your afterstep config file for
that, or you can put it in .xinitrc/.xsession.

If you want to specify WHERE things go (if you just leave it alone,
Afterstep will try and place them intelligently, but it's usually more
like 'StupidPlacement' than 'SmartPlacement' :) .. use the -geometry
option to most X programs to tell them where to go.
