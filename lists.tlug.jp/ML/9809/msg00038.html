I log into the Cobalt Qube, immediately go to change my password.  It asks me
for the current one, so I type it. It asks me for the new one, so I type it.
It asks me for verification, so I type it again.  Then it says I don't have
permission to do that.  So I go through the whole thing again, just in case it
was doing something weird like blowing me off on permissions for a typo.
Nope.  It really won't let me change my password.  Apparently that's the
default setting: users can't be trusted to change their own passwords :-(

Other details:

It's an RPM system
Kernel is 2.0.33
Apache is 1.2.5
