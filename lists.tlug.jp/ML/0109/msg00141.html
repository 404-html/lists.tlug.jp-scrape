I don't know where Stephen got this mutt-j being AWOL news, but their
web site, <http://www.emaillab.org/mutt/>, is very much alive and
a new patch came out only a few days ago.

mutt as it is works OK for Japanese; mutt-j fixes some subtle issues,
like encoding of text attachments (which becomes an issue when you
forward mail) and tolerance of non-compliant mailers, and adds some
extra features not included in the main version. Oh, what the heck,
I'll attach the relevant parts from the release note below, so see
for yourself.
