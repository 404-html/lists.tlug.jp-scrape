I dont know English how-to for the EM-OME.

However, linux-box detect EM-ONE/EM-ONE as USB-modem,
so we can do diversion from D02HW(and same ones)s how-to,
only 's/D02HW/EM-ONE/g'.

And its quick start guide. Currently, I use only ubuntu, if you use
another distro, I can check it(Please send Distro name and versions).
Please feel free to inquire.

-------------------------------------------------------
for Ubuntu 9.04/8.10
-------------------------------------------------------
Notes: Ubuntu 9.04/8.10 's NetworkManager could not valid handling USB-Modem.

0) connect EM-ONE with USB-cable.

1) install gnome-ppp
$ sudo apt-get install gnome-ppp

2) exec gnome-ppp
$ sudo gnome-ppp

3) configurations are below;
- username: em
- password: em
- Phone Number: *99***1#

4) Push [Setup] Button,
- device: /dev/ttyUSB0
- Speed: 115200

-------------------------------------------------------
for Ubuntu 8.04
-------------------------------------------------------
Use NetworkManager, parameters are same as gnome-ppp.
http://gihyo.jp/admin/serial/01/ubuntu-recipe/0043
