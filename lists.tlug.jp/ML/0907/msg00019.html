If you're looking for an alternative, I'd go with a Mac Mini.  I've
owned two in the past, and the cheap price, and rock solid hardware
have worked well for me.  My previous machine dual-booted both Mac OS
X and I think Ubuntu.  That worked out nicely enough, though my wife
didn't like the dual-booting, so I had to run Linux/BSD off of VMbox
on the second host we got.

Minis are not powerful, so forget about gaming, but for basic needs,
the price/stability and now Pentium architecture can't be beat.  :)
