checking out FreeNAS 7 (currently in RC1) http://www.freenas.org.


Hi Kyle, As a user of FreeNAS (using version 0.69.2), for home storage , I'm glad that you're bringing attention to FreeNAS which is a great product. But I still regard FreeBSD's (which FreeNAS is based off of) implementation of ZFS very beta, at least from looking at FreeBSD's ZFS "Known Issues" page. I know that I wouldn't want to be storing all my family videos/photos on it at the moment.

http://wiki.freebsd.org/ZFSKnownProblems

I really hope that FreeBSD's version of ZFS continues to mature nicely but I think if you're going to use ZFS at the moment, the *Solaris boxes are the safest way to go.
