On Wed, Jul 29, 2009 at 18:14, Martin Killmann<martin@example.com> wrote:
> TLUG,
>
> I have a fairly standard task, several hundred .html files where I need
> to do some simple text replacements with basic regular expressions.
>
> Now, under Linux, this would be done in 5 minutes with board tools, but
> scrapping my company PC at work and installing a decent OS is not an
> option right now, so I need to do this under Windows XP. Does someone
> here know a decent, easy to use Windows tool that can do text
> replacements across multiple files?
>
> I've tried a dozen programs I found on download.com and the best tool I
> found was something called Textcrawler, which is free and does what I
> need but is too buggy to use (i.e. it leaves crap characters across
> files, and the cleanup takes more time than replacing by hand).
> Yes, I could use Dreamweaver, but a small, fast, dedicated tool for the
> job would be nicer.

I used a Software called "BK ReplaceEm v2.0" http://www.orbit.org/replace/

It works for me, replaces text in files. I use this on windows servers
where I have to change some simple text (eg copyright notice) or db
configs scattered all over the place.

And yes, those simple things that can be done in less than 5 min on a
real OS take hours on this GUI vomit thing ... (but perhaps the MS
power shell thing can do that more easy? anyone tried that?)
