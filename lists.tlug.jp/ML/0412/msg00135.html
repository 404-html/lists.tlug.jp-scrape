This is part 2 in my ongoing printing saga.

Some people might be wondering what mups is...  
Microline Unix Printing System for Oki printers.

If you go to the okidata webpage you can fill in a form and they will
send you their mups cd and a copy of "CUPS 共通Unix印刷システム" which is the
japanese version of Micheal R. Sweet's Common UNIX Printing System.
Basically the whole package is the japanese way of saying "RTFM" before
you call support.

First problem i had with mups was that it only comes with rpms and some
packages for solaris. So just make sure i got everything right i
installed rh9  on an old box i and installed the mups rpms according to
the readme. Everything went fine and the printer worked beautifully with
all the options the windows driver has... printing the Mups test page
was a beautiful moment. I also managed to get it working on gentoo, by
copying the ppd file, the filter and another file i dropped in /usr/lib.
It was so easy it annoyed me, since Oki tell you to remove cups when you
install mups...

Anyway... everything started to go downhill when I tried to print a
postscript file as I was instucted by Stephen Turnbull in an earlier
thread. 
The printer basically did nothing, a little movement and then nothing.
Everything else works perfectly. I can print postscript files made by
evolution so at first i thought that dvipsk was the problem, but i made
some postscript files from tex and dvi files i downloaded that will
print. Which can really only mean that there is something wrong with the
tex files that _I_ am writing.... but even the simplest of things won't
work.

like this... (basically what Stephen told me originally)
This worked before, it works with xdvi and ghostview.

cat > tmp.tex << EOT
\documentclass{article}
\begin{document}

This is a test.

\end{document}
EOT

platex tmp
dvips tmp


I would love to hear from anybody can see what I am missing or has any
idea what i might be doing wrong.

But if I can get over this last little bump i'll have a great impact
printer that was cheap and is fully supported by Oki. 
... btw It is Microline 50hu.

Thanks and sorry for such a long post.
Patrick
