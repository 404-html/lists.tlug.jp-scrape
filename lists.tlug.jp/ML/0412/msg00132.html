>  simon> Is there any way of finding this out using the apt
>  simon> programs?  Or is it a "you get a feel for it as you along"
>  simon> type of thing?


Generally a command like this would give you good results:

    apt-cache search dns dig
    or
    apt-cache search dns | grep dig

However, in this case with dig, it doesn't turn up anything.  But in
general, it's a great way to find the programs you need.  

To get more detail on any particular program, try:

    apt-cache show dnsutils
