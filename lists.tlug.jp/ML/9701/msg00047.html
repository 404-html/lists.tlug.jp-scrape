I mentioned a Korean site for Debian Linux:

ftp.kreonet.re.kr:/pub/Linux/debian

Be aware if you use it that it does not permit passive FTP (what ever
that is) which is the default for the Debian 'dselect'.  So you need
to change that in the Debian dselect file location process.
