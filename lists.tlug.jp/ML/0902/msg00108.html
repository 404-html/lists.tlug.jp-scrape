On 土, 2009-02-14 at 02:09 +0900, Dave M G wrote:
> Is this information in any way attached to the file? Specifically, if
> I 
> upload it to the interweb, and someone downloads it on their
> computer, 
> does it still say "Owner: dave - Dave Gutteridge"?

I doubt it. Ownership and permissions are stored in the directory entry.
That wouldn't be transferred when up/downloaded. The downloader would
become the owner. The exception might be if the file was part of a tar
archive and it was extracted using the -p flag, but even so, only the
UID *number* would be preserved. The file would appear to be owned by
the person associated with that UID on the local computer. I think. I
may be completely wrong.
