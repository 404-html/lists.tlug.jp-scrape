scim-gtk2-immodule:

 Depends: scim-modules-socket but it is not going to be installed
 Depends: libgtk2.0-bin but it is not going to be installed

 Depends: libscim8c2a but it is not going to be installed

The first two have already been installed;  just minutes before.

If you examine the version of the scim package installed, using :
dpkg -l | grep scim

and check the version required by the scim-gtk2-module:
apt-cache show scim-gtk2-module

this way you can see the differences in versions (if there are any)

I was not recommending that you point it at another site, merely that the scim-gtk2-module might depend on a different version.

apt-get install {package-name}={version}
