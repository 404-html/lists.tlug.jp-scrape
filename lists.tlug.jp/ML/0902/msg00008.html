Recently I reinstalled the OS into my eee (901) using Ubuntu.
I had little problem with Xandros but just wanted to try EasyPeasy
(formerly Ubuntu Eee) because the distribution based on Ubuntu 8.10
says it has almost everything you need out of the box such as GCC.
Well honestly, another reason is that I removed qt3-mt and all the
dependent packages by mistake.

So anyway, I downloaded the ISO image from here:
  http://www.geteasypeasy.com/
and burned it into a DVD-R.  The installation is very easy
following the instructions on screen, and it comes with
wireless LAN driver that works fine from the start.

Using Japanese input method took some more work.
First LANG has to be set to "ja_JP.UTF-8".  I had to do

$ sudo apt-get install anthy scim-anthy scim-table-ja

then from the home menu choose system settings ->
language support and check "Japanese" in the list.
Finally restart the computer and you'll be fine with
typing Japanese using SCIM.

Has anyone tried DebianEeePC?
