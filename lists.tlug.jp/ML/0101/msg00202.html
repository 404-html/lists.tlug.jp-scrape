I don't know about the BOOTP questions, but the HD message means you have to
do a format of the disk. Sun disks use a disk label and need an "envelope"
partition for the whole disk. I usually use the formatting program from a
Solaris CD, but I think I've used the fdisk from various Linux-Solaris
distros as well.

I have Solaris disks and an external CD that I'd be glad to lend to you if
you want to try a CD install. I'm convinced that the network install is the
best thing, but setting it up......
