I set up my sister-in-law's machine for this earlier this month. I don't have
the script to hand, but it was very straightforward.  I did have one problem,
though, at the GOL end.  The first number I tried fed the machine one of two

When I shifted to a different number, everything went swimmingly.

Perhaps Jonathan will chime in with news of whether the problem got sorted
