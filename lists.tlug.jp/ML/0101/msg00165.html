Meaning you'd like them to be logged using a facility other
than "user"?  You could tell your MN-128-SOHO to use a
different facility:
  ip syslog facility 21
(which /usr/include/sys/syslog.h tells me is facility local5).
And then filter local5.* off to /var/log/router.

> Should I add something like this to allow whatever it is the router is
> wanting to do?
>    ip filter 31 pass out 211.123.38.193 0.0.0.0 udp route route remote 0

Do you need RIP?  I run my router with:
  ip rip off
