On Tue, Jan 16, 2001 at 03:47:44PM +0900, Antony Stace wrote:

> Can someone please give me a few tips on how to get this set up so I can
> indeed run xapps over a ssh session.

* Don't mess with the DISPLAY variable
* Ensure X forwarding is turned on in sshd's config file
* Ensure xauth is in your path
* Ensure .Xauthority is writeable
* Ensure your sshd binary was compiled with the proper path to xauth
