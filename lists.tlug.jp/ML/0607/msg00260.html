Is there anyone out there who knows C++, has Japanese input working on Linux and/or Windows, and would like to contribute to a worthy open source project? Scribus, the desktop publishing program, is having an issue with non-Latin characters: 

   http://bugs.scribus.net/view.php?id=3173
In case anyone was interested, it seems this may not be a bug in Scribus after all. I checked out the SCIM FAQ and found that apparently SCIM has a problem with Qt, or vice versa, but it can be "fixed" by setting 'XIM Input Style' to 'Over-the-spot' in Qt. No explanation of what might cause the problem, though.
