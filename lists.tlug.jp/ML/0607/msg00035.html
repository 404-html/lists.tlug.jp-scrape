Scott Robbins wrote:

> Step one.  During installation, did you install Japanese support? 
> If not, go back and do it. 

Amen. 

Selecting Japanese during the initial installation includes all 
kind of stuff that you might not know you need. 

I've never found anything in Red Hat for adding language support 
after the initial installation. If someone knows of it, please tell 
the list. 

What I _have_ done, is to install Centos multiple times, 
first English only, then English + Japanese. 
After each installation, I saved the output of rpm -qa, 
so I can diff the rpm lists and then _maybe_ you can just 
install the diffed rpms for Japanese support. 
If that works, then great. If not, consider reinstalling 
from scratch.
