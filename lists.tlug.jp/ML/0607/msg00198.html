One quick thing to keep in mind when running sshd on a nonstandard port-
nmap will still pick it up as ssh no matter what port you put it on.
Therefore it may be a good idea to create an iptables rule to allow
connections to that port from only the IP address you will be connecting
from. If you are not connecting from a static IP then maybe using knockd
will help.
