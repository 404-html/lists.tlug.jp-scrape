I wrote:

> Scott Robbins wrote:
> > Are you using scim-anthy, kinput2-canna or some other combo?
> 
> I'm using the boring defaults. 

The only special sauce I add, are little scripts to let me run a 
program in a different language when logged in in English. 

For example: 

   [jim@example.com cjk]$ cat *4.2/bin/ja
   GDM_LANG=ja_JP.UTF-8 GTK_IM_MODULE=iiim LANG=ja_JP.UTF-8 XMODIFIERS=@example.com=h??
   export GDM_LANG GTK_IM_MODULE LANG XMODIFIERS SUPPORTED
   jahttx &
    &
   [jim@example.com cjk]$

allows me to run gnome-terminal in Japanese while X is mostly English, 
by executing: 

   ja xmms

Programs that I want to use often in another language, have their own script. 
For example: 

   [jim@example.com cjk]$ cat *4.2/bin/twgnome-terminal
   GDM_LANG=zh_TW.UTF-8 GTK_IM_MODULE=iiim LANG=zh_TW.UTF-8 XMODIFIERS=@example.com=h??
   export GDM_LANG GTK_IM_MODULE LANG XMODIFIERS
   twhttx &
   gnome-terminal &
   [jim@example.com cjk]$

Granted, I really really need to clean up those scripts.
