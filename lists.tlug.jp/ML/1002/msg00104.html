Oh, come to think of it, since it's Ubuntu, I also have some notes on
how to deal with Gnome desktops:

  http://www.starling-software.com/en/blog/sysadmin/2009/02/24.alternate-window-managers-under-gnome-2.24.html

It's important to rememember that you're dealing with several components
here. As well as the session manager (which you don't generally see
much sign of), you have a window manager (Metacity), a file manager
(Nautilus), and the panel (that little bar with icons and so on). I
myself use Gnome and the panel, but a different window manager (fvwm)
and no file manager at all.

It is also, unfortunately, important to remember that every damn release
of Ubuntu changes how things work, and you're most likely going to have
to spend several hours reworking your configuration when you "upgrade."
