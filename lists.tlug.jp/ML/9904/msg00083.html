On Thu, 8 Apr 1999, Howard Abbey wrote:

> I'm still not sure I'd be willing to pay the extra $100 it would have

I'm pretty happy with the SCSI model.  My only quibble is that it gets
quite hot under moderate use.

> Do other ATAPI changers have multiple LUNS?!?

Apparently so -- I seem to recall the NEC drives doing the right thing
with respect to LUNs.  The best way to find out is to use the ide-scsi
driver.  If it responds to all seven LUNs, then it won't work properly ;)

There is a guy that wrote a patch that allowed ATAPI changers to work
properly.  I tried it last week ... it had compile problems that I
couldn't sort out.
