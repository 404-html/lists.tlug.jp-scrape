On Mon, 12 Apr 1999, Jonathan Byrne wrote:

> KDE is developed using the Qt libraries, which are not Free, but the
> stuff needed for KDE may be freely distributed, if I understand
> correctly (but isn't open source?).

Qt is distributed via the Troll Public License, which prohibits the
following:

* distribution of _any_ patches to Qt, even to fix bugs
* development of non-open source software with Qt (for UNIX platforms,
  that is)

Obviously, this is not free software.

Troll wised up and fixed their license so that it resembles the BSD
license, but no code has yet been issued under that license -- so their
new license is moot, in my opinion.

Is it legal to distribute a GPL'd program that uses Qt?  Probably not.
The licenses conflict.

These are some of the reasons why I use GNOME[1].

-- Chris

[1] The other reasons, when expressed, tend to ignite passionate flame
    wars, so I'll keep them private.
