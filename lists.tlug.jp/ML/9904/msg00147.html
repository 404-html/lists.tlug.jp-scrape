Basically setup includes several different areas:

Boot time:
----------
1. You need to patch (or configure, if it's available)  kernel to send
all boot messages to /dev/tty.. instead of monitor.

2. Configuring LILO to accept input from serial line instead of
keyboard.

After boot:
----------

3. Modifying /etc/inittab to enable logins from serial port. (Probably
you'll need also to change /etc/securetty as well to configure  ttyS0
as secure terminal).
