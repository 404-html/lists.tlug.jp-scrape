I saw this in some installation instructions:
  make all >& make.log &
  tail -f make.log

Isn't there an easier way to have stdout go to a file while still going to
screen? If there is nothing built into bash, surely there must be a utility
something like this?
  make all | logit make.log

Where "logit" would read a character from stdin then writes it to stdout
and to the file given in the commandline.
