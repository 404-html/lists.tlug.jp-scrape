Subba Rao wrote:
> 

> How can I trace what process or host is using the dial-up connection?

Are you using chat in verbose mode? If you are, you could either run
chat with -r <report file> or you could use -v which sends all text
received from the modem and the output strings sent to the modem to
SYSLOG (/var/log/messages?)

Also read the PPP-HOWTO which describes the dial-up connection process
in detail.
