However, at one point, while running Slack, I also tried,
unsuccessfully, to install kinput2 and canna.  I used the links from Mr.
Katz-Brown's page, which, IIRC, went to Asahi's site.  

However, there is Plamo Linux, a Japanese distro based on
Slackware--around the time I learned about it, I'd already gotten
involved with Gentoo, so never followed this up. However, I wonder if
you might have success using their kinput2 and canna packages. I don't
have Slackware on any boxes here, so can't test it, but if you feel
adventurous, kinput2 is at

ftp://www.nlcomputer.com/pub/Plamo/Plamo-3.0/plamo/xap1/kinput2.tgz
(Kterm is in that directory as well, as kterm.tgz)

Canna is at 

ftp://www.nlcomputer.com/pub/Plamo/Plamo-3.0/plamo/cn1/canna.tgz


If you give it a shot and it works, let us know, and I'll add it my
pages on Japanese in Linux.
