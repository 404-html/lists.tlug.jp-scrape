On Sun, Aug 18, 2002 at 08:25:22PM +0900, Ryan Shaw wrote:

>It's a Corega BAR SW-4P Broadband Access Router
>Switching Hub

OK, so you're using two routers, the Fujitsu and the Corega,
which is an integrated router and switch.  The Fujitsu probably
is, too, unless it has only a DSL connection on one side
and an Ethernet connection on the other (in which case it's
just a router).

>This sounds like the way to go, but I can't seem
>to figure out how to set it up to function as a 
>simple switch.

Since this device is a router with an integral switch attached,
the short answer is that you can't.

The easiest thing to do here is put the Corega back in its
box, and go buy a switch.  An 8-port Corega 10/100 switch
is only a few thousand yen, and they work well.

>There is a button on the back of this thing labeled
>"Cascade" that can be set to either "PC" or "Hub",
>could that be related? If I set it to "Hub" everything
>stops working.

That affects one port on the switch part, either the
first one or the last one (the instructions will say,
if it is not marked on the device).  What it does
is set that one port as either an uplink port to
another switch or hub (cascade) or a normal switch port
to connect it to a PC or other equipment.  Under the hood,
what is happening is it reverses the TX and RX pins on that
port when in cascade mode, since you would otherwise have
to use a crossover cable to connect two switches together.

>Maybe I should just go out and buy a true hub, but
>this thing also has packet filtering features that
>I would like to use if I ever get this set up. 

Does that mean the Fujitsu doesn't?

It's probably possible to make the Corega do what you want,
but it will require going over the docs to find it.  If you 
can't read them, it's time to bribe someone who can :-)
