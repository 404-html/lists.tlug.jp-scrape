You can use awk:

BEGIN { current = 1;
        text="";
}
$1 == current {
        if (current > 1)
                print text;
        text = $0;
        current++;
        next;
}
{
        text = text $0;
}
END {
        if (text != "")
                print text;
}


You will also probably need to add a space (something like:
	text = text " " $0;
It depends on what you need.
