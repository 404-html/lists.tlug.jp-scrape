But while I'm at it:
I was looking for a lighter alternative to my heavy old laptop
for travelling. It should be able to exchange data with my 
linux system and should be capable of Japanese.

Probably eventually I don't want to miss keybord input, but I also 
had a look at the 3Com Palms.
Searching the tlug archives and following the links there it looked like 
you basically leave the Palm OS on the Pilot (and descendants) and program 
applications for it with the emulator on your linux desktop. 
Same with the Japanese Palm OS.

But then a friend told me that he had seen Jeff Dionne presenting Linux
running _on_ one of the palms (in May in Cologne).
(You can see how far they got under the Linux Microcontroller Project
http://ryeham.ee.ryerson.ca/uClinux/).
So I was wondering, whether any Japanese display and input was on
the way as well.
