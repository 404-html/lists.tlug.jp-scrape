            ANNOUNCEMENT:  TLUG ADVOCACY LIST FORMED
---------------------------------------------------------------------
 
Tlug now has a new list for all members who are interested in 
non-technical discussions related to Linux and Open Source Code 
software in general, the "tlug-advocacy" list.  This brings to three
the number of general lists run by tlug, as follows.  All tlug lists 
are open to anybody who wishes to subscribe.

    tlug-advocacy
    tlug
    tlug-admin
 
The tlug-advocacy list (tlug-advocacy@example.com) and accompanying
digest (tlug-advocacy-digest) have been set up to reduce the load on the
tlug main technical list, and include (but are not limited to) the
following topics: 

    What is Linux
    What is Open Source Code Software
    Why does Open Source Code work?
    Jokes, General Conversation (hopefully linux related :-)
    
The main list (tlug@example.com) and accompanying digest
(tlug-digest@example.com) will now focus entirely on technical
issues, which are firmly limited to the following: 

    Specific Software issues concerning linux/unix OS's
    Specific Software issues concerning tools, programs, etc.
    Specific Software issues concerning Open Source Code
    Hardware-related issues
    Books / Book reviews
 
The admin list (tlug-admin@example.com) focuses on administration
issues of the tlug organization itself, and includes (but is not limited
to) the following topics: 
     
    Tlug server issues
    Tlug webpages
    Tlug mailing Lists
    Tlug meetings/Nomikais
    Tlug finances
    Tlug projects (CDRoms, T-Shirts, etc)

Feel free to join into any or all of the tlug lists.  

To subscribe or unsubscribe to any of these lists: 
    send a message to majordomo@example.com with the command 
    SUBSCRIBE <list> in the body (where <list> can be: tlug, tlug-digest,
    tlug-admin, or tlug-advocacy). To unsubscribe, use the command
    UNSUBSCRIBE <list>

Thanks to everyone on the lists for their support to date!
