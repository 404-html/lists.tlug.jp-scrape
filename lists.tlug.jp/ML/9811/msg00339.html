SuSE puts the X link in /var, huh?  I guess everyone likes to do it
differently :).  And the servers go in /usr/X11R6/bin/XF86/bin/ ??  odder
and odder.

anyway, the command to link is 'ln -s'.  use it like this:

ln -s original target

ie:

ln -s /usr/X11R6/bin/XF86/bin/XF86_SVGA /var/X11R6/bin/X
