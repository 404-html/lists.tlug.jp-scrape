Under RedHat there is a check box in the Network Configurator's Edit Interface
dialog which says "Allow any user to (de)activate interface" Checking this
should allow a normal user to bring the interface up or down.

I assume that it adds 

USERCTL=yes

at the end of the config file.

When you do "ifup ppp0" under RedHat and you are not root it launches 
"/usr/sbin/usernetctl" which is a suid program which allows a non-root user to 
control a network interface.

Since the filenames under TL seem similar to RedHat then maybe things work the
same.
