Hello all. 
I am using TL2.0J Standard on a Sharp Mebius laptop. 
Next big goal is to get it connected to GOL via PPP. 
I used the Network configuring icon from the xwindows 
desktop which seems to have generated the file below 
at   /etc/sysconfig/network-scripts/ifcfg-ppp0 . 
However, the modem icon is not working (click on it 
and nothing happens). ppp-on ppp0 results in an 
error message of something like 'users not allowed to 
control this interface'. 
windows 95 setting has the modem on com2 (would 
that mean ppp1?); I have added an entry for ppp1 
and tried connecting with that but haven't fared any better. 
btw, am I correct in not setting PAP when connecting 
to GOL? 
I would appreciate greatly any guidance tlug members 
can offer. 



PERSIST=yes
DEFROUTE=yes
ONBOOT=no
INITSTRING=ATZ
MODEMPORT=/dev/modem
LINESPEED=115200
ESCAPECHARS=no
DEFABORT=yes
HARDFLOWCTL=yes
DEVICE=ppp0
PPPOPTIONS=
DEBUG=no
PAPNAME=
REMIP=
IPADDR=
BOOTPROTO=none
MTU=
MRU=
DISCONNECTTIMEOUT=
RETRYTIMEOUT=
USERCTL=no
