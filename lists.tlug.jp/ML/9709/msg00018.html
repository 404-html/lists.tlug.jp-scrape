Hi all,

Somebody out there has apparently invented a new form of spam -
adding links to an 'add your own links page'.

The TLUG Linux links page has been under assault for the lsat
few days. Last night there were about 200 links added, mostly
for 'sexy Latin babes' and that kind of thing.

I've temporarily disabled the perl script running the links
page, but I'd like to keep it available if I can. I was thinking
about a second perl script that confirms an entry before the
link is added. Something like, "Please check the link title and
URL to make sure they are accurate."

Will this be enough to stop the spammers?

Thanks,

Jim S.
