Opera.  I'm surprised no one has mentioned it yet.  It's only drawback
is that it's a bit stricter when it comes to standards, and as so many
web sites ignore standards these days, some pages won't render properly.

of the cases.

might be better off simply using your own mail client to pull in (and
send out) gmail.
