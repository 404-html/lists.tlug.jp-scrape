Dave M G wrote:
> Okay, so obviously the first assumption is that the script is probably
> not running the command correctly. So I try running the script from
> the command line, and it works perfectly.
>
> So I'm baffled. Why would a script that runs perfectly from the
> command line not complete its task when executed from cron?

Because the environment cron is running it in is different.  The PATH
and other environment settings are usually minimal fro cron so you may
need to set them explicitly or provide the full path for executables in
your script.
