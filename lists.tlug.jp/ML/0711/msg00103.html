> Hi everyone, here is another question. I have a fedora 7 machine in my
> office that I want to use as a web server. I have apache 2.0 running,
> the configuration is the same as before on my redhat 9 machine and I can
> see the index page when I type in our url but there are no files listed.
> The files are in the right place and the apache pathway is correct.

Probably default configuration has the Indexes option switched off in
Fedora 7. Edit httpd.conf ( /etc/httpd/conf/ I think) and search for
"Options" and add:
   +Indexes

More information here:
 http://httpd.apache.org/docs/2.0/mod/core.html#options
 http://httpd.apache.org/docs/2.0/mod/mod_autoindex.html
