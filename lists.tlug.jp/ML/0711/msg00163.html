"Nguyen Vu Hung" <vuhung16plus@example.com> wrote:

> With the utility "file" or mplayer, I can get some information about
> an avi file, but the length of the .avi wouldn't show.
> 
> So how can I do that? ( Suppose that I can't use GUI )

mplayer -identify |grep ID_LENGTH

But be aware that the returned value might not be correct,
depending on the container and the program used for encoding.
