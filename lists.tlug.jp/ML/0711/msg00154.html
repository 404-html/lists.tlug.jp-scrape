I have given the libpst a try. :)  The 0.5.2 version.

It works ok, with a couple of issues.

One, it doesn't handle password protected files.
There is no option in the program to offer a password,
and if it tries to open a password protected file, it
gets an error when it tries to index it.

Second, it doesn't much like really large files.  It
will load them, and start processing them, then die with
a malloc error. ^^;

Poor thing runs out of heap and dies. :P

By large though, I mean the initial file I started with
was 198MB.  :)  When I broke it up into smaller files, it
was able to handle them.

When I make landfall again, I will go in and submit a bug
report, but I figured the people who helped me find it
might be interested in a summary of the results. ^_^

Not a neat and clean solution yet, but it does work. :)

Well, mostly.
