"Andrew Errington" <a.errington@example.com> wrote:

> * Since Jan 2007 YahooBB! have had a rolling program of blocking port 25
> across their network.  It seems it has finally reached me.
> * My hosting provider offers no other port than port 25
> * YahooBB!'s SMTP server only accepts mail with a YahooBB! return address.

I had this problem too, i used then SSMTP (SMTP over SSL, port 465)
or piped it trough an ssh tunnel (-L2525:127.0.0.1:25).

These days the Message submission protocoll (RFC2476, port 587)
should be also possible.

If your mail server provider doesnt support one of these,
bug him until he does :-)
