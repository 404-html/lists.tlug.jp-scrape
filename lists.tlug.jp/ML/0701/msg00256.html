I'd like to hear what people are using on this list for managing dynamic
IP adresses. Any advise would be appreciated.

I use no-ip.com. They have a dynamic dns update program which is available in most packaging systems (incl. debian/ubuntu (no-ip), gentoo (noip-updater) and fedora (noip)) or as source code from their website. Works pretty well and I've not had any problems with it.
