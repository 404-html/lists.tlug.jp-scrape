TLUG,

After much file by file removal, I've seemingly got the errors down to two packages that are blocking me from upgrading to Edgy.

It's insane but these two files are just help file documents, and yet some error with them is holding me up from installing an entire OS.

Anyway, ranting aside, below you can see my dilemma. I do the apt-get -f install thing, and it tells me that two help files have "unmet dependencies". I try to remove one, openoffice.org-help-ja, and it says that I can't because language-support-ja relies on it. So I try to remove language-support-ja and it tells me I can't because a bunch of files depend on it. I start to remove one of those files, and guess what, language-support-ja depends on it, and so the circle is closed and I can't get out.

Is there any way I can manually blow these help files out of the dpkg/apt-get system, or resolve their dependancies, or anything so that I can successfully complete the sudo apt-get -f install process?
