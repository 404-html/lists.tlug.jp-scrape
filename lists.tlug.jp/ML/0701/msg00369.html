I now have an elegant almost-complete solution to the problem
of HTML forms being prematurely submitted when Enter is pressed in a 
TEXT field. The solution involves JavaScript (as expected), but the 
form continues to work in the usual way if JavaScript is disabled.

The JavaScript solution involves disabling onSubmit and then forcing it
to happen with a special JavaScript button. A bit of trickery makes the
traditional "Submit" button become visible if JavaScript is disabled.

It's implemented in the form at:
http://www.csse.monash.edu.au/~jwb/cgi-bin/wwwnewword.cgi

The key JavaScript is at the top of the page, and the Submit buttons
at the bottom (you'll be able to see why only one is visible.)

Thanks for the suggestions and comments.
