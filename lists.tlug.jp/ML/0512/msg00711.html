Heh, I'm revising it after more experimentation.  :)   Instead, 
go to the jpninpt.html page linked above, and follow the link to the
Ubuntu wiki, the one with the hyper link "The older one."  ~That~ link
takes you to a link for using Japanese in the latest Ubuntu, Breezy
Badger, using uim instead of scim, and works in all applications though
you'll have to fiddle, do some right clicking and setting input to
anthy--the article explains it fairly well.

I found that using the scim-anthy package I originally recommended, I
was unable to input Japanese in firefox and some others--hrm, abiword?
Openoffice? I've already forgotten.   It was fine for terminals.


I find scim-anthy easier and more intuitive to use than the uim-anthy,
but unfortunately, the Deb derivatives don't seem to work that well with
it.  I'm sure it could be fixed, but as I usually only boot into a Just
Works(TM) distro for multimedia stuff that I can't get working well
elsehwere, I don't feel like trying to trouble shoot it with my limited
knowledge.
