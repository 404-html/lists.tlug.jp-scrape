Jim wrote:
> Godwin Stewart did not post a reference or source code for rename. 
> That's a clue that it might be a common standard command. 
>   
There is actually at least two rename command and they are significantly
different.  One is a rename  from http://rename.berlios.de/ that goes
under the name of renamexm under gentoo but seems to be setup as a
rename alternative under debian.  The second comes with the util-linux
package  http://www.kernel.org/pub/linux/utils/util-linux/

the first allows sed like syntax i.e.

rename 's/\.htm$/\.html/' *

while the second rename [match] [replacement] [glob]

rename .htm .html *.htm
