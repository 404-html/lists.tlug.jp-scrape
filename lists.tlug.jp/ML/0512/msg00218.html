Which reminds me, if you run Firefox and you don't have the Conquery
extension[1] installed, install it now! What it does is allows you to
select text, in any place that text can normally be selected (e.g.
your Gmail compose textentry, a webpage, etc.), then right-click that
text (and here is *one* place where I *love* the fact that Windows has
that Context Menu / Properties key on the keyboard--anyone know how to
make FluxBox / Firefox do that?), choose the new "Query to..." item
from the context menu, and select a search engine (the same list that
you get in the Quick Search box at the top right of the standard
Firefox UI). The results of said search then open in a new tab.
Perfect!
