I should mention that the KDE suite is the second app I download and
installed on my Z. The first was terminal. There was no tweaking, no
hacking needed.

Please give it a try. It really is a pleasure to use.

My SL-C3000, and it is running KO/Pi and KA/Pi flawlessly. As I mentioned
before, I had been trying to use the usb memory key version on my linux
and Win boxen, because I like the idea, but alas, it seems to cause
synching problems. The way the Z connects to the PC might be an issue,
too. I do not yet have a NIC for it, so I am using the Windows-only Zaurus
drive PC connection network type right now.

Not having luck with synching from the usb, I installed KO/Pi and KA/Pi on
the Win machine directly, and it synchs just fine. It really is lovely.
Even without a Zaurus, I would recommend KDE PIM/Pi for Windows and Linux
users as an all-around solution. The developers are working on a MacOSX
version. I really do like this software quite a lot.

That having been said, I still eagerly plan to try out Emacs planner mode.
