Think you will have the same problem with Ubuntu too. Maybe the
problem is not in your
GUI environment (Gnome, KDE), but on the lower level (the X server
itself) - like graphic card recognition, graphic modes etc. Take a
look on the /tmp directory. The messages that you are getting during
the X starting are logged there too. Use some web paste service (for
example [ http://rafb.net/paste/ ] ) and post the log file there. Then
maybe you can get more specific help for the people in this list.
