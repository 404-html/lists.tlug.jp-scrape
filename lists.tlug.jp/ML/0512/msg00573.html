"Lyle (Hiroshi) Saxon" wrote:

> I didn't realize that KDE was so heavy.  
> Is Gnome fundamentally lighter than KDE?  

By default, both are fat. 

In the current issue of Linux Journal, there is an article 
that explain how KDE was slimmed down for http://rule-project.org/
You should read that article. 

Xfce is a fine lightweight window manager 
and should receive strong consideration. 

It was available on older versions of Knoppix, 
along with about a total of five or six window managers 
on Knoppix 3.3. Newer versions of Knoppix have fewer 
window managers. The older Knoppix is a great to try 
other window managers out, without installing them. 
Use the free and swapon -s commands to keep track of 
how much memory is being used.
