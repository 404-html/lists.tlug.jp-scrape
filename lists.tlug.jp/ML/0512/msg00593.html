You can do VoIP fine with open sources software it just requires a bit
more effort to set everything up.  The biggest problem is dealing with
firewalls.  You need an intermediary if both sides are on a poorly
configured firewall.  You can use asterisk for this.  If you use iax2 it
shouldn't require any configuration for most firewalls.  Most TSP
support sip (though most won't offer an unbundled service), which is
also supported by asterisk.
