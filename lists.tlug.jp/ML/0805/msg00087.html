Unfortunately, PR-200NE is not an easy one to configure (or others too?).

There are several webpages in Japanese, such as:
http://okwave.jp/qa2926781.html


In summary,

1. Set static IP adress (not given by DHCP) on the server machine
2. Open the www ports to the server address in "static IP masq"  静的IPマスカレード
3. And open the www ports to the server address in "packet filtering config"  パケットフィルタ設定

I think this is the minimum.
