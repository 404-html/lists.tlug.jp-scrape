And one brief comment on Exim.  We run the GOL mail system on Exim and
it has been quite stable and has nice reporting features.  The value of
the human-readable and editable config file cannot be overstated, and
Exim's regex-based spam filtering capabilities are excellent.  There is 
also a very good and very thorough online manual freely available on the
Exim website.
