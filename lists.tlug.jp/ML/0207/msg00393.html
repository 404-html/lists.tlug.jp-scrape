Excuse the mail client.  

try a couple commands:
# ifconfig eth0

$ more /var/lib/dhcp/dhclient.leases

I am assuming this _is_ a dhcp setup.  If you see no IP address
listed in the first command, and the second doesn't show you a file
that starts like
lease
{ interface "eth0";
  .
  .
  .

Then you are not getting your dhcp address correctly.  If the first
has no IP address, but the second shows you a lease, see when the
lease expire[s|d].
