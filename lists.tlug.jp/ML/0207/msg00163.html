On Tue, Jul 09, 2002 at 08:33:17PM +0900, hatsuhiro wrote:
> How can we configure Linux to first start with the GUI and then ask for
> login.

You can boot your system to a runlevel which starts a login manager for
this. On most systems, this runlevel is 5.

To make this permanent, edit the file '/etc/inittab' and change this
line:

id:3:initdefault:

to

id:5:initdefault:

which will start up 'kdm' or 'gdm', depending on whats installed on your
system.
