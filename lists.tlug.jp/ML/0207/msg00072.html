But there is a short answer to the problem of MBR stealing.  During or after
installing Linux, make a boot floppy.  After installing Linux, back up the
MBR (e.g. using the dd command) and back up any other important boot
records.  After installing anything else, boot the Linux boot floppy and
restore any desired boot records.
