Unfortunately, while that is the worst that is _likely_ to happen it
is certainly not the worst that _could_ happen.  The worst that could
happen is a malicious server somewhere exploits a buffer overflow in
a client you are using which opens a connection back to their server.
With said connection they can get to any machine on your network 
using that box as the jump point for attack. This is a very unlikely
turn of events but it could happen.  Just be safe and don't do anything
as root that doesn't need the access. :)
