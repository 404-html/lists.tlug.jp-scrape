You can:
1. Change 6th column in /etc/fstab for the root partition to be
   0. But this will stop to check root partition at all:

/etc/fstab
/dev/hda1    /    ext2   defaults   0     0 <-  this maybe is 1 in the moment

or

2. Add ro to the boot parameters. I don't not what kind of
   boot manager you use but for example for grub (menu.lst cfg):
   kernel=(hd0,0)/vmlinuz root=/dev/hda1 ro
