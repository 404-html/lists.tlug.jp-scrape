Thanks! I wanted the following info, which is not in the man page nor the
Running Linux book I have.

The fifth field (0) is used by dump (a backup utility) to decide if a
filesystem should be backed up. If zero then dump will ignore that
filesystem. The sixth field (0) is used by fsck (the filesystem check
utility) to determine the order in which filesystems should be checked.
If zero then fsck won't check the filesystem.
