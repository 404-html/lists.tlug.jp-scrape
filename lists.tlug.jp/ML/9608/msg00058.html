I've added three links to the Japanese Resource section of the
TLUG home page (http://www.twics.com/~tlug/linuxinjapan.html).
BTW, Jim updated the graphic.  Looks nice now.  :-)

The links are:
http://www.skyia.com/~devriesp/japanese.html

which is unfortunately about Windows '95 and NT.  Zannen!  I put it
on because I know it is relevant to a lot of club members that are
running English Windows '95 or NT and want to add Japanese support
without installing Windows '95-J or NT-J.

Another link is:
http://www.rdt.monash.edu.au/~jwb/edict.html

which is about EDICT, a Japanese/English dictionary in machine
readable format that can be used on Linux.  :-)   As is the
natural state of justice in the world.  

The third links is:

http://www.wg.omron.co.jp/cgi-bin/j-e

Which is the gateway to an interesting Japanese <-> English
Dictionary server.  

I've been having to deal more with Japanese and am getting
my machine up to speed.
