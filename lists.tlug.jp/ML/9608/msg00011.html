500 Server Error means that my CGI application failed with an error
return (usually a Perl compilation error).  You may have found a new a
different way to generate that error, but that's where I'd look first.
Look on your console for errors from the shell or script interpreter.
Occasionally these things get into <ServerDirectory>/logs/errorlog (or
whatever it is).  I haven't figured out how to get the script
interpreter to redirect errors to a file, unfortunately.
