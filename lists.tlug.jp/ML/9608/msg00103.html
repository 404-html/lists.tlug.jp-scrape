And just to add to topic drift...  in case some readers haven't
tumbled across 'ncftp'...

The same effect can be had when using ncftp with the command
"page file.lsm"  This gets the file and runs it through your
default pager.

I recommend ncftp which has a number of nice features over a
plain command line ftp.  It will recursively fetch directories
effectively mirroring part of a tree (and can get all of the
permissions right), supports resuming a failed command (if the
serving host does :-), remembers where you were the last time
you visited a site, and other such creature comforts.  The latest
version (2.3.x) includes a "split screen" mode (which I don't
happen to like, but some do).

It is available for Linux (and Win95/NT among others).  (If
you use Win95, check out http://www.itribe.net/virtunix/ for
a variety of tools to make you more comfortable in a foreign
land.)
