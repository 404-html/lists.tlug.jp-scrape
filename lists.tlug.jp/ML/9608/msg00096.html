I *seem* to recall that the source for GNU make only had a single
Makefile in top directory.  You should be able to compile it with one
of these flags:

 -f FILE, --file=FILE, --makefile=FILE
                              Read FILE as a makefile

For example if you have the make source in ~/sources/ do:

   # cd ~/sources/
   # make -f ~/sources/Makefile

If this does not work, there must be a binary distribution of make with
the dirent fix in there.
