On Thu, Jun 08, 2006 at 02:23:20PM +0900, David Riggs wrote:
> Fluxbox is indeed lightweight and fast, thanks folks for the tip.
> 
> But it is pretty mouse centric (yes I will try ion next).
> 
> Is there a way in Fluxbox to bring up the applications menu with a key 
> stroke? Thanks Josh for the "keys" file, but I would like to be able to 
> pop up the menu and arrow-key around without the mouse. (In part because 
> all I have is a track point on this barebones R50e Thinkpad.)

Did you see my page too, before Josh shamelessly plugged his as an
answer?  :)  

Assuming you're running fluxbox 0.9 something, you want a key binding to 
RootMenu---for example, mine is

Mod4 KP_Enter :RootMenu

This can then be navigated with the arrow keys.

(It's also mentioned in the man page.  The fluxbox man page is quite
well written, and parts about keybindings are, IMHO clear even to the
newcomer.)


In the older versions (although 0.9.x is called development, it's
considered stable, the old 0.1.14 version hasn't been worked on in
years) you would have to use Mr. Kita's patch, also mentioned on my
page--however, I don't think too many distros still even have 0.1.14
available.
