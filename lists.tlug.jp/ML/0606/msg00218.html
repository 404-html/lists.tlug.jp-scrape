Now that FC5 is out, FC4 support declines dramatically.

Here comes FC6 (and the abandonment of FC5):
I don't think "drop dramatically" and "abandonment" are quite fair. The Fedora project continues to maintain one release prior to the "current" one, and the Fedora Legacy project provides security updates for earlier releases. There is a distinction between fixes and new functionality, which is where this thread started.
