Question is, did we determine the uname output was FC1?


The definitive way of determining which distro you are running is:

cat /etc/*-release

For example, on my box:

: jglover@example.com; cat /etc/*-release
Gentoo Base System version 1.6.13

Or on a RHEL3 box:

somebox% cat /etc/*-release
LSB_VERSION="1.3"
Red Hat Enterprise Linux AS release 3 (Taroon Update 5)
