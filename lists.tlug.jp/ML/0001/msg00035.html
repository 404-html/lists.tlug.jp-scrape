My pick-of-the-week (and then some!) in must-have software for a
Japanese Linux environment is Gjiten, which just had release 0.3
announced on Freshmeat.

Gjiten is an outstanding Gnome frontend for Jim Breen's Edict and other
dictionaries using this format.  Works great, searches E and J, and 
works with kinput2.  

Get it here:

http://www2.kobe-u.ac.jp/~9915104t/progs/gjiten/
