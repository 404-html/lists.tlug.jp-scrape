Gloss:  the idea is to alias the domain names to your own host.  The
name lookup will get the equivalent of localhost, query the httpd
server there (well, here, actually ;), fail to find the page, and go
merrily on.  Or if there's no httpd there, even better; the connection
is refused.

You need to have 

hosts:          files dns

in your /etc/nsswitch.conf file or the appropriate equivalent in
/etc/hosts.conf if you have a rather old libc.
