I like using pine for e-mail and can now, thanks to information in some
archived TLUG messages, use it to read and enter Japanese using kterm and
kinput2 -canna &.  

For some reason, though, when I try to reply to a message containing
Japanese, the Japanese in the message I am replying to doesn't display
properly in reply mode. Also, some messages that I get that contain
Japanese completely discombobble my display, forcing me to quit pine and
restart it before anything displays properly again. 

Anyone know why this happens and how to fix it?  I'm using pine 3.96,
which I got on a Japanese Slackware CD.  It's not pine-j, which I also
understand exists.  Is the solution to use that?

Finally, does anyone have an opinion on Nmail for Japanese?  (I've
downloaded it from the Japanese site (http://nmail.hiug.ne.jp/) but
apparently need to upgrade from libc5 before I can install it.)
