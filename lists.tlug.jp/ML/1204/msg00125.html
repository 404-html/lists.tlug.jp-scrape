Just a note to all from one of the mods....

Although proper (to we old folks) posting style seems to have gone out of fashion,
on tlug we do try to maintain it.  This means please do not top post.  (Although if 
using smart phones, we do make exception.s


A: Because it messes up the order in which people normally read text.
Q: Why is top-posting such a bad thing?
A: Top-posting.
Q: What is the most annoying thing in e-mail?



Please try to trim and quote inline.  By this we mean, remove
unnecessary parts of the message to which you're replying, and answer as it's 
a conversation.  For example, if someone has an email to which you are replying point
by point, answer point A below point A, point B below point B, and so on.

List member Godwin Stewart has two excellent articles about proper posting style. 


http://linux.sgms-centre.com/misc/netiquette.php

http://howto-pages.org/posting_style/

Please take a moment to read them. 

Thank you
