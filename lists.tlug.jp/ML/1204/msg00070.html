On Wed, 11 Apr 2012 11:16:45 +1000
Johan Berntsson <johan@example.com> wrote:

> How about swap partitions? Do the old 2*memory size rule still hold?

Not really. You can use any swap size you want.
But i advice you to make a swap that is larger than the maximum installable
ram size.

This is because there is a certain kernel bug which i've hit a few times
that fills swap _and_ ram when swap is smaller than ram. Interestingly,
i and my brother are the only ones who've ever hit that bug (i have not
heard of other reports) and it is only an issue if there is swap available
and if the system is under some specific load (which i dont know what it is)
