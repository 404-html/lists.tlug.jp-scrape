> Still haven't figured out how to disable the touchpad when typing. The

I'm not sure if this would help, but I had a strange problem with my
laptop + Ubuntu.  Sometimes I could dim the screen and sometimes I
could not.  The option to dim it disappeared every once in a while.

I finally figured out the "reason" (sort of).  I dual-boot it with
Windows and sometimes I use Windows and dim the screen there (using
the Fn key + one of the F-keys).  Once I do that, I can't adjust the
brightness within Linux (this key combination didn't work while in
Ubuntu).  Probably not while in the Grub screen, either, but I don't
think I tried that.

Anyway, my "solution" was to put it on maximum brightness every time I
log out of Windows.  I know you don't have Windows on your system, so
this doesn't help you at all...but there might be some hotkey
combination that you could try while in Grub to put the touchpad in a
state which could give Linux control over it...
