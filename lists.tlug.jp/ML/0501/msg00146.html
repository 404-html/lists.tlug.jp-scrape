    Viktor> Polling from a user space application may be a UNIX way
    Viktor> but not the one to be proud of, IMHO.

I don't see what's wrong with it if it's the only way to use the user
space application.

Maybe JG should have done a little more research; since GNOME and KDE
can do it, it's almost certainly possible to do better than polling.
Window/session managers are just user apps, after all.  Even though
the X server itself often runs as root, that's generally considered a
bug, and if it has special access to information about available mass
storage devices, I would be horrified!  Wouldn't you?

So I bet there's a signal-driven way to accomplish this, maybe even
from a sufficiently bloated shell (zsh comes to mind).
