The modem seems to be correctly detected, but it doesn't seem to
understand the AT command set.  Read the modem manual which should
tell how to configure it.

Install a program such as dip that allows you to talk to the modem
port directly and try configuring it by hand.

Read the modem manual to find out whether this "failed with XXXX baud
stuff" makes any sense; on USB it really doesn't AFAICT (USB runs at
megabaud speeds, not "2400", etc).  If the modem manual doesn't help,
bitch to the vendor.
