> Now that you mentioned this, I did install RH 8 to a ThinkPad 240 from 
> the
> hard drive (but the same one, not a separate one) when I couldn't find
> Linux drivers for the external SCSI hard drive. At that time the laptop
> still had Windoze on it, so I made a sector large enough for the three
> image files, copied them to it, and then just selected "install from 
> images
> on hard drive" (or whatever it's called), and installed without any

Another alternative is to do a net install using FTP or HTTP (Forget 
which one I did) - You just copy the proper image to a boot floppy from 
the CD and load the images to a local network server.
Seemed to go quite fast on a local network when I did it about a year 
ago.  NFS is probably an option, too.
