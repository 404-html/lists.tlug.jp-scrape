  • Subject: [tlug] im-ja v0.5
Announcement follows.

I've released im-ja v0.5 
(For people who don't know: im-ja is a Japanese input module for GTK2.)

Binary and source packages are available from the project webpage at
http://im-ja.sourceforge.net
Gentoo ebuild probably needs an update. 

Im-ja has stabilized a lot since the last release (especially canna mode),
now I can actually recommend it.

From the ChangeLog: 
    Now using Gconf schema.
    Gettextized, intltoolized.
    Canna mode rewritten from scratch to use libRKC only.
    Kanjipad updates.
    Canna and/or wnn mode can be disabled by configure.
    Gnome desktop entry.

Comments/suggestions are welcome.
