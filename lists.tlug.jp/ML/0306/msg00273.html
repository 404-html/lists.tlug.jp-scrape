On Fri, 2003-06-27 at 22:03, ATT wrote:
> Hi All,
>  
> I recently installed Redhat 9 on my PC which has the standard Japanese
> 106 keyboard.  I have been successful in inputting Japanese using
> kinput2 and canna that the Redhat installer loads automatically when
> you choose to install Japanese.  (of course I had to create scripts
> with standard XMODIFIERS="@example.com=kinput2" LANGUAGE=en_US LC_CTYPE=ja_JP
> program-name)  Anyway, I would like to reconfigure the toggle key from
> shift-spacebar to the standard Japanese toggle key. 
>  
> Does anyone know how to do this?

The file might be in a different place but on Gentoo it is

/etc/X11/app-defaults/Kinput2

and the line is

*KinputProtocol.XlcConversionStartKey:  Shift<Key>space \n\
                                        <Key>Zenkaku_Hankaku

This actually allows you to do both.
