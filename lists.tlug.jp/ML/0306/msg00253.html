It's not a floppy distro like FreeSCO, which is too bad, but 
understandable - you actually need a HD.  The install is done from a 
bootable CD iso that you download, though after the install, you don't 
really need the CD.
(I'd yank it to save power, since that kind of box is typically on all 
the time.)
I've never seen any router that lets you set up things like NAT as 
easily as IPCop does.
IPCop is a fork from the Smoothwall distro, but the guy behind 
SmoothWall has a reputation for being a real jerk, so I avoided it.
It does VPN, too, and Snort, Squid, DynDNS, DHCP and lots of other 
stuff.  No complaints about it at all.
Enjoy.
