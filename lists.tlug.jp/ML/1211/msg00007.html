There are two pretty good ones I've found for Android: Offline Dictionaries (for when I can't get a signal or am in a different country) and, of course, the WWWJDIC app.

You might have to reflash the kindle with cyanogen to make them work, but it's possible that you can install them using adb. I don't know exactly how far Amazon has modified Android for the Kindles.
