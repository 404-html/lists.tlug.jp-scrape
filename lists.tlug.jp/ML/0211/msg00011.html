On Sun, Nov 03, 2002 at 12:16:32PM +0900, Eric O. Flores wrote:
> Yes I try "jp_JP" it seems that the mandrake distribution uses "ja_Jp"

Really? It uses capital-J, small-p? If so you should file a bug report.
That's wrong. According to authoritative sources I've looked up in the
past (sorry, don't remember what--maybe the POSIX standard?) the correct
form is ja_JP (2-letter, lowercase language code, underscore, 2-letter,
uppercase country code).
