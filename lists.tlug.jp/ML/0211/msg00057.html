I've tried about 4 Linux distributions and I don't think any of them

> or maybe

> Gentoo. 
> The real Gentoo is the fastest penguin, but...
> 
> some Gentoo Linux users are telling me that one is 
> always compiling from source with Gentoo, and as a 
> result, it takes a long time to install, modify, 
> etc. 
> True? OB (odious bunk)? 

I'm running Gentoo now, and I am continually compiling from source, but
this is A Good Thing. And it's not particularly time consuming if you
have a fat network pipe and a fast processor. Not only that, but the
user side to upgrading a package is extremely simple. For example,

emerge evolution

will download the latest evolution sources AND any other packages it
depends on, and compile and install everything for you. 

In addition, it affords you lots of control over how your packages are
built, and opportunities for optimization which increases the speed of
executables.
