> > 3..Where can I get simple "httpd.conf" file ? The
> > default http.conf file have a lot of option which I
> > don't understand.
 
You might want to start with the 'highperformance.conf' file. As far as
I know, it is only included with the source, but is very simple and easy
to understand. You can add more functionality as you need it and learn
much at the same time.

> The basic httpd.conf file requires little tweaking in fact. I suggest you
> look at the manual installed with apache in order to get better acquainted
> with all the options.
> 
> http://localhost/manual/

That is the perfect place to find out about those options that you may
need. Also, try to get the 'Apache Pocket Reference' from O'Reilly. Much
more useful than those '21 days' or '24 hour' books.
