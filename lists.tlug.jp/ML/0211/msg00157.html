> modprobe, OTOH, *does* need to be done as root.

Only if you want it to do anything. ;)

Seriously, you can use the -d, -l, and -n flags as a normal user.

The only reason that I mention this is that I am a big believer in doing as
root only what absolutely must be done as root.

When I want to sod around with modules, I usually do all of the preliminary
stuff as myself, and then run only the insmod | modprobe command as root.
