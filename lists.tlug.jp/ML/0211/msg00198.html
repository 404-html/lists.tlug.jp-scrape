Hey guys, you should check this out;

http://unit.aist.go.jp/it/knoppix/
http://www.knopper.net/knoppix/

It's Knoppix Linux, a Debian-based Distribution. It's CD bootable, and
it's ability to auto-configure itself is simply amazing. Usually, in any
installation, it's a pain to have Linux detect and recognize video or
sound or, especially the internal Ethernet port (especially in notebook
PCs), reads and mounts NTFS partition... This thing does EVERYTHING.

It doesn't affect the HD, It has most of the software that people would
need, (OpenOffice, mozilla, etc.), Japanese configuration is already
there, and if you don't like it, just take it out of the CD-Drive and
that's it!

Interesting thing is that, this has something to do with the Japanese
government trying to use Open Source, which was a topic here a while ago.
We all know that it is impossible to expect laymen to actually install
Linux, so the scheme seemed a bit unrealistic. But with this thing, it
really seems feasible. You can actually make a Linux machine right there
by anyone. Maybe things might move a bit faster than expected...
