> >

yes, this is definitely something that RPM should allow.  it would be
easy, except for the fact that RPM doesn't package the .spec file in
binary RPMs, only in the source RPM.  So I guess if you had the .src.rpm
as a file, and you had the binary RPM installed, then you could repack it
with some kind of script.  But then again, if you have the .src.rpm you
can just --rebuild it anyway.
