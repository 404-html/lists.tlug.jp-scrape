Let me see... searching the archives... you wanted to
read the PDF files distributed at http://kanpou.pb-mof.go.jp
and it happened that they were in Shift-JIS?

Install Acrobad Reader 4.0 and on top of that download
and install the Asian Fonts Pack
http://www.adobe.com/products/acrobat/cjkfontpack.html
be sure to choose "Platform: Unix". You should get a
file named "jpnfont.tar.gz"
