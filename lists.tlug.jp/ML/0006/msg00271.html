I don't use any automation tool for archiving old mail, but sometimes
find the following couple of lines in .procmailrc very useful:

:0 c
backup

:0 ic
| cd backup && rm -f dummy `ls -t msg.* | sed -e 1,100d`


This keeps 100 latest messages in the backup directory
