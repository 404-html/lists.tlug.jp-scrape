tkinter is still the community's "default" GUI, despite its lack of object
orientation.  There is a lot of code and documentation written for it giving
a significant head start.  And it runs on X and Windows acceptably nowadays.
Have you read Fredik Lundh's _An Introduction to Tkinter_?
http://www.pythonware.com/library.htm  Part of _Python and Tkinter_ is also
available online http://www.manning.com/Grayson/index.html
