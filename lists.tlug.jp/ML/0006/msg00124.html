Did you try putting
<META HTTP-EQUIV="Expires" CONTENT="0">
in the header of file.html? That would (hopefully) cause the cached version
of file.html to expire immediately and force a fresh download each time it
is required. I don't have a web server handy with which to try this out right 
now, though.
