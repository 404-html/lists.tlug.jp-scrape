Quite some time ago, I put up a question about decrypting some Japanese
PDF files.  Two approaches were suggested -- a Perl module distributed for
that specific purpose, and Ghostscript backup up by decryption support on
the system.

Thanks to everyone who offered advice.  The Perl script sort of works on
the target files, but there are enough problems to make it worth giving
Ghostscript a try.

Ghostscript 5.50 fails with the following:

*****

Processing pages 1 through 1.
Page 1
Error: /undefined in /setsmoothness
Operand stack:
   0.02
Execution stack:
   %interp_exit   .runexec2   --nostringval--   --nostringval--

[snip]

*****

It looks like the simplest thing to do would be to define /setsmoothness
to be a noop function with one argument.  Does anyone know what the
Postscript for that would be?  The only postfix language I've ever dealt
with directly is BibTeX.  :-(
