When you installed it required you to enter a password (and
confirm it) for the root user.

If you've forgotten it already, you can use the "rescue" mode
of the RedHat CD-ROM to reset it.  (See Chapter 9 of the Red
Hat Linux Customization Guide.)


You will probably be able to get the X Window System configured
for your machine.  The easiest way (especially for an oddball
notebook) is to use Google to search for someone that has
already done it and posted his working XF86Config file.
