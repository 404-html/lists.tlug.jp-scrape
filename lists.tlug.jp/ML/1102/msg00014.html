On 04/02/11 02:02, Romeo Theriault wrote:

I hope this doesn't start a flame war :-), but in your own words, can you explain why you are using Git as opposed to (say) Subversion? I guess Git is great, but not for everyone?

Any thoughts by you or anyone else?
