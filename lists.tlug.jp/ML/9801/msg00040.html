At home I use ghostscript with a Epson MJ-800C. I'm not 100% sure, but
from what I can remember, here's what I did:

1) create an entry in /etc/printcap

       #--------------------------------------
       lp|mj|mjc800:\
               :sd=/var/spool/lpd/lp:\
               :mx#0:\
               :lp=/dev/lp0:\
               :if=/var/spool/lpd/lp/gdevmjc:\
               :sh:
       #--------------------------------------
       
	where lp is the default printer, mj and mjc800 are aliases
	      sd is the spool directory
	      mx#0 sets the maximum printable file size to zero (no limit)
	      lp is the output device
              if is the input filter
              sh supresses the banner page

2) create the file gdevmjc in /var/spool/lpd/lp

	#!/bin/sh
	gs -sDEVICE=mjc720 -q -NOPAUSE -sOutputFile=- -

	where gs invokes ghostscript
	      -sDEVICE=mjc720 tells it to use the Epson MJ driver (you will
               have to change this for your printer)

3) tell your applications to print to lp as Postscript
