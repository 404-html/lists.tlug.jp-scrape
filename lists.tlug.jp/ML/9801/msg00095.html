I am using an X-window FTP client called "LLNL XDIR".  It was developed
by a guy a Lawrence Livermore.  I forgot where I got it, I think
just form the sunsite ftp, but looking at the software, they have
an address:  http://www.llnl.gov/ia/xdir.html.  It is not in rpm
format though, so you will have to get a tar gzipped file and
install it yourself.  It is not too hard.  I think even it is already
compiled, so you don't have to do that even, just put some files
in the right places, etc.  

Anyway, it is pretty nice.  It opens up any number of windows on
your X desktop and you can drag files from one to another and it
will ftp for you.  When you drag a folder, it will get everything in
that folder.  Which is really nice for downloading things where you
need a whole directory structure.  You just have to drag the base
folder and you will get the whole structure contained in that folder.
