Craig Oda writes:
 > 
 > Hi,
 > I'm downloading tetex-0.4p18 right now from the redhat 5.0 
 > directory of ftp.lab.kdd.co.jp.  
 > 
 > I want to set up red hat so that I can use latex2html on my
 > linux-nihongo doc.  The thought occurred to me that I might
 > be able to include Japanese in the document itself this time
 > as text instead of graphics.  I was wondering if I can
 > install tetex and Japanese latex on the same system 
You just have to copy the CJK style files into a directory (e.g CJK/)
in the texmf/tex/latex subdirectory of your teTeX directory.
Additionally you need to add the your japanese fonts to the  fonts/
directory of teTeX.
After that you should be able to use latex for japanese documents.
I have not tried, if latex2html works with japanese documents.
(I also haven't tried to send e-mail with japanese characters, so the
following example might not come over as expected)
