One of the mildly annoying things about RedHat's rpm....

>>>>> "Rob" == Rob Bickel <rob@example.com> writes:

    >> I do have a couple of questions, in Debian, I can see the
    >> information of the package with
    >> 
    >> dpkg -I packagename.deb
    >> 
    >> can I get this with RPM?

    Rob> using rpm on a command line, if you type

    Rob> rpm -qi <packagename>

    Rob> You will get a description of the package.

    Rob> rpm -ql <packagename>

    Rob> You will get a list of files.

I think you can stack these, ie, `rpm -qil <package-name>'.

    Rob> The problem with these commands is that the package must
    Rob> already be installed.  I am not sure how to accomplish the
    Rob> same thing if the package is not installed except with glint.
 
rpm -q<options> -p /path/to/rpms/<package-name>.rpm

or something like that.
