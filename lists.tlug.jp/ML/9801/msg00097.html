On Sun, 11 Jan 1998, Shigeo Honda/Pastel Corp wrote:

> 
> I have Linux box installed REDHAT5.0 for Intel.
> 
> I'm looking for X-window FTP client software. Because, I have to Put and
> Get over 300 files, nearly 200MB for web server.  I tried 'mput', 'mget'
> command, however it was tremendous job that press 'y' key for each file.
> 
> So I need a X FTP client like 'WinFTP' for Windows95.  Does anyone know
> such FTP client? Or Is there anyway to write macros for ncftp? Get and Put
> whole directory at once? 

I like llnxdir, personally. It picks up whole directories with no trouble.
It is supposed to require motif, but should run fine with lesstif. The
whole rpm is llnxdir-2.0-1.i386.rpm and is on all the redhat mirrors. It's
a little old and hasn't been updated for a couple of years...
