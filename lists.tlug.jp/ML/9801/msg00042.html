> I just installed the Debian "imap4" package to get IMAP services.
> 
> Docs were limited to a 25-line man page and
> /usr/doc/imap-4/{changelogDebian.gz,copyright}.  Is that really all

I've been diddling with IMAP lately, and so far it does seem to be a
tad underdocumented. The home page is http://www.imap.org at UW, but
the pine page is a little more useful since it helps with setup of
the user's side of things --> http://www.washington.edu/pine. 

RH's package included a little bit of info, but not much more than
what you have. For a protocol that is being touted as the replacement
for POP, it seems like time to get more information together...
