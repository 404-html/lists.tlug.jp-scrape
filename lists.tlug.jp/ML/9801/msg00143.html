> I do have a couple of questions,  in Debian, I can see the information
> of the package with
> 
>    dpkg -I packagename.deb
> 
> can I get this with RPM?
You can with glint (The X-windows RPM manager).  In terms of using
rpm on a command line, if you type

rpm -qi <packagename>

You will get a description of the package.

If you type

rpm -ql <packagename>

You will get a list of files.

The problem with these commands is that the package
must already be installed.  I am not sure how to
accomplish the same thing if the package is not
installed except with glint.  With glint, if you
click on available to view the packages that are
in the directory that you designated to install from,
then click on a file and click on query, you can
see the above information for a package which is not
installed.
