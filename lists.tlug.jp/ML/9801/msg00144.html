On Wed, 14 Jan 1998, Craig Oda wrote:

> I do have a couple of questions,  in Debian, I can see the information
> of the package with
> 
>    dpkg -I packagename.deb
> 
> can I get this with RPM?

rpm -qip foo.rpm shows you the spec file of the package, which has
some info and notes about patches, etc.

> Also, I want to get a file listing before I unpack the rpm file,
> something similar to   tar ztvf filename.tgz
> 
> Is there an rpm command for this?

rpm -qlp foo.rpm shows the list of files inside. You can also see this
from glint.
