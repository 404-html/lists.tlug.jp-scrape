On Tue, 13 Jan 1998, Totoro wrote:

> I was reading up on Xemacs yesterday (scary things can be found in the
> docs, no?), and noticed that it runs fine on the console. Voila, yes,
> Xemacs without X! However, since kinput requires X, what input system
> is available for the console users? I haven't seen anything in that
> area. 

XEmacs can be compiled with the Wnn or Canna libs directly
into it.  Then the XEmacs program can talk directly to the
jserver or cannaserver conversion engines.  You start input
with C-o (default canna) or C-\ (default Wnn).  Alternately,
you can use the elisp skk package to talk to the skk server.

There are many ways to do this.  Emacsen are a powerful
and flexible breed.
