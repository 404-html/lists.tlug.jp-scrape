Exim rocks. Ok.

Installation went pretty well and its much simpler, faster and lighter
than sendmail. Just one thing. I tried this a few months back, but gave
up in the end. It would be good to have SMTP authorization working.

I built exim with support for both PAM and MD5 and the option
"host_auth_accept_relay = *" works fine, but Im not sure how to
configure the last part.

This is what it looks like:

######################################################################
#                   AUTHENTICATION CONFIGURATION                     #
######################################################################

fixed_login:
  driver = plaintext
  public_name = LOGIN
  server_prompts = User Name : Password
  server_condition = \
  ${if and {{eq{$1}{ph10}}{eq{$2}{secret}}}{yes}{no}}
  server_set_id = $1

fixed_plain:
  driver = plaintext
  public_name = PLAIN
  server_condition = "\
  ${if pam {$2:$3}{yes}{no}}"
  server_set_id = $2

On Netscape, the dialog box keeps asking me for the password and doesnt
go away until I kill Netscape. Havent tried M$ Outlook yet.

I dont need any fancy features like limiting to a single network, user
etc., so it shouldnt be that difficult.
