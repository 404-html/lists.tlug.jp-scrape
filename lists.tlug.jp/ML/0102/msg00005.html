Hey Viktor,

I haven't used omni but I did some work with ORBacus earlier
on and found them quite good. There support team is really good
and answers questions as fast as you can ask them (or they did
a couple of months ago anyway). It is free for development with
C and Java versions and also the full source is available.

	http://www.ooc.com/products/orbacus.html

As for the code, make sure you are using recent example code that
matches your ORB as they all do it slightly differently, especially
the old BOA stuff.

Cheers,
