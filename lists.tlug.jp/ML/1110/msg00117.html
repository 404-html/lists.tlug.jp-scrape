For desktop environments (as well as IDEs and text editors) I find I'm
really reluctant to change. Even a trial install does not help - I need
to spend hours getting to know the software. What would get my interest
is watching someone else using it, someone who likes it and has put in
the time to learn how to use it well. Not so much as snazzy demo, though
that can be helpful too, I was thinking more ten minutes as a fly on the
wall.

Anyone seen any videos like this?

(It is off the subject topic, but I'm really keen to see someone running
a C++ debug session with Eclipse/CDT and Linux. I just want to see how
smooth it would be if I put in the effort learning it - every time I try
Eclipse I feel lost after 5 minutes and go back to the commandline...
but I still have fond memories of how Visual C++ debugging sessions were
fun and efficient.)
