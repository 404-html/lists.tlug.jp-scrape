I'm trying to connect from home to my office server using ssh.com's 
windows client but I am having trouble.

I've installed the client on an office win2k machine and can connect to 
server but I can't get a connection from my home (yahooBB account).

#my YahooBB network is 219.192.108.0/24
#unfortunately I don't have a static IP

#Office LAN
AllowHosts                      \m192.168.255.0/24

password authentication failed. Connection from 219.192.108.18 denied. 
Authentication as user apache was attempted.

figure out why I can access the server from the office but not from the
