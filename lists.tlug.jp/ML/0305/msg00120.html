G'day all,

Can anyone suggest a player for Linux that can handle .avi files with 
"DIV3" coding.

xanim has worked for me with .avi files until now, but with some clips I 
got recently it says:

	"AVI Video Codec: Unknown DIV3(44495633) is unsupported by this
	executable.(E18)"

The only thing I have found that displays them OK is WMP with the latest
codecs installed. I'd rather not use WMP, for obvious reasons.

I did try a patched xanim with a divx codec, but it didn't work.

Failing that, an avi to mpg converter would be good. The olde avi2mpg
won't touch these.
