I've a blog post, which I've updated a few times, on the low-end cloud
providers:
http://darrendev.blogspot.jp/2012/04/ive-been-participating-in-hp-clouds.html
(I did try Azure, but as of 3 months ago, it did not work for me [1])

Things change regularly, but currently the Rackspace 512MB server is
$16/month:
   http://www.rackspace.com/cloud/public/servers/pricing/

I thought they said you could add a bigger disk, as a separate option,
but maybe that is still in beta, as I don't see it mentioned on the URL.

You just get a linux machine: you need to install the stuff you want and
keep it up to date. But it sounds like that is what you are after.
