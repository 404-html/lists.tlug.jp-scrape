On 2012-12-12 14:01 +0000 (Wed), Jason Frisch wrote:

> If you compare specs, [tsukaeru.net is] actually cheaper and have
> phone support (some english speakers), and don't use Xen so you get
> better performance with the same spec.

I don't know that there's a huge performance difference, but the
downside of using whatever it is they use that's the equivalant of
FreeBSD's jails is that you share a kernel with a lot of other folks, so
you have no choice about which kernel version you're using.

That said, I've had a tsukaeru.net VPS for a couple of years now, and
I've been quite happy with it. I chose them mainly because they were one
of the few providers in Japan that had English support at the time.

> Amazon Web Services is as barebones as it gets. Not sure whether it
> will fit your budget (depends on traffic, type of instance, etc..)....

I used one of the free tiny instances for a year a couple of years ago.
It's a little more complex to set up because you need to buy and use a
(virtual) NAT device between your server and the world in order to have
a static IP address. I ended up giving it up because it was rather more
expensive than the other options out there if you just wanted to run one
box with not too much stuff on it.
