I just have to recommend this item:

http://www.sysresccd.org/

System Rescue CD is an ISO image with 100 meg or so of pretty
much every system recovery tool you could want.  Package overview
is here:

http://www.sysresccd.org/systools.en.php

Quick summary: it includes both lilo and grub so you can fix your
borked MBR, it understands seemingly every file system you might
have on your Linux box(that includes reliable read/write support
for NTFS), has browsers, understands networking (nice if you need
to download something to fix your system or need external
documentation), and more.

- AND - the reason I downloaded it in the first place:

It has a Qtparted with integrated ntfsresize.  I just got another
Thinkpad that came with Win2K Pro on it and I wanted to non-destructively
partition the disk and wanted to do it with free-as-in-beer tools.
After googling all options, this one seemed like the best, and it 
worked like a champ.  If you've used Partition Magic, you'll be very
comfortable with this.  It's reliable and fast, and if anything, I
like it better than Partition Magic.

System Rescue CD is now an important part of my recovery toolbox,
I highly recommend it to everyone.
