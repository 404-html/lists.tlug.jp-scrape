> Anyway, I'd like to make it works with ghostscript, because I use to use
> a2ps to print from Mozilla. It would be more practical.

It could be that your version of Ghostscript is not built with CJK support.
From the Gentoo ebuild, it would seem that you might need these character
maps:

http://www.matsusaka-u.ac.jp/mirror/gs-cjk/adobe-cmaps-200204.tar.gz
http://www.matsusaka-u.ac.jp/mirror/gs-cjk/acro5-cmaps-2001.tar.gz

These patches:

http://mirror.clarkson.edu/pub/distributions/gentoo-portage/app-text/ghostscript/files/gs7.05.6-cjk.diff
http://mirror.clarkson.edu/pub/distributions/gentoo-portage/app-text/ghostscript/files/gs7.05.6-kochi-substitute.patch

And these fonts:

arphicfonts
kochi-substitute (probably the only ones you need for Japanese)
baekmuk-fonts

The ebuild does the following to build Ghostscript:

1. Unpacks ghostscript source
2. Unpacks patches (see above)
3. patch gs7.05.6-cjk.diff
4. patch gs7.05.6-kochi-substitute.patch
5. Build and install as usual (./configure && make && make instll)
6. mkdir /usr/share/ghostscript/Resource
7. cd /usr/share/ghostscript/Resource
8. tar xvzf adobe-cmaps-200204.tar.gz
9. tar xvzf acro5-cmaps-2001.tar.gz
