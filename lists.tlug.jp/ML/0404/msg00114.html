>> For a few years now, I've had a couple of freely searchable CJK-English

>> focuses on CJK-based Buddhist terminology. Both of these work off an XML

And a very fine system it is.

>> Recently, I've been getting more and more requests to provide a locally

>> generate a set of HTML output files, compile/index them, and package
>> them with some sort of search function.

I get heaps of such requests for the WWWJDIC server functionality to be

the files to run on a local Apache.

>> So, my question is: has anyone come across a platform-independent
