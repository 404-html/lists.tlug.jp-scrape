Answering my own question: nothing changed.

I found out that this is a FAQ, question D-3 in the XFree86
FAQ list. The problem seems to be with how XFree handles the
24bpp images internally. Apparently it maps the pixels to 
24 bits, while other X implementations map them to 32 bits
(the other machine I tested was running some commercial X server,
it wasn't XF86-3.3.2 after all).

More info at 
   http://alley.fel.tno.nl/~kscp1/xfree_24bits.html
