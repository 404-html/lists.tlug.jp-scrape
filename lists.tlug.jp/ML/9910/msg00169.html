Greetings, everybody--

Well, after struggling with them on and off for many, many months,
I've finally finished a couple of my pet projects. Well, no, not
really finished at all ... just packaged in a semi-usable state, with
a little bit of documentation. The two projects are:

* AHO-JDPS (A Homemade Japanese Document Processing System)
  Pun? What pun? Anyway, this is a set of data files, scripts, and
  examples designed to help ease the monstrous task of processing
  Japanese SGML documents with Jade, then (optionally) with Jadetex,
  and actually getting something that looks like a real document.

* Omega-J
  An effort to provide Japanese support for Omega. All there really
  is at the moment is an input character translation routine and some
  instructions and sample data for preparing fonts to use with Omega.
  The output ... contains the kanji you asked for, but is otherwise
  totally broken.

These magnificent creations can be found at

http://www.havenrock.com/docproc/nihongo/index.html

Enjoy ... and please let me know what you think. I value the insights
of the TLUG crew, so you are the very first to know about this stuff.

Now I'm off to bed.
