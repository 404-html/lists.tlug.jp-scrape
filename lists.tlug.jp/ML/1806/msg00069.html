And next in line will be Big5, closely followed by GBK[1], and then this:

https://bugs.python.org/issue33865

Punchline: Microsoft registered *only* "windows-874" with the IANA,
the IANA provides a canonical[2] (prepend "cs", delete separator
characters) alias "cswindows874", but Microsoft also documents (and
Windows implements) "874" and "CP874", and the mapping file they gave
Unicode is named "CP874.TXT".

And it appears that a bug in the Anaconda distribution is responsible
for causing the actual bug report (haven't followed up since it's
being closed NOTOURFSCKINGBUG :-).
