Josh Glover (jmglov@example.com) wrote:

> True enough. Unfortunately, as I pointed out in another mail, now you 
> are paying for Windows 2000. The server version.

I forgot to list that one as a reason to build it yourself.  Unless
you are buying a pre-built server from a vendor who ships Linux,
you are paying the the Microsoft Tax.

Since there were (still are?) vendor licensing agreements from MS that
required the vendor to pay a Windows license fee for every machine
they sold - regardless of whether or not Windows was installed on 
it - it's possible that you could be paying the MS tax anyway, unless
you buy from a vendor who either doesn't sell Windows machines, or
doesn't sell them with OEM Windows pre-loaded.  Under such a licensing
agreement, a vendor would have only two choices when shipping a Linux
box - eat the Microsoft Tax, or pass it on to you.  Profit margins are
thin on computers.  I expect they would pass it on.  If that means
nobody buys Linux boxes from them, then they can discontinue the
option and blame it on lack of demand.
