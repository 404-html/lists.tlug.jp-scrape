You can use Control-Alt-<F1/F2/F3/F4> etc. as well with the graphical
installer to switch to the console. On console 2 there is always a
shell prompt where you can do interactive work, on console 10 are the
kernel messages etc ...

> 2) That green color scheme has to go :-)

You mean in KDE? I don't like that much either.  If you want the
default KDE theme, just 'rm -rf ~/.kde/' but leave ~/.skel/ in place
and restart KDE.  If ~/.skel/kdebase3 is already there, KDE start
script will not apply the SuSE theme and you get the default KDE
theme.
