You don't need to build your own kernel. SGI provides kernel RPMs. You
can get a RedHat 7.2 install ISO image with XFS enabled; use it with the
standard RedHat install CDs. Very simple. That includes kernel 2.4.9 (or
is it 2.4.5?) and XFS version 1.02. But you can get a modified standard
RedHat 2.4.9 kernel rpm, or a 2.4.18 kernel rpm from the SGI web site
that include XFS version 1.1, which you can install after you do the
standard install.

I've been running XFS for about a year on four machines and haven't had
any troubles. I've had two power failures -- the fsck upon recovery of
power took less than a second and there was no loss of data or
filesystem corruption. I wouldn't use anything else now.
