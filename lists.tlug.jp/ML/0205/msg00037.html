Norman Diamond (ndiamond@example.com) wrote:

> When dhcpd assigns a lease, I want the client machine's hostname and IP
> address to be added to the DNS server's directory.  When a lease expires or
> if a client releases it earlier, I want the corresponding DNS entry to be
> removed.  Is there any tool that does this automatically?

There's a much simpler way to do it.  Just created forward and
reverse zone files for the range of IP addresses you have, such that
host one is 192.168.1.1, host 2 is 192.168.1.2, etc.  Regardless
of what IP address a given host is dynamically assigned, it will
have a DNS sentry associated with it.

Or if you'd like them to have the same entry every time, you
can do as BOTi suggested and assign static IPs via DHCP.
