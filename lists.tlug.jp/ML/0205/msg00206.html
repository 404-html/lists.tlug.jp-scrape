I don't like up2date either.  My preferred tools are:

autoupdate
  http://www.mat.univie.ac.at/~gerald/ftp/autoupdate/index.html
autorpm
  http://www.autorpm.org/
  
autoupdate provides a completely automated tool for doing RPM
updates.  autorpm can be run in fully automated mode, but
provides more knobs for customizing the behavior and optionally
accepting operator intervention.  Each optionally allows you to
build a "local download pool" on a master machine that then
gets deployed to the rest of the fleet.
