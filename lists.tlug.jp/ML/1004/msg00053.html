Nick Hastings wrote:

> I'm looking for suggestions for a printer. 

The first and last word when selecting a printer to 
use with Unix or Linux is: 

   Postscript

> - It should work (almost?) flawlessly with Linux and CUPS.

   Postscript

Postscript is the native language for printing in Unix and Linux, 
so using a Postscript printer avoids many compatibility issues. 

http://www.faqs.org/docs/Linux-HOWTO/Printing-Usage-HOWTO.html#ss3.2
