about 25 hours:

  PID USER      PR  NI  VIRT  RES  SHR S %CPU %MEM    TIME+  COMMAND

  PID USER      PR  NI  VIRT  RES  SHR S %CPU %MEM    TIME+  COMMAND

it gets kind of sluggish when it gets that bloated. And once it starts 
getting sluggish, if i don't restart it manually it will force the 
issue by dying of its own volition. I should probably restart it for 
other reasons anyway; that machine isn't being actively used at the 
moment (i logged in remotely to get the memory stats), so Firefox 
should be consuming 0% CPU. Probably one of the extensions ran away 
again. Or maybe it was javascript that went wild. I don't know; i just 
hope 3.0 fixes some of the memory and stability problems.
