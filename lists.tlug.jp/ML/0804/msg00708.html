On 2008-04-29 11:02 +0900 (Tue), Kenneth Burling wrote:

> Make sure that for every www.domainx.com you create a ServerAlias to
> domainx.com.

Incidently, these errors can be easier to debug if you make the default
domain one that serves up a page along the lines of, "Unrecognized
domain name on this server."

It's a general principle: if something must be specified, but might not
be or might be specified incorrectly, the default should be an error
notification that it wasn't specified correctly.
