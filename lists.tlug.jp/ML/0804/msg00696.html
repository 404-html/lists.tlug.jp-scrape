On Sunday 27 April 2008 20:15:01 CL wrote:
> I updated 7.10 to 8.04 and find that, since Thursday, the Ubuntu Japan
> site runs extremely slow, when it runs at all.

You can solve this problem by using a different mirror.  Assuming that you are 
using the GUI (Gnome or KDE), select "Software Sources" under the System 
menu, and do the following (after typing in your password):

* Under "Download from:" select "Other..."
* Click on the "Select Best Server" button
* Click on the "Choose Server" button once the test is complete
* Click on the "Close" button
* Click on the "Reload" button

Note that I have used the ftp.ecc.u-tokyo.ac.jp server for quite some time, 
but it has gotten really bad recently.  (It is slow to connect, but then the 
transfer is still reasonable.)  The fastest one for me now is ftp.jaist.ac.jp 
(or ftp.yz.yamagata-u.ac.jp).  Note that if you select a server by hand, be 
sure to select the disired protocol as well.

Cheers,
