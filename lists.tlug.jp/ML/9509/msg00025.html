In the september issue of Unix User there was a short note on NFS under
Linux. If I remember correctly, it recommends the options rsize=8192
and wsize=8192. These set the read/write buffer size to 8k, that is the
SunOS default, while the Linux default is 1024.

I didn't try these options yet, as I always forget to bring the magazine 
from home. But I installed JE (that I had downloaded by FTP) from a NFS 
partition in a Sun Sparcstation.
