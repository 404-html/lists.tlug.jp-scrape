Do you need both exclamation marks? If just one exclamation mark is
enough, find the magic key to get into the BIOS, where there will be an
option to change boot order. If you see the Grub menu you were too slow.

E.g. On this lenovo the magic key is DEL and you have to press it in the
half second window *before* the message flashes on screen (for 0.25
seconds) telling you to press DEL. :-)
