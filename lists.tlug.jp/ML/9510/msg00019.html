> I am using a communication program which come with the Linux package 
> named Seyon and wondering if anyone knows a better one.

I prefer Minicom (now up to version 1.7x) to Seyon.  It has a simple
text-based interface, but will run in an XTerm without problem.  The
functionality is a near clone of the popular Telix shareware program
for DOS.  The scripting language is neither as clean or powerful as
Telix's, but it is quite workable.
