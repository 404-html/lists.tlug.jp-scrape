> The question I was researching when I hit the archive glitch was:
> 
> What do I need to do to be able to let other users connect to my
> userid's X server and open displays?  If I have an X program that
> requires being run as root, I want to be able to SU and run it, but I
> get:  ** WARNING **: cannot open display:  and it dies.
> 
> TIA,

You must run xhost and tell the Xserver who can connect to Xserver.   If you 
are running Xserver on machine A and want to run a client program on machine B 
which is displayed on machine A, you must run xhost on machine A like follows:
xhost +B
Then, Xserver knows machine B wants to connect to the Xserver on machine A.  
Then, for example, to run xterm on machine B, which is displayed on machine A, 
invoke
xterm -display (name of machine A):0.0
on machine B.   but you seem to know this.

 Hope it helps.
