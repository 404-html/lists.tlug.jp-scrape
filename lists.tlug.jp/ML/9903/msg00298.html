What do the headers look like here?  This should not happen.  The username
is being explicitly set ...

> I don't have one. I set masquerading in sendmail.cf (DM......)

Ah.  That could explain why it's not working properly.  I don't really
recommend editing sendmail.cf -- there are too many little options that
can be misconfigured.  I find it easier to rebuild from a .mc file.

You may need something like this:

MASQUERADE_AS(tokyo.email.ne.jp)
MASQUERADE_DOMAIN(viy.sakae.whatever)

Here is the .mc file that we use on the tlug server, it may help:

OSTYPE(linux)dnl
DOMAIN(generic)dnl
define(PROCMAIL_MAILER_PATH, `/usr/bin/procmail')
FEATURE(local_procmail)
FEATURE(use_cw_file)
FEATURE(use_ct_file)
FEATURE(access_db)
MASQUERADE_AS(tlug.gr.jp)dnl
MASQUERADE_DOMAIN(`tlug.pht.co.jp,tlug.linux.or.jp')dnl
FEATURE(masquerade_envelope)dnl
FEATURE(allmasquerade)dnl
MAILER(local)dnl
MAILER(smtp)dnl
