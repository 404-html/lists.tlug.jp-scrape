In my case, to Japanize netscape, I downloaded communicator-ja-v408-export.x86-
unknown-linux2.0.tar.gz and unzipped and untarred.  This archive contains a 
shell script file named install-ja.sh, so simply run this.  This asks some 
questions and after answer those, japanization completes.   To execute 
Japanized netscape, you should run /usr/local/bin/jcommunicator instead of 
usual netscape.   So far, you don't need any JRPM.   Of course when you want 
to enter Japanese language, you will need JRPM.
   I don't know about Vine Linux, so I cannot answer that.  But if you want to 
use Japanese language on X Windows, you don't need any other font than those 
coming with X Windows.  So I suspect it should be trying to use Japanese on 
the normal text screen.
