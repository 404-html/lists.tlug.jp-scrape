>Which (cheap) Ethernet Adapter PC Card would you recommend me to buy in
>Akihabara to run OK with my RedHat 5.2 Linux box on a Toshiba P100 machine?
>
>Are those PCI (dark blue boxes) OK? (I don't wich module they use BTW...)

You mean Planex? I got a 10/100 card for around 3000 yen. RedHat 5.1 didn't
recognize it when installing, but you can get a module here:
  ftp://cesdis.gsfc.nasa.gov/pub/linux/drivers/via-rhine.c
  http://cesdis.gsfc.nasa.gov/linux/drivers/via-rhine.html
  http://cesdis.gsfc.nasa.gov/linux/misc/modules.html

The even cheaper 10-base card is a different chip, so might be recognized
okay - if you're lucky it will be NE2000 compatible.
