We'll have to work that out.  If spammers were clever and ambitious
enough to start using such a ploy, we would have to respond by requiring
the From: and Sender: fields to match, or to both be subscribed to
tlug-post.  At present, that probably isn't necessary.  These details
will be worked out as we implement it (assuming the vote favors it,
which it is currently doing by landslide).  Every effort will be made to
minimize inconvenience to tlug members.
