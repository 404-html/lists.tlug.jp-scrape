>to do so?  Is there an app that allows me to negotiate POP3 by hand -

telnet pop.xxx.co.jp 110

To login type:
user [yourname]
pass [your password]

Then you can use "list" to see size of all messages. "top N L" will show
the top L lines of message N.

Then "dele N" where N is the message number. If you get it wrong you can
type "rset" to undelete all deleted messages. The messages really get
deleted when you end the session with "quit".
