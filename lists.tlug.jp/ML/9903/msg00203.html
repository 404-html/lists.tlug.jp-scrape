in my case (wnn/egg) I used to visit file where all the special symbols
(key input sequence) is defined.

on my current system the file is /usr/local/lib/mule/19.28/lisp/egg-jsymbol.el
to contain something like: 
(defvar *ku1-alist* '(
	 ( "　" . "　")
	 ( "、" . "、")
	 ( "。" . "。")
	 ( "，" . "，")
	 ( "．" . "．")
	 ( "・" . "・")
	 ( "：" . "：")
	 ( "；" . "；")
	 ( "？" . "？")
	 ( "！" . "！")
...

I've just defined my own names for each frequently used signs: 
e.g., ''kuro-maru'' to be convereted to ''●''.
