Well, thing is that people treat these things like little computers.
When you flash them, it is basically a disk image that you are
flashing it with, like a CD but smaller, or larger depending on the
things capabilities.

If you look at it like that, then they can juse as easily create a
proprietary input system, or port an existing one.  If they did it
that way they would have no obligation to make it public.  It is just
another application running on their little computer system.
