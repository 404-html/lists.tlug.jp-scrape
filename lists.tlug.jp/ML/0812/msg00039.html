Hi Dave,

Dave M G wrote: 
TLUG,

Any suggestions on what I should do?


Check permissions once the device is mounted and make sure it is owned by you. If you are relying on whatever automount subsystem of the month to do this for you, this in general works (until it doesn´t). However, just to be sure you can always just force it from the command line:

sudo chown -R me /path/to/mountpoint


Then try writing some files. If it isn´t permissions then this might require some more sleuthing.
